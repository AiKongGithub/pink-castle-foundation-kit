# Video Summary Template

> **Format:** Video Summary (ภาพรวมแบบวิดีโอ)
> **Tool:** NotebookLM + Video Editor
> **Output:** Short-form video summary
> **Duration:** 5-10 minutes

---

## Purpose

Create a visual summary of source content that combines key points with engaging visuals, suitable for YouTube Shorts, TikTok, or educational content.

---

## Template Structure

### 1. Opening Hook (15-30 sec)
```
- Visual hook (text overlay or animation)
- Question or surprising fact
- Topic preview
```

### 2. Introduction (30 sec - 1 min)
```
- Topic overview
- What viewers will learn
- Why this matters
```

### 3. Main Content (3-6 min)
```
Section 1: [Key Point 1]
- Visual: [Screenshot/Diagram/Animation]
- Explanation: [30-60 sec narration]
- Example: [Real-world application]

Section 2: [Key Point 2]
- Visual: [Screenshot/Diagram/Animation]
- Explanation: [30-60 sec narration]
- Example: [Real-world application]

Section 3: [Key Point 3]
- Visual: [Screenshot/Diagram/Animation]
- Explanation: [30-60 sec narration]
- Example: [Real-world application]
```

### 4. Summary & CTA (1 min)
```
- Recap main points (with text overlay)
- Call to action
- Subscribe/Like reminder
```

---

## NotebookLM Prompt

```
Create a video script from this content that:
1. Has a clear visual structure with sections
2. Is engaging and educational
3. Is approximately 5-10 minutes long
4. Includes suggestions for visuals/animations
5. Uses conversational Thai language
```

## Visual Guidelines

### Recommended Visuals
| Section | Visual Type | Tools |
|---------|-------------|-------|
| Hook | Text overlay + animation | Canva, CapCut |
| Intro | Logo animation | Canva, After Effects |
| Content | Diagrams, Screenshots | Canva, Figma |
| Examples | Screen recordings | OBS, Loom |
| Summary | Bullet points | Canva, PowerPoint |

### Text Overlay Best Practices
- Font: Clear, readable (Noto Sans Thai)
- Size: Large enough for mobile
- Color: High contrast
- Duration: 3-5 seconds per point

## Quality Checklist

### Content Quality
- [ ] Covers main concepts from source
- [ ] Information is accurate
- [ ] Examples are relevant
- [ ] Logical flow

### Visual Quality
- [ ] Visuals support the content
- [ ] Text is readable on mobile
- [ ] Consistent branding
- [ ] Smooth transitions

### Engagement
- [ ] Strong hook in first 15 seconds
- [ ] Maintains viewer interest
- [ ] Clear call to action
- [ ] Appropriate pace

### Technical
- [ ] Duration is 5-10 minutes
- [ ] Audio is clear
- [ ] Thai pronunciation correct
- [ ] Video resolution is HD
