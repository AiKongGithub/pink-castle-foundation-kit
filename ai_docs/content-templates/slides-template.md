# Slides Template

> **Format:** Slides (ชุดสไลด์)
> **Tool:** NotebookLM + Google Slides/PowerPoint
> **Output:** Presentation deck
> **Quantity:** 10-15 slides per source

---

## Purpose

Create presentation slides that can be used for teaching, sharing, or presenting source content in a structured, visual format.

---

## Template Structure

### Slide Deck Outline

```
1. Title Slide
2. Agenda/Overview
3-10. Content Slides (1 concept per slide)
11. Summary/Key Takeaways
12. Q&A / Discussion
13. Call to Action
14. Thank You / Contact
```

### Slide Types

#### 1. Title Slide
```
[Main Title]
[Subtitle]
[Presenter/Source]
[Date]
[Logo]
```

#### 2. Agenda Slide
```
วันนี้เราจะเรียนรู้:
1. [Topic 1]
2. [Topic 2]
3. [Topic 3]
4. [Topic 4]
```

#### 3. Content Slide
```
[Slide Title]
• [Bullet point 1]
• [Bullet point 2]
• [Bullet point 3]
[Supporting visual/image]
```

#### 4. Quote Slide
```
"[Memorable quote from content]"
— [Source/Author]
```

#### 5. Data Slide
```
[Title]
[Chart/Graph/Statistics]
[Key insight in 1 sentence]
```

#### 6. Summary Slide
```
สรุปสิ่งที่ได้เรียนรู้:
✓ [Takeaway 1]
✓ [Takeaway 2]
✓ [Takeaway 3]
```

---

## NotebookLM Prompt

```
Create presentation slides from this content that:
1. Has 10-15 slides total
2. Follows standard deck structure (Title, Agenda, Content, Summary, CTA)
3. Has 1 main idea per slide
4. Uses 3-5 bullet points max per slide
5. Includes suggestions for visuals
6. Is in Thai language
7. Can be presented in 15-20 minutes
```

## Design Guidelines

### Colors
| Element | Color | Use |
|---------|-------|-----|
| Primary | Pink (#FF69B4) | Titles, Accents |
| Secondary | White (#FFFFFF) | Background |
| Text | Dark Gray (#333333) | Body text |
| Accent | Gold (#F1C40F) | Highlights |

### Typography
| Element | Font | Size |
|---------|------|------|
| Title | Noto Sans Thai Bold | 44-60pt |
| Subtitle | Noto Sans Thai | 32-36pt |
| Body | Noto Sans Thai | 24-28pt |
| Caption | Noto Sans Thai | 18-20pt |

### Layout Rules
- Max 5 bullet points per slide
- Max 6-8 words per bullet
- 40% text, 60% white space
- One main idea per slide
- Consistent alignment

---

## Quality Checklist

### Content Quality
- [ ] Covers all main concepts
- [ ] Logical flow
- [ ] 1 idea per slide
- [ ] Concise text (5 bullets max)

### Visual Quality
- [ ] Consistent design
- [ ] Readable fonts
- [ ] Appropriate images
- [ ] Not overcrowded

### Presentation
- [ ] 15-20 minute duration
- [ ] Good pacing
- [ ] Clear transitions
- [ ] Strong opening/closing

### Technical
- [ ] Works in Google Slides/PowerPoint
- [ ] Fonts embedded
- [ ] Images high quality
- [ ] Animations subtle (if any)
