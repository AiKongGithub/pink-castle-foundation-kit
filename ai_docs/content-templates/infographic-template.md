# Infographic Template

> **Format:** Infographic (อินโฟกราฟิก)
> **Tool:** NotebookLM + Canva/Figma
> **Output:** Visual summary image
> **Size:** Portrait (1080x1920) or Square (1080x1080)

---

## Purpose

Create visually appealing infographics that summarize key concepts from source content in a shareable, scannable format for social media.

---

## Template Structure

### Header Section (10-15%)
```
[Logo/Branding]
[Title - 5-8 words max]
[Subtitle - optional]
```

### Main Content (70-80%)
```
Section 1: [Key Point with Icon]
Section 2: [Key Point with Icon]
Section 3: [Key Point with Icon]
Section 4: [Key Point with Icon]
Section 5: [Key Point with Icon]
(3-7 sections typical)
```

### Footer Section (10-15%)
```
[Call to Action]
[Source/Credit]
[Logo/Website]
```

---

## Content Types

### 1. Process/Steps
```
Step 1 → Step 2 → Step 3 → Step 4 → Result
(Linear flow with arrows)
```

### 2. List/Tips
```
Tip 1 [Icon]
Tip 2 [Icon]
Tip 3 [Icon]
Tip 4 [Icon]
Tip 5 [Icon]
```

### 3. Comparison
```
[Option A]     vs     [Option B]
Feature 1      |      Feature 1
Feature 2      |      Feature 2
Feature 3      |      Feature 3
```

### 4. Statistics
```
[Big Number]% - Description
[Big Number]x - Description
[Big Number]+ - Description
```

### 5. Timeline
```
2020 ──●── Event 1
2021 ──●── Event 2
2022 ──●── Event 3
2023 ──●── Event 4
```

---

## NotebookLM Prompt

```
Create infographic content from this material that:
1. Has a clear title (5-8 words)
2. Contains 5-7 key points
3. Uses short, punchy text (10 words max per point)
4. Can be visualized with icons
5. Follows a logical flow
6. Is in Thai language
7. Includes suggested colors and icons
```

## Visual Guidelines

### Color Palette
| Element | Color | Hex |
|---------|-------|-----|
| Primary | Pink | #FF69B4 |
| Secondary | Purple | #9B59B6 |
| Accent | Gold | #F1C40F |
| Background | White | #FFFFFF |
| Text | Dark Gray | #333333 |

### Typography
| Element | Font | Size | Weight |
|---------|------|------|--------|
| Title | Noto Sans Thai | 48-60px | Bold |
| Section Title | Noto Sans Thai | 24-32px | Semi-bold |
| Body | Noto Sans Thai | 18-24px | Regular |
| Footer | Noto Sans Thai | 14-16px | Regular |

### Icon Style
- Style: Flat/Line icons
- Size: Consistent (64-96px)
- Color: Match section or accent
- Source: FontAwesome, Heroicons, Flaticon

### Layout
- Margins: 40-60px
- Section Spacing: 30-40px
- Alignment: Center or Left
- Visual Balance: Symmetric

## Quality Checklist

### Content Quality
