# Audio Overview Template

> **Format:** Audio Overview (ภาพรวมแบบเสียง)
> **Tool:** NotebookLM
> **Output:** Podcast-style audio summary
> **Duration:** 10-15 minutes

---

## Purpose

Transform source content into an engaging podcast-style audio overview that summarizes key concepts and takeaways.

---

## Template Structure

### 1. Introduction (1-2 min)
```
- Hook: Interesting fact or question from the content
- Topic introduction: What this episode covers
- Why it matters: Relevance to the listener
```

### 2. Key Concepts (5-8 min)
```
- Main point 1: [Concept + Explanation + Example]
- Main point 2: [Concept + Explanation + Example]
- Main point 3: [Concept + Explanation + Example]
- (Repeat for 3-5 main points)
```

### 3. Practical Applications (2-3 min)
```
- How to apply these concepts
- Step-by-step guidance
- Real-world examples
```

### 4. Key Takeaways (1-2 min)
```
- Summary of main points
- Action items for listeners
- Next steps or related topics
```

### 5. Closing (30 sec)
```
- Thank you message
- Call to action
- Preview of next content (if applicable)
```

---

## NotebookLM Prompt

```
Create a podcast-style audio overview from this content that:
1. Is engaging and conversational in tone
2. Covers the main concepts clearly
3. Includes practical examples
4. Is approximately 10-15 minutes when spoken
5. Uses natural Thai language (ภาษาพูด ไม่เป็นทางการมาก)
```

## Quality Checklist

### Content Quality
- [ ] Covers all main points from source
- [ ] Information is accurate
- [ ] Examples are relevant and clear
- [ ] Flow is logical and easy to follow

### Audio Quality
- [ ] Language is natural (ภาษาพูด)
- [ ] Pace is appropriate (not too fast/slow)
- [ ] Transitions are smooth
- [ ] Pronunciation is correct

### Engagement
- [ ] Introduction hooks the listener
- [ ] Maintains interest throughout
- [ ] Includes practical value
- [ ] Ends with clear takeaways

### Technical
- [ ] Duration is 10-15 minutes
- [ ] No factual errors
- [ ] Thai language is correct
- [ ] Suitable for target audience
