## Slide 6: Remarketing Windows

```
┌─────────────────────────────────────────────┐
│  Remarketing Windows — หน้าต่างเวลา        │
│                                             │
│  Conversion Rate                            │
│  ▓▓▓▓▓▓▓▓▓▓  │ 1-7 วัน   │ สูงสุด  │ 50% │
│  ▓▓▓▓▓▓▓     │ 8-14 วัน  │ -30%    │ 25% │
│  ▓▓▓▓▓       │ 15-30 วัน │ -50%    │ 15% │
│  ▓▓          │ 31-90 วัน │ ต่ำมาก  │ 10% │
│                                             │
│  Frequency Capping: สูงสุด 5-7 ครั้ง       │
│  ├── เกิน 7 ครั้ง = ความรำคาญ              │
│  └── เปลี่ยน Creative แทนยิงซ้ำ            │
│                                             │
└─────────────────────────────────────────────┘
```

**Speaker Notes:** ยิ่งนานจาก Touch Point แรก Conversion Rate ยิ่งลด ควรให้งบหนักใน Window 1-7 วัน Frequency Capping สำคัญมาก ถ้าคนเห็นโฆษณาเดิมเกิน 5-7 ครั้งแล้วยังไม่ซื้อ ให้เปลี่ยน Creative ใหม่แทน

## Slide 8: Budget Allocation by Moment

```
┌─────────────────────────────────────────────┐
│  Budget Allocation — แบ่งงบตาม Moment       │
│                                             │
│  ┌─────────────────────────────────────┐    │
│  │ Top of Funnel (I-want-to-know)      │    │
│  │ 30% ของงบ — Educate + Awareness    │    │
│  ├─────────────────────────────────────┤    │
│  │ Middle of Funnel (I-want-to-do)     │    │
│  │ 30% ของงบ — Tutorial + Engage      │    │
│  ├─────────────────────────────────────┤    │
│  │ Bottom of Funnel (I-want-to-buy)    │    │
│  │ 40% ของงบ — Proof + CTA + Close    │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  Bottom ได้งบมาก = Conversion Rate สูง      │
│  แต่อย่าทิ้ง Top = เติมคนเข้า Funnel       │
│                                             │
└─────────────────────────────────────────────┘
```

**Speaker Notes:** ให้น้ำหนักกับ Bottom of Funnel มากที่สุดเพราะ Conversion Rate สูงสุด แต่ต้องไม่ทิ้ง Top เพราะเป็นการเติมคนเข้า Funnel เปรียบเหมือนต้นไม้ Top = น้ำที่รด, Bottom = ผลไม้ที่เก็บเกี่ยว

```
┌─────────────────────────────────────────────┐
│  Always-On + Burst Strategy                 │
│                                             │
│  งบรายเดือน:                                │

│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓          │

│                                             │
│  Burst (30-40%)                             │

│                                             │
```

**Speaker Notes:** กลยุทธ์ Always-On + Burst ช่วยให้ Brand มีตัวตนตลอดเวลา ไม่หายไปจากสายตาลูกค้า และเมื่อเจอโอกาสพิเศษก็พร้อมเพิ่มงบได้ทันที ต้องวางแผนงบรายปีล่วงหน้า
