## เฉลยและคำอธิบาย

### ข้อ 1: **B** ✅
การลิงก์เพื่อให้ข้อมูลระหว่าง YouTube กับ Google Ads ไหลถึงกัน สามารถดู Video Analytics (Views, Watch time, Engagement) ใน Google Ads ได้โดยตรง และสร้าง Remarketing List จากคนที่ดูวิดีโอได้

### ข้อ 2: **C** ✅
Linked accounts อยู่ที่ Tools (ไอคอนเครื่องมือมุมขวาบน) ในหมวด Setup จะเห็น Linked accounts ซึ่งแสดงรายการบริการต่างๆ ที่สามารถเชื่อมต่อกับ Google Ads ได้รวมถึง YouTube

### ข้อ 3: **B** ✅
ใน YouTube Studio ต้องไปที่ Settings (เมนูซ้ายล่าง) → Channel → แท็บ Advanced settings → เลื่อนลงมาจะเห็นหมวด Linked accounts ที่แสดงคำขอลิงก์รอ Approve

### ข้อ 4: **C** ✅
ต้องเป็น Owner (เจ้าของช่อง) หรือ Admin เท่านั้นที่จะ Approve ได้ ผู้ที่มีสิทธิ์เป็น Editor หรือ Viewer จะไม่สามารถอนุมัติการลิงก์ได้

### ข้อ 5: **C** ✅
หลังลิงก์สำเร็จสามารถสร้าง Audience ได้หลายระดับ ทั้งคนที่ดูวิดีโอทั้งหมด คนที่ดูเกินจำนวนวินาทีที่กำหนด คนที่ Subscribe ช่อง และคนที่กด Like ให้ความยืดหยุ่นในการ Targeting ตามระดับความสนใจ

### ข้อ 6: **C** ✅
Video Remarketing มี Conversion Rate สูงกว่าโฆษณาปกติ เพราะกลุ่มเป้าหมายเคยดูวิดีโอของเราแล้ว รู้จักเราแล้ว มีความสนใจอยู่แล้ว จึงมีโอกาส Convert สูงกว่าคนที่ไม่เคยเห็นเรามาก่อน

### ข้อ 7: **ผิด** ✅
สามารถลิงก์ได้หลายช่อง YouTube กับ 1 บัญชี Google Ads ถ้ามีหลายช่องสำหรับหลายแบรนด์ก็ลิงก์ทั้งหมดเข้ามาจัดการจากบัญชีเดียวได้

### ข้อ 8: **ถูก** ✅
ถ้าใช้ Google Account เดียวกันทั้ง Google Ads และ YouTube บางกรณีระบบจะ Link ให้อัตโนมัติเลย ไม่ต้องไปอนุมัติที่ YouTube Studio แยก แต่ถ้าเป็นคนละ Account ต้องส่งคำขอแล้วไป Approve ที่ YouTube Studio

### ข้อ 9: **ผิด** ✅
การลิงก์ช่อง YouTube กับ Google Ads ไม่มีค่าใช้จ่ายเพิ่มเติม เป็นฟีเจอร์ที่ Google ให้ใช้ฟรี ทำได้จากทั้งสองแพลตฟอร์มโดยไม่ต้องเสียเงิน

### ข้อ 10: ✅
Tools → **Linked accounts** → YouTube → Link → **ใส่ URL ช่อง YouTube** → Send link request

นี่คือลำดับขั้นตอนการลิงก์จากฝั่ง Google Ads เริ่มจากเข้าเมนู Tools หา Linked accounts เลือก YouTube กด Link ใส่ URL ช่อง แล้วส่งคำขอ
