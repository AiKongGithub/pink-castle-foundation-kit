## เฉลยและคำอธิบาย

### ข้อ 1: **B** ✅
Landing Page คือหน้าที่คนไปถึงหลังคลิกโฆษณา ถ้าหน้านี้ทำงานไม่ดี เงินค่าโฆษณาทุกบาทจะสูญเปล่า เหมือนเปิดร้านเชิญคนมาแต่ประตูเปิดไม่ได้ ต้องตรวจสอบให้พร้อมก่อนจ่ายเงินยิงโฆษณา

### ข้อ 2: **B** ✅
Message Match หมายถึงข้อความในโฆษณาต้องตรงกับข้อความบน Landing Page ทั้ง Headline, ข้อเสนอ และธีม ลูกค้าคลิกเพราะสนใจสิ่งที่โฆษณาบอก ถ้าเข้ามาแล้วเจอเรื่องอื่น จะสับสนและออกจากหน้าทันที

### ข้อ 3: **B** ✅
หน้า Landing Page ควรโหลดไม่เกิน 3 วินาที ถ้าช้ากว่านี้ผู้เข้าชมจะกดปิดไปเลย ทุกวินาทีที่เพิ่มขึ้นจะลด Conversion Rate ลงอย่างมีนัยสำคัญ

### ข้อ 4: **C** ✅
Google PageSpeed Insights (pagespeed.web.dev) เป็นเครื่องมือฟรีจาก Google สำหรับตรวจความเร็วเว็บ ใส่ URL แล้วจะได้คะแนน 0-100 พร้อมคำแนะนำในการปรับปรุง Google Analytics ใช้วิเคราะห์ Traffic, Search Console ใช้ดู SEO, Tag Manager ใช้จัดการ Tag

### ข้อ 5: **C** ✅
Traffic จากโฆษณา YouTube ส่วนใหญ่มาจาก Mobile เพราะผู้คนดูวิดีโอ YouTube บนมือถือเป็นหลัก ถ้า Landing Page แสดงผลไม่ดีบน Mobile จะเสียลูกค้ากลุ่มใหญ่ที่สุดไป

### ข้อ 6: **B** ✅
Conversion Elements 3 อย่างหลักคือ Opt-in Form (ต้องกรอกและส่งได้จริง), CTA Button (ต้องกดแล้วทำงาน) และ Redirect ไป Thank You Page (ต้องไปถูกหน้า) ทั้งสามอย่างต้องทดสอบด้วยตัวเอง

### ข้อ 7: **ผิด** ✅
เมื่อ Message ไม่ Match ลูกค้าจะสับสนและออกจากหน้าทันที ไม่ใช่อ่านเนื้อหาทั้งหมดก่อนตัดสินใจ การตัดสินใจเกิดขึ้นใน 3 วินาทีแรกที่เข้ามาถึงหน้า ทำให้ Bounce Rate สูงและเสียเงินค่าคลิกฟรี

### ข้อ 8: **ผิด** ✅
ต้องทดสอบบนมือถือจริงเสมอ เพราะ Traffic จาก YouTube ส่วนใหญ่มาจาก Mobile หน้าที่สวยบน Desktop อาจพังบน Mobile ได้ ต้องเช็คว่าข้อความอ่านง่าย ปุ่มกดง่าย ฟอร์มกรอกสะดวก

### ข้อ 9: **ถูก** ✅
ถ้าไม่มี Google Tag หรือ Pixel บน Landing Page จะวัด Conversion ไม่ได้เลย ไม่รู้ว่าโฆษณาตัวไหนได้ผล ไม่สามารถ Optimize แคมเปญได้ จึงเป็นเหมือนยิงโฆษณาแบบตาบอดจริงๆ

### ข้อ 10: ✅
**Message Match** → Speed Test → **Mobile Responsive** → Conversion Elements → **Tracking**

นี่คือ 5 จุดตรวจหลักของ Landing Page ตามลำดับ ต้องเช็คให้ครบทุกจุดก่อนเริ่มยิงโฆษณา
