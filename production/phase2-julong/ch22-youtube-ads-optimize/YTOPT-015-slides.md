## SLIDE 4: Remarketing Campaigns — หลักการ

- **Remarketing = โฆษณาให้คนที่รู้จักเราแล้ว**
- Conversion Rate สูงกว่าคนใหม่ 3-5 เท่า
- Cost per Conversion ต่ำกว่ามาก
- เปรียบเทียบ: จีบคนที่เคยยิ้มให้ vs จีบคนแปลกหน้า

| ประเภท Remarketing | ระดับความตั้งใจ |
|-------------------|---------------|
| ดูวิดีโอแค่ 5 วินาที | ต่ำ |
| ดูวิดีโอจนจบ | ปานกลาง |
| Subscribe ช่อง | สูง |
| เข้าเว็บไซต์ | สูง |

## SLIDE 5: วิธีสร้าง Video Remarketing Lists

- **ขั้นตอน:**
  1. Tools & Settings → Audience Manager
  2. สร้าง Audience ใหม่ → YouTube Users
  3. เลือกประเภท: Viewed / Viewed to completion / Subscribed / Visited website
  4. ตั้ง Membership Duration: 30-90 วัน
  5. สร้างแคมเปญ Remarketing แยกจากแคมเปญปกติ
- **ข้อควรจำ:**
  - Membership ไม่ควรนานเกิน 90 วัน (คนลืมเราแล้ว)
  - แยก List ตามพฤติกรรม ปรับ Message ให้ตรงกลุ่ม

## SLIDE 6: Custom Intent Audiences

- **คืออะไร:** สร้าง Audience จากคนที่ค้นหา Keywords เฉพาะใน Google
- **ทำไมถึงเด็ด:** นำ Search Intent มาใช้กับ Video Ads
- **วิธีสร้าง:**
  1. Audience → Custom Segments
  2. เลือก "People who searched for any of these terms"
  3. ใส่ Keywords ที่เกี่ยวข้อง
- **เทคนิค Competitor:**
  - ใส่ URL ของเว็บคู่แข่ง 3-5 เว็บ
  - Google หาคนที่มีพฤติกรรมคล้ายผู้เข้าชมเว็บนั้น
- **คำเตือน:** เริ่มงบน้อยก่อน ดู Performance 2 สัปดาห์

---

## SLIDE 7: Performance Max (PMax)

```
เราใส่ Assets
    ↓
Google AI จัดสรร
    ↓
แสดงใน 6 Channels:
YouTube | Search | Display | Gmail | Maps | Discover
```

- **ข้อดี:** Reach กว้าง, AI Optimize ให้อัตโนมัติ
- **ข้อเสีย:** ควบคุมน้อย, ไม่รู้ว่าโฆษณาไปที่ไหน
- **เงื่อนไข:** ต้องมี 30+ Conversions/เดือน จากแคมเปญปกติก่อน

## SLIDE 8: วิธีใช้ Performance Max ให้ดีที่สุด

| Asset Type | จำนวนแนะนำ |
|------------|-----------|
| Headlines | 5-15 ตัว |
| Descriptions | 2-5 ตัว |

- ใช้เป็นแคมเปญเสริม ไม่ใช่แคมเปญหลัก
- เปรียบเทียบ ROAS กับแคมเปญปกติเสมอ

## SLIDE 10: Scaling Strategy — 4 กฎ

| กฎ | รายละเอียด |
|----|-----------|
| 1. เพิ่มงบ 20-30%/สัปดาห์ | อย่าเพิ่ม 100% → Reset Learning Phase |
| 2. ขยาย Audience | Similar Audiences, Custom Intent ที่กว้างขึ้น |
| 3. Clone แคมเปญที่ดี | เปลี่ยน Targeting/Creative เล็กน้อย |
| 4. Monitor Plateau Effect | CPC เพิ่ม >20% → ชะลอ Scale |

```
แคมเปญทำงานดี
    ↓ เพิ่มงบ 20-30%/สัปดาห์
ยังดีอยู่?
    ├── ใช่ → เพิ่มต่อ + ขยาย Audience
    └── ไม่ (Plateau) → หยุด Scale + Clone แคมเปญใหม่
```
