### Q1: Capture Page เปรียบเทียบได้กับอะไร?

A) ห้องนอนในบ้าน
B) ประตูหน้าบ้าน
C) หลังคาบ้าน
D) รั้วบ้าน

✅ **คำตอบ: B) ประตูหน้าบ้าน**

**อธิบาย:** Capture Page เปรียบเหมือนประตูหน้าบ้านที่รับลูกค้าเข้ามา ส่วน Thank You Page เหมือนห้องรับแขก ถ้าประตูเปิดไม่ได้หรือไปผิดห้อง ลูกค้าจะหายไปเลย

---

### Q2: ใน Kartra จะตั้งค่า Redirect หลังกรอกฟอร์มที่ไหน?

A) Dashboard → Settings → General
B) Form Element → After Submission Settings
C) Page Settings → SEO
D) My Pages → URL Settings

✅ **คำตอบ: B) Form Element → After Submission Settings**

**อธิบาย:** เข้า Page Builder → เลือก Capture Page → คลิก Form Element → After Submission Settings → เลือก Redirect to Page → เลือก Thank You Page ที่สร้างไว้

---

### Q3: สำหรับ Youtube Ads Campaign ควรใช้ Opt-in แบบใด?

A) Double Opt-in เพื่อกรองลูกค้าคุณภาพ
B) Triple Opt-in เพื่อความปลอดภัยสูงสุด
C) Single Opt-in เพื่อให้ Flow ลื่นไหล
D) ไม่ต้อง Opt-in คนเข้าได้เลย

✅ **คำตอบ: C) Single Opt-in เพื่อให้ Flow ลื่นไหล**

**อธิบาย:** Single Opt-in ทำให้กรอกฟอร์มแล้วเข้า Thank You Page ได้ทันที ไม่ต้องยืนยันอีเมล ช่วยไม่เสีย Lead ระหว่างทาง ซึ่งเหมาะกับ Youtube Ads Campaign ที่ต้องการ Flow ลื่นไหล

---

### Q4: Redirect แบบใดแนะนำสำหรับ Youtube Ads Campaign?

A) External Redirect เพราะยืดหยุ่นกว่า
B) JavaScript Redirect เพราะเร็วกว่า
C) Meta Refresh Redirect เพราะง่ายที่สุด
D) Internal Redirect เพราะง่ายและเร็วกว่า

✅ **คำตอบ: D) Internal Redirect เพราะง่ายและเร็วกว่า**

**อธิบาย:** Internal Redirect ส่งไปยังหน้าอื่นภายใน Kartra เดียวกัน แค่เลือกจาก Dropdown ง่ายกว่าและเร็วกว่า External Redirect ที่ต้องใส่ Full URL ให้ถูก 100%

---

### Q5: ทำไมไม่ควร Redirect หลายชั้น?

A) เพราะ Google ห้าม
B) เพราะทำให้เพจโหลดช้า คนอาจปิดหน้าไปก่อน
C) เพราะ Kartra ไม่รองรับ
D) เพราะจะทำให้ Pixel ไม่ทำงาน

✅ **คำตอบ: B) เพราะทำให้เพจโหลดช้า คนอาจปิดหน้าไปก่อน**

**อธิบาย:** Redirect หลายชั้นเพิ่มเวลาโหลดหน้า ถ้าเพจโหลดช้า คนจะปิดหน้าไปก่อนถึง Thank You Page ควร Redirect ครั้งเดียวจาก Capture Page ไป Thank You Page เลย

---

### Q6: Tracking Pixel คืออะไร?

A) รูปภาพขนาดเล็กที่ใส่ในอีเมล
B) โค้ดเล็กๆ ที่ฝังในหน้าเว็บเพื่อส่งสัญญาณกลับไป Google Ads
C) ปลั๊กอินของ Chrome สำหรับวัดผล
D) โปรแกรมแยกต่างหากที่ต้องติดตั้งบนคอมพิวเตอร์

✅ **คำตอบ: B) โค้ดเล็กๆ ที่ฝังในหน้าเว็บเพื่อส่งสัญญาณกลับไป Google Ads**

**อธิบาย:** Tracking Pixel คือโค้ดเล็กๆ ที่ฝังไว้ในหน้าเว็บ เมื่อมีคนเข้ามาถึงหน้านั้น Pixel จะส่งสัญญาณกลับไปบอก Google Ads ว่า "มีคนมาถึงหน้านี้แล้ว" ทำให้วัดผลแคมเปญได้

---

### Q7: Google Ads Conversion Tag ควรติดตั้งที่หน้าไหน?

A) Capture Page เท่านั้น
B) Thank You Page เท่านั้น
C) ทั้ง Capture Page และ Thank You Page
D) หน้า Homepage ของเว็บไซต์

✅ **คำตอบ: B) Thank You Page เท่านั้น**

**อธิบาย:** Google Ads Conversion Tag ติดบน Thank You Page เพื่อนับจำนวน Conversion (Lead) เพราะคนที่มาถึง Thank You Page คือคนที่กรอกฟอร์มเสร็จแล้ว = ได้ Lead 1 คน

---

### Q8: Google Analytics Tag ควรติดตั้งที่หน้าไหน?

A) Capture Page เท่านั้น
B) Thank You Page เท่านั้น
C) ทั้ง Capture Page และ Thank You Page
D) ไม่จำเป็นต้องติดตั้ง

✅ **คำตอบ: C) ทั้ง Capture Page และ Thank You Page**

**อธิบาย:** Google Analytics Tag ติดบนทั้ง 2 หน้า เพื่อดูพฤติกรรมผู้ใช้ เช่น มีคนเข้า Capture Page กี่คน กรอกฟอร์มกี่คน อัตรา Conversion เป็นเท่าไร

---

### Q9: เครื่องมือใดใช้เช็คว่า Tracking Pixel ทำงานถูกต้อง?

A) Google Search Console
B) Google Tag Assistant
C) Google Ads Editor
D) Google Keyword Planner

✅ **คำตอบ: B) Google Tag Assistant**

**อธิบาย:** Google Tag Assistant เป็นเครื่องมือที่ช่วยเช็คว่า Tracking Pixel ที่ติดตั้งทำงานถูกต้องหรือไม่ ต้องเช็คก่อนปล่อยแคมเปญเสมอ

---

### Q10: ทำไมต้องทดสอบ Conversion Flow บน Mobile?

A) เพราะ Google ให้คะแนน Mobile สูงกว่า
B) เพราะ Kartra ทำงานบน Mobile เท่านั้น
C) เพราะคนส่วนใหญ่ดู Youtube บนมือถือ
D) เพราะ Pixel ทำงานได้เฉพาะบน Mobile

✅ **คำตอบ: C) เพราะคนส่วนใหญ่ดู Youtube บนมือถือ**

**อธิบาย:** คนส่วนใหญ่ดู Youtube บนมือถือ เมื่อคลิกลิงก์จากโฆษณาจะเปิด Capture Page บนมือถือ ถ้า Redirect ไม่ทำงานหรือเพจโหลดช้าบนมือถือ ก็จะเสีย Lead ทันที

---

### Q11: Page Speed ที่แนะนำสำหรับ Capture Page และ Thank You Page คือเท่าไร?

A) ไม่เกิน 1 วินาที
B) ไม่เกิน 3 วินาที
C) ไม่เกิน 10 วินาที
D) ไม่มีข้อจำกัด

✅ **คำตอบ: B) ไม่เกิน 3 วินาที**

**อธิบาย:** ถ้าเพจโหลดช้ากว่า 3 วินาที อาจเสีย Lead ไปเยอะ ใช้ Google PageSpeed Insights เช็คความเร็วทั้ง 2 หน้า ถ้าช้ากว่า 3 วินาทีต้อง Optimize

---

### Q12: ข้อใดเป็นลำดับขั้นตอนที่ถูกต้องของ Conversion Flow?

A) Thank You Page → Capture Page → Youtube Ad → Pixel
B) Youtube Ad → Capture Page → Redirect → Thank You Page → Pixel นับ Conversion
C) Capture Page → Youtube Ad → Thank You Page → Redirect
D) Pixel → Youtube Ad → Thank You Page → Capture Page

✅ **คำตอบ: B) Youtube Ad → Capture Page → Redirect → Thank You Page → Pixel นับ Conversion**

**อธิบาย:** Conversion Flow ที่สมบูรณ์: คนดูโฆษณาบน Youtube → คลิกลิงก์ → มาถึง Capture Page → กรอกฟอร์ม → Redirect ไป Thank You Page → Pixel ส่งสัญญาณ → Google Ads บันทึก Lead
