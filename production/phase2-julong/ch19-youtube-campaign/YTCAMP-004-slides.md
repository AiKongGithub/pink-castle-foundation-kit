## Slide 6: Google Tag Manager (GTM)

```
╔══════════════════════════════════════════╗
║                                          ║
║     GTM = กล่องเครื่องมือ               ║
║     tagmanager.google.com               ║
║                                          ║
║     แทนที่จะฝัง Code ทีละตัว...         ║
║                                          ║
║     ❌ วิธีเก่า:                        ║
║     ┌──────────────────────┐            ║
║     │ เว็บ ← Google Ads   │            ║
║     │ เว็บ ← GA4          │            ║
║     │ เว็บ ← FB Pixel     │            ║
║     │ (แก้โค้ดทุกครั้ง!)   │            ║
║     └──────────────────────┘            ║
║                                          ║
║     ✅ วิธี GTM:                        ║
║     ┌──────────────────────┐            ║
║     │ เว็บ ← GTM (1 code) │            ║
║     │   GTM ← ทุก Tag     │            ║
║     │   (จัดการผ่าน UI!)   │            ║
║     └──────────────────────┘            ║
║                                          ║
╚══════════════════════════════════════════╝
```

**Speaker Notes:** GTM เปรียบเหมือนกล่องเครื่องมือ ฝัง Code อันเดียวแล้วจัดการ Tag ทั้งหมดผ่าน Dashboard ง่ายต่อการเพิ่ม แก้ไข หรือลบ Tag
