## Slide 8: หน้าที่ของแต่ละชั้น

```
+==========================================+
|                                          |
|     หน้าที่ของแต่ละชั้น                  |
|                                          |
|     CAMPAIGN = "ทำอะไร ใช้เงินเท่าไร"    |
|     +--------------------------------+   |
|     | - เลือก Objective              |   |
|     | - กำหนด Budget (รายวัน/รวม)    |   |
|     | - ตั้ง Schedule (เริ่ม-จบ)      |   |
|     +--------------------------------+   |
|                                          |
|     AD GROUP = "ยิงใคร ราคาเท่าไร"       |
|     +--------------------------------+   |
|     | - เลือก Targeting (กลุ่มเป้าหมาย)|  |
|     | - ตั้ง Bidding Strategy         |   |
|     +--------------------------------+   |
|                                          |
|     AD = "โฆษณาจริงที่คนเห็น"            |
|     +--------------------------------+   |
|     | - อัปโหลดวิดีโอ                |   |
|     | - ใส่ข้อความ + CTA             |   |
|     +--------------------------------+   |
|                                          |
+==========================================+
```

**Speaker Notes:** แต่ละชั้นมีหน้าที่ชัดเจน Campaign ตอบคำถาม "ทำอะไร ใช้เงินเท่าไร" Ad Group ตอบ "ยิงใคร ราคาเท่าไร" Ad ตอบ "คนเห็นอะไร" ทั้ง 3 ชั้นต้องสอดคล้องกัน

## Slide 10: Budget เบื้องต้น

```
+==========================================+
|                                          |
|     Budget เบื้องต้น                     |
|                                          |
|     +--------------------------------+   |
|     | ผู้เริ่มต้น                     |   |
|     | 300-500 บาท/วัน                |   |
|     | = 10,000-15,000 บาท/เดือน      |   |
|     +--------------------------------+   |
|                                          |
|     +--------------------------------+   |
|     | Project Fortress               |   |
|     | 30,000 บาท/เดือน               |   |
|     | = 1,000 บาท/วัน               |   |
|     +--------------------------------+   |
|                                          |
|     หลักการ:                             |
|     เริ่มน้อย --> ทดสอบ --> เห็นผล       |
|     --> ค่อยเพิ่ม                        |
|                                          |
+==========================================+
```

**Speaker Notes:** Budget เริ่มต้นแนะนำ 300-500 บาท/วัน ของ Project Fortress ตั้งไว้ 30,000 บาท/เดือน หลักการสำคัญคือเริ่มน้อยทดสอบก่อน แล้วค่อยเพิ่มเมื่อเห็นผลลัพธ์ที่ดี

## Slide 11: เปรียบเทียบทหารสามก๊ก

```
+==========================================+
|                                          |
|     เปรียบเทียบ Campaign = กองทัพ        |
|                                          |
|     +-----+     +----------+            |
|     |     |     | CAMPAIGN |            |
|     | แม่ |     | กองทัพ   |            |
|     | ทัพ | ==> | Budget   |            |
|     |     |     | Objective|            |
|     +-----+     +----------+            |
|        |              |                  |
|     +-----+     +----------+            |
|     | กอง |     | AD GROUP |            |
|     | ร้อย| ==> | Targeting|            |
|     |     |     | Bidding  |            |
|     +-----+     +----------+            |
|        |              |                  |
|     +-----+     +----------+            |
|     |ทหาร |     | AD       |            |
|     | แต่ | ==> | Video    |            |
|     | ละคน|     | Text+CTA |            |
|     +-----+     +----------+            |
|                                          |
|     จัดกองทัพให้ดีก่อนออกรบ!             |
|                                          |
+==========================================+
```

**Speaker Notes:** เปรียบเทียบให้เห็นภาพชัด Campaign เหมือนกองทัพที่แม่ทัพบัญชาการ Ad Group เหมือนกองร้อยที่นายร้อยนำทีม Ad เหมือนทหารแต่ละคนที่ออกรบจริง จัดกองทัพให้ดีก่อนออกรบ
