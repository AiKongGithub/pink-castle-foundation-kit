## Slide 4: Single vs Double Opt-in

```
╔══════════════════════════════════════════╗
║                                          ║
║     📧 Opt-in Confirmation               ║
║                                          ║
║     ┌─────────────────────────────────┐  ║
║     │ Single Opt-in ✅ (แนะนำ)        │  ║
║     │ กรอกฟอร์ม → Thank You Page ทันที│  ║
║     │ Flow ลื่นไหล ไม่เสีย Lead       │  ║
║     └─────────────────────────────────┘  ║
║     ┌─────────────────────────────────┐  ║
║     │ Double Opt-in ⚠️               │  ║
║     │ กรอกฟอร์ม → ยืนยันอีเมล →     │  ║
║     │ Thank You Page                  │  ║
║     │ เสีย Lead ระหว่างทาง            │  ║
║     └─────────────────────────────────┘  ║
║                                          ║
║     → Youtube Ads ใช้ Single Opt-in!    ║
║                                          ║
╚══════════════════════════════════════════╝
```

**Speaker Notes:** สำหรับ Youtube Ads Campaign แนะนำ Single Opt-in เพราะ Flow ลื่นไหลกว่า Double Opt-in จะเสีย Lead ระหว่างขั้นตอนยืนยันอีเมล

## Slide 5: Internal vs External Redirect

```
╔══════════════════════════════════════════╗
║                                          ║
║     🔀 ประเภท Redirect                  ║
║                                          ║
║     ┌──────────────────────────┐         ║
║     │ Internal Redirect ✅      │         ║
║     │ ภายใน Kartra เดียวกัน    │         ║
║     │ ง่าย: เลือกจาก Dropdown  │         ║
║     │ เร็วกว่า                 │         ║
║     └──────────────────────────┘         ║
║     ┌──────────────────────────┐         ║
║     │ External Redirect ⚠️     │         ║
║     │ ไป URL ภายนอก           │         ║
║     │ ต้องใส่ Full URL ถูก 100%│         ║
║     │ อาจช้ากว่า              │         ║
║     └──────────────────────────┘         ║
║                                          ║
║     ⚡ Redirect ครั้งเดียว! ห้ามหลายชั้น ║
║                                          ║
╚══════════════════════════════════════════╝
```

**Speaker Notes:** Internal Redirect ง่ายและเร็วกว่า แนะนำสำหรับ Youtube Ads สิ่งสำคัญคือ Redirect ครั้งเดียว อย่าผ่านหลายหน้า

## Slide 6: Tracking Pixel คืออะไร

```
╔══════════════════════════════════════════╗
║                                          ║
║     📡 Tracking Pixel                    ║
║                                          ║
║     คืออะไร?                             ║
║     ┌────────────────────────────────┐   ║
║     │ โค้ดเล็กๆ ฝังในหน้าเว็บ       │   ║
║     │ เมื่อคนเข้ามา → ส่งสัญญาณ    │   ║
║     │ กลับไปบอก Google Ads          │   ║

║     └────────────────────────────────┘   ║
║                                          ║

║                                          ║
╚══════════════════════════════════════════╝
```

**Speaker Notes:** Tracking Pixel คือตัวส่งสัญญาณ ถ้าไม่มี Pixel ก็ไม่มีข้อมูล ไม่มีข้อมูลก็วัดผลไม่ได้ เท่ากับเสียเงินโฆษณาฟรี

## Slide 8: วิธีติดตั้ง Pixel ใน Kartra

```
╔══════════════════════════════════════════╗
║                                          ║
║     🛠️ ติดตั้ง Pixel ใน Kartra          ║
║                                          ║
║     ┌───────────────────────────────┐    ║
║     │ 1. Page Settings              │    ║
║     │ 2. Tracking Code              │    ║
║     │ 3. วาง Pixel Code:            │    ║
║     │    ┌─────────────────────┐    │    ║
║     │    │ Header Code (GA Tag)│    │    ║
║     │    ├─────────────────────┤    │    ║
║     │    │ Body Code (Conv Tag)│    │    ║
║     │    └─────────────────────┘    │    ║
║     └───────────────────────────────┘    ║
║                                          ║
║     ⚡ เคล็ดลับ:                         ║
║     ติดตั้ง Pixel ก่อน → เช็ค Tag        ║
║     Assistant → แล้วค่อยปล่อยแคมเปญ     ║
║                                          ║
╚══════════════════════════════════════════╝
```

**Speaker Notes:** ใน Kartra เข้า Page Settings แล้วหา Tracking Code วาง Pixel Code ในช่อง Header หรือ Body ที่ต้องระวังคือต้องติดตั้งและเช็คก่อนปล่อยแคมเปญ
