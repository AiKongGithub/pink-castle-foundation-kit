|   Q1: งบน้อยเริ่มยังไง?                           |
|                                                  |
|   งบเริ่มต้น: 300-500 บาท/วัน                    |
|   Bidding: CPV (จ่ายเมื่อดูจริง > 30 วินาที)     |

|   แผน:                                           |
|   1. Campaign เดียวก่อน                          |

|                                                  |

> **Speaker Notes:** ไม่ต้องมีงบหลักหมื่น เริ่มเล็กๆ CPV จ่ายเฉพาะเมื่อคนดูจริง ถ้าข้ามไม่เสียเงิน Remarketing ใช้เงินน้อยกว่าแต่ Conversion Rate สูงกว่า

SLIDE 3:
```
+--------------------------------------------------+
|   Q2: โฆษณาไม่ผ่านการอนุมัติ                      |
|                                                  |
|   4 สาเหตุหลัก:                                  |
|   1. Misleading Content (เกินจริง)               |
|      เช่น "รวยใน 7 วัน" "การันตี 100%"          |
|   2. Trademark Issues (แบรนด์คนอื่น)            |
|   3. Copyright (เพลง/ภาพมีลิขสิทธิ์)             |
|   4. Prohibited Content (ต้องมีเอกสาร)           |
|                                                  |
|   วิธีแก้: อ่าน Violation → แก้ → Appeal         |
|   ระยะเวลา: 1-2 วันทำการ                        |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** ป้องกันดีกว่าแก้ อ่าน Google Ads Policy ก่อนทำโฆษณา ถ้าโดน Disapprove อ่าน Violation ให้ละเอียดแล้วแก้ตรงจุด

SLIDE 4:
```
+--------------------------------------------------+
|   Q3: CPV สูงเกินไปทำยังไง?                       |
|                                                  |
|   CPV ที่ดี: 0.10 - 0.50 บาท                     |
|                                                  |
|   | สาเหตุ            | วิธีแก้              |    |
|   |-------------------|---------------------|    |
|   | Targeting แคบเกิน  | ขยาย Audience        |    |
|   | Creative ไม่ดี     | ปรับ 5 วินาทีแรก     |    |
|   | ช่วงแข่งขันสูง     | เลี่ยงเทศกาล/เพิ่มงบ |    |
|                                                  |
|   สัญญาณ: View Rate < 15% = Creative ไม่ดี     |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** View Rate ต่ำกว่า 15% แปลว่า Creative ไม่ดึงดูดพอ Google คิดเงินแพงขึ้น Reach ต่ำแปลว่า Targeting แคบเกินไป

SLIDE 5:
```
+--------------------------------------------------+
|   Q4: Conversion Tracking ไม่ทำงาน               |
|                                                  |
|   Checklist 4 จุด:                               |
|   [ ] Google Tag ติดตั้งถูกหน้า (ทุกหน้า)        |
|   [ ] Conversion Action ถูกประเภท                |
|   [ ] Google Tag Assistant = Tag Fire ✓          |
|   [ ] GTM Trigger ถูก + Published Version        |
|                                                  |
|   จุดพลาดบ่อย:                                    |
|   Attribution Window ตั้งสั้นเกิน (ควร 30 วัน)    |
|   → Conversion รายงานน้อยผิดปกติ                 |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** Attribution Window ปกติตั้ง 30 วัน ถ้าตั้ง 1 วัน คนดูโฆษณาวันนี้แล้ว Convert อีก 3 วันถัดมาจะไม่นับ Conversion ที่รายงานจะน้อยกว่าจริง

SLIDE 6:
```
+--------------------------------------------------+
|   Q5: เลือก TrueView หรือ Bumper?                |
|                                                  |
|   | Format    | ความยาว  | Bidding | เหมาะกับ |  |
|   |-----------|---------|---------|---------|   |
|   | TrueView  | ไม่จำกัด | CPV     | Aware + |  |
|   | In-stream | ข้ามได้   |         | Consider|  |
|   | Bumper    | 6 วินาที | CPM     | Brand   |  |
|   |           | ข้ามไม่ได้|         | Recall  |  |
|   | Discovery | ไม่จำกัด | CPV     | Channel |  |
|   |           | หน้าค้นหา |         | Growth  |  |
|                                                  |
|   มือใหม่: TrueView ก่อน → Bumper เสริม         |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** TrueView ให้ข้อมูลมากกว่า รู้ว่าคนดูถึงตรงไหน ส่วน Bumper เน้นให้คนจำแบรนด์ได้ เริ่ม TrueView ก่อนแล้วค่อยเพิ่ม Bumper

SLIDE 7:
```
+--------------------------------------------------+
|   Q6: Remarketing ต้องรอนานแค่ไหน?               |
|                                                  |
|   ขั้นต่ำ: 100 Users ก่อนใช้งานได้               |
|                                                  |
|   | Traffic/วัน | ระยะเวลาสะสม     |              |
|   |------------|-----------------|              |
|   | 50 คน/วัน  | 2-3 วัน          |              |
|   | 20 คน/วัน  | 5-7 วัน          |              |
|   | 10 คน/วัน  | 10-14 วัน        |              |
|                                                  |
|   ระหว่างรอ: วิ่ง Prospecting Campaign ก่อน      |
|   ครบ 100 → เปิด Remarketing เพิ่ม              |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** อย่ารอสะสม List เฉยๆ ทำ Prospecting ดึงคนใหม่เข้าเว็บคู่ไป พอ List ครบ 100 ก็เปิด Remarketing เพิ่มอีก Campaign

```
+--------------------------------------------------+
|   Q7: วิดีโอยาวแค่ไหนดี?                          |
|                                                  |
|   | เป้าหมาย      | ความยาวแนะนำ         |       |
|   |---------------|---------------------|       |
|   | Bumper Ads    | 6 วินาที (ตายตัว)     |       |
|   | Awareness     | 15-30 วินาที         |       |

|   ***** 5 วินาทีแรกสำคัญที่สุด *****            |

+--------------------------------------------------+
```
> **Speaker Notes:** ไม่มีคำตอบตายตัวว่ายาวแค่ไหน ขึ้นอยู่กับเป้าหมายและเนื้อหา แต่ 5 วินาทีแรกเป็นตัวตัดสินเสมอ ลงทุนเวลาออกแบบ Hook ให้มากที่สุด

SLIDE 9:
```
+--------------------------------------------------+
|   Q8: คู่แข่งยิง Ads เหมือนกัน                    |
|                                                  |
|   กลยุทธ์ที่ 1: Creative Quality                  |
|   - Hook ดึงดูดใน 5 วินาทีแรก                    |
|   - View Rate สูง → Google ให้ราคาถูก           |
|                                                  |
|   กลยุทธ์ที่ 2: Differentiation                   |
|   - คู่แข่งเน้นราคา → เราเน้นคุณภาพ             |
|   - คู่แข่งเน้น Feature → เราเน้น Emotion       |
|                                                  |
|   กลยุทธ์ที่ 3: Custom Intent Audiences           |
|   - Target คนค้นหาคำเกี่ยวกับคู่แข่ง             |
|   - แสดงโฆษณาของเราแทน                         |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** การแข่งขันเป็นเรื่องปกติ สิ่งที่ชนะคือ Creative ที่ดีกว่า Google ให้รางวัลโฆษณาที่ View Rate สูงด้วยราคาที่ถูกลง

SLIDE 10:
```
+--------------------------------------------------+
|   Q9: ทำเอง vs จ้างเอเจนซี่                      |
|                                                  |
|   | งบ/เดือน     | คำแนะนำ              |        |
|   |-------------|---------------------|        |
|   | < 30,000    | ทำเอง               |        |
|   | 30K - 100K  | ตามเวลา+ความรู้      |        |
|   | > 100,000   | พิจารณาจ้างเอเจนซี่  |        |
|                                                  |
|   สำคัญ: แม้จ้างเอเจนซี่                         |
|   ก็ต้องเข้าใจพื้นฐานเสมอ                        |
|   → ตรวจสอบงานได้                               |
|   → ถามคำถามถูก                                 |
|   → ไม่ถูกหลอก                                  |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** ค่าเอเจนซี่ปกติ 15-25% ของงบโฆษณา ถ้างบ 30K ค่าเอเจนซี่ 5-7K เหลืองบจริงแค่ 23K ไม่คุ้ม แต่งบ 100K ค่าเอเจนซี่ 15-25K ยังเหลืองบเยอะ

```
+--------------------------------------------------+
|   Q10: วัดผลสำเร็จอย่างไร — KPI Benchmarks      |
|                                                  |
|   | Metric     | ค่าที่ดี  | ต้อง Optimize |      |
|   |------------|---------|-------------|      |
|   | View Rate  | > 15%   | < 15%       |      |
|   | CPV        | < 1 บาท  | > 1 บาท     |      |
|   | CTR        | > 0.5%  | < 0.5%      |      |
|   | ROAS       | > 3x    | < 3x        |      |
|                                                  |
|   ROAS > 3x = คุ้มค่า                            |
|   ลงไป 1 บาท ได้กลับ 3 บาท                      |
|                                                  |
|   ROAS < 3x → Optimize                          |
|   Creative / Targeting / Landing Page            |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** ROAS เป็น North Star Metric ที่ต้องดู แต่ต้องดูร่วมกับ View Rate CPV CTR ด้วย ถ้า ROAS ต่ำต้องไล่หาว่าจุดไหนเป็นปัญหา
