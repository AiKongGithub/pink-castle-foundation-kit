SLIDE 5:
```
+--------------------------------------------------+
|   เกณฑ์ตรวจ 4 ระดับ                               |
|                                                  |
|   | ระดับ | ส่วนที่ครบ | คุณภาพ            |      |
|   |-------|----------|------------------|      |

|   - Conversion Tracking ต้อง Work จริง           |

+--------------------------------------------------+
```
> **Speaker Notes:** ความเข้าใจสำคัญกว่าความครบ ถ้าอธิบายได้ว่าทำไมเลือก Targeting แบบนี้ ทำไมวิดีโอเป็นแบบนี้ แสดงว่าเข้าใจจริง
