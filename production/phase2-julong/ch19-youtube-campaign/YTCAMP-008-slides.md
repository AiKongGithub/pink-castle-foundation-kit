SLIDE 5:

SLIDE 6:
```
+--------------------------------------------------+
|   Optimization Tips — หลัง Launch                 |
|                                                  |
|   1. Bid Adjustment                              |
|      → เพิ่ม Bid กลุ่มที่ดี ลดกลุ่มที่ไม่ดี      |
|      → เริ่มหลังวิ่ง 3-7 วัน                      |
|                                                  |
|   2. Audience Refinement                         |
|      → สร้าง Lookalike จากคนที่ Convert           |
|                                                  |
|   3. Ad Rotation                                 |
|      → หมุนเวียน Creative ปิดตัวที่ไม่ดี          |
|                                                  |
|   กฎ: ปรับทีละอย่าง รอ 3-5 วัน                   |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** กฎเหล็กของ Optimization คือปรับทีละตัวแปร รอดูผล 3-5 วัน แล้วค่อยปรับต่อ ถ้าปรับหลายอย่างพร้อมกันจะไม่รู้ว่าอะไรทำให้ดีขึ้น

SLIDE 7:
```
+--------------------------------------------------+
|   Budget Scaling — เพิ่มงบอย่างไรให้ถูกวิธี       |
|                                                  |
|   กฎทอง: เพิ่มทีละ 20-30% ต่อครั้ง              |
|                                                  |
|   ตัวอย่าง:                                      |
|   500 → 650 → 850 → 1,100 บาท/วัน              |
|      ↑ 5-7 วัน ↑ 5-7 วัน ↑ 5-7 วัน              |
|                                                  |
|   ห้าม: เพิ่มทีเดียว 2-3 เท่า                    |
|   → Google Relearn ใหม่ → ผลแย่ลงชั่วคราว       |
|                                                  |
|   เริ่ม Scale: หลัง Campaign ดี 7-14 วัน         |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** Gradual Scaling คือหัวใจของ Budget Management เพิ่มทีละนิดรักษา Algorithm Learning ที่สะสมมา

SLIDE 8:
```
+--------------------------------------------------+
|   เมื่อไรต้อง Pause Campaign?                     |
|                                                  |
|   สัญญาณอันตราย:                                 |
|   - CPA สูงกว่าเป้า 50%+ (หลังวิ่ง 7 วัน)       |
|   - View Rate < 5% ต่อเนื่อง                    |
|   - Budget หมดเร็วแต่ไม่มี Conversion            |
|                                                  |
|   Pause → วิเคราะห์ 3 จุด:                       |
|   1. Targeting (คนถูกกลุ่มไหม?)                  |
|   2. Creative (วิดีโอดึงดูดไหม?)                  |
|   3. Landing Page (หน้าพร้อม Convert ไหม?)       |
|                                                  |
|   แก้แล้ว → เปิดใหม่                             |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** อย่าปล่อยให้ Campaign ที่ไม่ดีวิ่งต่อ ยิ่งปล่อยยิ่งเสียเงิน ต้องกล้า Pause แล้ววิเคราะห์สาเหตุ
