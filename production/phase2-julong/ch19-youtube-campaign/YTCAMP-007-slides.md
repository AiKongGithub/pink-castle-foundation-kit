SLIDE 4:
```
+--------------------------------------------------+
|   Bidding Strategy — เลือกให้ตรงเป้าหมาย        |
|                                                  |
|   | Strategy | จ่ายเมื่อ     | เหมาะกับ       |  |
|   |----------|---------------|----------------|  |
|   | CPV      | ดูเกิน 30 วิ  | เน้น Engagement|  |
|   | CPM      | ทุก 1K แสดง   | เน้น Reach     |  |
|   | CPA      | เกิด Convert  | เน้น Action    |  |
|                                                  |
|   ถาม: เป้าหมายคือ ดู / เห็น / ทำ?              |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** ก่อนเลือก Bidding ถามตัวเองว่าต้องการอะไร — ดู (CPV), เห็น (CPM), ทำ Action (CPA) คำตอบจะบอกว่าควรเลือก Bidding แบบไหน

SLIDE 5:
```
+--------------------------------------------------+
|   Targeting Layers ซ้อน — แม่นยำแต่ระวัง Reach   |
|                                                  |
|   Layer 1: Audience (คนสนใจเทคโนโลยี)           |
|       +                                          |
|   Layer 2: Topic (Business Software)             |
|       +                                          |
|   Layer 3: Keyword ("AI tools for business")     |
|       =                                          |
|   ผลลัพธ์: แม่นยำสูง แต่ Reach แคบ              |
|                                                  |
|   กฎ: ซ้อน 2-3 ชั้นพอ                            |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** การซ้อน Layers ทำให้เข้าถึงคนที่ตรงกลุ่มมาก แต่ถ้าซ้อนเกิน 3 ชั้น Reach จะน้อยจนไม่ได้ข้อมูลเพียงพอสำหรับ Google Optimize

---

SLIDE 6:
```
+--------------------------------------------------+
|   Ad Creative Setup — 5 ขั้นตอน                  |
|                                                  |
|   1. วาง Video URL จาก Youtube                   |
|   2. ใส่ Headline (สั้น กระชับ ดึงดูด)            |
|   3. ใส่ Description (บอกประโยชน์)               |
|   4. Companion Banner (300x60 px)                |
|   5. CTA Overlay (ข้อความ + Final URL)           |
|                                                  |
|   ** วิดีโอต้อง Upload ลง Youtube ก่อน **        |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** วิดีโอต้องอยู่บน Youtube ก่อน ไม่สามารถอัปโหลดตรงเข้า Google Ads ได้ ตั้ง Unlisted ก็ได้ ไม่จำเป็นต้อง Public

SLIDE 8:
```
+--------------------------------------------------+
|   Call-to-Action Overlay                         |
|                                                  |
|   ตั้งค่า 2 อย่าง:                               |
|   1. ข้อความ CTA (ปุ่มบนวิดีโอ)                 |
|   2. Final URL (ลิงก์ปลายทาง)                   |
|                                                  |
|   | เป้าหมาย   | CTA ที่เหมาะ              |    |
|   |------------|--------------------------|    |
|   | Awareness  | "เรียนรู้เพิ่มเติม"         |    |
|   | Conversion | "สมัครฟรี" / "ซื้อเลย"     |    |
|   | Traffic    | "เข้าชมเว็บ"              |    |
|                                                  |
+--------------------------------------------------+
```
> **Speaker Notes:** CTA ต้องสอดคล้องกับ Campaign Objective ข้อความที่ไม่ตรงจะทำให้คนสับสนและ CTR ต่ำ

SLIDE 11:
```
+--------------------------------------------------+

|                                                  |
|   1. กด Preview                                  |
```
