### SLIDE 2: ปัญหาของ Course Creator ส่วนใหญ่

## Single Point of Failure

**สถานการณ์:**
- สร้างคอร์ส 1 ตัว

- ขายไม่ออก = จบ

**ทางออก:**
> สร้าง **หลายช่องทางรายได้** ที่เชื่อมโยงกัน

*อย่าวาง "ไข่ทั้งหมด" ในตะกร้าใบเดียว*

### SLIDE 4: Recurring Revenue — พลังของ Membership

## Membership = เครื่องพิมพ์เงิน

**ตัวอย่างคำนวณ:**
```
สมาชิก 100 คน x 990 บาท =  99,000 บาท/เดือน
สมาชิก 300 คน x 990 บาท = 297,000 บาท/เดือน
สมาชิก 500 คน x 990 บาท = 495,000 บาท/เดือน
```

**สิ่งที่ต้องให้สมาชิก:**
- Content ใหม่ทุกเดือน
- Community exclusive
- Live Q&A session
- เข้าถึง Resource library

### SLIDE 5: Coaching — รายได้ต่อหัวสูงสุด

## Coaching: High-Ticket Revenue

**1-on-1 Coaching**
- ราคา: 15,000 - 100,000+ บาท/คน
- ข้อดี: รายได้สูงต่อหัว
- ข้อจำกัด: ใช้เวลาตัวเอง ไม่ scale ได้

**Group Coaching**
- ราคา: 5,000 - 30,000 บาท/คน
- สอน 10-20 คนพร้อมกัน
- **Win-Win**: ลูกค้าจ่ายถูกกว่า, Trainer ได้เงินรวมมากกว่า

---

### SLIDE 6: Upsell & Cross-sell

## ลูกค้าเก่า = ขุมทอง

> ลูกค้าเก่าซื้อซ้ำง่ายกว่าหาลูกค้าใหม่ **5-7 เท่า**

**Upsell** (เสนอสินค้าเพิ่มหลังซื้อ):
- Workbook / Template
- คอร์สขั้นสูง
- เข้าถึง Community

**Cross-sell** (สินค้าต่างหมวดที่เกี่ยวข้อง):
- ซื้อคอร์สการตลาด → เสนอคอร์ส Copywriting
- ซื้อคอร์สออกแบบ → เสนอคอร์ส Branding

### SLIDE 7: Affiliate — ทีมขายไร้เงินเดือน

## Affiliate Marketing

**คุณแนะนำคนอื่น:**
- แนะนำเครื่องมือ/บริการ → ได้ค่าคอมมิชชัน
- เช่น Hosting, Email Platform, Design Tools

**คนอื่นแนะนำคุณ:**
- สร้าง Affiliate Program ของตัวเอง
- ให้ค่าคอมมิชชัน 20-50%
- = ทีมขายที่จ่ายเฉพาะเมื่อขายได้
- ไม่มีเงินเดือนประจำ ไม่มีความเสี่ยง

---

### SLIDE 8: Value-Based Pricing

## ตั้งราคาจากคุณค่า ไม่ใช่ต้นทุน

**ตัวอย่าง:**
```
คอร์สช่วยลูกค้าหาเงินเพิ่ม 50,000 บาท/เดือน
ราคาคอร์ส = 9,900 บาท
ROI ของลูกค้า = 50,000/9,900 = 5x ในเดือนแรก!
```

**กฎ:** ราคาคอร์ส ควรเป็น **1/5 - 1/10** ของผลลัพธ์ที่ลูกค้าจะได้

*ถ้าลูกค้ารู้สึก "แพง" = คุณยังสื่อสารคุณค่าไม่ชัด*

### SLIDE 9: Tiered Pricing + Decoy Effect

## เสนอ 3 ตัวเลือก

| | Basic | Standard | Premium |
|---|---|---|---|
| ราคา | 2,990 | **5,990** | 12,990 |
| วิดีโอ | 20 ชม. | 20 ชม. | 20 ชม. |
| Workbook | - | ได้ | ได้ |
| Community | - | ได้ | ได้ |
| Q&A Live | - | - | ได้ |
| 1-on-1 | - | - | 2 ครั้ง |

**คนส่วนใหญ่เลือก Standard**
> Premium = Decoy (Anchor) ทำให้ Standard ดูคุ้ม

### SLIDE 10: Premium Offer

## ขาย Transformation ไม่ใช่ Information

**4 รูปแบบ Premium Offer:**
1. **Mastermind Group** — หลักหมื่น-แสน/คน
2. **Done-For-You** — ทำให้เลย
3. **VIP Coaching** — 1-on-1 ระยะยาว
4. **Certification** — หลักสูตรได้ใบรับรอง

**Key:**
> Information หาฟรีได้บน Internet
> Transformation ต้องมีคนนำทาง + ระบบ + Accountability

### SLIDE 13: ข้อผิดพลาดที่ต้องหลีกเลี่ยง

## 5 ข้อผิดพลาด Revenue Strategy

1. **พึ่งพาคอร์สเดียว** → สร้าง Product Ecosystem
2. **ตั้งราคาจากต้นทุน** → ใช้ Value-Based Pricing
3. **ไม่มี Membership** → สร้าง Recurring Revenue
4. **ไม่ Upsell** → เสนอสินค้าเพิ่มหลังซื้อ
5. **กลัวตั้งราคาสูง** → ลูกค้าจ่ายแพง = ตั้งใจเรียนมากกว่า
