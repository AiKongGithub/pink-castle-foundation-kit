## SEGMENT 2: Video Hosting — จะเก็บวิดีโออยู่ที่ไหน? (03:00 - 08:00)

**Host B:** เริ่มจากเรื่อง Video Hosting ก่อนเลยค่ะ มีตัวเลือกหลายแบบ แต่ละแบบมีข้อดีข้อเสียต่างกัน

**Host A:** ตัวเลือกแรกคือ Self-Hosted บน Kartra ครับ Kartra มี Built-in Video Hosting ให้ อัพโหลดวิดีโอเข้าไปตรงๆ ข้อดีคือจัดการง่าย ไม่ต้องใช้เครื่องมืออื่น แต่ข้อเสียคือพื้นที่จำกัดตาม Plan ที่เลือก และ Bandwidth อาจไม่เพียงพอถ้ามีสมาชิกเยอะ

**Host B:** ตัวเลือกที่สองคือ Third-party Hosting ค่ะ เช่น Vimeo Pro หรือ Wistia ซึ่งเป็น Professional Video Hosting ที่ออกแบบมาสำหรับ Course Creator โดยเฉพาะ ข้อดีคือ Player สวย มีระบบ Analytics ดูว่าสมาชิกดูถึงตรงไหน Bandwidth ไม่จำกัด แต่มีค่าใช้จ่ายเพิ่ม

**Host A:** ตัวเลือกที่สามคือ YouTube Unlisted ครับ อัพโหลดขึ้น YouTube แต่ตั้งเป็น Unlisted แล้ว Embed ใน Membership ข้อดีคือฟรี Bandwidth ไม่จำกัด แต่ข้อเสียคือ ลูกค้าอาจ Share Link ได้ YouTube มีโฆษณา (ถ้าไม่ใช่ Premium) และดูไม่มืออาชีพ

**Host B:** อาจารย์แนะนำว่าถ้าเริ่มต้นให้ใช้ Kartra Built-in ก่อนค่ะ แล้วพอมีสมาชิกเยอะค่อยย้ายไป Vimeo Pro เพราะ Kartra Embed Vimeo ได้ง่ายมาก และ Vimeo ไม่มีโฆษณา ดูเป็นมืออาชีพกว่า YouTube

## SEGMENT 3: File Formats และ Compression (08:00 - 13:00)

**Host A:** มาเรื่อง Technical กันบ้างครับ File Format ที่แนะนำคือ MP4 ด้วย H.264 Codec เพราะรองรับทุก Browser และทุก Device ทั้ง Desktop, iOS, Android

**Host B:** Resolution ที่แนะนำค่ะ สำหรับคอร์สออนไลน์ 1080p (1920x1080) ถือว่าเพียงพอแล้ว ไม่จำเป็นต้อง 4K เพราะ ไฟล์ใหญ่เกินไป โหลดช้า ใช้ Bandwidth เยอะ และคนส่วนใหญ่ดูบนจอเล็กอยู่แล้ว 1080p ชัดพอและไฟล์ขนาดเหมาะสม

**Host A:** เรื่อง Compression สำคัญมากครับ วิดีโอดิบจาก Camera มักมีขนาดใหญ่มาก ต้อง Compress ก่อนอัพโหลด เครื่องมือที่แนะนำคือ HandBrake ซึ่งเป็น Free Software ตั้ง Bitrate ที่ 5-8 Mbps สำหรับ 1080p จะได้ไฟล์ที่คุณภาพดีและขนาดเล็กพอ

**Host B:** Audio ก็สำคัญไม่แพ้ภาพค่ะ ใช้ AAC Codec, Bitrate 128-256 kbps, Stereo คนยอมรับภาพที่ไม่คมได้ แต่ถ้าเสียงแย่จะปิดทันที อาจารย์เน้นว่าลงทุนกับ Microphone ดีๆ คุ้มกว่าลงทุนกับกล้องแพงๆ เพราะเสียงมีผลต่อ Retention มากกว่าภาพ

**Host A:** และอย่าลืม Framerate ครับ 30fps เป็นมาตรฐานสำหรับ Talking Head วิดีโอ ถ้าเป็น Screen Recording 24fps ก็พอ ไม่ต้อง 60fps เว้นแต่เป็นเนื้อหาที่มี Motion เยอะ

## SEGMENT 4: Thumbnail และการจัดระเบียบบทเรียน (13:00 - 18:00)

**Host B:** มาถึง Thumbnail Creation ค่ะ Thumbnail เป็นภาพปกของวิดีโอ เป็นสิ่งแรกที่สมาชิกเห็น ถ้า Thumbnail น่าสนใจ สมาชิกจะคลิกเข้าดู

**Host A:** หลักการออกแบบ Thumbnail ที่ดีครับ หนึ่ง ใช้ Font ขนาดใหญ่ที่อ่านง่าย สอง มีใบหน้าผู้สอน (Face) เพราะสมองมนุษย์สนใจใบหน้ามากที่สุด สาม ใช้สีที่สอดคล้องกับแบรนด์ สี่ แสดงหมายเลขบทเรียนให้ชัดเจน และห้า ขนาดแนะนำ 1280x720 pixels

**Host B:** การจัดระเบียบบทเรียนค่ะ ต้องจัดเป็นระบบที่ชัดเจน ใน Kartra จัดได้แบบ Hierarchy คือ Course → Module → Lesson ตัวอย่าง Course "การตลาดออนไลน์" → Module 1 "พื้นฐาน" → Lesson 1.1 "Marketing คืออะไร", Lesson 1.2 "กลุ่มเป้าหมาย"

**Host A:** การตั้งชื่อบทเรียนสำคัญมากครับ ต้องบอกให้ชัดว่าเรียนจบแล้วจะได้อะไร เช่น แทนที่จะตั้งว่า "บทที่ 3" ให้ตั้งว่า "วิธีสร้าง Content ที่มีคนแชร์" ชื่อที่ดีทำให้สมาชิกรู้สึกว่าทุกบทเรียนมีคุณค่าและอยากกดเข้าดู

**Host B:** เรื่อง Lesson Duration ค่ะ อาจารย์แนะนำว่าวิดีโอแต่ละตอนไม่ควรยาวเกิน 15-20 นาที ถ้ายาวกว่านั้นให้แบ่งเป็นหลาย Part เพราะ Attention Span ของคนส่วนใหญ่อยู่ที่ 10-15 นาที วิดีโอสั้นมี Completion Rate สูงกว่าวิดีโอยาวมาก

## SEGMENT 5: Video Player Settings และ Mobile Optimization (18:00 - 22:00)

**Host A:** มาถึง Video Player Settings ครับ ใน Kartra สามารถปรับแต่ง Player ได้หลายอย่าง เช่น Auto-play เปิดหรือปิด (แนะนำปิดเพราะรบกวนผู้ใช้) แถบ Progress Bar ให้เห็นหรือไม่ ปุ่ม Speed Control (1x, 1.5x, 2x) ซึ่งสมาชิกชอบมาก และปุ่ม Download ถ้าต้องการให้ดาวน์โหลดได้

**Host B:** Playback Speed ที่สำคัญมากค่ะ สมาชิกหลายคนชอบดูวิดีโอเร็วขึ้น 1.5x หรือ 2x ถ้า Player ไม่มีตัวเลือกนี้ จะรู้สึกเสียเวลา เป็น Feature เล็กๆ ที่ส่งผลต่อ User Experience มาก

**Host A:** Mobile Optimization ครับ ส่วนนี้สำคัญมากเพราะคนส่วนใหญ่ดูวิดีโอบนมือถือ ต้องเช็ค Player แสดงผลดีบนจอเล็ก ปุ่มต่างๆ กดได้ง่ายด้วยนิ้ว วิดีโอ Landscape เมื่อหมุนมือถือต้องเป็น Full Screen อัตโนมัติ และ Adaptive Bitrate ปรับคุณภาพตามความเร็วอินเทอร์เน็ต

**Host B:** อาจารย์ยังแนะนำเรื่อง Offline Viewing ด้วยค่ะ ถ้าเป็นไปได้ ให้สมาชิกดาวน์โหลดวิดีโอไว้ดูแบบ Offline บนมือถือได้ เพราะไม่ใช่ทุกที่จะมีอินเทอร์เน็ตแรง โดยเฉพาะสมาชิกในต่างจังหวัด Feature นี้เพิ่ม Engagement ได้มาก

**Host A:** สรุปเรื่องวิดีโอครับ ใช้ MP4 H.264, 1080p, Compress ให้ดี, Audio คุณภาพสูง, Thumbnail น่าสนใจ, จัดระเบียบชัดเจน, Player Settings เหมาะสม, และ Mobile-friendly ทำครบทุกข้อแล้วสมาชิกจะมีประสบการณ์การเรียนที่ดีเยี่ยม

**Host B:** ตอนต่อไปเราจะไปเรื่อง Single Opt-in Page ซึ่งเป็นหน้าแรกที่ดึงคนเข้ามาใน Funnel สวัสดีค่ะ!

**Host A:** สวัสดีครับ!
