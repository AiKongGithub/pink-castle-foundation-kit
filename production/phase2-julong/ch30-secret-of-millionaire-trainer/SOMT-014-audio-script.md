## SEGMENT 1: เปิดรายการ — จาก "คอร์สเปล่า" สู่ "คอร์สเต็ม" (00:00 - 05:00)

**Host A:** สวัสดีครับทุกคน PinkCastle Podcast กลับมาแล้ว วันนี้เป็นตอนที่ยาวมากครับ เกือบ 1 ชั่วโมง เพราะเนื้อหาสำคัญมาก นั่นคือ "วิธีใส่บทเรียนในระบบสมาชิก"

**Host B:** ใช่ค่ะ ก่อนหน้านี้เราสร้างระบบสมาชิก สร้างหน้า Opt-in, Sale Page, Thank You Page เสร็จหมดแล้ว แต่ระบบยังเป็น "บ้านเปล่า" ค่ะ ยังไม่มีเนื้อหาข้างใน ตอนนี้คือการเปลี่ยน "บ้านเปล่า" ให้เป็น "บ้านที่มีชีวิต"

**Host A:** การใส่บทเรียนไม่ใช่แค่อัพโหลดวิดีโอแล้วจบครับ ต้องวางแผนโครงสร้างเนื้อหา จัดลำดับบทเรียน ใส่เนื้อหาข้อความ แนบไฟล์ดาวน์โหลด และตั้งค่าการนำทางให้สมาชิกเรียนรู้ได้อย่างเป็นระบบ

**Host B:** คิดง่ายๆ ว่าระบบสมาชิกคือ "โรงเรียนออนไลน์" ค่ะ ถ้าบทเรียนไม่เป็นระเบียบ สมาชิกจะสับสน หมดกำลังใจ และไม่ต่ออายุสมาชิก แต่ถ้าบทเรียนจัดอย่างดี สมาชิกจะเรียนรู้ได้อย่างมีประสิทธิภาพ เห็นผลลัพธ์ แล้วอยู่กับเราไปนานๆ

## SEGMENT 2: Content Structure Planning — Module, Lesson, Sub-lesson (05:00 - 15:00)

**Host A:** ก่อนจะเริ่มใส่เนื้อหา ต้องวางแผนโครงสร้างก่อนครับ โครงสร้างมาตรฐานมี 3 ระดับ Module อยู่ระดับบนสุด เป็นหัวข้อใหญ่ เหมือนบทในหนังสือ

**Host B:** ภายใน Module จะมี Lessons ค่ะ แต่ละ Lesson คือบทเรียนย่อยที่สอนเรื่องใดเรื่องหนึ่ง เหมือนหัวข้อย่อยในบท และภายใน Lesson อาจมี Sub-lessons อีก สำหรับเนื้อหาที่ลงลึกเป็นพิเศษ

**Host A:** ตัวอย่างครับ สมมติสร้างคอร์ส "Digital Marketing Mastery" Module 1 อาจเป็น "พื้นฐานการตลาดดิจิทัล" ภายในมี Lesson 1.1 "Marketing Funnel คืออะไร" Lesson 1.2 "Customer Journey" Lesson 1.3 "เลือก Platform ที่ใช่" แล้ว Module 2 อาจเป็น "Facebook Ads" มี Lesson 2.1 2.2 2.3 ไปเรื่อยๆ

**Host B:** หลักการวางโครงสร้างที่ดีมี 4 ข้อค่ะ ข้อแรก Logical Flow เนื้อหาต้องเรียงจากง่ายไปยาก จากพื้นฐานไปขั้นสูง ข้อสอง Bite-sized แต่ละ Lesson ไม่ควรยาวเกิน 15-20 นาที คนจะจำไม่ได้ถ้ายาวเกินไป

**Host A:** ข้อสาม Complete แต่ละ Lesson ต้อง "จบในตัว" ครับ สอนเรื่องเดียว มี Intro อธิบาย แล้วจบด้วย Action Item หรือ Assignment ที่ทำได้ทันที ข้อสี่ Consistent Format ทุก Lesson ใช้รูปแบบเดียวกัน เช่น เปิดด้วย "วันนี้จะเรียนเรื่อง..." จบด้วย "สิ่งที่ต้องทำหลังจากนี้คือ..."

**Host B:** จำนวน Module ที่เหมาะสมสำหรับคอร์สทั่วไปคือ 4-8 Modules ค่ะ แต่ละ Module มี 3-7 Lessons รวมแล้วคอร์สหนึ่งอาจมี 20-40 Lessons ไม่ต้องเยอะมากจนสมาชิกรู้สึก Overwhelm

## SEGMENT 3: Video Lesson Setup (15:00 - 25:00)

**Host A:** มาถึงขั้นตอนหลักครับ การใส่ Video Lesson ซึ่งเป็นรูปแบบเนื้อหาที่นิยมที่สุดในคอร์สออนไลน์ เพราะคนเรียนรู้จากวิดีโอได้ดีกว่าข้อความอย่างมาก

**Host B:** สิ่งที่ต้องเตรียมก่อนอัพโหลดค่ะ หนึ่ง ไฟล์วิดีโอที่ตัดต่อเรียบร้อย สอง Thumbnail รูปปกของแต่ละ Lesson สาม ชื่อ Lesson ที่ชัดเจน บอกว่าจะเรียนอะไร สี่ Description สั้นๆ สรุปเนื้อหาใน 2-3 ประโยค

**Host A:** Video Hosting มี 2 ตัวเลือกหลักครับ ตัวเลือกแรกคือ Built-in Hosting ของแพลตฟอร์ม เช่น Kartra มี Video Hosting ในตัว สะดวกแต่พื้นที่จำกัด ตัวเลือกที่สอง External Hosting เช่น Vimeo Pro, Wistia, YouTube (Unlisted) อัพโหลดที่นั่นแล้ว Embed เข้ามา

**Host B:** ข้อดีของ Vimeo Pro คือ สามารถซ่อน Player Controls ได้ ป้องกัน Download มี Analytics บอกว่าสมาชิกดูถึงตรงไหน ข้อเสียคือมีค่าใช้จ่ายรายเดือน ส่วน YouTube Unlisted ฟรีแต่คนอาจ Share Link ได้ ขึ้นอยู่กับงบประมาณและความต้องการค่ะ

**Host A:** เวลาอัพโหลดวิดีโอเข้าระบบ ต้องตั้งค่าเหล่านี้ครับ หนึ่ง Video Title ตรงกับ Lesson Name สอง Auto-play ปิดไว้ ให้สมาชิกคลิกเล่นเอง สาม Playback Speed ให้สมาชิกเร่งหรือชะลอได้ สี่ Closed Captions ถ้ามี จะช่วยคนที่ชอบอ่านมากกว่าฟัง

## SEGMENT 4: Text Content และ Downloadable Resources (25:00 - 40:00)

**Host B:** นอกจากวิดีโอ ยังต้องมี Text Content ประกอบด้วยค่ะ Text Content ทำหน้าที่ 3 อย่าง หนึ่ง เป็น Summary ของวิดีโอ สองเป็น Step-by-step Guide ที่สมาชิกทำตามได้ สาม เป็น Quick Reference ที่กลับมาดูทีหลังได้โดยไม่ต้องเปิดวิดีโอ

**Host A:** การจัดรูปแบบ Text Content ที่ดี ใช้ Heading ชัดเจน H2 H3 แบ่งส่วน ใช้ Bullet Points สำหรับรายการ ใช้ Bold สำหรับคำสำคัญ ใส่ Screenshots ประกอบขั้นตอน และมี Link ไปยัง Resource ที่เกี่ยวข้อง

**Host B:** มาถึง Downloadable Resources ค่ะ สิ่งที่แนบให้สมาชิกดาวน์โหลดได้ ยกตัวอย่างเช่น PDF Workbook สำหรับทำ Assignment, Template Files เช่น Spreadsheet หรือ Canva Template, Checklist สรุปขั้นตอน, Mind Map สรุปเนื้อหา, และ Swipe Files ตัวอย่างที่ใช้ได้เลย

**Host A:** Downloadable Resources มีค่ามากครับ เพราะทำให้สมาชิกรู้สึกว่า "ได้ของจริง" ไม่ใช่แค่ดูวิดีโอ และยังเป็นเหตุผลที่ทำให้สมาชิกต่ออายุ เพราะทุกเดือนอาจมี Resource ใหม่ๆ ให้ดาวน์โหลด

**Host B:** วิธีแนบไฟล์ในระบบ ส่วนใหญ่จะมีปุ่ม "Add Attachment" หรือ "Upload File" ใต้เนื้อหาของแต่ละ Lesson ค่ะ อัพโหลดไฟล์แล้วตั้งชื่อให้ชัดเจน เช่น "Module1-Lesson3-Funnel-Template.xlsx" อย่าตั้งชื่อว่า "file1.pdf" เพราะสมาชิกจะสับสน

## SEGMENT 5: Lesson Ordering, Navigation และ Progress Tracking (40:00 - 50:00)

**Host A:** การจัดลำดับบทเรียนสำคัญมากครับ สมาชิกต้องเรียนตามลำดับที่ถูกต้อง ไม่ใช่กระโดดข้ามไปเรียนบทหลังก่อน

**Host B:** มี 2 แนวทางค่ะ แนวแรกคือ Linear Learning บังคับให้เรียนตามลำดับ ต้องจบ Lesson 1 ก่อนถึงจะปลด Lesson 2 เหมาะกับคอร์สที่เนื้อหาต่อเนื่องกัน

**Host A:** แนวที่สอง Free Navigation ให้สมาชิกเลือกเรียน Lesson ไหนก็ได้ตามต้องการ เหมาะกับคอร์สแบบ Reference ที่แต่ละ Lesson จบในตัว ไม่จำเป็นต้องเรียนตามลำดับ

**Host B:** ส่วนใหญ่แนะนำ Hybrid Approach ค่ะ คือบังคับให้เรียน Module ตามลำดับ แต่ภายใน Module ให้เลือก Lesson ได้อิสระ เช่น ต้องจบ Module 1 ก่อนถึงจะเปิด Module 2 แต่ Lesson 1.1, 1.2, 1.3 เรียนก่อนหลังได้

**Host A:** เรื่อง Navigation ครับ ระบบต้องมี Sidebar หรือ Navigation Menu ที่แสดงรายการ Module และ Lesson ทั้งหมด สมาชิกต้องเห็นภาพรวมว่ามีอะไรบ้าง เรียนถึงไหนแล้ว และอะไรถัดไป

**Host B:** Preview/Locked Content Display ก็สำคัญค่ะ Lesson ที่ยังไม่ปลดล็อกควรแสดงชื่อและ Description ให้เห็น แต่มีไอคอนกุญแจหรือข้อความ "Locked" เพื่อให้สมาชิกรู้ว่ามีเนื้อหาอะไรรออยู่ สร้าง Anticipation

**Host A:** สุดท้าย Progress Tracking ครับ ระบบต้องแสดงความก้าวหน้าของสมาชิก เช่น "คุณเรียนจบ 40% ของคอร์สแล้ว" Progress Bar ที่เพิ่มขึ้นทุกครั้งที่จบ Lesson เป็น Motivation ที่ทรงพลัง เหมือน Progress Bar ในเกมที่ทำให้อยากเล่นต่อ

## SEGMENT 6: สรุปและปิดรายการ (50:00 - 56:53)

**Host B:** สรุปทั้งหมดค่ะ การใส่บทเรียนในระบบสมาชิกประกอบด้วย 5 ขั้นตอนหลัก

**Host A:** หนึ่ง Content Structure Planning วางโครงสร้าง Module → Lesson → Sub-lesson สอง Video Lesson Setup อัพโหลดวิดีโอ ตั้งค่า Thumbnail Title Description สาม Text Content Formatting ใส่เนื้อหาข้อความประกอบ สี่ Downloadable Resources แนบไฟล์ให้ดาวน์โหลด ห้า Lesson Ordering, Navigation และ Progress Tracking จัดลำดับ นำทาง และติดตามความก้าวหน้า

**Host B:** สิ่งที่ต้องจำไว้คือ คอร์สที่ดีไม่ได้ดีแค่เนื้อหา แต่ดีที่การจัดระเบียบค่ะ เนื้อหาเดียวกัน ถ้าจัดอย่างดีจะให้ผลลัพธ์คนละเรื่อง สมาชิกจะเรียนจบ เห็นผลลัพธ์ และบอกต่อ

**Host A:** ตอนหน้าเราจะต่อด้วยการใส่บทเรียนย่อยและบทเรียนที่ 2 ครับ ไปลงลึกเรื่องโครงสร้างแบบ Hierarchical อย่าพลาด สวัสดีครับ!
