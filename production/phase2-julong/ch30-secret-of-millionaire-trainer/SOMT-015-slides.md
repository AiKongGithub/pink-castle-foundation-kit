### SLIDE 3: Sub-lessons คืออะไร?
## "บทเรียนภายในบทเรียน"
```
Module 1: Facebook Ads
  └── Lesson 1: ตั้งค่า Business Manager
        ├── Sub-lesson 1.1: สร้าง Account
        ├── Sub-lesson 1.2: เพิ่ม Facebook Page
        └── Sub-lesson 1.3: ตั้งค่า Payment
```

**ข้อดี:**
- แก้ปัญหา Lesson ที่ยาวเกินไป
- Navigation ดีขึ้น — ข้ามไปส่วนที่ต้องการได้
- ไม่ต้อง Scrub หาจุดในวิดีโอยาว

### SLIDE 4: วิธีสร้าง Sub-lessons
## 2 วิธีในระบบสมาชิก

| วิธี | คำอธิบาย |
|------|---------|
| **Indent Method** | สร้าง Lesson → ลาก (Drag) ให้อยู่ภายใต้ Lesson แม่ |
| **Child Lesson** | สร้าง Lesson → กำหนด Parent Lesson |

**กฎเหล็ก:**
- ไม่เกิน 3 ระดับ (Module → Lesson → Sub-lesson)
- ใช้ระบบเลข: 1.1, 1.2, 1.3
- ชื่อบอกเนื้อหาเฉพาะเจาะจง

### SLIDE 5: Content Types 6 ประเภท
## ไม่ใช่ทุก Lesson ต้องเป็นวิดีโอ

| Type | ลักษณะ | เหมาะกับ |
|------|--------|---------|
| Video | บรรยาย/Screen Record | เนื้อหาที่ต้อง "ดู" |
| Text | เนื้อหาข้อความ | Reference Material |
| Audio | Podcast-style | ฟังระหว่างทำอื่น |
| PDF | เอกสาร Download | Workbook, Case Study |
| Interactive | Quiz/Assignment | Active Learning |
| Live Replay | บันทึก Live Q&A | คำถาม-ตอบ |

---

### SLIDE 6: ผสม Content Types
## ความหลากหลาย = Engagement สูง

```
Lesson 1: Video (บรรยาย)        ← ดูและฟัง
Lesson 2: Video + Workbook (PDF) ← ดู + ทำ
Lesson 3: Quiz + Discussion      ← ทดสอบ + พูดคุย
Lesson 4: Live Replay            ← ดูถามตอบ
Lesson 5: Audio + Text Summary   ← ฟังแล้วอ่าน
```

**แนะนำ: อย่างน้อย 3 ประเภทต่อ Module**

---

### SLIDE 7: Transition ระหว่าง Lessons
## ทำให้เนื้อหาต่อเนื่อง

**จบ Lesson 1:**
> "ตอนต่อไปเราจะเรียนเรื่อง [หัวข้อ Lesson 2]..."

**เริ่ม Lesson 2:**
> "จากตอนที่แล้วเราเรียนเรื่อง [สรุป Lesson 1]... วันนี้จะไปต่อเรื่อง..."

**ข้อจำ:** Lesson Duration ไม่จำเป็นต้องยาวเท่ากัน

---

### SLIDE 8: Quizzes — Active Learning Tool
## ไม่ใช่แค่ทดสอบ แต่ช่วยจำได้ดีขึ้น

| ประเภท | จำนวนข้อ | ตำแหน่ง |
|--------|---------|---------|
| Knowledge Check | 3-5 | ท้าย Lesson |
| Module Assessment | 10-15 | ท้าย Module |
| Final Exam | 20-30 | ท้ายคอร์ส |

**Retrieval Practice:**
> การทำ Quiz ช่วยจำได้ดีกว่าอ่าน/ดูซ้ำ **2-3 เท่า**

---

### SLIDE 9: Quiz Settings
## 4 สิ่งที่ต้องตั้งค่า

| Setting | คำอธิบาย | แนะนำ |
|---------|---------|-------|
| **Passing Score** | เกณฑ์ผ่าน | 70-80% |
| **Retry Policy** | ทำซ้ำได้กี่ครั้ง | Unlimited (Knowledge Check) |
| **Feedback** | คำอธิบายเมื่อตอบผิด | ให้ความรู้เพิ่ม ไม่ใช่แค่ "ผิด" |
| **Timer** | จำกัดเวลา | Optional (Final Exam เท่านั้น) |

---

### SLIDE 10: Quiz Design Tips
## เคล็ดลับข้อสอบที่ดี
- **เน้น Key Takeaways** — ไม่ใช่รายละเอียดปลีกย่อย
- **Plausible Distractors** — ตัวเลือกผิดต้องดูสมเหตุ
- **Feedback ให้ความรู้เพิ่ม** — ไม่ใช่แค่ "ถูก/ผิด"

**รูปแบบคำถามที่นิยม:**
Multiple Choice | True/False | Fill-in-blank | Matching | Short Answer

---

### SLIDE 11: Completion Requirements
## เลือกตามประเภทคอร์ส

| ระดับ | วิธี | เหมาะกับ |
|-------|------|---------|
| View-based | ดูวิดีโอจบ | Casual / Inspiration |
| Quiz-based | ผ่าน Quiz | Skill Development |
| Assignment-based | ส่ง Assignment | Workshop |
| **Combined** | ดู + Quiz + Assignment | **Certification** |

> *อย่าใช้ Combined กับคอร์ส Casual — ยากเกินจนสมาชิกเลิกเรียน*

---

### SLIDE 12: Certificate of Completion
## Motivator + Marketing Tool

**6 องค์ประกอบ:**
1. ชื่อคอร์ส
2. ชื่อสมาชิก (Dynamic)
3. วันที่จบ (Dynamic)
4. ลายเซ็นผู้สอน
5. Logo Brand
6. Certificate Number

**Dynamic = ดึงข้อมูลอัตโนมัติ → ไม่ต้องทำด้วยมือ**

---

### SLIDE 13: Shareable Certificate
## ทุกการแชร์ = โฆษณาฟรี

```
สมาชิกจบคอร์ส → ได้ Certificate
      ↓
แชร์บน LinkedIn / Facebook
      ↓
เพื่อนเห็น Brand + ชื่อคอร์ส
      ↓
สนใจสมัครเรียนด้วย!
```

**Trigger:** จบคอร์สครบ + ผ่าน Quiz → **Generate Certificate อัตโนมัติ**

---

### SLIDE 14: Course Completion Flow
## จากเริ่มเรียนถึง Certificate

```
เริ่มเรียน → Lesson + Sub-lessons
                   ↓
            Knowledge Check Quiz (ท้าย Lesson)
                   ↓
            Module Assessment (ท้าย Module)
                   ↓
            จบทุก Module → Final Exam
                   ↓
            ผ่านเกณฑ์ → Certificate!
                   ↓
            แชร์ → โฆษณาฟรี!
```
