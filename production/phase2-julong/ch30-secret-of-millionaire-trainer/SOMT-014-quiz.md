## เฉลยพร้อมคำอธิบาย

### ข้อ 1: ✅ B) Module → Lesson → Sub-lesson
> Module คือหัวข้อใหญ่ (เหมือนบท) Lesson คือบทเรียนย่อยในแต่ละ Module Sub-lesson คือเนื้อหาเจาะลึก เรียงจากใหญ่ไปเล็ก

### ข้อ 2: ✅ B) 15-20 นาที
> Lesson ที่ยาวเกินไปทำให้คนจำไม่ได้และหมดกำลังใจ Bite-sized 15-20 นาที ทำให้เรียนจบได้ง่าย Progress Bar ขยับบ่อย เป็น Motivation

### ข้อ 3: ✅ B) ซ่อน Player Controls ได้ ป้องกัน Download มี Analytics
> Vimeo Pro มีค่าใช้จ่าย ~$20/เดือน แต่ข้อดีคือ ป้องกันการ Download, ซ่อน Controls ที่ไม่ต้องการ และมี Analytics บอกว่าสมาชิกดูถึงตรงไหน

### ข้อ 4: ✅ B) บังคับเรียน Module ตามลำดับ แต่ Lesson ใน Module ให้เลือกอิสระ
> Hybrid Approach สร้างสมดุลระหว่าง Structure (ต้องเรียน Module ตามลำดับ) กับ Flexibility (เลือก Lesson ภายใน Module ได้อิสระ) เป็นแนวทางที่แนะนำ

### ข้อ 5: ✅ C) Social Media Login Credentials
> Social Media Credentials ไม่ใช่ Downloadable Resource ที่ให้สมาชิก ประเภทที่ถูกต้องคือ PDF Workbook, Template Files, Checklist, Mind Map และ Swipe Files

### ข้อ 6: ✅ C) 3 อย่าง (Summary, Step-by-step Guide, Quick Reference)
> Text Content ทำหน้าที่ 3 อย่าง: สรุปเนื้อหาวิดีโอ (Summary) ให้คำแนะนำทำตามทีละขั้น (Step-by-step) และเป็นแหล่งอ้างอิงกลับมาดูทีหลังโดยไม่ต้องเปิดวิดีโอ (Quick Reference)

### ข้อ 7: ✅ ถูก
> Consistent Format ทำให้สมาชิกรู้ว่าจะเจออะไรทุกครั้ง ลดความสับสน สร้างความคุ้นเคย และทำให้เรียนรู้ได้เร็วขึ้นเพราะไม่ต้องปรับตัวทุกครั้ง

### ข้อ 8: ✅ ผิด
> ควรแสดงชื่อและ Description ของ Lesson ที่ยังไม่ปลดล็อก พร้อมไอคอน "Locked" เพื่อสร้าง Anticipation ให้สมาชิกรู้ว่ามีอะไรรออยู่และอยากเรียนต่อ

### ข้อ 9: ✅ ถูก
> Progress Bar เป็น Gamification ที่ทรงพลัง เมื่อเห็นเปอร์เซ็นต์เพิ่มขึ้น สมาชิกรู้สึก Accomplish และอยากเรียนต่อจนจบ เหมือน Progress Bar ในเกม

### ข้อ 10: ✅ 4 ถึง 8
> คอร์สทั่วไปควรมี 4-8 Modules แต่ละ Module มี 3-7 Lessons ไม่เยอะจนสมาชิก Overwhelm แต่ครอบคลุมเนื้อหาทั้งหมด
