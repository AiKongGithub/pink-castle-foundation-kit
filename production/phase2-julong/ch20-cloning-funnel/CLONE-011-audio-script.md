## ช่วงที่ 1: ทบทวนสิ่งที่เรียนมาทั้ง 10 ตอน

**Host A:** ก่อนจะไปดูการบ้าน เรามาทบทวนกันก่อนครับว่า 10 ตอนที่ผ่านมาเราเรียนอะไรกันมาบ้าง ตอนที่ 1-2 เราเรียนเรื่อง Clone Funnel และ Import Campaign คือการนำ Template สำเร็จรูปเข้ามาใช้งาน ตอนที่ 3-8 คือการปรับแต่ง 6 หมวดหลัก ได้แก่ Capture Page, Headline, รูปภาพ, Pop up, Thank You Page และเนื้อหาอื่นๆ ตอนที่ 9-10 เข้าสู่เรื่อง Automation คือ Thank You Page กับ Sequence ครับ ทั้งหมดนี้คือกระบวนการสร้าง Funnel ครบวงจรครับ

**Host B:** ใช่ค่ะ ถ้าเปรียบเทียบเหมือนสร้างบ้าน ตอน 1-2 คือวางโครงสร้าง ตอน 3-8 คือตกแต่งภายใน และตอน 9-10 คือติดตั้งระบบไฟฟ้าประปา วันนี้ตอนที่ 11 คือการตรวจรับบ้าน อาจารย์จะเดินตรวจทุกห้อง ดูว่าอะไรเรียบร้อย อะไรต้องแก้ไข ก่อนจะส่งมอบบ้านให้เข้าอยู่ได้ค่ะ สิ่งที่สำคัญคือนักเรียนทุกคนได้ลงมือทำจริงครบทุกขั้นตอนก่อนมาถึงตอนนี้ค่ะ

## ช่วงที่ 2: ตรวจผลงานนักเรียน — จุดแข็งและจุดอ่อน

**Host A:** เข้าสู่ไฮไลท์ของตอนนี้เลยครับ อาจารย์เปิดผลงาน Funnel ของนักเรียนทีละคน สิ่งที่น่าสังเกตคือนักเรียนที่ทำได้ดีมักมีจุดร่วมกันครับ หนึ่ง Branding สม่ำเสมอ ทั้ง Funnel ใช้สีเดียวกัน โลโก้ตำแหน่งเดียวกัน ฟอนต์สไตล์เดียวกัน ดูเป็นมืออาชีพมาก สอง Headline ตรง Pain Point คือข้อความแรกที่ลูกค้าเห็นต้องพูดถึงปัญหาที่เขามีจริงๆ แล้วบอกว่าเราช่วยได้ สาม รูปภาพคุณภาพดี ไม่ใช้รูปจาก Template เดิม เปลี่ยนเป็นรูปของตัวเอง หรือรูปที่เกี่ยวข้องกับธุรกิจจริงครับ

**Host B:** ฝั่งจุดอ่อนที่พบบ่อยก็น่าสนใจมากค่ะ อันดับหนึ่งเลยคือ Branding ไม่สม่ำเสมอ Capture Page ใช้สีฟ้า Thank You Page ใช้สีชมพู Pop up เป็นสีเขียว ดูเหมือนคนละเว็บเลย อันดับสองคือ Headline ไม่ตรง Pain Point เขียนเป็นแนวบรรยาย เช่น เราให้บริการที่ดี แทนที่จะพูดถึงปัญหาของลูกค้าตรงๆ อันดับสามคือไม่ Test บน Mobile หลายคนออกแบบบน Desktop สวยมาก แต่เปิดบนมือถือแล้วเละ ข้อความล้น ปุ่มเล็กเกินกด รูปภาพไม่ Responsive ค่ะ

**Host A:** และที่สำคัญมากครับ Pop up Timing ไม่เหมาะสม บางคนตั้งให้ Pop up ขึ้นทันทีที่เข้าหน้า ลูกค้ายังไม่ทันอ่านอะไรเลยก็มีหน้าต่างมาบัง ทำให้รำคาญแล้วกดปิดไป ควรตั้ง Delay อย่างน้อย 5-10 วินาที หรือใช้ Exit Intent แทนครับ อีกอย่างคือ Sequence ไม่ปรับเนื้อหา Clone มาแล้วใช้ข้อความเดิมทั้งหมด ไม่เปลี่ยนชื่อแบรนด์ ไม่เปลี่ยนลิงก์ ถ้าลูกค้าได้รับอีเมลที่เขียนถึงแบรนด์อื่น ความน่าเชื่อถือหายหมดเลยครับ

## ช่วงที่ 3: ข้อผิดพลาดที่พบบ่อยและวิธีแก้ไข

**Host B:** มาสรุปข้อผิดพลาดทั้งหมดที่พบจากการตรวจการบ้านนะคะ อาจารย์แบ่งเป็น 6 หมวดหลักค่ะ หมวดแรก Branding ไม่สม่ำเสมอ วิธีแก้คือทำ Brand Guide ง่ายๆ กำหนดสี 2-3 สี ฟอนต์ 1-2 แบบ ใช้เหมือนกันทั้ง Funnel หมวดสอง Headline ไม่ตรง Pain Point วิธีแก้คือเริ่มจากปัญหาของลูกค้า ไม่ใช่เริ่มจากสินค้าของเรา เขียนเป็น Pain Point แล้วตามด้วย Solution ค่ะ

**Host A:** หมวดสาม ไม่ Test บน Mobile วิธีแก้คือทุกครั้งที่แก้ไขอะไร กด Preview บนมือถือด้วยทุกครั้ง Kartra มีโหมด Mobile Preview ให้ใช้ หมวดสี่ Pop up Timing ไม่เหมาะ วิธีแก้คือตั้ง Delay อย่างน้อย 5-10 วินาที หรือใช้ Exit Intent Trigger ดีกว่าครับ หมวดห้า Sequence ไม่ปรับ วิธีแก้คืออ่าน Email ทุกฉบับใน Sequence เปลี่ยนชื่อแบรนด์ ลิงก์ และเนื้อหาให้เป็นของเราทั้งหมด และหมวดหก Tags ผิด วิธีแก้คือตรวจสอบ Tag ทุกจุดใน Automation ว่า Assign ถูก Tag หรือไม่ครับ

**Host B:** อาจารย์ย้ำมากเลยค่ะว่า ข้อผิดพลาดเหล่านี้ทุกข้อแก้ได้ง่าย ใช้เวลาแค่ 15-30 นาทีต่อหมวด แต่ถ้าไม่แก้ มันจะทำลายความน่าเชื่อถือทั้งหมดที่สร้างมาค่ะ ลองคิดดูว่าลูกค้าเห็นว่า Funnel ของเราไม่เป็นมืออาชีพ เขาจะไว้ใจให้ข้อมูลหรือจ่ายเงินมั้ย คำตอบคือไม่แน่นอนค่ะ

## ช่วงที่ 4: Best Practices — Checklist ก่อนเปิด Funnel

**Host A:** มาถึงส่วนที่เป็นประโยชน์มากที่สุดครับ อาจารย์สรุปเป็น Checklist 7 หมวดก่อนเปิด Funnel ให้ลูกค้าจริง หมวดที่ 1 Branding ตรวจโลโก้ สีหลัก ฟอนต์ สม่ำเสมอทั้ง Funnel หรือไม่ หมวดที่ 2 Copy ตรวจ Headline ตรง Pain Point หรือไม่ ข้อความ CTA ชัดเจนหรือไม่ เนื้อหาเป็นภาษาของแบรนด์เราหรือไม่ หมวดที่ 3 Media ตรวจรูปภาพคุณภาพดีหรือไม่ วิดีโอเล่นได้หรือไม่ Responsive บน Mobile หรือไม่ครับ

**Host B:** หมวดที่ 4 Email ตรวจ Sequence ทุกฉบับเปลี่ยนเนื้อหาเป็นของเราแล้วหรือไม่ ลิงก์ในอีเมลถูกต้องหรือไม่ ส่ง Test Email ดูแล้วหรือไม่ หมวดที่ 5 Forms ตรวจฟอร์มกรอกได้จริงหรือไม่ ข้อมูลเก็บเข้าระบบถูกต้องหรือไม่ Redirect ไปถูกหน้าหรือไม่ หมวดที่ 6 Automations ตรวจ Tags ถูกต้องหรือไม่ Trigger ทำงานหรือไม่ Email Sequence เชื่อมถูกหรือไม่ และหมวดที่ 7 Testing ตรวจทั้ง Desktop และ Mobile ทดสอบ Flow ทั้งหมดตั้งแต่เข้าหน้าแรกจนถึง Thank You Page แล้วหรือไม่ค่ะ

**Host A:** Checklist นี้ถ้าผ่านหมดทั้ง 7 หมวด Funnel ของเราพร้อมเปิดใช้งานจริงครับ อาจารย์แนะนำให้ Print ออกมาติดข้างจอ ทุกครั้งที่สร้างหรือแก้ไข Funnel ให้เช็คทุกข้อก่อนกด Publish ครับ เพราะ Funnel ที่ดีไม่ใช่แค่สวย แต่ต้องทำงานได้ถูกต้องทุกจุดครับ

## ช่วงที่ 5: คำถาม-ตอบ และ Next Steps

**Host B:** ช่วงท้ายเป็นคำถาม-ตอบค่ะ นักเรียนถามปัญหาที่เจอจริง คำถามที่น่าสนใจมากคือ ถ้า Funnel Clone มาแล้วรู้สึกว่าไม่เหมาะกับธุรกิจเรา ควรปรับหรือสร้างใหม่ อาจารย์ตอบว่าให้ปรับครับ ไม่ต้องสร้างใหม่ เพราะโครงสร้าง Funnel ที่ Clone มามันถูกออกแบบมาแล้วว่า Flow ดี สิ่งที่ต้องเปลี่ยนคือเนื้อหา ไม่ใช่โครงสร้างค่ะ

**Host A:** อีกคำถามที่ดีครับคือ ต้องมีสินค้าพร้อมก่อนสร้าง Funnel ไหม อาจารย์ตอบว่าไม่จำเป็นครับ เริ่มจาก Lead Magnet ก่อนได้ สร้าง Capture Page เก็บอีเมล สร้างความสัมพันธ์ผ่าน Sequence แล้วค่อยเสนอขายทีหลัง Funnel ไม่จำเป็นต้องมี Checkout Page ตั้งแต่แรกครับ

**Host B:** และคำถามสุดท้ายค่ะ หลังจากจบ Chapter 20 นี้แล้วต้องไปเรียนอะไรต่อ อาจารย์แนะนำว่า Chapter 21 คือเรื่องระบบอีเมลค่ะ ซึ่งจะต่อยอดจาก Sequence ที่เรียนมา ไปสู่กลยุทธ์ Email Marketing แบบเต็มรูปแบบ ตั้งแต่การเขียน Subject Line การแบ่ง Segment ไปจนถึงการวิเคราะห์ผลค่ะ
