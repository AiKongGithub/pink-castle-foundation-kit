## SLIDE 3: Sequence คืออะไร? (What is a Sequence)
```
┌─────────────────────────────────────────────┐
│  Sequence คืออะไร?                          │
│  ─────────────────────────                  │
│                                             │

│                                             │
└─────────────────────────────────────────────┘
```

## SLIDE 4: 3 ประเภทของ Sequence (3 Types)
```
┌─────────────────────────────────────────────┐
│  3 ประเภทของ Sequence                       │
│  ─────────────────────────                  │
│                                             │
│  ┌──────────────┬──────────────────────┐    │
│  │ Time-based   │ ส่งตามเวลา           │    │
│  │              │ Day 0, Day 1, Day 3  │    │
│  │              │ เริ่มต้นง่าย ★★★     │    │
│  ├──────────────┼──────────────────────┤    │
│  │ Event-based  │ ส่งเมื่อเกิดเหตุการณ์ │    │
│  │              │ คลิกลิงก์ เข้าชมเว็บ  │    │
│  │              │ Personalize สูง       │    │
│  ├──────────────┼──────────────────────┤    │
│  │ Conditional  │ ส่งตามเงื่อนไข        │    │
│  │              │ เปิด/ไม่เปิดอีเมล     │    │
│  │              │ แยกเส้นทางลูกค้า      │    │
│  └──────────────┴──────────────────────┘    │
│                                             │
│  เริ่มจาก Time-based แล้วค่อยซับซ้อนขึ้น    │
│                                             │

```

## SLIDE 5: องค์ประกอบ 4 Node (4 Node Types)
```
┌─────────────────────────────────────────────┐
│  4 องค์ประกอบ (Node) ของ Sequence           │
│  ─────────────────────────                  │
│                                             │
│  ┌─────────────────────────────────┐        │
│  │ 1. Email Node                   │        │
│  │    ตัวอีเมลที่ส่ง (หัวข้อ+เนื้อหา+CTA) │  │
│  ├─────────────────────────────────┤        │
│  │ 2. Delay Node                   │        │
│  │    กำหนดเวลารอ (1 วัน, 3 วัน)  │        │
│  ├─────────────────────────────────┤        │
│  │ 3. Condition Node               │        │
│  │    แยกเส้นทาง (เปิด/ไม่เปิด)    │        │
│  ├─────────────────────────────────┤        │
│  │ 4. Action Node                  │        │
│  │    ติด Tag / ย้ายกลุ่ม / Subscribe │      │
│  └─────────────────────────────────┘        │
│                                             │
│  4 Node รวมกัน = ระบบ Automation ทรงพลัง    │
│                                             │
└─────────────────────────────────────────────┘
```

## SLIDE 6: วิธีสร้าง Sequence ใน Kartra (How to Create)

## SLIDE 10: 4 Metrics ที่ต้องติดตาม (Key Metrics)
```
┌─────────────────────────────────────────────┐
│  4 Metrics ที่ต้องติดตาม                     │
│  ─────────────────────────                  │
│                                             │
│  1. Open Rate (เกณฑ์: > 20%)               │
│     → วัด Subject Line ดึงดูดไหม            │
│     ถ้าต่ำ: ปรับหัวข้อให้น่าสนใจ             │
│                                             │
│  2. Click Rate (เกณฑ์: > 3%)               │
│     → วัดเนื้อหา + CTA น่าคลิกไหม          │
│     ถ้าต่ำ: ปรับ CTA ให้ชัดเจน              │
│                                             │
│  3. Unsubscribe Rate (เกณฑ์: < 2%)         │
│     → วัดว่าส่งถี่เกิน/ไม่ตรงกลุ่มไหม       │
│     ถ้าสูง: ลดความถี่ ปรับเนื้อหา            │
│                                             │
│  4. Conversion Rate (เกณฑ์: > 1-3%)        │
│     → เป้าหมายสุดท้าย กี่% ซื้อ/ทำ Action   │
│     ★ สำคัญที่สุด!                          │
│                                             │
└─────────────────────────────────────────────┘
```
