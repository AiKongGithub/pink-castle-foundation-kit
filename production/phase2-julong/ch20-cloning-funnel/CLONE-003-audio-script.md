### Segment 1: ทำไมต้อง Import Campaign?

**Host A:** สวัสดีครับทุกท่าน กลับมาพบกันอีกครั้งกับซีรีส์ Cloning Sale Funnel ครับ วันนี้เราจะมาพูดถึงเรื่องที่น่าตื่นเต้นมาก — การ Import Campaign เข้า Kartra

**Host B:** ใช่ครับ ตอนที่แล้วเราพูดถึงระบบแห่งความสำเร็จ วันนี้เราจะลงมือทำจริงเลย — เอา Campaign ที่คนอื่นสร้างไว้แล้ว Import เข้ามาใช้งานเลย

**Host A:** พูดง่ายๆ ก็คือ "ไม่ต้องสร้างใหม่ตั้งแต่ศูนย์" ใช่มั้ยครับ?

**Host B:** ถูกต้องเลยครับ นี่คือ Power ของ Kartra — มีคนที่ประสบความสำเร็จแล้ว สร้าง Funnel ที่ได้ผลแล้ว แล้วเขาแชร์ให้เราผ่าน Share Link เราก็แค่กดนำเข้า แล้วปรับแต่งให้เป็นของเรา

**Host A:** แล้วทำไมวิธีนี้ถึงดีกว่าสร้างใหม่ล่ะครับ?

**Host B:** เหตุผลหลักๆ 3 ข้อครับ ข้อแรก **ประหยัดเวลามหาศาล** — Campaign ที่ดีใช้เวลาสร้าง 2-4 สัปดาห์ แต่ Import ใช้เวลาแค่ 5 นาที ข้อสอง **ใช้ Template ที่พิสูจน์แล้ว** — ไม่ต้องลองผิดลองถูก เพราะเป็น Funnel ที่คนอื่นทดสอบมาแล้วว่าได้ผล ข้อสาม **เรียนรู้จากโครงสร้าง** — ได้เห็นว่าคนเก่งเขาวาง Funnel ยังไง

**Host A:** เหมือนกับการ Copy งานศิลปะแล้วปรับให้เป็นสไตล์เราเอง?

**Host B:** ใช่เลยครับ ในโลกธุรกิจเรียกว่า "Modeling" — ดูว่าคนที่สำเร็จเขาทำยังไง แล้วนำมาปรับใช้กับธุรกิจเรา

### Segment 2: ขั้นตอนการ Import ทีละขั้น

**Host A:** มาดูขั้นตอนจริงๆ กันเลยครับ เริ่มจากอะไรก่อน?

**Host B:** ขั้นตอนหลักมี 5 ขั้นครับ

**Host B:** **ขั้นที่ 1: ได้รับ Share Link** — คนที่แชร์ Campaign จะส่ง URL ให้เรา มีรูปแบบเป็น `app.kartra.com/share/xxxxx` Link นี้เปรียบเหมือนกุญแจที่เปิดห้องเก็บ Campaign

**Host A:** ได้ Link มาแล้ว ขั้นต่อไปล่ะครับ?

**Host B:** **ขั้นที่ 2: คลิก Link แล้ว Login** — เปิด Link ใน Browser ระบบจะถาม Login ถ้ายังไม่ได้ Login Kartra ก็ใส่ Email + Password

**Host B:** **ขั้นที่ 3: เลือก Components** — หน้าจอจะแสดงว่า Campaign นี้มีอะไรบ้าง — Pages กี่หน้า, Forms กี่อัน, Sequences กี่ชุด, Automations กี่ตัว เราเลือกได้ว่าจะ Import ทั้งหมดหรือเลือกเฉพาะบางส่วน

**Host A:** แนะนำให้เลือกทั้งหมดมั้ยครับ?

**Host B:** สำหรับมือใหม่ แนะนำ Import ทั้งหมดเลยครับ เพราะ Components มันเชื่อมกัน ถ้าขาดชิ้นใดชิ้นหนึ่งอาจทำงานไม่สมบูรณ์

**Host B:** **ขั้นที่ 4: กด Import** — กดปุ่ม Import แล้วรอ ใช้เวลาไม่กี่นาที ขึ้นอยู่กับขนาด Campaign

**Host B:** **ขั้นที่ 5: ตรวจสอบ** — หลัง Import เสร็จ ต้องเข้าไปตรวจว่าทุก Component เข้ามาครบ แล้วทำงานได้ถูกต้อง

**Host A:** 5 ขั้นตอน ฟังดูไม่ยากเลยครับ!

---

### Segment 3: Components ที่ได้รับจากการ Import

**Host A:** มาดูรายละเอียดกันครับว่า Import แล้วจะได้อะไรบ้าง?

**Host B:** Campaign ใน Kartra ประกอบด้วย 5 ส่วนหลักครับ

**Host B:** **1. Pages (หน้าเว็บ)** — ได้ทั้ง Capture Page, Sales Page, Thank You Page, Checkout Page ทุกหน้าจะมา Design สำเร็จรูป มี Layout, สี, ตำแหน่งองค์ประกอบพร้อม

**Host B:** **2. Forms (แบบฟอร์ม)** — ฟอร์มรับข้อมูล เช่น ชื่อ, อีเมล, เบอร์โทร ฟอร์มจะเชื่อมกับ Page อัตโนมัติ

**Host B:** **3. Email Sequences (ลำดับอีเมล)** — ชุดอีเมลอัตโนมัติที่ส่งตามลำดับ เช่น Welcome Email → Value Email → Offer Email ได้ทั้งเนื้อหาและ Timing

**Host A:** รวมถึงเนื้อหาอีเมลด้วยเลยหรือครับ?

**Host B:** ใช่ครับ ได้ทั้ง Subject Line, Body, CTA — แต่เราต้องแก้ให้เป็นเนื้อหาของเรานะครับ

**Host B:** **4. Automations (กฎอัตโนมัติ)** — เช่น "ถ้าคนกรอกฟอร์ม → ส่ง Email Sequence → ติด Tag" ระบบอัตโนมัติทั้งหมดจะมาด้วย

**Host B:** **5. Tags (แท็ก)** — แท็กที่ใช้จัดกลุ่มคน เช่น "lead", "buyer", "interested" ต้องตรวจว่าไม่ซ้ำกับ Tags ที่เรามีอยู่แล้ว

**Host A:** เหมือนได้ระบบมาทั้งชุดเลยนะครับ!

---

### Segment 4: ข้อควรระวังและสิ่งที่ต้องปรับหลัง Import

**Host A:** มีอะไรที่ต้องระวังมั้ยครับ?

**Host B:** มีหลายเรื่องสำคัญมากครับ

**Host B:** **ข้อ 1: Tags ซ้ำ** — ถ้า Campaign ใช้ Tag ชื่อ "buyer" แล้วเรามี Tag ชื่อ "buyer" อยู่แล้ว อาจเกิดความสับสน ต้องตรวจก่อน Import ว่ามี Tags ไหนซ้ำ

**Host B:** **ข้อ 2: Domain/URL** — หน้าเว็บที่ Import มาจะใช้ URL ของเรา ต้องตรวจว่า Domain ตั้งค่าถูกต้อง

**Host B:** **ข้อ 3: เนื้อหาเป็นของคนอื่น** — ทุกอย่างที่ Import มา ไม่ว่าจะเป็น Text, รูป, วิดีโอ — ยังเป็นเนื้อหาของคนที่แชร์ ต้องเปลี่ยนให้เป็นของเราก่อนใช้งานจริง

**Host A:** อันนี้สำคัญมากเลยครับ ห้ามใช้เนื้อหาคนอื่นโดยตรง

**Host B:** ถูกต้องครับ **ข้อ 4: ทดสอบหลัง Import** — ต้อง Test ทุกอย่าง กรอกฟอร์มทดสอบ, ตรวจ Email ว่าส่งจริง, ตรวจ Automation ว่าทำงาน

**Host B:** **ข้อ 5: Payment Gateway** — ถ้า Campaign มีหน้า Checkout ต้องเชื่อมกับ Payment Gateway ของเรา (Stripe, PayPal) ไม่ได้มาอัตโนมัติ

**Host A:** สรุปคือ Import ง่าย แต่ห้ามลืมปรับแต่ง!

---

### Segment 5: Checklist หลัง Import + สรุป

**Host A:** มาสรุปเป็น Checklist กันครับ

**Host B:** Checklist หลัง Import มี 8 ข้อครับ

**Host B:** ✅ **1.** ตรวจ Pages ว่ามาครบทุกหน้า
✅ **2.** ตรวจ Forms ว่าเชื่อมกับ Pages ถูกต้อง
✅ **3.** ตรวจ Email Sequences ว่ามีครบทุกฉบับ
✅ **4.** ตรวจ Automations ว่าทำงานถูกต้อง
✅ **5.** ตรวจ Tags ว่าไม่ซ้ำกับที่มีอยู่
✅ **6.** เปลี่ยนเนื้อหาทั้งหมดเป็นของเรา
✅ **7.** เชื่อม Payment Gateway (ถ้ามี Checkout)
✅ **8.** ทดสอบ Funnel ทั้งหมดตั้งแต่ต้นจนจบ

**Host A:** 8 ข้อ ทำครบแล้ว Campaign พร้อมใช้งาน!

**Host B:** ใช่ครับ และตอนหน้าเราจะมาลงรายละเอียดเรื่อง "วิธีปรับแต่งแคมเปญ Kartra" ซึ่งจะสอนวิธีแก้ไขทุก Component ให้เป็นของเราอย่างละเอียด

**Host A:** น่าสนใจมากครับ ขอบคุณทุกท่านที่ติดตาม แล้วพบกันตอนหน้าครับ!
