| 3 | Mobile Testing คือขั้นตอนที่ขาดไม่ได้ | หลายนักเรียนออกแบบ Funnel บน Desktop สวยมาก แต่ไม่เคย Preview บน Mobile เลย ผลคือข้อความล้นจอ ปุ่มเล็กเกินกด รูปภาพไม่ Responsive ในขณะที่ผู้ใช้ส่วนใหญ่เปิดเว็บจากมือถือ | กด Mobile Preview ทุกครั้งที่แก้ไขอะไรก็ตาม Kartra มีโหมด Mobile Preview ในตัว ตรวจข้อความ ปุ่ม รูปภาพ และ Flow ทั้งหมดบน Mobile ก่อน Publish | สูง |
| 4 | Pop up Timing ที่ผิดทำลายโอกาสมากกว่าสร้าง | Pop up ที่ขึ้นทันทีเมื่อลูกค้าเข้าหน้าทำให้รำคาญและกดปิดทันที ลูกค้ายังไม่ทันอ่านเนื้อหาก็มีหน้าต่างมาบัง สูญเสียโอกาสที่จะดึงความสนใจ | ตั้ง Delay อย่างน้อย 5-10 วินาที ให้ลูกค้าอ่านเนื้อหาก่อน หรือใช้ Exit Intent Trigger ที่ขึ้นเมื่อลูกค้ากำลังจะปิดหน้า ซึ่งเป็นจังหวะที่เหมาะที่สุด | สูง |

| 6 | Checklist 7 หมวดคือเครื่องมือป้องกันข้อผิดพลาด | อาจารย์สรุป Best Practices เป็น Checklist 7 หมวด ได้แก่ Branding, Copy, Media, Email, Forms, Automations, Testing ครอบคลุมทุกจุดที่ต้องตรวจก่อน Publish ถ้าผ่านหมด Funnel พร้อมเปิดใช้งานจริง | Print Checklist ติดข้างจอ ใช้ทุกครั้งที่สร้างหรือแก้ไข Funnel ตรวจทีละหมวดอย่างเป็นระบบ ใช้เวลา 1-2 ชั่วโมงต่อ Funnel ก่อน Publish | สูง |
