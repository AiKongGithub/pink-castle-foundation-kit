### Q1: Campaign Import ใน Kartra คืออะไร?

A) การสร้าง Campaign ใหม่ตั้งแต่ศูนย์
B) การนำเข้า Campaign สำเร็จรูปผ่าน Share Link
C) การ Copy เนื้อหาจากเว็บไซต์อื่น
D) การซื้อ Template จาก Kartra Marketplace

✅ **คำตอบ: B) การนำเข้า Campaign สำเร็จรูปผ่าน Share Link**

**อธิบาย:** Campaign Import คือการใช้ Share Link ที่คนอื่นแชร์มาเพื่อนำเข้า Campaign ทั้งระบบ (Pages, Forms, Sequences, Automations, Tags) เข้ามาใน Account ของเรา

---

### Q2: การ Import Campaign ประหยัดเวลาเท่าไหร่เมื่อเทียบกับสร้างใหม่?

A) 10%
B) 50%
C) 75%
D) 90%+

✅ **คำตอบ: D) 90%+**

**อธิบาย:** สร้าง Campaign จากศูนย์ใช้ 2-4 สัปดาห์ แต่ Import ใช้เพียง 5 นาที ประหยัดเวลา 90% ขึ้นไป (ยังต้องใช้เวลา 1-2 วันปรับแต่งเนื้อหา)

---

### Q3: ขั้นตอนแรกของการ Import Campaign คืออะไร?

A) เลือก Components ที่ต้องการ
B) กด Import
C) ได้รับ Share Link
D) ทดสอบ Funnel

✅ **คำตอบ: C) ได้รับ Share Link**

**อธิบาย:** ขั้นตอนแรกคือได้รับ Share Link (URL รูปแบบ app.kartra.com/share/xxxxx) จากผู้ที่แชร์ Campaign จากนั้นจึงคลิก Link → Login → เลือก Components → Import

---

### Q4: Campaign Import ได้ Components กี่ชิ้น?

A) 3 (Pages, Forms, Emails)
B) 5 (Pages, Forms, Sequences, Automations, Tags)
C) 7 (Pages, Forms, Sequences, Automations, Tags, Products, Memberships)
D) 2 (Pages, Forms)

✅ **คำตอบ: B) 5 (Pages, Forms, Sequences, Automations, Tags)**

**อธิบาย:** Campaign Import ให้ 5 Components หลัก: Pages (หน้าเว็บ), Forms (แบบฟอร์ม), Email Sequences (อีเมลอัตโนมัติ), Automations (กฎอัตโนมัติ), Tags (แท็กจัดกลุ่ม)

---

### Q5: ทำไมมือใหม่ควร Import ทั้ง Campaign แทนที่จะเลือกเฉพาะบางส่วน?

A) เพราะ Import บางส่วนไม่ได้
B) เพราะ Components เชื่อมกัน ขาดชิ้นใดอาจทำงานไม่สมบูรณ์
C) เพราะ Import ทั้งหมดเร็วกว่า
D) เพราะ Kartra บังคับให้ Import ทั้งหมด

✅ **คำตอบ: B) เพราะ Components เชื่อมกัน ขาดชิ้นใดอาจทำงานไม่สมบูรณ์**

**อธิบาย:** Page เชื่อมกับ Form, Form เชื่อมกับ Automation, Automation เชื่อมกับ Sequence ถ้าขาดส่วนใดส่วนหนึ่ง ระบบอาจทำงานไม่ครบ

---

### Q6: ข้อใดเป็นข้อควรระวังสำคัญที่สุดหลัง Import?

A) เปลี่ยนสีพื้นหลัง
B) เปลี่ยนเนื้อหาทั้งหมดเป็นของตัวเอง
C) เพิ่ม Social Media Links
D) เปลี่ยน Font

✅ **คำตอบ: B) เปลี่ยนเนื้อหาทั้งหมดเป็นของตัวเอง**

**อธิบาย:** เนื้อหาที่ Import มายังเป็นของคนที่แชร์ ห้ามใช้โดยตรงทั้งด้านกฎหมายลิขสิทธิ์และด้าน Branding ต้องเปลี่ยน Text, รูป, วิดีโอ ทั้งหมดให้เป็นของตัวเอง

---

### Q7: Tags ซ้ำหลัง Import จะเกิดปัญหาอะไร?

A) Campaign จะไม่ถูก Import
B) คนอาจถูกจัดกลุ่มผิดและได้รับ Email ผิดชุด
C) ไม่มีปัญหาใดๆ
D) Tags จะถูกลบอัตโนมัติ

✅ **คำตอบ: B) คนอาจถูกจัดกลุ่มผิดและได้รับ Email ผิดชุด**

**อธิบาย:** ถ้า Campaign ใหม่มี Tag "buyer" และเรามี Tag "buyer" อยู่แล้ว ระบบจะใช้ Tag เดียวกัน ทำให้คนจากทั้ง 2 Campaign ปนกัน ได้รับ Email ผิดชุด

---

### Q8: Payment Gateway หลัง Import ต้องทำอย่างไร?

A) มาอัตโนมัติ ไม่ต้องทำอะไร
B) ต้องเชื่อม Stripe/PayPal เอง
C) ต้องซื้อ Payment Plugin เพิ่ม
D) ใช้ได้เฉพาะ PayPal

✅ **คำตอบ: B) ต้องเชื่อม Stripe/PayPal เอง**

**อธิบาย:** Payment Gateway ไม่ได้มากับ Campaign Import ต้องเข้า Settings → Integrations เชื่อม Stripe หรือ PayPal เอง แล้วตั้งค่าราคาที่ Checkout Page

---

### Q9: Post-Import Checklist มีกี่ข้อ?

A) 5 ข้อ
B) 8 ข้อ
C) 10 ข้อ
D) 12 ข้อ

✅ **คำตอบ: B) 8 ข้อ**

**อธิบาย:** Checklist 8 ข้อ: (1) ตรวจ Pages (2) ตรวจ Forms (3) ตรวจ Sequences (4) ตรวจ Automations (5) ตรวจ Tags (6) เปลี่ยนเนื้อหา (7) เชื่อม Payment (8) ทดสอบ Funnel

---

### Q10: วิธีทดสอบ Funnel หลัง Import ที่ดีที่สุดคืออะไร?

A) ดูจากหน้า Admin เท่านั้น
B) เดินผ่าน Funnel เหมือนเป็นลูกค้าจริง
C) ให้ AI ตรวจสอบให้
D) ไม่ต้องทดสอบ ถ้า Import สำเร็จก็ใช้ได้

✅ **คำตอบ: B) เดินผ่าน Funnel เหมือนเป็นลูกค้าจริง**

**อธิบาย:** ต้อง Test ทุก Path — กรอกฟอร์ม, รับ Email, คลิก Link, เข้า Thank You Page เหมือนเป็นลูกค้าจริง เพราะการ Import ไม่รับประกัน 100%

---

### Q11: เปรียบเทียบ Import vs สร้างใหม่ ข้อใดถูกต้อง?

A) สร้างใหม่เร็วกว่า Import
B) Import ปรับแต่งไม่ได้
C) Import เหมาะกับมือใหม่ สร้างใหม่เหมาะกับผู้เชี่ยวชาญ
D) ไม่มีข้อแตกต่าง

✅ **คำตอบ: C) Import เหมาะกับมือใหม่ สร้างใหม่เหมาะกับผู้เชี่ยวชาญ**

**อธิบาย:** Import ช่วยมือใหม่เริ่มต้นเร็ว ได้ Template ที่ดี ส่วนผู้เชี่ยวชาญอาจต้องการออกแบบเองเพื่อความยืดหยุ่นสูงสุด แต่แม้แต่ผู้เชี่ยวชาญก็ใช้ Import เป็น Reference ได้

---

### Q12: สรุปข้อใดอธิบาย Campaign Import ได้ดีที่สุด?

A) Import แล้วใช้ได้เลย ไม่ต้องทำอะไรเพิ่ม
B) Import ง่ายแต่ห้ามลืมปรับแต่งและทดสอบก่อนเผยแพร่
C) Import ได้แค่ Pages เท่านั้น
D) Import ต้องจ่ายเงินเพิ่ม

✅ **คำตอบ: B) Import ง่ายแต่ห้ามลืมปรับแต่งและทดสอบก่อนเผยแพร่**

**อธิบาย:** ขั้นตอน Import ง่ายมาก (5 ขั้น, 5 นาที) แต่สิ่งสำคัญที่สุดคือการปรับแต่งเนื้อหาให้เป็นของตัวเองและทดสอบ Funnel ทั้งหมดก่อนเปิดใช้งานจริง
