### Segment 1: ภาพรวมการปรับแต่ง

**Host A:** สวัสดีครับ กลับมาต่อกันกับ Cloning Sale Funnel ครับ ตอนที่แล้วเราสอน Import Campaign สำเร็จ วันนี้มาถึงขั้นตอนสำคัญ — การปรับแต่งให้เป็นของเรา

**Host B:** ใช่ครับ Import ได้ "โครงสร้าง" มา แต่ "เนื้อหา" ยังเป็นของคนอื่น ถ้าใช้ตรงๆ ลูกค้าจะรู้ว่า Copy มา ทำลายความน่าเชื่อถือทันที

**Host A:** แล้วต้องปรับอะไรบ้างครับ?

**Host B:** มี 6 หมวดหลักครับ — **Branding**, **Copy**, **Images/Videos**, **Email Sequences**, **Forms**, และ **Automations** แล้วปิดท้ายด้วย **Testing** ก่อนเปิดใช้จริง

**Host A:** ฟังดูเยอะเลย!

**Host B:** แต่ถ้าทำเป็นระบบ ใช้เวลาแค่ 1-2 วันครับ เทียบกับสร้างใหม่ 2-4 สัปดาห์ ยังประหยัดมากครับ

---

### Segment 2: Branding + Copy

**Host A:** เริ่มจาก Branding ก่อนเลยครับ

**Host B:** Branding มี 3 อย่างหลัก — **Logo**, **Color Scheme**, **Font** ใส่ Logo ทุกหน้า ใช้สีเดียวกันตลอด Funnel ฟอนต์ก็ต้องสม่ำเสมอ Kartra มี Global Styles เปลี่ยนทีเดียวได้ทุกหน้าเลยครับ

**Host A:** Tip ดีมาก! แล้ว Copy ล่ะครับ?

**Host B:** Copy คือส่วนสำคัญที่สุดครับ แบ่งเป็น 3 ส่วน — **Headlines** ต้องตรงกับ Pain Points ของกลุ่มเป้าหมายเรา **Body Text** ใช้น้ำเสียงของเรา เน้นประโยชน์ที่ลูกค้าจะได้ แล้วก็ **CTA** ต้องใช้ภาษาเชิงประโยชน์

**Host A:** CTA เชิงประโยชน์คืออะไรครับ?

**Host B:** แทนที่จะเขียนว่า "Click Here" หรือ "Submit" ให้เขียนว่า "รับสิทธิ์เลย" หรือ "เริ่มฟรีวันนี้" บอกสิ่งที่ลูกค้าจะได้รับ CTR จะสูงกว่ามากครับ

---

### Segment 3: Images/Videos + Email Sequences

**Host A:** มาต่อเรื่อง Media กันครับ

**Host B:** หลักการง่ายมากครับ — **"ของจริงคุณภาพต่ำ ดีกว่า Stock Photo คุณภาพสูง"** ลูกค้าไว้ใจรูปจริงมากกว่า ถ่ายรูปสินค้าด้วยมือถือก็ได้ ขอให้เป็นของจริง รวบรวม Testimonial จากลูกค้าจริง ถ่ายวิดีโอแนะนำตัวสั้นๆ

**Host A:** แล้ว Email Sequences ล่ะครับ?

**Host B:** มี 4 สิ่งที่ต้องปรับครับ — **ชื่อผู้ส่ง** เปลี่ยนเป็นชื่อเราหรือชื่อธุรกิจ **Subject Line** ต้องน่าเปิดอ่าน **เนื้อหา** ต้องตรงกับ Value Ladder ของเรา แล้วก็ **Timing** — Welcome ทันที, Day 1 ให้คุณค่า, Day 3 เสนอขาย

**Host A:** ถ้าไม่ปรับ Email จะเกิดอะไรขึ้นครับ?

**Host B:** ลูกค้าจะได้รับอีเมลที่ไม่เกี่ยวกับเรา Unsubscribe Rate จะพุ่ง และเสียลูกค้าทันทีครับ

---

### Segment 4: Forms + Automations

**Host A:** มาดู Forms กับ Automations กันครับ

**Host B:** Forms มี 3 จุดที่ต้องตรวจ — **ฟิลด์** อะไรจำเป็น อะไรตัดออกได้ สำหรับ Lead Magnet ขอแค่ชื่อกับอีเมลก็พอ ยิ่งขอน้อย Conversion ยิ่งสูง แล้วก็ **ข้อความ Confirmation** และ **Redirect URL** ต้องชี้ไปหน้าที่ถูกต้อง

**Host A:** Automations ล่ะครับ ฟังดูซับซ้อน?

**Host B:** Automations มี 3 องค์ประกอบครับ — **Trigger** คือเงื่อนไขเริ่มทำงาน (เช่น คนกรอกฟอร์ม) **Action** คือสิ่งที่เกิดขึ้น (เช่น ส่งอีเมล) **Tags** คือป้ายกำกับ (เช่น ติด tag "lead") ต้องตรวจทุก Rule ว่า Trigger ชี้ถูกที่ Action ส่งถูกฉบับ Tags ตรงกับระบบของเรา

**Host A:** Tip สำหรับจัดการ Tags?

**Host B:** วาดแผนผัง Tag System ของเราก่อนเริ่มปรับครับ จะได้เห็นภาพรวมว่า Tags ไหนใช้ทำอะไร ไม่งงเวลาปรับ

---

### Segment 5: Testing + สรุป

**Host A:** มาถึงขั้นสุดท้ายครับ — Testing!

**Host B:** Testing เป็น **ขั้นตอนบังคับ ห้ามข้าม** ครับ ต้องเดินผ่านทั้ง Funnel เหมือนเป็นลูกค้าจริง — กรอกฟอร์ม ตรวจว่าได้รับอีเมล คลิกลิงก์ทุกจุด ตรวจ Automations ทำงานถูก Tags ติดถูก แล้วเช็คทั้ง Desktop และ Mobile

**Host A:** ทำไมต้อง Mobile ด้วยครับ?

**Host B:** เพราะ Layout บน Mobile อาจต่างจาก Desktop มาก ปุ่มอาจเล็กเกินไป ฟอร์มอาจแสดงผลผิด 70% ของคนดูเว็บใช้ Mobile

**Host A:** สรุปวันนี้สั้นๆ ได้มั้ยครับ?

**Host B:** สรุปง่ายมากครับ — "โครงสร้างประหยัดเวลา แต่เนื้อหาต้องเป็นของเรา" ปรับ 6 หมวด + Testing = Campaign พร้อมใช้ ตอนหน้าจะลงรายละเอียดเรื่องปรับแต่ง Capture Page โดยเฉพาะครับ

**Host A:** ขอบคุณครับ แล้วพบกันตอนหน้า!
