## SLIDE 4: สูตร Revenue Formula

# Revenue = Traffic x Conversion Rate x AOV

```
+--------------------------------------------------+
|                                                  |
|   Traffic      x   Conversion   x   AOV          |
|   (คนเข้ามา)       Rate (% ซื้อ)    (ยอดเฉลี่ย)   |
|                                                  |
|   1,000 คน    x   5%          x   1,000 บาท     |
|                                                  |
|   = 50,000 บาท/เดือน                             |
|                                                  |
|   ปรับปรุงทั้ง 3 ตัว → รายได้เพิ่มแบบทวีคูณ        |
|                                                  |
+--------------------------------------------------+
```

> **ตัวอย่าง:** เพิ่ม Traffic 20%, CR 20%, AOV 20% → รายได้เพิ่ม **73%** (1.2 x 1.2 x 1.2 = 1.73)

## SLIDE 10: วัดผล KPIs

# 5 KPIs ที่ต้องติดตาม

| KPI | ความหมาย | เป้าหมาย |
|-----|---------|---------|
| Revenue | รายได้รวม | เพิ่มทุกเดือน |
| Profit Margin | อัตรากำไร | > 50% |
| CAC | ค่าหาลูกค้าใหม่ | ยิ่งต่ำยิ่งดี |
| LTV | มูลค่าตลอดชีพลูกค้า | > 3x CAC |
| Churn Rate | อัตราลูกค้าหาย | < 5% |

```
+------------------------------------------+
|  กฎทอง: LTV ต้องมากกว่า 3 เท่าของ CAC   |
|  ถ้า LTV < CAC = ขาดทุนไปเรื่อยๆ        |
+------------------------------------------+
```

## SLIDE 11: Urgency/Scarcity — กระตุ้นการตัดสินใจ

# Urgency + Scarcity เพิ่ม Conversion

```
Urgency (ความเร่งด่วนด้านเวลา):
+--------------------------------------------------+
| "โปรหมดภายใน 24 ชั่วโมง"                          |
| "ราคาพิเศษเฉพาะวันนี้เท่านั้น"                     |
+--------------------------------------------------+

Scarcity (ความขาดแคลนด้านจำนวน):
+--------------------------------------------------+
| "เหลืออีก 3 ที่เท่านั้น"                           |
| "รับจำกัด 50 คนแรก"                               |
+--------------------------------------------------+
```

- กระตุ้นให้ลูกค้า **ตัดสินใจเร็วขึ้น**
- ต้องใช้ **อย่างจริงใจ** ไม่ควรหลอกลวง
- ผลลัพธ์: Conversion Rate **เพิ่มขึ้น 20-40%**
