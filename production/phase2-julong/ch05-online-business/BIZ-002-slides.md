## SLIDE 5: Irresistible Offer

# สร้าง Offer ที่ปฏิเสธไม่ลง

```
+--------------------------------------------------+
|                                                  |
|  1. สิ่งที่ลูกค้าต้องการ  --> ตรง Pain Points     |
|                                                  |
|  2. ราคาที่คุ้มค่า        --> ได้มากกว่าจ่าย      |
|                                                  |
|  3. Bonus               --> ของแถมเพิ่มคุณค่า    |
|                                                  |
|  4. Guarantee            --> ลดความเสี่ยง         |
|                                                  |
+--------------------------------------------------+
```

> **ตัวอย่าง:** "คอร์สมูลค่า 15,000 บาท ราคาเพียง 2,990 + eBook ฟรี + กลุ่ม VIP + คืนเงิน 30 วัน"

## SLIDE 6: MVP — Minimum Viable Product

# MVP — สร้างเร็ว Launch เร็ว เรียนรู้เร็ว

```
+-------------+     +-------------+     +-------------+
|   สร้าง     |     |   Launch    |     |   ปรับปรุง   |
|   MVP       | --> |   ขายจริง   | --> |   จาก       |
|   2-4 สัปดาห์ |     |   ทดสอบตลาด  |     |   Feedback   |
+-------------+     +-------------+     +-------------+
       ^                                       |
       |                                       |
       +---------------------------------------+
                   วนซ้ำ ปรับปรุง
```

- ไม่ต้องสมบูรณ์แบบ 100%
- Launch ก่อน ปรับทีหลัง
- Feedback จากลูกค้าจริง ดีกว่าการคาดเดา

> **"ถ้าไม่อายเวอร์ชันแรก แปลว่า launch ช้าเกินไป"**

## SLIDE 7: กลยุทธ์ตั้งราคา

# 4 กลยุทธ์ตั้งราคา

| กลยุทธ์ | วิธีคิด | ตัวอย่าง |
|---------|--------|---------|
| Cost-based | ต้นทุน + กำไร | 100 + 150 = 250 |
| Value-based | คุณค่าที่ลูกค้าได้ | ช่วยหาเงินเพิ่ม 10K → ขาย 3K |
| Price Anchoring | เทียบกับของแพงกว่า | "ที่ปรึกษา 5K/ชม. vs คอร์ส 2,990" |
| Psychological | ลงท้ายด้วย 7 หรือ 9 | 997, 2,490, 4,997 |

> **ผู้สอนแนะนำ:** ใช้ **Value-based Pricing** เพราะกำไรสูงกว่า + สะท้อนคุณค่าจริง

## SLIDE 9: Guarantee — ลดความเสี่ยง

# Guarantee เพิ่มยอดขายได้อย่างไร

```
ก่อนมี Guarantee:
+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+
| ลูกค้า |  สนใจ  | ----> | กลัวเสียเงิน | ----> | ไม่ซื้อ  |
+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+

หลังมี Guarantee:
+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+
| ลูกค้า |  สนใจ  | ----> | มี Guarantee | ----> | กล้าซื้อ  |
+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+
```

- ลูกค้าที่ขอคืนเงินจริง = **น้อยมาก**
- ยอดขายที่เพิ่มขึ้น = **คุ้มกว่ามาก**
- แสดงว่าเราเชื่อมั่นในสินค้าตัวเอง

## SLIDE 10: เริ่มขาย 10 คนแรก

# 10 คนแรก — จุดเปลี่ยนของธุรกิจ

```
ขายคนรู้จัก -----> ขอ Feedback -----> ขอ Testimonial
      |                  |                    |
      v                  v                    v
 พิสูจน์ว่า          ปรับปรุง             Social Proof
 ขายได้จริง          สินค้า              ดึงลูกค้าใหม่
```

**ทำไม 10 คนแรกสำคัญที่สุด:**
- พิสูจน์ว่าสินค้าขายได้จริง
- ได้ feedback เพื่อปรับปรุง
- ได้ Testimonial สำหรับขยายต่อ
- เปลี่ยนจาก "ไม่มีใครซื้อ" เป็น "มีคนซื้อแล้ว"

## SLIDE 11: Social Proof — พลังแห่งคำชม

# Testimonial แรก — มีค่ามหาศาล

```
+--------------------------------------------------+
|                                                  |
|  ไม่มี Testimonial:                               |
|  "จะซื้อดีไหม? ไม่มีใครเคยลอง..."               |
|                                                  |
|  มี Testimonial:                                  |
|  "มีคนซื้อแล้วและชอบ! น่าลองนะ"                  |
|                                                  |
+--------------------------------------------------+
```

**วิธีใช้ Social Proof:**
1. Screenshot รีวิวจากลูกค้า
2. โพสต์เป็น Content บน Social Media
3. ใส่ใน Sales Page / หน้าขาย
4. ให้ลูกค้าเก่าแนะนำคนใหม่
