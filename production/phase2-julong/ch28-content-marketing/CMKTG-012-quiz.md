## คำถาม

### Q1. Blog Post List ทำหน้าที่อะไรในบล็อก?
- A) เป็นหน้าแสดงบทความเดี่ยว
- B) เป็นหน้ารวมบทความทั้งหมด ทำหน้าที่เป็น "ประตูหน้า" ของบล็อก
- C) เป็นหน้า Dashboard สำหรับผู้ดูแลระบบ
- D) เป็นหน้าตั้งค่า SEO ของบล็อก

### Q2. Grid Layout เหมาะกับบล็อกประเภทใดมากที่สุด?
- A) บล็อกข่าวสารการเงิน
- B) บล็อกบทวิเคราะห์เชิงลึก
- C) บล็อกท่องเที่ยว อาหาร ออกแบบ ที่เน้นภาพ
- D) บล็อกกฎหมาย

### Q3. จำนวนบทความที่แนะนำต่อหน้าสำหรับ Pagination คือเท่าไร?
- A) 1-3 บทความ
- B) 6-12 บทความ
- C) 20-30 บทความ
- D) ไม่จำกัด แสดงทั้งหมดในหน้าเดียว

### Q4. Pagination แบบ Load More Button มีข้อดีอย่างไร?
- A) SEO ดีกว่า Numbered Pagination
- B) ใช้ Bandwidth น้อยกว่า
- C) ให้ประสบการณ์ผู้ใช้ที่ดีกว่า โดยเฉพาะบนมือถือ เพราะไม่ต้องรีโหลดหน้า
- D) โหลดเร็วกว่าทุกกรณี

### Q5. จำนวน Category ที่เหมาะสมสำหรับ Blog Post List คือเท่าไร?
- A) 1-2 หมวด
- B) 5-8 หมวด
- C) 15-20 หมวด
- D) ไม่จำกัด ยิ่งเยอะยิ่งดี

### Q6. Featured Post ควรออกแบบอย่างไร?
- A) เหมือนบทความอื่นทุกประการ ไม่ต้องทำให้โดดเด่น
- B) ใช้รูปภาพขนาดใหญ่กว่า ชื่อเรื่องตัวใหญ่กว่า มีป้าย "แนะนำ" กำกับ
- C) ซ่อนไว้ ให้ผู้อ่านค้นหาเอง
- D) แสดงเฉพาะชื่อเรื่อง ไม่มีรูปภาพ

### Q7. Widget ใดที่สำคัญที่สุดสำหรับธุรกิจบน Sidebar ของหน้า Blog Post List?
- A) ปฏิทิน
- B) Tag Cloud
- C) Email Opt-in Form
- D) นาฬิกา

### Q8. SEO Settings ข้อใดที่ต้องตั้งค่าสำหรับหน้า Blog Post List?
- A) Title Tag, Meta Description, URL, Open Graph Image, Schema Markup
- B) เฉพาะ Title Tag เท่านั้น
- C) ไม่ต้องทำ SEO เพราะเป็นหน้ารวม ไม่ใช่บทความ
- D) เฉพาะ URL เท่านั้น

### Q9. บน Mobile ถ้าใช้ Grid Layout ควรแสดงกี่คอลัมน์?
- A) 3 คอลัมน์เหมือน Desktop
- B) 2 คอลัมน์
- C) 1 คอลัมน์
- D) 4 คอลัมน์

### Q10. จุดเด่นของการเชื่อม Blog กับ Funnel ใน Kartra คืออะไร?
- A) ทำให้หน้าเว็บโหลดเร็วขึ้น
- B) Blog และ Funnel อยู่ในแพลตฟอร์มเดียว สร้าง CTA ลิงก์ไปยัง Opt-in Page ได้ทันที ข้อมูลเชื่อมต่อกันหมด
- C) ทำให้บทความยาวขึ้น
- D) ช่วยแก้ไขบทความอัตโนมัติ

### Q11. BAM (Behavioral Adaptive Marketing) ใน Kartra ทำงานอย่างไร?
- A) ส่งอีเมลอัตโนมัติทุกวัน
- B) จดจำพฤติกรรมผู้อ่านแล้วแสดง CTA ที่ตรงกับความสนใจ
- C) สร้างบทความอัตโนมัติจาก AI
- D) ลบบทความที่ไม่มีคนอ่าน

### Q12. ข้อใดคือเส้นทางที่ถูกต้องในการเปลี่ยนผู้อ่านเป็นลูกค้าผ่าน Blog + Funnel?
- A) Blog → ซื้อทันที → จบ
- B) Blog → อ่านบทความ → เห็น CTA → Opt-in Page → กรอกอีเมล → Email Sequence → ลูกค้า
- C) Blog → แชร์โซเชียล → รอลูกค้ามาเอง
- D) Blog → ลง Google Ads → ลูกค้า

## Answer Key

| ข้อ | คำตอบ | คำอธิบาย |
|-----|-------|---------|
| Q1 | ✅ **B** | Blog Post List คือหน้ารวมบทความทั้งหมด เป็นหน้าแรกที่ผู้อ่านเห็นเมื่อเข้า Blog |
| Q2 | ✅ **C** | Grid Layout แสดงเป็นการ์ดเรียงตาราง เน้นภาพ เหมาะกับบล็อกท่องเที่ยว อาหาร ออกแบบ |
| Q3 | ✅ **B** | 6-12 บทความต่อหน้า เป็นจำนวนที่สมดุลระหว่างปริมาณเนื้อหาและความเร็วในการโหลด |
| Q4 | ✅ **C** | Load More Button โหลดบทความเพิ่มในหน้าเดิม ไม่ต้องรีโหลด ดีกว่าบนมือถือ |
| Q5 | ✅ **B** | 5-8 หมวดหมู่ เพียงพอต่อการจัดหมวด ไม่มากจนสับสน ชื่อสั้นกระชับ |
| Q6 | ✅ **B** | Featured Post ต้องโดดเด่นจากรายการ ใช้รูปใหญ่ ชื่อใหญ่ มีป้ายกำกับ เพิ่ม CTR |
| Q7 | ✅ **C** | Email Opt-in Form สำคัญที่สุดสำหรับธุรกิจ เพราะเก็บ Lead ที่ติดต่อกลับได้ |
| Q8 | ✅ **A** | ต้องตั้งค่าทุกจุด: Title Tag, Meta Description, URL, Open Graph Image, Schema Markup |
| Q9 | ✅ **C** | บน Mobile ควรแสดง 1 คอลัมน์เพื่อให้อ่านง่าย รูปภาพไม่เล็กเกินไป |
| Q10 | ✅ **B** | Kartra เป็น All-in-One ทำให้ Blog กับ Funnel เชื่อมกันได้อย่างไร้รอยต่อ |
| Q11 | ✅ **B** | BAM จดจำว่าผู้อ่านสนใจเรื่องอะไร แล้วแสดง CTA ที่ตรงใจ เพิ่ม Conversion |
| Q12 | ✅ **B** | เส้นทางที่ถูกต้อง: อ่าน → CTA → Opt-in → Lead Magnet → Email Nurture → ลูกค้า |
