## SLIDE 3: ภาพรวมขั้นตอน
```
╔══════════════════════════════════════════╗
║   Content Builder: ขั้นตอนสร้างบทความ   ║
╚══════════════════════════════════════════╝
```

```
สร้างบทความด้วย Content Builder
 [1] เข้าถึง Builder      ← My Content → New Post
 [2] เลือก Template       ← เริ่มจากแบบเรียบง่าย
 [3] Drag & Drop          ← วาง Components 5 ตัว
 [4] Typography           ← จัดรูปแบบตัวอักษร

```

- ทุกขั้นตอนเชื่อมกัน: เนื้อหาดี + Typography ดี + SEO ดี = บทความคุณภาพ
- เป้าหมาย: สร้างบทความที่ดีทั้งสำหรับผู้อ่านและ Google

## SLIDE 4: เข้าถึง Content Builder + เลือก Template
```
╔══════════════════════════════════════════╗
║   ขั้นตอน 1-2: เข้า Builder + Template  ║
╚══════════════════════════════════════════╝
```

**วิธีเข้าถึง:**
```
Kartra Dashboard → My Content (Blog) → New Post
                                          ↓
                               ┌─────────────────────┐
                               │  [1] Settings        │
                               │  (ชื่อ, หมวด, แท็ก)  │
                               │  [2] Content Builder │
                               │  (สร้างเนื้อหา)      │
                               └─────────────────────┘
```

**เลือก Template:**

| แบบ | ลักษณะ | เหมาะกับ |
|-----|--------|---------|
| Clean Minimal | เรียบง่าย ไม่มี Sidebar | มือใหม่ บทความสั้น |
| With Sidebar | มีแถบด้านข้าง | บล็อกที่มี Widget |
| Full Width | เต็มความกว้างจอ | บทความ Long-form |
| Hero Section | มีรูปใหญ่ด้านบน | บทความเด่น Feature |

**คำแนะนำ**: มือใหม่เลือก Clean Minimal → ค่อยเพิ่ม Component ทีละตัว

## SLIDE 5: Text Component — หัวใจของบทความ
```
╔══════════════════════════════════════════╗
║   Text Component: เขียนเนื้อหาทั้งหมด   ║
╚══════════════════════════════════════════╝
```

**ใช้สร้าง:**
- Heading H1, H2, H3, H4
- ย่อหน้า (Paragraph)
- Bullet Point / Numbered List

**Heading Hierarchy (สำคัญมากสำหรับ SEO):**
```
H1: ชื่อบทความ (มีแค่ 1 ตัว)
  H2: หัวข้อหลักที่ 1
    H3: หัวข้อย่อย 1.1
    H3: หัวข้อย่อย 1.2
  H2: หัวข้อหลักที่ 2
    H3: หัวข้อย่อย 2.1
```

**วิธีใช้**: ลากจาก Component Panel → วางบน Canvas → พิมพ์ได้เลย

## SLIDE 6: Image + Video Components
```
╔══════════════════════════════════════════╗
║   Image & Video: Multimedia ดึงดูดคน    ║
╚══════════════════════════════════════════╝
```

**Image Component:**

| การตั้งค่า | คำแนะนำ |
|-----------|---------|
| แหล่งรูป | อัปโหลดจากเครื่อง หรือใส่ URL |
| Alt Text | ใส่ทุกรูป (SEO + Accessibility) |
| ขนาด | ปรับให้เหมาะกับ Layout |

**Video Component:**

| วิธี | เหมาะกับ | ข้อดี |
|------|---------|-------|
| Embed YouTube/Vimeo | วิดีโอสาธารณะ | ประหยัด Storage + ได้ View |
| Upload ตรง | วิดีโอเฉพาะสมาชิก | ควบคุมได้เต็มที่ |

**ข้อดีของ Video**: เพิ่ม Time on Page → Google เห็นว่าผู้อ่านสนใจ

## SLIDE 7: Button + Divider Components
```
╔══════════════════════════════════════════╗
║   Button & Divider: นำทาง + จัดระเบียบ  ║
╚══════════════════════════════════════════╝
```

**Button Component:**
```
┌─────────────────────────────────┐
│  ✅ ดี: "ดาวน์โหลดฟรี"         │  → Action-Oriented
│  ✅ ดี: "สมัครเลย"              │  → บอกว่าได้อะไร
│  ❌ ไม่ดี: "คลิกที่นี่"          │  → ไม่สื่อ Action
└─────────────────────────────────┘
```
- สีปุ่มตัดกับพื้นหลัง ให้เห็นชัด
- Link ปลายทาง: สมัครอีเมล / หน้าสินค้า / ดาวน์โหลด / บทความอื่น

**Divider Component:**
- เส้นคั่นแบ่งส่วนเนื้อหา
- บทความยาว **ต้องมี** Divider ไม่งั้นเนื้อหาเป็นก้อนใหญ่อ่านยาก
- ช่วยให้ผู้อ่านเห็นว่าแต่ละ Section เริ่มต้นตรงไหน

## SLIDE 8: Typography — จัดรูปแบบตัวอักษร
```
╔══════════════════════════════════════════╗
║   Typography: ดูเป็นมืออาชีพ            ║
╚══════════════════════════════════════════╝
```

**การตั้งค่าที่แนะนำ:**

| องค์ประกอบ | ค่าแนะนำ | เหตุผล |
|-----------|---------|--------|
| Body Text Size | 16-18px | อ่านง่ายบนทุกอุปกรณ์ |
| Line Height | 1.5-1.8 เท่า | ไม่แน่นเกินไป สบายตา |
| Heading Size | H1 > H2 > H3 | ลำดับชัดเจน |
| Font Weight | Bold เฉพาะ Heading/คำสำคัญ | ไม่ Bold ทั้งบทความ |
| Font Color | สีเข้มอ่านง่าย | หลีกเลี่ยงสีอ่อนบนพื้นขาว |

**กฎทอง**: ความสม่ำเสมอ — ใช้ Font Style เดียวกันตลอดบทความ อย่าเปลี่ยนไปมา

## SLIDE 9: Internal Links — เชื่อมบทความภายใน
```
╔══════════════════════════════════════════╗
║   Internal Links: เชื่อมเนื้อหาใน Blog  ║
╚══════════════════════════════════════════╝
```

**ข้อดี 3 ประการ:**
```
[1] ผู้อ่านค้นพบเนื้อหาที่เกี่ยวข้อง
          ↓
[2] เพิ่ม Pages per Session (Google ชอบ)
          ↓
[3] กระจาย Link Juice ทุกหน้าแข็งแรงขึ้น (SEO)
```

**วิธีใส่ใน Content Builder:**
1. Select ข้อความที่ต้องการ
2. กดปุ่ม Link
3. เลือก URL ของบทความปลายทาง

**เคล็ดลับ:**
- Anchor Text สื่อความหมาย: "อ่านเพิ่มเรื่อง Email Marketing" ดีกว่า "คลิกที่นี่"
- แนะนำ **3-5 Internal Links** ต่อบทความ

## SLIDE 10: SEO On-page — 3 จุดที่ต้องตั้งค่า
```
╔══════════════════════════════════════════╗
║   SEO On-page: Title + Meta + URL Slug  ║
╚══════════════════════════════════════════╝
```

| จุด | คำแนะนำ | ตัวอย่าง |
|-----|---------|---------|
| Title | ไม่เกิน 60 ตัวอักษร, มี Keyword, ดึงดูดคลิก | "วิธีสร้างบทความด้วย Kartra ฉบับมือใหม่" |
| Meta Description | 150-160 ตัวอักษร, Keyword + CTA เบาๆ | "เรียนรู้วิธีสร้างบทความ Drag & Drop ใน 10 นาที พร้อมเทคนิค SEO ที่ใช้ได้จริง" |
| URL Slug | สั้น, ภาษาอังกฤษ, คั่นด้วย -, มี Keyword | yoursite.com/blog/content-builder-guide |

**ข้อห้ามสำคัญ**: URL Slug ห้ามเปลี่ยนหลัง Publish → เกิด Broken Links

## SLIDE 11: Preview + Publish
```
╔══════════════════════════════════════════╗
║   Preview + Publish: ก่อนเผยแพร่        ║
╚══════════════════════════════════════════╝
```

**Preview Checklist:**

| อุปกรณ์ | สิ่งที่ต้องตรวจ | ผ่าน? |
|---------|----------------|:---:|
| Desktop | Layout ถูกต้อง, รูปแสดงครบ | [ ] |
| Tablet | อ่านง่าย, ปุ่มกดสะดวก | [ ] |
| Mobile | Typography อ่านชัด, ไม่ล้นจอ | [ ] |

**Publish Checklist:**

| รายการ | ตรวจสอบ | ผ่าน? |
|--------|---------|:---:|
| Title (SEO) | มี Keyword, ไม่เกิน 60 ตัวอักษร | [ ] |
| Meta Description | 150-160 ตัวอักษร, มี CTA | [ ] |
| URL Slug | สั้น, ภาษาอังกฤษ, มี Keyword | [ ] |
| Featured Image | อัปโหลดแล้ว | [ ] |
| Category + Tags | เลือกแล้ว | [ ] |
| Internal Links | มี 3-5 ลิงก์ | [ ] |
| อ่านจบ 1 รอบ | อ่านจากต้นจนจบ | [ ] |

## SLIDE 12: สรุป + Next Steps
```
╔══════════════════════════════════════════╗
║   สรุป: Content Builder Workflow        ║
╚══════════════════════════════════════════╝
```

**Workflow สร้างบทความ:**
```
New Post → เลือก Template → Drag & Drop 5 Components
    ↓
Text + Image + Video + Button + Divider
    ↓
จัด Typography → ใส่ Internal Links
    ↓
ตั้งค่า SEO (Title + Meta + URL Slug)
    ↓
Preview 3 ขนาดจอ → อ่านจบ 1 รอบ → Publish
```

**จำง่าย — 5 Components หลัก:**
- **T**ext — เนื้อหา
- **I**mage — รูปภาพ
- **V**ideo — วิดีโอ
- **B**utton — ปุ่ม CTA
- **D**ivider — เส้นคั่น
