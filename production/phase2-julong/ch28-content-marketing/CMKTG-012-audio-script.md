# ปรับแต่ง Template Blog Post List — CMKTG-012 Audio Script
> Format: Audio Script (Host A/B)
> Source: SWP3 Ch28 Content Marketing Mastery ตอนที่ 12
> Production: PinkCastle Academy | จูล่ง CTO

**Host A:** สวัสดีครับทุกคน กลับมาพบกันอีกครั้งกับซีรีส์ Content Marketing Mastery วันนี้เป็นตอนที่ 12 แล้วนะครับ ตอนที่แล้วเราพูดถึง Template สำหรับหน้าบทความเดี่ยว วันนี้เรามาพูดถึงอีกหน้าหนึ่งที่สำคัญไม่แพ้กัน นั่นก็คือ **Template Blog Post List** หรือหน้ารวมบทความทั้งหมดครับ

**Host B:** ถูกต้องค่ะ หลายคนอาจจะคิดว่าหน้ารวมบทความไม่สำคัญ แค่แสดงรายชื่อบทความเรียงๆ ก็พอ แต่จริงๆ แล้วหน้านี้คือ **ประตูหน้าของบล็อก** ค่ะ เป็นหน้าแรกที่ผู้อ่านจะเห็นเมื่อคลิกเข้ามาที่เมนู "Blog" หรือ "บทความ" ถ้าหน้านี้ไม่ดึงดูด ไม่เป็นระเบียบ ผู้อ่านก็จะกดปิดไปเลยโดยไม่ได้อ่านบทความแม้แต่ชิ้นเดียว

**Host A:** วันนี้เราจะมาพูดถึงการปรับแต่ง Template Blog Post List ใน Kartra กันครับ Kartra เป็นแพลตฟอร์ม All-in-One สำหรับธุรกิจออนไลน์ ที่มีระบบสร้างบล็อกในตัว โดยเราจะครอบคลุมหัวข้อหลักๆ ตั้งแต่ Layout Grid กับ List, Pagination, Category Filter, Featured Post, Sidebar Widgets, SEO Settings, Responsive Design และที่สำคัญคือวิธีเชื่อม Blog กับ Funnel ครับ

**Host B:** เริ่มจากเรื่อง **Layout** กันก่อนเลยค่ะ ใน Kartra มีรูปแบบการแสดงรายการบทความหลักๆ 2 แบบ คือ **Grid Layout** กับ **List Layout** Grid Layout จะแสดงบทความเป็นการ์ดเรียงเป็นตาราง เหมือนหน้า Pinterest หรือ Instagram ดูสวยงาม ทันสมัย เหมาะกับบล็อกที่เน้นภาพเป็นหลัก

**Host A:** ส่วน List Layout จะแสดงบทความเป็นแถวๆ จากบนลงล่างครับ แต่ละแถวจะมีรูปภาพด้านซ้าย แล้วชื่อเรื่องกับ Excerpt ด้านขวา ดูเป็นทางการกว่า เหมาะกับบล็อกที่เนื้อหาเยอะ อยากให้ผู้อ่านเห็น Preview ของแต่ละบทความได้ชัดเจน

**Host B:** คำถามที่หลายคนถามคือ ควรเลือกแบบไหนดีค่ะ คำตอบขึ้นอยู่กับประเภทคอนเทนต์และกลุ่มเป้าหมายค่ะ ถ้าคอนเทนต์เน้นภาพเยอะ เช่น บล็อกท่องเที่ยว อาหาร ออกแบบ ก็ใช้ Grid ถ้าคอนเทนต์เน้นเนื้อหาเชิงลึก เช่น บทความการตลาด การเงิน ให้ใช้ List จะดีกว่า บางคนก็ใช้แบบผสมได้ค่ะ

**Host A:** ต่อมาเรื่อง **Pagination** ครับ Pagination คือระบบแบ่งหน้า เมื่อบทความมีจำนวนเยอะขึ้น เราไม่ควรแสดงทั้งหมดในหน้าเดียว เพราะหน้าจะโหลดช้ามาก ใน Kartra เราสามารถตั้งค่าได้ว่าจะแสดงกี่บทความต่อหน้า แนะนำ 6-12 บทความต่อหน้าครับ ไม่มากไป ไม่น้อยไป

**Host B:** Pagination มี 2 แบบหลักค่ะ แบบแรกคือ **Numbered Pagination** คือมีเลขหน้า 1, 2, 3, 4 ให้คลิก แบบที่สองคือ **Load More Button** คือปุ่ม "โหลดเพิ่มเติม" ที่กดแล้วบทความจะเพิ่มมาในหน้าเดิมโดยไม่ต้องรีโหลดหน้า แบบ Load More จะให้ประสบการณ์ผู้ใช้ที่ดีกว่า โดยเฉพาะบนมือถือค่ะ

**Host A:** มาถึงเรื่อง **Category Filter** ครับ ถ้าบล็อกมีบทความหลายหมวดหมู่ การมี Category Filter จะช่วยให้ผู้อ่านกรองเฉพาะหมวดที่สนใจได้เลย ใน Kartra เราสร้าง Category ได้ไม่จำกัด แล้วผูกบทความแต่ละชิ้นเข้ากับหมวดหมู่ที่เหมาะสม จากนั้นหน้า Blog Post List จะมีปุ่มกรองให้ผู้อ่านกดเลือกได้

**Host B:** เคล็ดลับสำคัญเรื่อง Category ค่ะ อย่าสร้างหมวดหมู่มากเกินไปนะคะ 5-8 หมวดหมู่ถือว่าเหมาะสม ถ้ามากกว่านี้ผู้อ่านจะสับสน และชื่อหมวดหมู่ควรสั้นกระชับ เข้าใจง่าย เช่น "การตลาด", "SEO", "อีเมล", "คอนเทนต์" ไม่ควรใช้ชื่อยาวเกินไป

**Host A:** ทีนี้มาพูดถึง **Featured Post** ครับ Featured Post คือบทความเด่นที่เราอยากให้ผู้อ่านเห็นก่อนเป็นอันดับแรก อาจจะเป็นบทความใหม่ล่าสุด บทความยอดนิยม หรือบทความที่เรากำลังโปรโมทอยู่ ใน Kartra เราสามารถปักหมุดบทความให้อยู่ด้านบนสุดของรายการได้

**Host B:** การออกแบบ Featured Post ควรให้โดดเด่นกว่าบทความอื่นๆ ค่ะ เช่น ใช้รูปภาพขนาดใหญ่กว่า ชื่อเรื่องตัวใหญ่กว่า มีคำอธิบายยาวกว่า หรือมีป้าย "แนะนำ" กำกับ เพื่อดึงความสนใจของผู้อ่าน การมี Featured Post ช่วยเพิ่ม Click-Through Rate ได้อย่างมากค่ะ

**Host A:** ต่อมาเรื่อง **Sidebar Widgets** ครับ เช่นเดียวกับหน้าบทความเดี่ยว หน้ารวมบทความก็ควรมี Sidebar ที่มี Widget ที่เป็นประโยชน์ Widget ที่แนะนำสำหรับหน้า Blog Post List ก็จะมี Search Bar สำหรับค้นหาบทความ, Category List รายชื่อหมวดหมู่ทั้งหมด, Recent Posts บทความล่าสุด, Email Opt-in Form แบบฟอร์มสมัครอีเมล และ Social Media Follow Buttons ปุ่มติดตามโซเชียล

**Host B:** สิ่งสำคัญคือ Sidebar ต้องไม่รกจนเกินไปค่ะ เลือก 3-5 Widget ที่จำเป็นจริงๆ และจัดเรียงตามลำดับความสำคัญ Widget ที่สำคัญที่สุดไว้บนสุด เช่น Search Bar หรือ Email Opt-in ส่วน Widget ที่สำคัญน้อยกว่าวางไว้ด้านล่าง

**Host A:** มาถึงเรื่อง **SEO Settings** ครับ หน้า Blog Post List ก็ต้องทำ SEO เช่นกัน ไม่ใช่แค่บทความแต่ละชิ้นเท่านั้น สิ่งที่ต้องตั้งค่าคือ Title Tag ของหน้า Blog, Meta Description ที่อธิบายว่าบล็อกของเราเกี่ยวกับอะไร, URL ที่สั้นและสื่อความหมาย เช่น yoursite.com/blog และ Open Graph Image สำหรับการแชร์บนโซเชียลมีเดียครับ

**Host B:** อีกเรื่องที่เกี่ยวกับ SEO คือ **Schema Markup** ค่ะ ถ้าเป็นไปได้ ควรเพิ่ม Blog Posting Schema ให้กับหน้า Blog Post List ซึ่งจะช่วยให้ Google เข้าใจว่าหน้านี้เป็นรายการบทความ และอาจแสดงผลเป็น Rich Snippet ในผลค้นหา ช่วยเพิ่ม Click-Through Rate จากหน้า Google ได้

**Host A:** เรื่อง **Responsive Design** ครับ สำคัญมากเลย ผู้ใช้กว่า 60-70% เข้าเว็บผ่านมือถือ หน้า Blog Post List ต้องแสดงผลสวยงามบนทุกอุปกรณ์ ถ้าใช้ Grid Layout บนมือถือ ควรเปลี่ยนเป็นแสดง 1 คอลัมน์ ถ้าใช้ List Layout ต้องดูให้มั่นใจว่ารูปภาพไม่ล้นจอ และตัวอักษรอ่านง่าย

**Host B:** ใน Kartra มีโหมด Preview ให้ดูผลลัพธ์บน 3 ขนาดหน้าจอค่ะ คือ Desktop, Tablet และ Mobile ทุกครั้งที่ปรับแต่ง Template ต้องตรวจสอบทั้ง 3 ขนาดเสมอ โดยเฉพาะบนมือถือ ต้องดูว่าปุ่มกดง่าย ตัวอักษรอ่านชัด รูปภาพโหลดเร็ว และ Category Filter ใช้งานสะดวก

**Host A:** และเรื่องสุดท้ายที่สำคัญมากคือ **วิธีเชื่อม Blog กับ Funnel** ครับ นี่คือจุดเด่นของ Kartra เลย เพราะมันเป็นระบบ All-in-One คือบล็อกกับ Funnel อยู่ในแพลตฟอร์มเดียวกัน เราสามารถสร้าง CTA ในบทความที่ลิงก์ไปยัง Landing Page หรือ Opt-in Page ใน Funnel ได้เลย ไม่ต้องใช้เครื่องมือแยกต่างหาก

**Host B:** ยกตัวอย่างค่ะ สมมติเราเขียนบทความเรื่อง "5 เทคนิค Email Marketing" ท้ายบทความเราใส่ CTA ว่า "ดาวน์โหลด Email Template ฟรี" เมื่อผู้อ่านคลิก ก็จะเข้าสู่ Opt-in Page ที่เราสร้างใน Kartra Funnel ผู้อ่านกรอกอีเมล ได้รับ Lead Magnet อัตโนมัติ แล้วเข้าสู่ Email Sequence ต่อเนื่อง ทุกอย่างเชื่อมต่อกันหมดค่ะ

**Host A:** และยังมี **Behavioral Adaptive Marketing** หรือ BAM ด้วยครับ ระบบจะจำได้ว่าผู้อ่านคนนี้อ่านบทความหมวดไหนมาก แล้วแสดงเนื้อหาที่ตรงกับความสนใจ เช่น ถ้าคนอ่านบทความ SEO บ่อย ก็จะเห็น CTA เกี่ยวกับ SEO ไม่ใช่ Email Marketing ระบบฉลาดขนาดนี้เพราะข้อมูลทั้งหมดอยู่ในที่เดียว

**Host B:** สรุปแล้ว การปรับแต่ง Template Blog Post List ใน Kartra ประกอบด้วย 7 ส่วนหลักค่ะ คือ Layout (Grid/List), Pagination, Category Filter, Featured Post, Sidebar Widgets, SEO Settings และ Responsive Design บวกกับเทคนิคเชื่อม Blog กับ Funnel เพื่อเปลี่ยนผู้อ่านเป็นลูกค้าค่ะ

**Host A:** ทุกส่วนต้องทำงานร่วมกันอย่างลงตัวครับ เป้าหมายคือทำให้หน้ารวมบทความของเราเป็นประตูที่เชิญชวนให้คนเข้ามาอ่าน ค้นหาได้ง่าย แล้วก็เปลี่ยนจากผู้อ่านเป็น Subscriber หรือลูกค้าในที่สุด

**Host B:** ถูกต้องค่ะ ครั้งหน้าเราจะมาพูดถึงการสร้างบทความด้วย Content Builder ตอน 1 กันค่ะ อย่าลืมติดตามนะคะ แล้วพบกันใหม่ค่ะ
