## คำถาม

### Q1. Content Builder ใน Kartra เข้าถึงได้อย่างไร?
- A) ไปที่ Settings → Advanced → Content Builder
- B) ไปที่ My Content หรือ Blog → กด New Post → เข้าสู่ Content Builder
- C) ดาวน์โหลดโปรแกรมแยกต่างหากจากเว็บไซต์ Kartra
- D) ต้องเขียนโค้ด HTML เพื่อเปิด Builder

### Q2. สำหรับมือใหม่ ควรเลือก Template แบบใดในการเริ่มต้น?
- A) Template ที่ซับซ้อนที่สุด เพราะมี Component ครบ
- B) Template แบบเรียบง่าย แล้วค่อยเพิ่ม Component ทีละตัว
- C) ไม่ต้องใช้ Template เริ่มจากหน้าเปล่าเสมอ
- D) Template ที่มี Animation เยอะที่สุด

### Q3. Components หลัก 5 ชนิดที่ใช้สร้างบทความมีอะไรบ้าง?
- A) Header, Footer, Sidebar, Menu, Logo
- B) Text, Image, Video, Button, Divider
- C) Font, Color, Size, Border, Shadow
- D) Title, Subtitle, Body, Caption, Quote

### Q4. Heading Hierarchy ที่ถูกต้องคือข้อใด?
- A) H3 สำหรับชื่อบทความ, H1 สำหรับหัวข้อย่อย
- B) ใช้ H1 ทุกหัวข้อเพื่อให้ตัวใหญ่สม่ำเสมอ
- C) H1 สำหรับชื่อบทความ (มีแค่ 1 ตัว), H2 หัวข้อหลัก, H3 หัวข้อย่อย
- D) ไม่จำเป็นต้องใช้ Heading ใช้ Bold แทนได้

### Q5. ทำไมต้องใส่ Alt Text ให้รูปภาพทุกรูป?
- A) เพื่อให้รูปภาพโหลดเร็วขึ้น
- B) เพื่อช่วย SEO (Google เข้าใจว่ารูปเกี่ยวกับอะไร) และช่วยผู้ใช้ Screen Reader
- C) เพื่อทำให้รูปภาพมีขนาดเล็กลง
- D) เพื่อป้องกันไม่ให้คนอื่นขโมยรูป

### Q6. เมื่อไหร่ควร Embed วิดีโอจาก YouTube แทนที่จะอัปโหลดตรง?
- A) ไม่ควร Embed จาก YouTube เลย ควรอัปโหลดเสมอ
- B) เมื่อวิดีโออยู่บน YouTube อยู่แล้ว เพราะประหยัด Storage และได้ YouTube View เพิ่ม
- C) เมื่อวิดีโอเป็นเนื้อหาเฉพาะสมาชิก
- D) เมื่อวิดีโอมีความยาวน้อยกว่า 1 นาที

### Q7. ข้อความบน Button ที่ดีควรเป็นแบบใด?
- A) "คลิกที่นี่"
- B) "ปุ่ม"
- C) "ดาวน์โหลดฟรี" หรือ "สมัครเลย" (Action-Oriented)
- D) "ลิงก์"

### Q8. Body Text ขนาดเท่าไหร่ที่แนะนำ และ Line Height ควรเป็นเท่าไร?
- A) 10-12px, Line Height 1.0 เท่า
- B) 16-18px, Line Height 1.5-1.8 เท่า
- C) 24-30px, Line Height 3.0 เท่า
- D) ไม่จำกัด ขึ้นอยู่กับอารมณ์

### Q9. Internal Links คืออะไร และแนะนำกี่ลิงก์ต่อบทความ?
- A) ลิงก์ไปยังเว็บไซต์อื่น แนะนำ 20-30 ลิงก์
- B) ลิงก์ไปยังหน้า Social Media แนะนำ 1 ลิงก์
- C) ลิงก์ในบทความที่เชื่อมไปยังบทความอื่นในบล็อกเดียวกัน แนะนำ 3-5 ลิงก์
- D) ลิงก์สำหรับดาวน์โหลดไฟล์ แนะนำ 10 ลิงก์

### Q10. SEO Title สำหรับบทความควรมีความยาวเท่าไร?
- A) ไม่เกิน 10 ตัวอักษร
- B) ไม่เกิน 60 ตัวอักษร ต้องมี Keyword หลักและดึงดูดให้คลิก
- C) ไม่เกิน 200 ตัวอักษร ใส่ Keyword ให้มากที่สุด
- D) ไม่จำกัดความยาว ยิ่งยาวยิ่งดี

### Q11. URL Slug ที่ดีควรเป็นอย่างไร?
- A) ยาวมากเพื่อใส่ Keyword ทั้งหมด เช่น /blog/how-to-create-article-content-builder-kartra-step-by-step-guide-2026-seo
- B) สั้น ใช้ภาษาอังกฤษ มี Keyword หลัก คั่นด้วยขีด เช่น /blog/content-builder-guide
- C) ใช้ภาษาไทยทั้งหมด เช่น /blog/สร้างบทความ-คอนเทนท์
- D) ใช้ตัวเลขอย่างเดียว เช่น /blog/12345

### Q12. ก่อน Publish บทความ ต้องทำอะไรเป็นสิ่งสำคัญที่สุด?
- A) กด Publish ทันที ไม่ต้อง Preview
- B) Preview เฉพาะบน Desktop อย่างเดียว
- C) Preview ทั้ง 3 ขนาดจอ (Desktop, Tablet, Mobile) และอ่านทั้งบทความจากต้นจนจบอย่างน้อยหนึ่งรอบ
- D) ส่งให้เพื่อนอ่านก่อน ไม่ต้อง Preview เอง

## Answer Key

| ข้อ | คำตอบ | คำอธิบาย |
|-----|-------|---------|
| Q1 | ✅ **B** | เข้าถึง Content Builder ผ่าน My Content หรือ Blog → กด New Post ระบบจะเปิดหน้า Settings และ Content Builder |
| Q2 | ✅ **B** | Template เรียบง่ายช่วยให้เข้าใจหลักการ Content Builder ได้เร็ว ไม่สับสนกับ Component ที่มากเกินไป |
| Q3 | ✅ **B** | Text, Image, Video, Button, Divider คือ 5 Components หลักที่ใช้สร้างบทความ ใช้ Drag & Drop ลากมาวาง |
| Q4 | ✅ **C** | H1 มีแค่ 1 ตัวสำหรับชื่อบทความ, H2 หัวข้อหลัก, H3 หัวข้อย่อย ลำดับถูกต้องช่วย SEO |
| Q5 | ✅ **B** | Alt Text ช่วย Google เข้าใจเนื้อหารูปภาพ (SEO) และช่วยผู้ใช้ Screen Reader เข้าถึงเนื้อหาได้ |
| Q6 | ✅ **B** | วิดีโอที่อยู่บน YouTube อยู่แล้ว Embed ดีกว่า ประหยัด Storage + ได้ YouTube View อัปโหลดตรงเหมาะกับวิดีโอเฉพาะสมาชิก |
| Q7 | ✅ **C** | ข้อความ Action-Oriented บอกว่ากดแล้วได้อะไร เช่น "ดาวน์โหลดฟรี" ชัดเจนและดึงดูดมากกว่า "คลิกที่นี่" |
| Q8 | ✅ **B** | Body Text 16-18px อ่านง่ายทุกอุปกรณ์, Line Height 1.5-1.8 เท่า ไม่แน่นเกินไป สบายตา |
| Q9 | ✅ **C** | Internal Links เชื่อมบทความภายในบล็อกเดียวกัน ช่วยผู้อ่านค้นพบเนื้อหา + เพิ่ม Pages per Session + กระจาย Link Juice |
| Q10 | ✅ **B** | Title ไม่เกิน 60 ตัวอักษร มี Keyword หลัก และดึงดูดให้คนอยากคลิก แสดงบนหน้า Google |
| Q11 | ✅ **B** | URL Slug สั้น ภาษาอังกฤษ มี Keyword คั่นด้วยขีด ไม่มีอักขระพิเศษ ห้ามเปลี่ยนหลัง Publish |
| Q12 | ✅ **C** | Preview 3 ขนาดจอ ตรวจรูปภาพ ลิงก์ ปุ่ม Typography และอ่านจบทั้งบทความอย่างน้อย 1 รอบก่อน Publish |
