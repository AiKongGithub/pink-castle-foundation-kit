## Answer Key

| ข้อ | คำตอบ | คำอธิบาย |
|-----|-------|---------|
| Q1 | **B** | Thrive Architect เป็น Visual Page Builder ที่ทำงานบน WordPress แบบ Drag-and-Drop |
| Q2 | **A** | Hero Section ประกอบด้วย Title, Featured Image, Date, Author |
| Q3 | **C** | 16-18px เป็นขนาดที่อ่านง่ายสบายตาบนหน้าจอ |
| Q4 | **D** | วิดีโอ Autoplay เสียงดังจะรบกวนผู้อ่านและทำให้ปิดหน้าทันที |
| Q5 | **C** | 3 ตำแหน่งเชิงกลยุทธ์: หลังย่อหน้าแรก กลางบทความ และท้ายบทความ |
| Q6 | **C** | 1200x630px เป็นขนาดมาตรฐานที่คมชัดและเหมาะกับ Open Graph |
| Q7 | **B (ไม่จริง)** | Author Box สำคัญมาก ช่วยสร้างความน่าเชื่อถือและความเชื่อมต่อกับผู้อ่าน |
| Q8 | **A (จริง)** | Related Posts ช่วยให้ผู้อ่านอยู่ในเว็บนานขึ้น ลด Bounce Rate |
| Q9 | **B (ไม่จริง)** | Responsive Design จำเป็นมาก ผู้ใช้ 60%+ เข้าเว็บผ่านมือถือ |
| Q10 | **Boxed** | 3 แบบ คือ Full-width, Boxed, Split |
