## สรุปภาพรวม

บทเรียนนี้สอนการสร้างบทความด้วย Content Builder ใน Kartra ตั้งแต่ขั้นตอนแรกจนถึงการ Publish โดยครอบคลุมวิธีเข้าถึง Builder, การเลือก Template, การใช้ Drag & Drop วาง Components หลัก 5 ชนิด (Text, Image, Video, Button, Divider), การจัดรูปแบบ Typography, การใส่ Internal Links, การตั้งค่า SEO On-page (Title, Meta Description, URL Slug) และการ Preview + Publish ระยะเวลาบทเรียน 33 นาที 55 วินาที

## ประเด็นสำคัญ

| หัวข้อ | รายละเอียด |
|--------|------------|
| เครื่องมือหลัก | Kartra Content Builder — เครื่องมือ Drag & Drop สร้างบทความ |
| แนวคิดหลัก | สร้างบทความคุณภาพได้โดยไม่ต้องเขียนโค้ด ด้วย Components สำเร็จรูป |
| เป้าหมาย | สร้างบทความที่สวยงาม อ่านง่าย และ SEO-Friendly |
| ระดับความยาก | เริ่มต้น |

## ขั้นตอนการสร้างบทความ

| ขั้นตอน | การดำเนินการ | ผลลัพธ์ |
|---------|-------------|---------|
| 1 | เข้าถึง Content Builder ผ่าน My Content → New Post | เปิดหน้าสร้างบทความใหม่ |
| 2 | เลือก Template เริ่มต้น (แนะนำแบบเรียบง่าย) | โครงสร้างพื้นฐานพร้อมใช้ |
| 3 | ลาก Text Component สร้างเนื้อหา + Heading H1-H3 | เนื้อหาหลักของบทความ |
| 4 | ลาก Image Component ใส่รูปภาพ + Alt Text | บทความดึงดูดสายตามากขึ้น |
| 5 | ลาก Video Component Embed วิดีโอ YouTube/Vimeo | เพิ่ม Time on Page |
| 6 | ลาก Button Component ใส่ CTA | นำทางผู้อ่านไปทำ Action |
| 7 | ลาก Divider Component แบ่งส่วนเนื้อหา | บทความอ่านง่ายขึ้น |
| 8 | จัดรูปแบบ Typography (Font, Size, Line Height) | ดูเป็นมืออาชีพ สม่ำเสมอ |
| 9 | ใส่ Internal Links 3-5 จุด | เชื่อมบทความภายในบล็อก |
| 10 | ตั้งค่า SEO: Title, Meta Description, URL Slug | บทความติดอันดับ Google |
| 11 | Preview บน Desktop, Tablet, Mobile | ตรวจสอบการแสดงผลทุกจอ |
| 12 | Publish เผยแพร่บทความ | บทความไปอยู่บนหน้า Blog |

## Components หลัก 5 ชนิด

| Component | หน้าที่ | เคล็ดลับ |
|-----------|---------|---------|
| Text | เขียนเนื้อหา หัวข้อ รายการ | ใช้ Heading Hierarchy H1→H2→H3 ถูกลำดับ |
| Image | รูปภาพประกอบบทความ | ใส่ Alt Text ทุกครั้ง ช่วย SEO |
| Video | วิดีโอ Embed หรือ Upload | Embed YouTube ประหยัด Storage + ได้ View |
| Button | ปุ่ม CTA นำทางผู้อ่าน | ข้อความ Action-Oriented สีตัดกับพื้นหลัง |
| Divider | เส้นคั่นแบ่งส่วนเนื้อหา | ใช้แบ่ง Section ในบทความยาว |

## เคล็ดลับสำคัญ

- **Template**: เริ่มจากแบบเรียบง่ายก่อน แล้วค่อยเพิ่ม Component ทีละตัว
- **Typography**: Body Text 16-18px, Line Height 1.5-1.8 เท่า ใช้ Style สม่ำเสมอ
- **Internal Links**: ใส่ 3-5 ลิงก์ต่อบทความ Anchor Text ต้องสื่อความหมาย
- **SEO Title**: ใส่ Keyword หลัก ไม่เกิน 60 ตัวอักษร ดึงดูดให้คลิก
- **Meta Description**: 150-160 ตัวอักษร มี Keyword + CTA เบาๆ
- **URL Slug**: สั้น ใช้ภาษาอังกฤษ คั่นด้วยขีด ไม่เปลี่ยนหลัง Publish
- **Preview**: ตรวจสอบทั้ง 3 ขนาดจอ อ่านจบทั้งบทความก่อน Publish

## ผลกระทบต่อธุรกิจ

| ตัวชี้วัด | ไม่ใช้ Content Builder อย่างถูกวิธี | ใช้ Content Builder อย่างถูกวิธี |
|-----------|-----------------------------------|---------------------------------|
| เวลาสร้างบทความ | 2-3 ชั่วโมง | 30-60 นาที |
| SEO Score | ต่ำ (ขาด On-page) | สูง (Title+Meta+URL ครบ) |
| Time on Page | ต่ำ (ไม่มี Video/Image) | สูง (Multimedia + อ่านง่าย) |
| Internal Link Clicks | ไม่มี | 5-15% ของผู้อ่าน |
| Mobile Readability | แย่ | ดี (Preview ก่อน Publish) |

## สิ่งที่ต้องทำต่อ

1. ฝึกสร้างบทความ 1 ชิ้น ใช้ Components ครบ 5 ตัว
2. ตั้งค่า SEO On-page ให้ครบ (Title, Meta Description, URL Slug)
3. ใส่ Internal Links อย่างน้อย 3 จุด
4. Preview ทุกขนาดจอก่อน Publish
5. เตรียมบทความสำหรับตอนถัดไป: ตรวจการบ้าน (ครั้งที่ 2)
