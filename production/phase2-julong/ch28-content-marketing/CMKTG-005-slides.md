## SLIDE 3: BOFU คืออะไร?
```
+==========================================+
|     BOFU — Bottom of Funnel             |
+==========================================+
|                                          |
|   ชั้นล่างสุดของ Marketing Funnel        |
|   เป้าหมาย: ปิดการขาย (Conversion)      |
|                                          |
|   +----------------------------------+   |
|   |  BOFU leads มี conversion rate   |   |
|   |  สูงถึง 20-30%                   |   |
|   |  (เทียบกับ TOFU แค่ 1-2%)        |   |
|   +----------------------------------+   |
|                                          |
|   คนที่อยู่ชั้น BOFU:                    |
|   * ผ่าน TOFU → รู้จักเราแล้ว           |
|   * ผ่าน MOFU → เชื่อมั่นเราแล้ว        |
|   * ต้องการแค่ แรงผลักดันสุดท้าย!        |
|                                          |
|   BOFU = ขจัดข้อกังวลสุดท้าย             |
|   ทำให้มั่นใจ 100% ว่าเลือกเราถูกแล้ว   |
|                                          |
+==========================================+
```

> BOFU = แรงผลักดันสุดท้ายเพื่อกดปุ่มซื้อ

## SLIDE 10: BOFU KPI — 3 ตัวชี้วัดหลัก
```
+==========================================+
|      BOFU KPI — วัดการปิดการขาย          |
+==========================================+
|                                          |
|   +-----------+-----------------------+  |
|   | KPI       | รายละเอียด             |  |
|   +-----------+-----------------------+  |
|   | Conversion| อัตราเปลี่ยน lead      |  |
|   | Rate      | → ลูกค้า (20-30%)     |  |
|   +-----------+-----------------------+  |
|   | Revenue   | รายได้จริง AOV         |  |
|   |           | + CLV                 |  |
|   +-----------+-----------------------+  |
|   | CAC       | ต้นทุนได้ลูกค้า 1 คน   |  |
|   |           | (ค่าใช้จ่าย/ลูกค้าใหม่) |  |
|   +-----------+-----------------------+  |
|                                          |
|   สูตรสำคัญ:                             |
|   +----------------------------------+   |
|   | CLV > CAC x 3                    |   |
|   | (ต่ำกว่า 3 เท่า = ต้องปรับปรุง!)  |   |
|   +----------------------------------+   |
|                                          |
+==========================================+
```

> CLV/CAC Ratio อย่างน้อย 3:1 = Funnel ที่ยั่งยืน

## SLIDE 11: ภาพรวม Full Funnel
```
+==========================================+
|       Full Funnel — TOFU + MOFU + BOFU  |
+==========================================+
|                                          |
|  +--------+   +--------+   +--------+   |
|  | TOFU   | → | MOFU   | → | BOFU   |   |
|  | รู้จัก  |   | เชื่อมั่น|   | ซื้อ!   |   |
|  +--------+   +--------+   +--------+   |
|                                          |
|  TOFU เครื่องมือ:                         |
|  Blog, Social Media, Video, Infographic  |
|  KPI: Traffic, Reach, Views              |
|                                          |
|  MOFU เครื่องมือ:                         |
|  Case Study, Webinar, E-book, Email Seq  |
|  KPI: Leads, Subscribers, Open Rate      |
|                                          |
|  BOFU เครื่องมือ:                         |
|  Free Trial, Demo, Testimonial,         |
|  Comparison, Guarantee, Pricing Page     |
|  KPI: Conversion, Revenue, CAC          |
|                                          |
|  +----------------------------------+   |
|  | ทั้ง 3 ชั้นต้องสมดุล             |   |
|  | ขาดชั้นใดชั้นหนึ่ง Funnel รั่ว!  |   |
|  +----------------------------------+   |
|                                          |
+==========================================+
```

> 3 ชั้น 1 เป้าหมาย: พาลูกค้าจากคนแปลกหน้าสู่ลูกค้าจริง
