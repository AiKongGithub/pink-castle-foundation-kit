## SLIDE 5: ปรับ Content Area — Typography
```
╔══════════════════════════════════════════════════╗
║   📝 Content Area — Typography                   ║
╠══════════════════════════════════════════════════╣
║                                                  ║
║   Font Settings:                                 ║
║   ┌─────────────────────────────────────┐        ║
║   │ H1 (Title)    → 32-40px, Bold      │        ║
║   │ H2 (Section)  → 24-28px, Bold      │        ║
║   │ H3 (Sub)      → 20-22px, Semi-bold │        ║
║   │ Body          → 16-18px, Regular   │        ║
║   │ Line Height   → 1.6-1.8            │        ║
║   └─────────────────────────────────────┘        ║
║                                                  ║
║   ⚠️ กฎสำคัญ:                                   ║
║   • Font เดียวกันทั้งบทความ                      ║
║   • ความกว้าง Content: 700-800px                 ║
║   • Paragraph Spacing: 20-24px                   ║
║   • สีตัวอักษร: #333 หรือ #2D2D2D (ไม่ดำสนิท)   ║
║                                                  ║
╚══════════════════════════════════════════════════╝
```

**หมายเหตุผู้บรรยาย:** Typography ที่ดีทำให้อ่านสบาย ลด Bounce Rate ได้จริง

## SLIDE 6: ปรับ Content Area — Media & Visual
```
╔══════════════════════════════════════════════════╗
║   🖼️ Content Area — Media Elements              ║
╠══════════════════════════════════════════════════╣
║                                                  ║
║   Featured Image:                                ║
║   • ขนาด: 1200×628px (OG Image ready)           ║
║   • Alt Text: ใส่ Keyword + อธิบายรูป            ║
║   • Compression: ไม่เกิน 200KB                   ║
║                                                  ║
║   In-Content Images:                             ║
║   • ทุก 300-400 คำ ใส่รูป/กราฟิก 1 รูป          ║
║   • Infographic, Screenshot, Diagram             ║
║   • Caption ใต้รูป เพิ่ม Context                  ║
║                                                  ║
║   Embedded Media:                                ║
║   • YouTube Video — ใช้ Embed Code               ║
║   • Quote Box — ไฮไลท์คำพูดสำคัญ                 ║
║   • Code Block — สำหรับบทความเทคนิค              ║
║                                                  ║
╚══════════════════════════════════════════════════╝
```

**หมายเหตุผู้บรรยาย:** Visual Elements เพิ่ม Engagement และทำให้บทความน่าอ่าน

## SLIDE 9: SEO On-page Settings
```
╔══════════════════════════════════════════════════╗
║   🔍 SEO On-page Settings                       ║
╠══════════════════════════════════════════════════╣
║                                                  ║
║   ✅ Title Tag                                    ║
║      → Keyword + Benefit (60-70 ตัวอักษร)        ║
║                                                  ║
║   ✅ Meta Description                             ║
║      → สรุป + CTA (150-160 ตัวอักษร)             ║
║                                                  ║
║   ✅ URL Slug                                     ║
║      → สั้น, มี Keyword, ใช้ขีด (-)               ║
║      → ❌ /post-12345                             ║
║      → ✅ /email-marketing-guide                  ║
║                                                  ║
║   ✅ Heading Hierarchy                            ║
║      → H1 > H2 > H3 (ไม่ข้ามลำดับ)              ║
║                                                  ║
║   ✅ Image Alt Text                               ║
║      → Keyword + อธิบายรูป                       ║
║                                                  ║
║   ✅ Internal Links                               ║
║      → 3-5 ลิงก์ไปบทความที่เกี่ยวข้อง            ║
║                                                  ║
╚══════════════════════════════════════════════════╝
```

**หมายเหตุผู้บรรยาย:** SEO ทำให้บทความถูกค้นพบบน Google ต้องตั้งค่าทุกครั้ง

## SLIDE 11: Pre-Publish Checklist
```
╔══════════════════════════════════════════════════╗
║   ✅ Pre-Publish Checklist                       ║
╠══════════════════════════════════════════════════╣
║                                                  ║
║   □ Header: Logo + Menu ถูกต้อง                  ║
║   □ Title: H1 มี Keyword                        ║
║   □ Featured Image: ขนาด + Alt Text              ║
║   □ Typography: Font/Size สม่ำเสมอ              ║
║   □ Images: ทุก 300-400 คำ + Compressed          ║
║   □ Sidebar: Email Opt-in + Popular Posts        ║
║   □ CTA: ท้ายบทความ + Lead Magnet               ║
║   □ Related Posts: 3-4 บทความ                   ║
║   □ SEO: Title Tag + Meta + URL Slug             ║
║   □ Internal Links: 3-5 ลิงก์                    ║
║   □ Mobile Preview: ทดสอบ 3 ขนาดหน้าจอ          ║
║   □ Load Speed: รูปไม่เกิน 200KB ต่อรูป          ║
║                                                  ║
║   ผ่านครบ 12 ข้อ → Publish ได้!                  ║
║                                                  ║
╚══════════════════════════════════════════════════╝
```

**หมายเหตุผู้บรรยาย:** ใช้ Checklist นี้ทุกครั้งก่อน Publish ลดข้อผิดพลาดได้ 90%
