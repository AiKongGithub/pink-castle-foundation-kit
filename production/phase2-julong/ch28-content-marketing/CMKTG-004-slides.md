## SLIDE 11: เครื่องมือวัดผล
```
+==========================================+
|         เครื่องมือวัดผล TOFU & MOFU      |
+==========================================+
|                                          |
|   วัดผล TOFU:                            |

|   * Google Search Console — keyword rank |
|   * Social Media Analytics — reach       |
|                                          |
|   วัดผล MOFU:                            |

|   +----------------------------------+   |
|   | UTM Parameters                   |   |

|   | &utm_campaign=tofu_blog01        |   |
|   |                                  |   |
|   | = รู้ว่า traffic มาจากไหน        |   |
|   +----------------------------------+   |
|                                          |
|   + Content Scoring                      |
|   = ให้คะแนนทุกชิ้นเนื้อหาตามผลลัพธ์    |
|                                          |
+==========================================+
```
