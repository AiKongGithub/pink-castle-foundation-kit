## SLIDE 7: Lead Magnet คืออะไร?
```
╔══════════════════════════════════════════╗
║           Lead Magnet                   ║
║   ของมีค่าที่แลกกับข้อมูลติดต่อ         ║
╠══════════════════════════════════════════╣
║                                          ║
║   ┌─────────────┐    ┌─────────────┐    ║
║   │             │    │             │    ║
║   │   E-book    │ ⇄  │   อีเมล     │    ║
║   │   Webinar   │    │   เบอร์โทร  │    ║
║   │   Template  │    │   ชื่อ       │    ║
║   │             │    │             │    ║
║   └─────────────┘    └─────────────┘    ║
║    (เราให้ฟรี)        (เขาให้เรา)        ║
║                                          ║
║   พอได้ข้อมูลแล้ว → Nurture ด้วย Email  ║
║   ส่งเนื้อหาดีๆ ต่อเนื่อง → สร้างความไว้ใจ ║
║                                          ║
╚══════════════════════════════════════════╝
```

> Lead Magnet ต้องเป็นสิ่งที่กลุ่มเป้าหมายต้องการจริงๆ ไม่ใช่แจกมั่ว
