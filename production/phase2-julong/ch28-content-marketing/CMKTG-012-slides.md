## SLIDE 4: Layout — Grid vs List
```
╔══════════════════════════════════════════╗
║   Layout: Grid vs List                   ║
╚══════════════════════════════════════════╝
```

| ด้าน | Grid Layout | List Layout |
|------|-------------|-------------|
| รูปแบบ | การ์ดเรียงเป็นตาราง | แถวจากบนลงล่าง |
| จุดเน้น | ภาพเป็นหลัก | เนื้อหาเป็นหลัก |
| เหมาะกับ | ท่องเที่ยว, อาหาร, ออกแบบ | การตลาด, การเงิน, เทคนิค |
| ความรู้สึก | ทันสมัย เหมือน Pinterest | เป็นทางการ จัดระเบียบ |
| Preview | เห็นภาพรวมได้เยอะ | เห็นรายละเอียดแต่ละชิ้น |

**กฎเลือก**: ดูรูปเยอะ → Grid | อ่านเยอะ → List | ผสมได้

## SLIDE 5: Pagination — แบ่งหน้าอย่างฉลาด
```
╔══════════════════════════════════════════╗
║   Pagination: จัดการบทความจำนวนมาก      ║
╚══════════════════════════════════════════╝
```

**ทำไมต้อง Pagination?**
- บทความ 50+ ชิ้นในหน้าเดียว = โหลดช้า = คนหนี
- แนะนำ **6-12 บทความต่อหน้า**

| แบบ | วิธีทำงาน | ข้อดี | ข้อเสีย |
|-----|----------|-------|---------|
| Numbered (1,2,3...) | เปลี่ยนหน้า | SEO ดีกว่า, Google index ได้ | ต้องรีโหลดหน้า |
| Load More Button | เพิ่มในหน้าเดิม | UX ดีกว่าบนมือถือ | SEO ยากกว่า |

**คำแนะนำ**: ใช้ Load More สำหรับผู้ใช้ + Numbered สำหรับ SEO (ถ้าเลือกได้ทั้งคู่)

## SLIDE 6: Category Filter — กรองง่าย หาเจอเร็ว
```
╔══════════════════════════════════════════╗
║   Category Filter: นำทางผู้อ่าน         ║
╚══════════════════════════════════════════╝
```

**หลักการสร้าง Category:**
- จำกัด **5-8 หมวดหมู่** (มากกว่านี้ = สับสน)
- ชื่อ **สั้นกระชับ** เข้าใจได้ทันที
- **ไม่ซ้ำซ้อน** — แต่ละหมวดต้องแตกต่างชัดเจน
- ผูกบทความทุกชิ้นเข้ากับหมวดหมู่

**ตัวอย่างหมวดหมู่ที่ดี:**
```
[การตลาด] [SEO] [อีเมล] [คอนเทนต์] [โซเชียล] [เครื่องมือ]
```

**ตัวอย่างที่ไม่ดี:**
```
[การตลาดออนไลน์] [ดิจิทัลมาร์เก็ตติ้ง] [โปรโมทสินค้าผ่านอินเทอร์เน็ต]
→ ซ้ำซ้อน + ชื่อยาวเกินไป
```

## SLIDE 7: Featured Post — บทความเด่นดึงสายตา
```
╔══════════════════════════════════════════╗
║   Featured Post: ปักหมุดบทความเด่น      ║
╚══════════════════════════════════════════╝
```

**Featured Post คืออะไร?**
- บทความที่ปักหมุดให้อยู่ **ด้านบนสุด** ของรายการ
- โดดเด่นกว่าบทความอื่นๆ ในรายการ

**วิธีทำให้โดดเด่น:**
```
┌─────────────────────────────────────┐
│  ★ แนะนำ                           │
│  ┌─────────────────────────────────┐│
│  │    [รูปภาพขนาดใหญ่]            ││
│  │    ชื่อเรื่อง (ตัวใหญ่)         ││
│  │    คำอธิบายยาวกว่าปกติ...       ││
│  │    [อ่านเพิ่มเติม →]            ││
│  └─────────────────────────────────┘│
└─────────────────────────────────────┘
```

- ใช้สำหรับ: บทความใหม่, ยอดนิยม, กำลังโปรโมท
- **สำคัญ**: อัปเดตสม่ำเสมอ อย่าให้บทความเดิมค้างนานเกินเดือน

## SLIDE 8: Sidebar Widgets — เครื่องมือด้านข้าง
```
╔══════════════════════════════════════════╗
║   Sidebar Widgets: นำทาง + เก็บ Lead    ║
╚══════════════════════════════════════════╝
```

**Widget 5 อย่างที่แนะนำ (เรียงตามสำคัญ):**

| ลำดับ | Widget | หน้าที่ |
|-------|--------|---------|
| 1 | Search Bar | ค้นหาบทความที่ต้องการทันที |
| 2 | Email Opt-in Form | เก็บ Lead — สำคัญที่สุดสำหรับธุรกิจ |
| 3 | Category List | ดูหมวดหมู่ทั้งหมดได้ง่าย |
| 4 | Recent Posts | แสดงบทความล่าสุด 3-5 ชิ้น |
| 5 | Social Follow Buttons | เพิ่ม Followers บนโซเชียล |

**กฎ**: จำกัด **3-5 Widget** | Widget สำคัญที่สุด **อยู่บนสุด** | อย่ารก

## SLIDE 9: SEO Settings — ทำให้ Google เจอ
```
╔══════════════════════════════════════════╗
║   SEO: หน้า Blog List ก็ต้อง SEO       ║
╚══════════════════════════════════════════╝
```

**5 จุดที่ต้องตั้งค่า:**

| จุด | ตัวอย่าง | ผลลัพธ์ |
|-----|---------|---------|
| Title Tag | "บทความการตลาด - Pink Castle Blog" | ชื่อที่แสดงบน Google |
| Meta Description | "รวมบทความ Content Marketing, SEO, Email..." | คำอธิบายใต้ชื่อบน Google |
| URL | yoursite.com/blog | สั้น สื่อความหมาย |
| Open Graph Image | รูป 1200x630px ของบล็อก | รูปเมื่อแชร์บนโซเชียล |
| Schema Markup | Blog Posting Schema | Rich Snippet บน Google |

**Schema Markup ช่วยให้**: Google แสดงผลเป็น Rich Snippet → CTR สูงขึ้น

## SLIDE 10: Responsive Design — ดูดีทุกจอ
```
╔══════════════════════════════════════════╗
║   Responsive: 60-70% ใช้มือถือ         ║
╚══════════════════════════════════════════╝
```

**Checklist ตรวจสอบ 3 ขนาด:**

| อุปกรณ์ | สิ่งที่ต้องตรวจ | ผ่าน? |
|---------|----------------|:---:|
| Desktop | Layout แสดงตามที่ออกแบบ, Sidebar อยู่ถูกที่ | [ ] |
| Tablet | Grid ลดคอลัมน์, ปุ่มกดง่าย | [ ] |
| Mobile | Grid เป็น 1 คอลัมน์, ตัวอักษรอ่านชัด, Category Filter สะดวก | [ ] |

**ข้อควรระวัง:**
- Grid 3 คอลัมน์บน Desktop → 2 บน Tablet → 1 บน Mobile
- Sidebar อาจซ่อนหรือย้ายไปด้านล่างบนมือถือ
- ปุ่มขนาดขั้นต่ำ 44x44px สำหรับนิ้วกด

## SLIDE 11: เชื่อม Blog กับ Funnel
```
╔══════════════════════════════════════════╗
║   Blog + Funnel: จุดเด่นของ Kartra      ║
╚══════════════════════════════════════════╝
```

**เส้นทางผู้อ่าน → ลูกค้า:**
```
Blog Post List → อ่านบทความ → เห็น CTA → คลิก
     ↓
Opt-in Page (Funnel) → กรอกอีเมล → ได้ Lead Magnet
     ↓
Email Sequence → Nurture → เสนอขาย → ลูกค้า
```

**BAM (Behavioral Adaptive Marketing):**
- ระบบจำพฤติกรรมผู้อ่าน
- แสดง CTA ที่ตรงกับความสนใจ
- คนอ่าน SEO บ่อย → เห็น CTA เรื่อง SEO
- คนอ่าน Email Marketing → เห็น CTA เรื่อง Email

**ทุกอย่างเชื่อมกันใน Kartra แพลตฟอร์มเดียว**
