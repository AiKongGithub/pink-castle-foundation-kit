## Insights Table

| # | Insight | รายละเอียด | ระดับสำคัญ | การนำไปใช้ |
|---|---------|-----------|-----------|-----------|
| 1 | Content Builder คือกระดาษเปล่าที่มีเครื่องมือครบ | Kartra Content Builder ใช้ระบบ Drag & Drop ลาก Components จาก Panel มาวางบน Canvas ทำให้สร้างบทความได้โดยไม่ต้องเขียนโค้ด แม้ไม่มีพื้นฐานเว็บก็ทำได้ | สูงมาก | เข้าถึงผ่าน My Content → New Post แล้วเริ่มลาก Components ได้ทันที |
| 2 | Template เริ่มต้นต้องเลือกให้เหมาะ | Kartra มี Template สำเร็จรูปหลายแบบ (Minimal, Sidebar, Full Width, Hero) เลือกแบบใกล้เคียงแล้วปรับแต่งทีหลัง สำหรับมือใหม่แนะนำแบบเรียบง่าย | สูง | เลือก Template เรียบง่ายก่อน แล้วค่อยเพิ่ม Component ทีละตัว |
| 3 | Text Component คือหัวใจของบทความ | ใช้สร้างทุกอย่างที่เป็นข้อความ ทั้ง Heading H1-H4, ย่อหน้า, Bullet Point, Numbered List สิ่งสำคัญคือ Heading Hierarchy ต้องถูกลำดับเพราะช่วย SEO | สูงมาก | H1 มีแค่ 1 ตัว (ชื่อบทความ), H2 หัวข้อหลัก, H3 หัวข้อย่อย ห้ามข้ามลำดับ |
| 4 | รูปภาพทุกรูปต้องมี Alt Text | Image Component ใส่รูปภาพประกอบบทความ Alt Text ช่วย SEO ให้ Google เข้าใจว่ารูปเกี่ยวกับอะไร และช่วยผู้ใช้ Screen Reader สามารถอัปโหลดจากเครื่องหรือใส่ URL ก็ได้ | สูง | อัปโหลดรูป + ใส่ Alt Text ที่อธิบายรูปภาพทุกครั้ง ห้ามเว้นว่าง |
| 5 | Video เพิ่ม Time on Page อย่างมาก | Video Component Embed วิดีโอจาก YouTube/Vimeo หรืออัปโหลดตรง Time on Page สูงทำให้ Google เห็นว่าผู้อ่านสนใจ ถ้ามีวิดีโอบน YouTube อยู่แล้วให้ Embed ดีกว่าอัปโหลดซ้ำ | สูง | Embed YouTube = ประหยัด Storage + ได้ View, Upload ตรง = วิดีโอเฉพาะสมาชิก |
| 6 | ปุ่ม CTA ต้อง Action-Oriented | Button Component นำพาผู้อ่านไปทำ Action ข้อความบนปุ่มต้องบอกว่ากดแล้วได้อะไร เช่น "ดาวน์โหลดฟรี" ดีกว่า "คลิกที่นี่" สีตัดกับพื้นหลังให้เห็นชัด | ปานกลาง | เขียนข้อความ Action-Oriented + เลือกสีปุ่มที่ตัดกับพื้นหลัง |
| 7 | Typography ที่ดีทำให้บทความดูเป็นมืออาชีพ | Body Text 16-18px, Line Height 1.5-1.8 เท่า, Font Style สม่ำเสมอตลอดบทความ อย่าเปลี่ยน Font ไปมา ความสม่ำเสมอสำคัญที่สุด | สูง | กำหนด Font Setting ตั้งแต่ต้น แล้วใช้เหมือนกันตลอดบทความ |
| 8 | Internal Links กระจาย Link Juice ทั่วบล็อก | ลิงก์ภายในบทความที่เชื่อมไปยังบทความอื่นในบล็อก ช่วยผู้อ่านค้นพบเนื้อหา เพิ่ม Pages per Session และกระจายพลัง SEO ให้ทุกหน้า Anchor Text ต้องสื่อความหมาย | สูง | ใส่ 3-5 Internal Links ต่อบทความ Anchor Text อธิบายว่าคลิกไปเจออะไร |
| 9 | SEO On-page 3 จุดขาดไม่ได้ | Title (ไม่เกิน 60 ตัวอักษร มี Keyword), Meta Description (150-160 ตัวอักษร มี Keyword + CTA), URL Slug (สั้น ภาษาอังกฤษ คั่นด้วยขีด) ทั้ง 3 จุดต้องตั้งค่าทุกบทความ | สูงมาก | ตั้งค่า Title + Meta + URL Slug ให้ครบก่อนกด Publish ทุกครั้ง |
| 10 | Preview ก่อน Publish เสมอ | ตรวจสอบ 3 ขนาดจอ (Desktop, Tablet, Mobile) ดูรูปภาพ ลิงก์ ปุ่ม Typography อ่านทั้งบทความจากต้นจนจบอย่างน้อยหนึ่งรอบ URL Slug ห้ามเปลี่ยนหลัง Publish เพราะจะเกิด Broken Links | สูงมาก | Preview 3 ขนาดจอ → อ่านจบ 1 รอบ → ตรวจ Checklist → กด Publish |
