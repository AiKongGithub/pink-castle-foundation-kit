## Card 1

**Q:** ทำไมความเร็วในการโหลดเว็บไซต์ถึงสำคัญ?

**A:** เพราะส่งผลต่อ 2 เรื่องหลัก คือ (1) ประสบการณ์ผู้ใช้ — ถ้าเว็บโหลดนานเกิน 3 วินาที ผู้เยี่ยมชมกว่าครึ่งจะกดปิดทันที และ (2) SEO — Google ใช้ความเร็วเป็นปัจจัยจัดอันดับ เว็บช้าจะถูกลดอันดับในผลการค้นหา

---

## Card 2

**Q:** WP-Rocket คืออะไร?

**A:** WP-Rocket เป็น Premium Caching Plugin สำหรับ WordPress ที่ช่วยเพิ่มความเร็วเว็บไซต์ด้วยฟีเจอร์หลายด้าน ได้แก่ Page Caching, Browser Caching, GZIP Compression, Lazy Loading, Minify CSS/JS และ Database Optimization

---

## Card 3

**Q:** Page Caching คืออะไร และทำงานอย่างไร?

**A:** Page Caching คือการสร้างสำเนา (Static Version) ของหน้าเว็บไว้ล่วงหน้า เมื่อผู้เยี่ยมชมเข้ามา เว็บจะแสดงสำเนาที่เตรียมไว้แล้ว แทนที่จะประมวลผลใหม่ทุกครั้ง ทำให้โหลดเร็วขึ้นอย่างมาก

---

## Card 4

**Q:** Lazy Loading คืออะไร? ช่วยเรื่องความเร็วเว็บได้อย่างไร?

**A:** Lazy Loading คือเทคนิคที่โหลดรูปภาพเฉพาะตอนที่ผู้ใช้เลื่อนหน้าจอลงไปถึงตำแหน่งนั้นเท่านั้น แทนที่จะโหลดรูปทั้งหมดพร้อมกันตั้งแต่เปิดหน้าเว็บ ช่วยลดเวลาโหลดหน้าแรกลงอย่างมาก

---

## Card 5

**Q:** GZIP Compression ทำหน้าที่อะไร?

**A:** GZIP Compression ทำหน้าที่บีบอัดไฟล์ของเว็บไซต์ (HTML, CSS, JavaScript) ให้มีขนาดเล็กลงก่อนส่งไปยังเบราว์เซอร์ของผู้ใช้ ช่วยลดปริมาณข้อมูลที่ต้องดาวน์โหลด ทำให้โหลดเร็วขึ้น

---

## Card 6

**Q:** Minify CSS/JS หมายถึงอะไร? มีข้อควรระวังอะไรบ้าง?

**A:** Minify คือการลดขนาดไฟล์ CSS และ JavaScript โดยตัดส่วนที่ไม่จำเป็นออก เช่น ช่องว่าง, ข้อคิดเห็น (comments), การเว้นบรรทัด ข้อควรระวังคืออาจทำให้ CSS หรือ JS บางตัวทำงานผิดปกติ ต้องทดสอบหลังเปิดใช้งานทุกครั้ง

---

## Card 7

**Q:** Database Optimization ใน WP-Rocket ทำอะไรได้บ้าง?

**A:** ล้างข้อมูลที่ไม่จำเป็นออกจากฐานข้อมูล WordPress เช่น Post Revisions ที่สะสมไว้เยอะ, Spam Comments, Transient Options และข้อมูลขยะอื่นๆ ช่วยให้ฐานข้อมูลเบาขึ้นและเว็บทำงานเร็วขึ้น

---

## Card 8

**Q:** WP-Rocket แตกต่างจาก W3 Total Cache อย่างไร?

**A:** W3 Total Cache เป็น Plugin ฟรีที่มีฟีเจอร์เยอะ แต่ตั้งค่าซับซ้อนมากจนมือใหม่อาจสับสน ส่วน WP-Rocket เป็น Premium Plugin ที่ใช้งานง่ายมาก ทำงานได้ทันทีหลังเปิดใช้งาน และมีฟีเจอร์เพิ่มเติมเช่น Database Optimization และ Lazy Loading ในตัว

---

## Card 9

**Q:** LiteSpeed Cache มีข้อจำกัดอะไร เมื่อเทียบกับ WP-Rocket?

**A:** LiteSpeed Cache เป็น Plugin ฟรีที่มีฟีเจอร์ดี แต่ต้องใช้กับ LiteSpeed Web Server เท่านั้นจึงจะใช้งานได้เต็มประสิทธิภาพ ถ้า Hosting ไม่ใช่ LiteSpeed จะใช้ฟีเจอร์ได้ไม่ครบ ส่วน WP-Rocket รองรับทุก Hosting Provider

---

## Card 10

**Q:** ขั้นตอนการติดตั้ง WP-Rocket มีอะไรบ้าง?

**A:** (1) ซื้อ License จาก wp-rocket.me และดาวน์โหลดไฟล์ Plugin (2) เข้า WordPress Admin ไปที่ Plugins > Add New > Upload Plugin (3) อัปโหลดไฟล์ zip และกด Install Now (4) กด Activate (5) เข้าไปปรับแต่งเพิ่มเติมที่ Settings > WP Rocket

---

## Card 11

**Q:** ควรทดสอบความเร็วเว็บไซต์ด้วยเครื่องมืออะไร?

**A:** เครื่องมือที่แนะนำคือ Google PageSpeed Insights และ GTmetrix โดยควรทดสอบทั้งก่อนและหลังติดตั้ง WP-Rocket เพื่อเปรียบเทียบผลลัพธ์อย่างชัดเจน

---

## Card 12

**Q:** นอกจากการติดตั้ง Caching Plugin แล้ว ต้องทำอะไรอีกเพื่อเพิ่มความเร็วเว็บไซต์?

**A:** ต้องดูแลเรื่องอื่นควบคู่ด้วย ได้แก่ (1) ลดขนาดรูปภาพให้เหมาะสม (2) เลือก Hosting ที่มีคุณภาพ (3) จำกัดจำนวน Plugin ที่ใช้ให้น้อยที่สุดเท่าที่จำเป็น เพราะ Plugin มากเกินไปจะทำให้เว็บช้าลง
