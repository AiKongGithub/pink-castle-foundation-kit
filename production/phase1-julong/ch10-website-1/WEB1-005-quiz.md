### ข้อ 1: (Multiple Choice) เว็บโฮสติ้ง (Web Hosting) คืออะไร?

A) บริการสร้างเว็บไซต์อัตโนมัติ
B) บริการให้เช่าพื้นที่บนเซิร์ฟเวอร์สำหรับเก็บไฟล์เว็บไซต์
C) โปรแกรมสำหรับออกแบบหน้าเว็บ
D) บริการจดทะเบียนชื่อเว็บไซต์

### ข้อ 2: (Multiple Choice) Shared Hosting เปรียบเสมือนอะไร?

A) บ้านเดี่ยว — ได้ใช้ทั้งหลัง
B) หอพักรวม — ใช้ทรัพยากรร่วมกับคนอื่น

D) โรงแรม — เช่าห้องชั่วคราว

### ข้อ 3: (True/False) Cloud Hosting กระจายข้อมูลอยู่บนเซิร์ฟเวอร์ตัวเดียว ถ้าตัวนั้นล่ม เว็บก็ล่มตามไปด้วย

A) True — Cloud Hosting ใช้เซิร์ฟเวอร์ตัวเดียวเหมือน Dedicated
B) False — Cloud Hosting กระจายข้อมูลบนหลายเซิร์ฟเวอร์ ถ้าตัวหนึ่งล่ม ตัวอื่นรับช่วงต่อได้

### ข้อ 4: (Multiple Choice) ข้อใดเป็นจุดเด่นของ Hostinger สำหรับผู้ใช้คนไทย?

A) มี support ภาษาไทยตลอด 24 ชั่วโมง

D) ให้ Dedicated Server ในราคา Shared Hosting

### ข้อ 5: (Multiple Choice) Uptime ของโฮสติ้งควรมีค่าอย่างน้อยเท่าไหร่?

B) 95%
C) 99%
D) 99.9%

### ข้อ 6: (True/False) ทุกเว็บไซต์ในยุคปัจจุบันควรมี SSL Certificate เพราะ Google ให้อันดับค้นหาที่ดีกว่ากับเว็บที่มี SSL

A) True — SSL เป็นปัจจัยด้าน SEO และสร้างความน่าเชื่อถือ
B) False — SSL จำเป็นเฉพาะเว็บที่รับข้อมูลบัตรเครดิตเท่านั้น

### ข้อ 7: (Multiple Choice) SSD Storage ดีกว่า HDD Storage อย่างไรในเรื่องโฮสติ้ง?

A) ถูกกว่า HDD
B) เก็บข้อมูลได้มากกว่า HDD
C) อ่านและเขียนข้อมูลเร็วกว่า HDD หลายเท่า ทำให้เว็บโหลดเร็วขึ้น
D) ปลอดภัยกว่า HDD เพราะเข้ารหัสข้อมูลอัตโนมัติ

### ข้อ 8: (True/False) การซื้อโฮสติ้งสัญญาระยะยาว (2-3 ปี) มักจะได้ราคาต่อเดือนที่ถูกกว่าการซื้อรายเดือน

A) True — ผู้ให้บริการให้ส่วนลดเมื่อซื้อระยะยาว
B) False — ราคาเท่ากันไม่ว่าจะซื้อสั้นหรือยาว

### ข้อ 9: (Multiple Choice) ตำแหน่งเซิร์ฟเวอร์ที่เหมาะสมที่สุดสำหรับเว็บไซต์ที่มีกลุ่มเป้าหมายเป็นคนไทยคือที่ใด?

A) สหรัฐอเมริกา
B) สิงคโปร์หรือญี่ปุ่น
C) สหราชอาณาจักร
D) ออสเตรเลีย

### ข้อ 10: (Fill-in-Blank) หลังจากซื้อโฮสติ้งเรียบร้อย ต้องนำค่า __________ จากอีเมลของโฮสติ้งไปตั้งค่าที่ผู้จดโดเมน เพื่อเชื่อมโดเมนกับโฮสติ้ง

| ข้อ | ประเภท | คำตอบ | คำอธิบายสั้น |

| 1 | MC | **B** | โฮสติ้ง = บริการให้เช่าพื้นที่เซิร์ฟเวอร์เก็บไฟล์เว็บ |
| 2 | MC | **B** | Shared Hosting = หอพักรวม ใช้ทรัพยากรร่วมกัน |

| 5 | MC | **D** | Uptime ต้อง 99.9% ขึ้นไปเป็นมาตรฐาน |
| 6 | T/F | **A (True)** | SSL เป็นปัจจัย SEO และสร้างความน่าเชื่อถือ |
| 7 | MC | **C** | SSD เร็วกว่า HDD หลายเท่า ทำให้เว็บโหลดเร็ว |
| 8 | T/F | **A (True)** | สัญญาระยะยาวได้ส่วนลดราคาต่อเดือน |
| 9 | MC | **B** | สิงคโปร์/ญี่ปุ่นใกล้ไทย ทำให้เว็บเร็วสำหรับคนไทย |
