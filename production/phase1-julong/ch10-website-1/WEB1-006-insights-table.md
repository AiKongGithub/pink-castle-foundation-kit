| # | Insight (ข้อค้นพบ) | Detail (รายละเอียด) | Application (การนำไปใช้) | Impact Level (ระดับผลกระทบ) |
|---|---|---|---|---|
| 1 | cPanel ทำให้จัดการโฮสติ้งง่ายโดยไม่ต้องเขียนโค้ด | เป็น GUI ที่แบ่งหมวดหมู่ชัดเจน (Files, Databases, Domains, Email, Security, Software) ใช้งานเหมือนสมาร์ทโฟน | เรียนรู้หมวดหมู่หลักของ cPanel ให้คุ้นเคย จะทำให้จัดการเว็บได้อย่างมั่นใจ | สูง |
| 2 | Softaculous ทำให้ติดตั้ง WordPress ได้ภายในไม่กี่คลิก | เป็น Auto Installer ที่สร้างฐานข้อมูล ตั้งค่าการเชื่อมต่อ และติดตั้งไฟล์ WordPress ให้อัตโนมัติ ใช้เวลารวม 8-10 นาที | ใช้ Softaculous แทนการติดตั้งแบบ manual เสมอ ประหยัดเวลาและลดโอกาสผิดพลาด | สูง |
| 3 | ช่อง In Directory กำหนดตำแหน่งติดตั้ง WordPress | ช่องว่าง = WordPress เป็นหน้าหลัก, พิมพ์ชื่อ = อยู่ในโฟลเดอร์ย่อย (เช่น /blog) ค่าเริ่มต้นมักเป็น "wp" ต้องลบออก | ลบค่าในช่อง In Directory ให้เป็นช่องว่าง ถ้าต้องการให้ WordPress เป็นเว็บไซต์หลัก | สูง |
| 4 | Admin Username ห้ามใช้ "admin" | "admin" เป็นชื่อที่แฮกเกอร์ลองเป็นอันดับแรกในการโจมตีแบบ Brute Force ใช้ชื่ออื่นที่เดาได้ยาก | ตั้ง Admin Username เป็นชื่อที่ไม่ซ้ำและจำได้ พร้อมรหัสผ่านที่แข็งแรง (ตัวเล็ก ตัวใหญ่ ตัวเลข สัญลักษณ์) | สูง |
| 5 | Table Prefix ควรเปลี่ยนจากค่าเริ่มต้น wp_ | wp_ เป็นค่าที่ทุกคนรู้ แฮกเกอร์ใช้ SQL Injection โจมตีฐานข้อมูลได้ง่ายถ้ารู้ชื่อตาราง | เปลี่ยน Table Prefix เป็นค่าอื่น เช่น pc_ หรือ mysite_ ตั้งแต่ตอนติดตั้ง เปลี่ยนทีหลังยากกว่ามาก | กลาง |
| 6 | WordPress Dashboard คือศูนย์บัญชาการเว็บไซต์ | เข้าผ่าน yourdomain.com/wp-admin มีเมนูครบทุกอย่าง (Posts, Pages, Appearance, Plugins, Media, Settings) | จดจำ URL wp-admin ไว้ และบุ๊คมาร์คในเบราว์เซอร์ จะได้เข้าถึงได้เร็ว | สูง |
| 7 | Permalink ต้องตั้งเป็น Post Name ทันที | ค่าเริ่มต้น WordPress ใช้ URL แบบ ?p=123 ซึ่งอ่านยากและไม่ดีต่อ SEO ต้องเปลี่ยนเป็น Post Name ก่อนเริ่มสร้างเนื้อหา | ไปที่ Settings > Permalinks > Post Name ทันทีหลังติดตั้ง เพราะเปลี่ยนภายหลังจะทำให้ลิงก์เก่าเสีย | สูง |
| 8 | WordPress รองรับภาษาไทยสมบูรณ์ | เลือกภาษาไทยได้ตั้งแต่ตอนติดตั้ง หรือเปลี่ยนภายหลังที่ Settings > General > Site Language | เลือกภาษาไทยเพื่อให้เมนูและอินเทอร์เฟซเป็นไทย ช่วยให้ใช้งานได้คล่องขึ้น | กลาง |
| 9 | cPanel ยังสร้างอีเมลธุรกิจได้ | สร้างอีเมลที่ใช้ชื่อโดเมนของเรา เช่น info@yourdomain.com ดูเป็นมืออาชีพมากกว่าใช้ Gmail | สร้างอีเมลธุรกิจอย่างน้อย 1 บัญชี เช่น contact@ หรือ info@ เพื่อรับติดต่อจากลูกค้า | กลาง |
| 10 | ทุกอย่างสำเร็จได้ภายใน 10 นาทีโดยไม่ต้องเขียนโค้ด | จาก cPanel สู่ WordPress พร้อมใช้งาน ใช้เวลาจริงประมาณ 8-10 นาที เครื่องมือ one-click ทำให้ทุกคนสร้างเว็บได้ | อย่ากลัวว่าจะยาก ลงมือทำเลย ทำตาม step-by-step ถ้าติดปัญหาตรงไหน ติดต่อ Support ของโฮสติ้ง | สูง |
