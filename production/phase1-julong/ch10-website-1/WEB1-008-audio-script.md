## ช่วงที่ 1: Plugin คืออะไร?

**Host A:** เริ่มจากพื้นฐานก่อนเลยครับ Plugin ก็คือส่วนเสริมหรือโปรแกรมเล็กๆ ที่เราติดตั้งเพิ่มลงไปใน WordPress เพื่อเพิ่มความสามารถให้เว็บไซต์ เปรียบเทียบง่ายๆ ก็เหมือนแอปในมือถือครับ มือถือเปล่าๆ ทำได้ไม่กี่อย่าง แต่พอลงแอปก็ทำได้หลายอย่างขึ้น Plugin ก็เหมือนกันครับ

**Host B:** เปรียบเทียบได้ดีมากค่ะ WordPress เปล่าๆ ก็เขียนบล็อกได้ จัดหน้าได้ แต่ถ้าอยากให้มีฟอร์มติดต่อ มีระบบ SEO มีการป้องกันแฮกเกอร์ มีระบบสำรองข้อมูล สิ่งเหล่านี้ต้องใช้ Plugin ทั้งนั้นค่ะ และในคลังของ WordPress มี Plugin ให้เลือกมากกว่า 60,000 ตัวเลยนะคะ

**Host A:** ถูกต้องครับ และนี่คือเหตุผลว่าทำไม Plugin ถึงสำคัญมาก มันทำให้ WordPress สามารถปรับแต่งได้ตามความต้องการของแต่ละเว็บไซต์ เว็บไซต์ร้านค้าก็ใช้ Plugin แบบหนึ่ง เว็บไซต์บล็อกก็ใช้อีกแบบหนึ่ง เว็บไซต์พอร์ตโฟลิโอก็อีกแบบ ยืดหยุ่นมากครับ

## ช่วงที่ 2: วิธีค้นหาและติดตั้ง Plugin

**Host B:** มาดูวิธีติดตั้ง Plugin กันค่ะ ง่ายมากเลย เข้าไปที่หน้า Admin ของ WordPress แล้วไปที่เมนู Plugins แล้วกด Add New ค่ะ จะเจอช่องค้นหาให้พิมพ์ชื่อ Plugin ที่ต้องการ

**Host A:** พอเจอ Plugin ที่ต้องการแล้ว กด Install Now ครับ รอแป๊บนึง แล้วกด Activate เท่านี้ก็ใช้งานได้แล้วครับ ง่ายมากจริงๆ แต่ก่อนจะกด Install สิ่งที่ควรดูคือ Rating หรือคะแนน จำนวนดาวน์โหลด และวันที่อัปเดตล่าสุด ถ้า Plugin ไม่อัปเดตมานานเป็นปี ก็ควรหลีกเลี่ยงครับ

**Host B:** จุดนี้สำคัญมากค่ะ การประเมินคุณภาพ Plugin ก่อนติดตั้งเป็นสิ่งที่ต้องทำเสมอ ดูจำนวน Active Installations ว่ามีคนใช้เยอะไหม ดู Rating ว่าได้กี่ดาว ดู Last Updated ว่าอัปเดตล่าสุดเมื่อไร และดู Tested up to ว่ารองรับ WordPress เวอร์ชันล่าสุดหรือเปล่าค่ะ

---

## ช่วงที่ 3: Plugin Security — Wordfence / iThemes

**Host A:** เข้าสู่ Plugin ที่ต้องมีตัวแรกเลยครับ คือ Plugin ด้าน Security หรือความปลอดภัย ตัวที่แนะนำคือ Wordfence Security หรือ iThemes Security ครับ สอง Plugin นี้เป็นตัวท็อปเลยในหมวดความปลอดภัย

**Host B:** Wordfence มีระบบ Firewall ป้องกันการโจมตี มีระบบสแกนมัลแวร์ มีระบบบล็อก IP ที่น่าสงสัย ส่วน iThemes Security ก็มีฟีเจอร์คล้ายกัน แต่เน้นเรื่อง Brute Force Protection คือป้องกันคนที่พยายามเดารหัสผ่านค่ะ เลือกตัวใดตัวหนึ่งก็พอนะคะ ไม่ต้องลงทั้งสองตัว

**Host A:** จุดนี้สำคัญมากครับ อย่าลง Plugin ที่ทำหน้าที่เดียวกันหลายตัว เพราะอาจจะขัดกัน (conflict) ได้ เลือกตัวหนึ่งแล้วใช้ให้เต็มที่ดีกว่าครับ

---

## ช่วงที่ 4: Plugin SEO — Yoast / RankMath

**Host B:** ตัวที่สองที่ต้องมีคือ Plugin SEO ค่ะ สองตัวยอดนิยมคือ Yoast SEO กับ RankMath ค่ะ Plugin เหล่านี้จะช่วยให้เราปรับแต่ง Title, Meta Description, Sitemap และตรวจสอบว่าเนื้อหาของเราเป็นมิตรกับ Google หรือเปล่า

**Host A:** Yoast SEO เป็นตัวที่มีผู้ใช้มากที่สุด มีมานานและเสถียรมากครับ ส่วน RankMath เป็นตัวใหม่กว่า ฟีเจอร์ฟรีเยอะกว่า มี Schema Markup มี Keyword Tracking ในเวอร์ชันฟรีด้วย ถ้าเป็นมือใหม่ผมแนะนำ Yoast ครับ ถ้าอยากได้ฟีเจอร์เยอะกว่าในเวอร์ชันฟรีก็ RankMath ครับ

## ช่วงที่ 5: Contact Form, Analytics, Backup

**Host B:** Plugin ต่อมาคือ Contact Form ค่ะ แนะนำ WPForms เพราะใช้งานง่ายมาก ลากวางได้เลย มีเทมเพลตฟอร์มให้เลือกเยอะ ใช้สร้างฟอร์มติดต่อ ฟอร์มสมัครสมาชิก หรือแม้แต่แบบสอบถามก็ได้ค่ะ

**Host A:** ตัวต่อมาคือ Analytics ครับ ถ้าเราต้องการดูสถิติว่ามีคนเข้าเว็บเท่าไร มาจากไหน ดูหน้าไหนบ้าง ก็ต้องเชื่อม Google Analytics กับ WordPress ซึ่งมี Plugin ช่วยอย่าง MonsterInsights หรือ Site Kit by Google ทำให้เราดูสถิติได้จากหน้า Dashboard ของ WordPress เลยครับ

**Host B:** และตัวสุดท้ายที่ขาดไม่ได้เลยคือ Backup Plugin ค่ะ แนะนำ UpdraftPlus ค่ะ เป็น Plugin สำรองข้อมูลที่ดีที่สุดตัวหนึ่ง ตั้งค่าให้สำรองข้อมูลอัตโนมัติได้ เช่น ทุกวัน ทุกสัปดาห์ แล้วส่งไปเก็บที่ Google Drive หรือ Dropbox ได้ด้วยค่ะ

**Host A:** เรื่อง Backup นี้สำคัญมากครับ มีคำพูดที่ว่า "ไม่มี Backup ก็เหมือนไม่มีเว็บ" เพราะถ้าเว็บถูกแฮก ข้อมูลหาย หรือเราทำอะไรผิดพลาด ถ้ามี Backup ก็กู้คืนได้ทันที ถ้าไม่มีก็ต้องทำใหม่ทั้งหมดครับ

## สรุปปิดรายการ

**Host B:** สรุปวันนี้นะคะ เราได้เรียนรู้ว่า Plugin คือส่วนเสริมที่เพิ่มความสามารถให้ WordPress ติดตั้งง่ายจากหน้า Admin และ Plugin ที่ต้องมีสำหรับทุกเว็บไซต์มี 5 ประเภท ได้แก่ Security, SEO, Contact Form, Analytics และ Backup ค่ะ

**Host A:** และอย่าลืมประเมินคุณภาพ Plugin ก่อนติดตั้งทุกครั้งนะครับ ดู Rating ดูจำนวนผู้ใช้ ดูวันอัปเดตล่าสุด ในตอนถัดไปเราจะมาพูดถึง Plugin ขั้นสูงอีก เช่น Plugin เร่งความเร็ว Plugin แชร์โซเชียล และ Page Builder ครับ อย่าลืมติดตามนะครับ สวัสดีครับ
