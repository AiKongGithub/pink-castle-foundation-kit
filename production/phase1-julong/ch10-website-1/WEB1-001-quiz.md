## เฉลย (Answer Key)

| ข้อ | คำตอบ | คำอธิบายสั้น |
|-----|-------|-------------|
| 1 | **B** | CMS = Content Management System ระบบจัดการเนื้อหา |
| 2 | **C** | WordPress ครองส่วนแบ่งตลาดมากกว่า 40% ของเว็บไซต์ทั่วโลก |
| 3 | **B** | เท็จ — WordPress เป็น Open Source ที่ฟรี ไม่มีค่าลิขสิทธิ์ |
| 4 | **C** | WooCommerce เป็นปลั๊กอินสร้างร้านค้าออนไลน์ยอดนิยมของ WordPress |
| 5 | **C** | WordPress ทำได้หลายประเภท ทั้งบล็อก เว็บธุรกิจ ร้านค้า Landing Page |
| 6 | **B** | เท็จ — กลับกัน Self-hosted เหมือนซื้อบ้าน WordPress.com เหมือนเช่าคอนโด |
| 7 | **B** | Self-hosted ติดตั้งปลั๊กอินได้ไม่จำกัดและแก้ไขโค้ดได้อย่างอิสระ |
| 8 | **C** | WordPress มีปลั๊กอินมากกว่า 60,000 ตัวให้เลือกใช้ |
| 9 | **B** | เท็จ — WordPress.com แบบฟรีมีข้อจำกัดด้านปลั๊กอินและการแก้โค้ด |
| 10 | **B** | ค่าใช้จ่ายหลักคือค่าโฮสติ้งและค่าโดเมนเนม ตัว WordPress เองฟรี |
