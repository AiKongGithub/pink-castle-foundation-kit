## เปิดรายการ

**Host A:** สวัสดีครับทุกคน กลับมาพบกันอีกครั้งกับรายการ PinkCastle Academy Podcast สองตอนที่แล้วเราเรียนเรื่องจดโดเมนและซื้อโฮสติ้งกันไปแล้ว วันนี้ถึงเวลาที่ทุกคนรอคอยครับ เราจะมาทำความรู้จักกับ cPanel และลงมือ ติดตั้ง WordPress กันจริงๆ ครับ

**Host B:** ตื่นเต้นมากเลยค่ะ ตอนนี้ผู้เรียนมีทั้งโดเมนและโฮสติ้งแล้ว เหลือแค่ติดตั้ง WordPress ก็จะมีเว็บไซต์เป็นของตัวเองแล้วค่ะ และต้องบอกเลยว่ามันง่ายมากๆ ค่ะ

---

## ช่วงที่ 1: cPanel คืออะไร

**Host A:** ก่อนจะติดตั้ง WordPress เรามาทำความรู้จักกับ cPanel กันก่อนครับ cPanel คือแผงควบคุมเว็บโฮสติ้งที่ได้รับความนิยมมากที่สุดในโลก เปรียบเสมือนห้องควบคุมของบ้านเรา ที่เราสามารถจัดการทุกอย่างเกี่ยวกับโฮสติ้งได้จากที่เดียวครับ

**Host B:** ง่ายๆ เลยค่ะ cPanel ทำให้เราไม่ต้องพิมพ์คำสั่งยากๆ ทุกอย่างเป็นแบบกราฟิก มีไอคอนให้คลิก ใช้งานง่ายเหมือนใช้สมาร์ทโฟนค่ะ ไม่ต้องมีความรู้ด้านเทคนิคเลยก็ใช้ได้ค่ะ

**Host A:** วิธีเข้า cPanel ก็ง่ายครับ โดยปกติจะเข้าผ่าน URL ของเว็บเราตามด้วย :2083 เช่น yourdomain.com:2083 หรือเข้าผ่านลิงก์ที่โฮสติ้งส่งมาทางอีเมลก็ได้ครับ จากนั้นก็ล็อกอินด้วย username และ password ที่ได้มาตอนซื้อโฮสติ้งครับ

---

## ช่วงที่ 2: หน้าตาและเมนูหลักของ cPanel

**Host B:** พอล็อกอินเข้ามาแล้ว เราจะเห็นหน้า Dashboard ของ cPanel ค่ะ มันจะแบ่งเป็นหมวดหมู่ต่างๆ เช่น หมวด Files สำหรับจัดการไฟล์ มี File Manager ให้อัปโหลดและจัดการไฟล์ได้ค่ะ

**Host A:** หมวด Databases สำหรับจัดการฐานข้อมูล MySQL ซึ่ง WordPress ต้องใช้ฐานข้อมูลในการเก็บเนื้อหา บทความ ข้อมูลผู้ใช้ ทุกอย่างครับ หมวด Domains สำหรับจัดการโดเมน เพิ่ม Subdomain หรือจัดการ DNS ก็ทำได้ที่นี่ครับ

**Host B:** หมวด Email สำหรับสร้างอีเมลที่ใช้ชื่อโดเมนของเรา เช่น info@yourdomain.com ดูเป็นมืออาชีพมากค่ะ และหมวด Security สำหรับจัดการ SSL และการรักษาความปลอดภัยต่างๆ ค่ะ

**Host A:** แต่สิ่งที่เราจะใช้มากที่สุดวันนี้คือหมวด Software ครับ เพราะตรงนี้จะมีเครื่องมือติดตั้ง WordPress แบบ one-click ที่เรียกว่า Softaculous หรือ Installatron ครับ

---

## ช่วงที่ 3: ติดตั้ง WordPress ด้วย Softaculous

**Host B:** มาถึงไฮไลท์ของวันนี้แล้วค่ะ การติดตั้ง WordPress ผ่าน Softaculous ค่ะ ขั้นตอนแรก ใน cPanel ให้มองหาไอคอน Softaculous Apps Installer หรือ WordPress ในหมวด Software แล้วคลิกเข้าไปค่ะ

**Host A:** จะเจอหน้า Softaculous ที่แสดง WordPress พร้อมปุ่ม Install ครับ กดปุ่ม Install เลย จากนั้นจะมีแบบฟอร์มให้กรอกข้อมูลครับ อย่างแรกคือ Choose Installation URL เลือก Protocol เป็น https:// ถ้ามี SSL แล้ว และเลือกโดเมนที่ต้องการติดตั้งครับ

**Host B:** ส่วน In Directory นี่สำคัญนะคะ ถ้าต้องการให้ WordPress เป็นหน้าหลักของเว็บ ให้ลบคำว่า wp ออกให้เป็นช่องว่าง แต่ถ้าต้องการติดตั้งในโฟลเดอร์ย่อย เช่น yourdomain.com/blog ก็พิมพ์ blog ลงไปค่ะ

**Host A:** ถัดมาคือตั้งค่าฐานข้อมูลครับ Softaculous จะสร้างฐานข้อมูลให้อัตโนมัติ เราแค่ตั้งชื่อ Table Prefix ซึ่งโดยปกติจะเป็น wp_ ก็ได้ครับ แต่ถ้าเรื่องความปลอดภัย แนะนำให้เปลี่ยนเป็นอย่างอื่น เช่น pc_ หรืออะไรก็ได้ที่จำได้ครับ

---

## ช่วงที่ 4: ตั้งค่าเริ่มต้น WordPress

**Host B:** ส่วน Site Settings ให้กรอก Site Name ชื่อเว็บไซต์ และ Site Description คำอธิบายสั้นๆ ค่ะ ไม่ต้องกังวลเรื่องนี้มาก เพราะเปลี่ยนได้ทีหลังค่ะ

**Host A:** ส่วนที่สำคัญที่สุดคือ Admin Account ครับ ตั้ง Admin Username อย่าใช้คำว่า admin นะครับ เพราะแฮกเกอร์จะลองชื่อนี้เป็นอันดับแรก ใช้ชื่ออื่นที่จำได้ ตั้ง Password ให้แข็งแรง ใช้ตัวเล็ก ตัวใหญ่ ตัวเลข และสัญลักษณ์ผสมกัน แล้วก็ใส่ Admin Email ที่ใช้งานจริงครับ

**Host B:** สุดท้ายเลือกภาษา สำหรับคนไทยเลือก Thai ได้เลยค่ะ WordPress รองรับภาษาไทยสมบูรณ์ค่ะ จากนั้นกดปุ่ม Install รอสักครู่ ระบบจะติดตั้ง WordPress ให้อัตโนมัติค่ะ

**Host A:** เสร็จแล้วจะได้ลิงก์ 2 ลิงก์ครับ ลิงก์แรกคือหน้าเว็บไซต์ของเรา เช่น yourdomain.com ลิงก์ที่สองคือหน้า Admin Dashboard เช่น yourdomain.com/wp-admin ที่เราจะใช้จัดการเนื้อหาทั้งหมดครับ

---

## ช่วงที่ 5: เข้าสู่ WordPress Admin Dashboard

**Host B:** พอคลิกเข้า wp-admin แล้ว ล็อกอินด้วย username และ password ที่ตั้งไว้ จะเจอหน้า Dashboard ของ WordPress ค่ะ ตรงนี้คือศูนย์บัญชาการของเว็บไซต์เราเลยค่ะ

**Host A:** ในหน้า Dashboard จะมีเมนูด้านซ้ายที่สำคัญ ได้แก่ Posts สำหรับเขียนบทความ Pages สำหรับสร้างหน้าเว็บ Appearance สำหรับเปลี่ยนธีมและดีไซน์ Plugins สำหรับเพิ่มฟีเจอร์ Settings สำหรับตั้งค่าทั่วไป และ Media สำหรับจัดการรูปภาพและไฟล์ครับ

**Host B:** แนะนำให้ตั้งค่าเริ่มต้นก่อนค่ะ ไปที่ Settings แล้วตั้ง Permalink Structure เป็น Post Name เพื่อให้ URL ของบทความอ่านง่ายและดีต่อ SEO ค่ะ เช่น yourdomain.com/how-to-start-business แทนที่จะเป็น yourdomain.com/?p=123 ค่ะ

**Host A:** เท่านี้ก็มีเว็บไซต์ WordPress เป็นของตัวเองแล้วครับ พร้อมที่จะเริ่มสร้างเนื้อหา ออกแบบหน้าตา และเพิ่มฟีเจอร์ต่างๆ ได้เลยครับ

## สรุปปิดรายการ

**Host B:** สรุปวันนี้นะคะ เราได้ทำความรู้จัก cPanel ที่เป็นแผงควบคุมโฮสติ้ง ติดตั้ง WordPress ด้วย Softaculous แบบ one-click ตั้งค่าฐานข้อมูล ตั้งชื่อเว็บ สร้าง Admin Account และล็อกอินเข้า Dashboard สำเร็จแล้วค่ะ

**Host A:** ตอนนี้ทุกคนมีครบแล้วครับ ทั้งโดเมน โฮสติ้ง และ WordPress สิ่งที่เหลือคือการเริ่มสร้างเนื้อหาและออกแบบเว็บไซต์ให้สวยงาม ซึ่งจะเป็นเรื่องของบทเรียนถัดไปครับ

**Host B:** ยินดีด้วยค่ะ ถ้าทำตามมาจนถึงตอนนี้ คุณก็มีเว็บไซต์เป็นของตัวเองแล้วค่ะ อย่าลืมติดตามตอนถัดไปนะคะ สวัสดีค่ะ

**Host A:** ขอบคุณที่รับฟังครับ สวัสดีครับ
