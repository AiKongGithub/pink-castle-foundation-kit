## ตาราง 5: กระบวนการเรียน SWP3 (Process Table)

| ขั้นตอน | กิจกรรม | ระยะเวลาแนะนำ | ผลลัพธ์ |
|---------|---------|---------------|---------|
| 1. เตรียมตัว | ดูบทปฐมนิเทศ ตั้ง mindset | 1 วัน | เข้าใจภาพรวมและความคาดหวัง |
| 2. วางแผน | กำหนดตารางเรียน เตรียมเครื่องมือ | 1-2 วัน | มีแผนการเรียนชัดเจน |
| 3. เรียนทฤษฎี | ดูวิดีโอ จดบันทึก | ต่อเนื่อง | เข้าใจเนื้อหาแต่ละบท |
| 4. ลงมือปฏิบัติ | ทำตามที่เรียน สร้างของจริง | ต่อเนื่อง | มีผลงานจริง |
| 5. ถามตอบ | ใช้ Q&A เมื่อติดปัญหา | ตามต้องการ | แก้ปัญหาได้ ไม่หยุดชะงัก |
| 6. แลกเปลี่ยน | เข้าร่วม community | สัปดาห์ละ 2-3 ครั้ง | ได้แรงบันดาลใจ ไอเดียใหม่ |
| 7. ทบทวน | กลับมาดูบทที่ผ่านมา | เดือนละครั้ง | เสริมความเข้าใจ ปิดจุดอ่อน |

## ตาราง 8: ปัจจัยความสำเร็จ vs ปัจจัยล้มเหลว (Comparison Table)

| ปัจจัยความสำเร็จ | ปัจจัยล้มเหลว |
|-----------------|--------------|
| เรียนตามลำดับ ไม่ข้ามบท | ข้ามบทไปเรียนเฉพาะที่สนใจ |
| ลงมือทำทันทีหลังเรียนแต่ละบท | ดูวิดีโอจบแล้วไม่ทำอะไร |
| เข้าร่วม community สม่ำเสมอ | เรียนคนเดียว ไม่ถามใคร |
| ตั้งตารางเรียนและทำตามวินัย | เรียนเมื่อมีอารมณ์เท่านั้น |
| ยอมรับว่าต้องใช้เวลา 1-3 ปี | คาดหวังผลลัพธ์ภายใน 1 เดือน |
| จดบันทึกและทบทวน | ปล่อยให้ลืมแล้วไม่กลับมาดู |
| ถาม Q&A เมื่อติดปัญหา | เก็บปัญหาไว้จนหมดกำลังใจ |
| ให้เวลา 1-2 ชั่วโมง/วัน | ทุ่มทั้งวันแล้วหยุดยาว (burn out) |
