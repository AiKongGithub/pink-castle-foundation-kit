SLIDE 3: [Context]
# Affiliate Marketing คืออะไร?
> **ช่วยโปรโมทสินค้าคนอื่น → ได้คอมมิชชันเมื่อมีคนซื้อ**

- ไม่ต้องสร้างสินค้าเอง
- ไม่ต้องจัดการชำระเงินหรือส่งสินค้า
- เหมาะสำหรับ **ผู้เริ่มต้น** ที่อยากสร้างรายได้ออนไลน์
- เชื่อมโยงกับ **Tripwire** ในระบบ SWP3

SLIDE 4: [Step-by-Step]
# ขั้นตอนสมัคร Affiliate
```
1. เข้าเว็บไซต์ MLMOnlineSchool
2. หาเมนู "Affiliate Program" (footer)
3. กรอกข้อมูล: ชื่อ, อีเมล, เบอร์โทร, บัญชีธนาคาร
4. ระบุช่องทางโปรโมท
5. รอการอนุมัติ (24-48 ชม.)
6. รับอีเมลยืนยัน → ล็อกอิน Dashboard
7. คัดลอก Affiliate Link → เริ่มโปรโมท!
```

SLIDE 7: [How It Works]
# Affiliate Link ทำงานอย่างไร?
```
คุณแชร์ลิงก์ → ลูกค้าคลิก → Cookie ติดตาม
    → ลูกค้าซื้อสินค้า → ระบบบันทึกยอดขาย
        → คุณได้รับคอมมิชชัน!
```

- **Tracking Code** ฝังอยู่ในลิงก์
- ระบุว่าลูกค้ามาจาก Affiliate คนใด
- บันทึกอัตโนมัติ ไม่ต้องทำเอง

SLIDE 9: [Ethics]
# การโปรโมทอย่างมีจริยธรรม
### 5 หลักการสำคัญ

1. **เลือกสินค้าที่เชื่อมั่น** — ทดลองใช้ก่อนแนะนำ
2. **รีวิวตรงไปตรงมา** — บอกทั้งข้อดีและข้อจำกัด
3. **สร้าง Content คุณภาพ** — แนะนำอย่างเป็นธรรมชาติ
4. **เปิดเผย Affiliate Disclosure** — ทุกครั้ง!
5. **ไม่พูดเกินจริง** — อย่าให้สัญญาที่ทำไม่ได้
