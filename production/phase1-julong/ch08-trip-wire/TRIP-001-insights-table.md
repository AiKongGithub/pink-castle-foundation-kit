## ตาราง 1: องค์ประกอบหลักของ Tripwire

| องค์ประกอบ | รายละเอียด | ความสำคัญ |
|-----------|-----------|----------|
| ราคาต่ำ | $1-$47 (35-1,600 บาท) | ทำให้ตัดสินใจซื้อได้ง่าย |
| คุณค่าสูงเกินราคา | มูลค่ารับรู้สูงกว่าราคา 10x | สร้างความประทับใจและความไว้วางใจ |

| ส่งมอบทันที | ดิจิทัล ดาวน์โหลดได้เลย | ลดระยะเวลารอคอย |

## ตาราง 2: เปรียบเทียบ Tripwire กับ Lead Magnet

| เกณฑ์ | Lead Magnet | Tripwire |
|-------|-------------|----------|
| ราคา | ฟรี (แลกอีเมล) | $1-$47 |
| วัตถุประสงค์ | เก็บข้อมูลติดต่อ | เปลี่ยนเป็นลูกค้าจ่ายเงิน |
| ตำแหน่งใน Funnel | Top of Funnel (TOFU) | Middle-Top of Funnel |
| ระดับความผูกพัน | ต่ำ (ให้อีเมลเท่านั้น) | ปานกลาง-สูง (จ่ายเงินจริง) |
| คุณภาพเนื้อหา | ดี | ดีมาก (เกินราคา) |
| โอกาสซื้อ Core Offer | ต่ำ-ปานกลาง | สูง |
| ตัวอย่าง | Ebook ฟรี, Checklist | Mini-Course $7, Templates $17 |

| 1 | Lead Magnet | ฟรี | ดึงดูด Lead | Ebook, Checklist, Webinar ฟรี |

## ตาราง 5: Frontend vs Backend Profit Strategy

| ด้าน | Frontend | Backend |
|------|----------|---------|
| สินค้า | Lead Magnet + Tripwire | Core Offer + Premium |
| กำไรต่อชิ้น | น้อยมาก / ขาดทุน | สูง |
| เป้าหมายหลัก | สร้างฐานลูกค้า (Customer Acquisition) | สร้างรายได้ (Revenue Generation) |
| ระยะเวลาคืนทุน | ไม่คาดหวังคืนทุน | ระยะกลาง-ยาว |
| ตัวชี้วัด | Conversion Rate, CAC | LTV, Backend Conversion Rate |
| กลยุทธ์คล้าย | Loss Leader (ห้างลดราคาดึงคนเข้าร้าน) | Upsell, Cross-sell, Recurring |

## ตาราง 6: ตัวชี้วัดสำคัญ (KPIs) ของ Tripwire

| ตัวชี้วัด | คำอธิบาย | เกณฑ์ที่ดี |
|-----------|----------|-----------|
| Tripwire Conversion Rate | % ของ Lead ที่ซื้อ Tripwire | 5-15% |
| Backend Conversion Rate | % ของลูกค้า Tripwire ที่ซื้อ Core Offer | 10-30% |
| Customer Acquisition Cost (CAC) | ต้นทุนในการได้ลูกค้า 1 คน | ต่ำกว่า LTV |
| Customer Lifetime Value (LTV) | มูลค่าตลอดชีพของลูกค้า | สูงกว่า CAC 3x+ |
| Break-even ROAS | จุดคุ้มทุนของค่าโฆษณา | 1.0+ รวม Backend |
| Refund Rate | อัตราคืนเงิน Tripwire | ต่ำกว่า 5% |

## ตาราง 7: ข้อผิดพลาดที่พบบ่อยและวิธีแก้ไข

| ข้อผิดพลาด | ผลกระทบ | วิธีแก้ไข |
|-----------|---------|----------|
| ตั้งราคาสูงเกินไป | Conversion Rate ต่ำ | ลดราคาให้อยู่ช่วง $7-$17 |
| คุณค่าต่ำเกินไป | ลูกค้าผิดหวัง ไม่ซื้อต่อ | เพิ่มเนื้อหาให้คุณค่า 10x ของราคา |
| ไม่เชื่อมกับ Core Offer | Backend Conversion ต่ำ | ออกแบบ Tripwire ให้เป็น "ตัวอย่าง" ของ Core |
| ไม่มี Upsell ต่อ | เสียโอกาสขาย | สร้าง Thank You Page พร้อม Upsell |
| ไม่ติดตามผล | ไม่รู้ว่าได้ผลหรือไม่ | ตั้ง KPI และวัดผลทุกสัปดาห์ |
