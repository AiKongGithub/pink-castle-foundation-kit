SLIDE 3: [Definition]
# Tripwire คืออะไร?
> **ข้อเสนอราคาต่ำ ($1-$47) ที่ออกแบบมาให้ "ปฏิเสธไม่ลง"**

- เป้าหมาย: เปลี่ยนผู้สนใจ (Lead) เป็นลูกค้า (Buyer)
- ไม่ใช่เครื่องมือทำกำไร แต่เป็นเครื่องมือ **สร้างฐานลูกค้า**
- ราคาต่ำพอให้ตัดสินใจได้ทันที ไม่ต้องคิดมาก

---

SLIDE 4: [Key Insight]
# หลักจิตวิทยาของ Tripwire
### "กำแพงการซื้อครั้งแรก"
- คนเราลังเลมากที่สุดตอนจ่ายเงินครั้งแรก
- เมื่อจ่ายแล้วครั้งหนึ่ง → โอกาสซื้อซ้ำเพิ่มขึ้น **หลายเท่า**
- Tripwire = ตัว "ทำลายกำแพง" นั้น

> **"ขอแค่ให้เขาหยิบกระเป๋าสตางค์ออกมาครั้งหนึ่ง"**

SLIDE 5: [Comparison]
# Tripwire vs Lead Magnet
| | Lead Magnet | Tripwire |
|---|---|---|
| **ราคา** | ฟรี | $1-$47 |
| **แลกกับ** | อีเมล/เบอร์โทร | เงินจริง |
| **สถานะ** | ผู้สนใจ (Lead) | ลูกค้า (Buyer) |
| **ความผูกพัน** | ต่ำ | ปานกลาง-สูง |
| **คุณค่า** | ดี | ดีมาก (เกินราคา) |

SLIDE 9: [Strategy]
# Frontend vs Backend Profit
### กำไรจริงอยู่ที่ Backend

| | Frontend | Backend |
|---|---|---|
| **สินค้า** | Lead Magnet + Tripwire | Core + Premium |
| **กำไร** | น้อยมาก/ขาดทุน | **กำไรหลัก** |
| **เป้าหมาย** | สร้างฐานลูกค้า | สร้างรายได้ |

SLIDE 10: [Analogy]
# Loss Leader Strategy
### เปรียบเทียบกับโลกจริง

- **ห้างสรรพสินค้า** ลดราคาสินค้าบางตัวจนขาดทุน → ดึงคนเข้าร้าน → ขายสินค้าอื่นกำไร
- **Tripwire** ขายราคาถูกจนแทบไม่มีกำไร → ดึงลูกค้าเข้าระบบ → ขาย Core/Premium กำไร

> **"ยอมเสียหน้า เพื่อชนะหลัง"**

SLIDE 12: [Best Practices]
# Tripwire ที่ดีต้องมีอะไรบ้าง?
1. **ราคาต่ำ** — ตัดสินใจได้ทันที
2. **คุณค่าสูงเกินราคา** — ประทับใจตั้งแต่ครั้งแรก
3. **เชื่อมกับ Core Offer** — เปิดทางขายถัดไป
4. **แก้ปัญหาเฉพาะจุด** — เห็นผลเร็ว
5. **ส่งมอบทันที** — ดิจิทัล ดาวน์โหลดเลย

---

SLIDE 13: [Common Mistakes]
# ข้อผิดพลาดที่ต้องหลีกเลี่ยง
- **ราคาสูงเกินไป** → Conversion ต่ำ
- **คุณค่าต่ำเกินไป** → ลูกค้าผิดหวัง ไม่ซื้อต่อ
- **ไม่เชื่อมกับ Core Offer** → Backend Conversion ต่ำ
- **ไม่มี Upsell** → เสียโอกาสขาย
- **ไม่วัดผล** → ไม่รู้ว่าได้ผลหรือไม่
