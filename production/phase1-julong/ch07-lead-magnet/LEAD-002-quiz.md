## เฉลยและคำอธิบาย

### ข้อ 1: **B** ✅
15-20 หน้า คือจุดที่พอดีสำหรับ E-book ที่เป็น Lead Magnet ไม่สั้นเกินจนดูไม่มีเนื้อหา (3-5 หน้า) และไม่ยาวเกินจนคนขี้เกียจอ่าน (50+ หน้า) ช่วงที่ยอมรับได้คือ 10-30 หน้า

### ข้อ 2: **B** ✅
Checklist คือรายการขั้นตอนที่ต้องทำ ติ๊กไปทีละข้อ (เช่น "15 ขั้นตอนเปิดร้านออนไลน์") ส่วน Cheat Sheet คือสรุปเนื้อหาสำคัญแบบย่อ ดูได้เร็ว (เช่น "สูตรลับ FB Ads ในหน้าเดียว") ทั้งสองใช้เวลาสร้างใกล้เคียงกัน

### ข้อ 3: **C** ✅
Checklist แนะนำสำหรับมือใหม่เพราะสร้างง่ายที่สุด ใช้เวลาแค่ 2-3 ชั่วโมง และมี Conversion Rate สูงเพราะดูแล้วไม่น่ากลัว คนตัดสินใจดาวน์โหลดง่าย

### ข้อ 4: **C** ✅
ข้อเสียหลักของ Resource List คือต้องอัปเดตเป็นระยะ เพราะลิงก์อาจหมดอายุ เครื่องมือบางอันอาจปิดตัว หรือราคาอาจเปลี่ยน Resource List ที่ล้าสมัยจะทำลายความน่าเชื่อถือ

### ข้อ 5: **B** ✅
Swipe File ต้องมาจากผลงานจริงที่ประสบความสำเร็จ ผู้ที่มีประสบการณ์และมีผลงานสะสมจึงเหมาะที่สุด มือใหม่อาจยังไม่มี Copy หรือ Template ที่พิสูจน์แล้วว่าได้ผล

### ข้อ 6: **B** ✅
Template ทรงพลังเพราะประหยัดเวลาให้คนไม่ต้องเริ่มจากศูนย์ แค่ปรับแต่งตาม Template ที่เราให้ นอกจากนี้คนจะใช้ Template บ่อยๆ ทำให้จำแบรนด์เราได้ในระยะยาว

### ข้อ 7: **ผิด** ✅
การออกแบบสำคัญมาก เพราะคนตัดสินคุณภาพจากหน้าตาก่อน ถ้าดีไซน์ดู Professional คนจะรู้สึกว่าเนื้อหาก็น่าจะดีด้วย ส่งผลต่อ Perceived Value โดยตรง

### ข้อ 8: **ถูก** ✅
Resource List ที่ดีต้องไม่ใช่แค่ลิสต์ลิงก์เฉยๆ ต้องมีคำอธิบายสั้นๆ บอกว่าเหมาะกับอะไร ราคาเท่าไหร่ ข้อดี-ข้อเสียคืออะไร ยิ่งมีรายละเอียดมากยิ่งมีคุณค่า

### ข้อ 9: **ผิด** ✅
Checklist ใช้เวลาสร้างประมาณ 2-3 ชั่วโมงเท่านั้น ขณะที่ E-book ใช้เวลา 1-2 สัปดาห์ Checklist จึงเป็นตัวเลือกที่ดีสำหรับผู้ที่ต้องการ Lead Magnet ตัวแรกอย่างรวดเร็ว

### ข้อ 10: ✅
**Checklist** → Resource List → **Swipe File** → Template → **E-book**

เรียงจากเวลาสร้างน้อยที่สุด: Checklist (2-3 ชม.) → Resource List (ครึ่งวัน-1 วัน) → Swipe File (1-2 วัน) → Template (1-3 วัน) → E-book (1-2 สัปดาห์)
