## ตาราง 2: Video Training — รายละเอียด

| หัวข้อ | รายละเอียด |
|--------|-----------|
| รูปแบบ | วิดีโอ 3-5 ตอน (ส่งทางอีเมล/แพลตฟอร์ม) |
| ตัวอย่าง | "3 ตอน สร้าง Landing Page ใน 1 ชม.", "Mini-Course 5 วัน FB Ads จากศูนย์" |
| ค่ารับรู้ | สูงที่สุด (คอร์สวิดีโอปกติขายหลักพัน) |
| จุดเด่น | Personal Connection + Repurpose ได้หลายรูปแบบ |
| ข้อเสีย | ใช้เวลาและทรัพยากรมาก แก้ไขยากถ้าล้าสมัย |
| Repurpose | คลิปสั้น Social Media / บทความ Blog / Podcast / Infographic |
| อุปกรณ์ที่ต้องมี | กล้อง ไมโครโฟน ไฟ ซอฟต์แวร์ตัดต่อ |

## ตาราง 3: Webinar / Workshop — สถิติและกลยุทธ์

| หัวข้อ | รายละเอียด |
|--------|-----------|
| ความยาว | 60-90 นาที |
| อัตราลงทะเบียน→เข้าร่วม | 30-50% |
| Conversion Rate (ขายท้าย Webinar) | 5-20% (สูงกว่า Lead Magnet อื่น) |
| เหมาะกับ | High-Ticket Sales (สินค้าราคา 5,000+ บาท) |
| สิ่งที่ต้องเตรียม | สคริปต์, Slide, ซ้อม, เทคโนโลยี, ระบบ Replay |
| เคล็ดลับ | สอนจริง 70% → เสนอขาย 30% ท้ายเซสชัน |

## ตาราง 4: Free Trial / Demo — การปรับใช้ตามธุรกิจ

| ประเภทธุรกิจ | รูปแบบ Free Trial | ตัวอย่าง | ระยะเวลา |
|-------------|-----------------|---------|---------|
| ซอฟต์แวร์ (SaaS) | ทดลองใช้ฟรี | "ใช้ฟรี 14 วัน" | 7-30 วัน |
| บริการที่ปรึกษา | ปรึกษาฟรี | "ปรึกษาฟรี 30 นาที" | 30-60 นาที |
| คอร์สออนไลน์ | เข้าถึงบทเรียนฟรี | "เรียนบทแรกฟรี" | 1-3 บท |
| สินค้าจับต้องได้ | ตัวอย่างฟรี | "รับ Sample ฟรี" | 1 ชิ้น |
| ธุรกิจบริการ | วิเคราะห์ฟรี | "วิเคราะห์ธุรกิจคุณฟรี" | 1 ครั้ง |

## ตาราง 5: Quiz / Assessment — หลักจิตวิทยาและเครื่องมือ

| หัวข้อ | รายละเอียด |
|--------|-----------|
| หลักจิตวิทยา | Self-Referencing Effect — คนรักเรื่องเกี่ยวกับตัวเอง |
| ตัวอย่าง Quiz | "คุณเป็นนักการตลาดประเภทไหน?", "ธุรกิจพร้อมขยายแค่ไหน?" |
| จำนวนคำถาม | 5-15 ข้อ (ไม่นานเกิน 5 นาที) |
| ข้อมูลที่ได้ | ความสนใจ, ปัญหา, ระดับความรู้, กลุ่มเป้าหมาย |
| เครื่องมือ | Typeform, Interact, Outgrow, Google Forms |
| การใช้ข้อมูล | แบ่งกลุ่ม (Segment) → ส่ง Email เฉพาะกลุ่ม |

## ตาราง 6: Audio vs Video vs Text — เปรียบเทียบสื่อ

| เกณฑ์ | Audio (Podcast) | Video (Training) | Text (E-book) |
|-------|----------------|-----------------|---------------|
| ความสะดวก | สูงสุด (ฟังทุกที่) | ปานกลาง (ต้องดูจอ) | ปานกลาง (ต้องอ่าน) |
| Personal Connection | สูง (เสียง = ความใกล้ชิด) | สูงที่สุด (เห็นหน้า+เสียง) | ต่ำ |
| การแสดงข้อมูลซับซ้อน | ต่ำ | สูง | สูง |
| เวลาสร้าง | ปานกลาง | สูง | ปานกลาง |
| การแก้ไข | ง่าย (อัดใหม่) | ยาก (ถ่ายใหม่) | ง่ายที่สุด |
| Shareability | ปานกลาง | สูง | ปานกลาง |

## ตาราง 8: เลือก Lead Magnet ตาม Niche

| Niche / ธุรกิจ | แนะนำ Lead Magnet | เหตุผล |
|---------------|------------------|--------|
| SaaS / ซอฟต์แวร์ | Free Trial + Webinar | ลองใช้จริง + สาธิตฟีเจอร์ |
| คอร์สออนไลน์ | Video Training + Webinar | สร้างความไว้ใจ + ขาย High-Ticket |
| E-commerce | Checklist + Quiz | Quick Win + แนะนำสินค้าเฉพาะตัว |
| ที่ปรึกษา / Coach | Free Trial (ปรึกษาฟรี) + E-book | ทดลองบริการ + แสดงความเชี่ยวชาญ |
| สื่อ / Content | Podcast + Infographic | สร้าง Community + Viral |
| Freelance | Template + Swipe File | เครื่องมือจริง + ผลงานจริง |

## Insight สำคัญ

> **ไม่จำเป็นต้องทำทุกประเภท — เลือก 1-2 แบบที่เหมาะที่สุด แล้วทำให้ดีเยี่ยม**

| Insight | รายละเอียด |
|---------|-----------|
| Video = ค่ารับรู้สูงสุด | แต่ต้องแลกกับเวลาและทรัพยากร |
| Quiz = Engagement สูงสุด | และยังเก็บข้อมูลลูกค้าได้ดีที่สุด |
| Webinar = Conversion สูงสุด | เหมาะกับ High-Ticket Sales โดยเฉพาะ |
| เลือกตาม 3 ปัจจัย | Niche + กลุ่มเป้าหมาย + ทรัพยากร |
| ทดสอบและวัดผล | ดูว่าแบบไหน Convert ดีที่สุดสำหรับธุรกิจเรา |
