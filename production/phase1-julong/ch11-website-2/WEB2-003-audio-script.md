## เปิดรายการ

**Host A:** สวัสดีครับทุกคน กลับมาพบกันอีกครั้งกับ PinkCastle Academy Podcast ในตอนนี้เราจะมาเจาะลึกเรื่องการปรับแต่ง General Settings ของ WordPress อย่างละเอียดครับ ในตอนที่แล้วเราพูดถึงการตั้งค่าพื้นฐาน แต่วันนี้จะเป็นเรื่องการปรับแต่งรูปลักษณ์ ทั้ง Logo สีสัน ฟอนต์ Header Footer และอื่นๆ ครับ

**Host B:** ใช่ค่ะ วันนี้จะเป็นเรื่องที่เห็นผลทันตาเลยค่ะ เพราะเป็นการทำให้เว็บไซต์มีบุคลิกเฉพาะตัว มีแบรนด์ของเราชัดเจน ผ่าน WordPress Customizer ที่สามารถดูตัวอย่างได้แบบ live เลยค่ะ

---

## ช่วงที่ 1: Site Identity — โลโก้และไอคอน

**Host A:** มาเริ่มจากสิ่งแรกที่ผู้เข้าชมเห็นกันก่อนเลยครับ นั่นก็คือ Site Identity ซึ่งอยู่ใน WordPress Customizer ประกอบด้วย 3 อย่างหลักครับ อย่างแรกคือ Logo คือรูปโลโก้ของเว็บไซต์ที่จะแสดงที่มุมซ้ายบนของ Header ครับ

**Host B:** อย่างที่สองคือ Site Icon หรือที่เรียกว่า Favicon ค่ะ เป็นรูปเล็กๆ ที่แสดงบนแท็บของเบราว์เซอร์ แนะนำขนาด 512x512 พิกเซลค่ะ รูปสี่เหลี่ยมจัตุรัส อาจเป็นย่อจากโลโก้ หรือตัวอักษรแรกของแบรนด์ก็ได้ค่ะ

**Host A:** อย่างที่สามคือ Site Title Display ครับ ซึ่งเราสามารถเลือกได้ว่าจะแสดงชื่อเว็บเป็นข้อความควบคู่กับโลโก้ หรือจะแสดงแค่โลโก้อย่างเดียว ถ้าโลโก้มีชื่อแบรนด์อยู่แล้วก็ไม่ต้องแสดงชื่อซ้ำครับ

---

## ช่วงที่ 2: Colors & Typography — สีและฟอนต์

**Host B:** ช่วงนี้สำคัญมากค่ะ เพราะสีและฟอนต์เป็นสิ่งที่สร้างอารมณ์และบุคลิกให้กับเว็บไซต์ ในส่วน Colors เราจะกำหนดสีหลักของแบรนด์ค่ะ ตั้งแต่สีพื้นหลัง สีข้อความ สีลิงก์ สีของปุ่ม ทุกอย่างควรสอดคล้องกับ Brand Identity ของเราค่ะ

**Host A:** ส่วนเรื่อง Typography หรือฟอนต์ ก็เป็นเรื่องที่ต้องเลือกให้ดีครับ ส่วนใหญ่จะเลือก 2 ฟอนต์ คือฟอนต์สำหรับหัวข้อ Heading และฟอนต์สำหรับเนื้อหา Body ฟอนต์หัวข้ออาจเลือกแบบที่โดดเด่นหน่อย ส่วนฟอนต์เนื้อหาต้องอ่านง่าย ไม่เล็กเกินไปครับ

**Host B:** เคล็ดลับคือ อย่าใช้เกิน 2-3 ฟอนต์ค่ะ ถ้าใช้เยอะเกินจะทำให้เว็บดูรก ไม่เป็นมืออาชีพ และยังทำให้เว็บโหลดช้าด้วยค่ะ เลือกฟอนต์จาก Google Fonts ได้เลย มีให้เลือกเป็นร้อยๆ แบบค่ะ

---

## ช่วงที่ 3: Header & Footer — ส่วนหัวและส่วนท้าย

**Host A:** มาถึงส่วน Header ครับ ซึ่งเป็นแถบด้านบนสุดของเว็บไซต์ที่แสดงทุกหน้า มีหลายเรื่องที่ปรับแต่งได้ครับ เรื่องแรกคือ Menu Placement ว่าจะวางเมนูนำทางตรงไหน ด้านข้างโลโก้ ใต้โลโก้ หรือเป็น Hamburger Menu ครับ

**Host B:** เรื่องที่สองคือ Logo Size ค่ะ ปรับขนาดให้พอดี ไม่ใหญ่เกินจนกินพื้นที่ ไม่เล็กเกินจนมองไม่เห็น และเรื่องที่สามที่น่าสนใจมากคือ Sticky Header ค่ะ คือทำให้ Header ติดอยู่ด้านบนตลอดเวลาแม้จะเลื่อนหน้าลงไป ช่วยให้ผู้ใช้เข้าถึงเมนูได้ง่ายค่ะ

**Host A:** ส่วน Footer ก็สำคัญไม่แพ้กันครับ เป็นส่วนท้ายของเว็บไซต์ที่แสดงทุกหน้าเหมือนกัน สิ่งที่ควรมีคือ Widgets ที่แสดงข้อมูลสำคัญ เช่น ข้อมูลติดต่อ เมนูลิงก์ด่วน ถัดมาคือ Copyright Text ข้อความลิขสิทธิ์ และ Social Links ลิงก์ไปยัง Social Media ของเราครับ

---

## ช่วงที่ 4: Sidebar และ Homepage Layout

**Host B:** อีกส่วนที่ปรับได้คือ Sidebar ค่ะ ซึ่งเป็นแถบด้านข้างของหน้าเว็บ สามารถเลือกได้ว่าจะมี Sidebar ด้านซ้าย ด้านขวา หรือไม่มีเลย ขึ้นอยู่กับประเภทเนื้อหาค่ะ สำหรับบทความบล็อกมี Sidebar อาจช่วยแสดงสิ่งที่เกี่ยวข้อง แต่สำหรับ Landing Page แนะนำให้ไม่มี Sidebar เลยค่ะ เพื่อไม่ให้เสียสมาธิ

**Host A:** และสุดท้ายคือ Homepage Layout Options ครับ ซึ่งกำหนดว่าหน้าแรกจะแสดงผลอย่างไร เต็มหน้าจอ หรือมีขอบ มีส่วนแนะนำ Hero Section หรือเปล่า ทั้งหมดนี้ปรับแต่งได้ผ่าน WordPress Customizer ครับ

**Host B:** จุดเด่นของ WordPress Customizer คือสามารถดูตัวอย่างแบบ Live Preview ได้เลยค่ะ คือเราปรับอะไร ก็เห็นผลทันทีที่หน้าจอด้านขวา ไม่ต้องกดบันทึกแล้วเปิดหน้าเว็บดูทุกครั้ง ประหยัดเวลามากค่ะ

---

## สรุปปิดรายการ

**Host B:** สรุปวันนี้นะคะ การปรับแต่ง General Settings ด้าน Visual ประกอบด้วย Site Identity สำหรับ Logo และ Favicon, Colors & Typography สำหรับสีและฟอนต์, Header สำหรับเมนูและ Sticky Header, Footer สำหรับข้อมูลติดต่อและ Social Links, Sidebar และ Homepage Layout ทั้งหมดทำได้ผ่าน WordPress Customizer ที่มี Live Preview ค่ะ

**Host A:** จำไว้นะครับ สีและฟอนต์ต้องสอดคล้องกับแบรนด์ อย่าใช้เกิน 2-3 ฟอนต์ และใช้ประโยชน์จาก Live Preview เพื่อดูผลก่อนบันทึก ในตอนถัดไปเราจะมาปรับแต่ง Style Layout กันครับ อย่าพลาดนะครับ

**Host B:** แล้วพบกันตอนหน้าค่ะ สวัสดีค่ะ

**Host A:** สวัสดีครับ
