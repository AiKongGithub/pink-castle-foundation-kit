## เฉลย (Answer Key)

| ข้อ | คำตอบ | คำอธิบายสั้น |
|-----|-------|-------------|
| 1 | **C** | WordPress Comment ในตัวเก็บข้อมูลทั้งหมดในฐานข้อมูลของเว็บเรา ต่างจาก Disqus/Facebook ที่ข้อมูลอยู่ภายนอก |
| 2 | **A** | Comment spam ที่มี spam links ทำให้ Google อาจลงโทษเว็บไซต์ ลดอันดับ SEO |
| 3 | **B** | Akismet ใช้ cloud-based AI กรอง spam comments อัตโนมัติ แม่นยำเกือบ 100% |
| 4 | **C** | OG tags กำหนดว่าเมื่อแชร์ลิงก์จะแสดงรูป หัวข้อ คำอธิบายอะไร |
| 5 | **B** | Social Follow = ปุ่มติดตาม Social Media ของเรา / Social Sharing = ปุ่มแชร์บทความ |
| 6 | **B** | Auto-posting คือแชร์บทความไป Social Media อัตโนมัติเมื่อ publish |
| 7 | **C** | Yoast SEO และ Rank Math จัดการ OG tags อัตโนมัติจาก Featured Image และ Meta Description |
| 8 | **C** | Social Login ให้ล็อกอินด้วย Facebook/Google ไม่ต้องสมัครใหม่ ลด friction |
| 9 | **Settings > Discussion** | การตั้งค่า Comment Moderation อยู่ที่ Settings > Discussion ในหน้า Admin |
| 10 | **A** | ถ้าไม่ตั้ง OG tags Social Media จะดึงข้อมูลแบบสุ่ม อาจแสดงรูปและข้อความที่ไม่ต้องการ |
