## เฉลย (Answer Key)

| ข้อ | คำตอบ | คำอธิบายสั้น |
|-----|-------|-------------|
| 1 | **C** | Comment spam สร้างโดย bot อัตโนมัติที่สแกนเว็บ WordPress ทั่วโลก ไม่สนเว็บใหญ่หรือเล็ก |
| 2 | **A** | Spam links ทำให้ Google อาจลงโทษเว็บ ส่งผลต่ออันดับ SEO โดยตรง |
| 3 | **B** | Akismet ทำงานแบบ cloud-based ส่งคอมเมนต์ไปตรวจกับฐานข้อมูล spam ขนาดใหญ่ |
| 4 | **C** | สำหรับเว็บส่วนบุคคล/บล็อกไม่ทำเงิน ขอ API Key ฟรีได้ที่ akismet.com |
| 5 | **B** | Akismet มากับ WordPress อยู่แล้ว แค่ Activate แล้วใส่ API Key ก็ใช้ได้ |
| 6 | **B** | False Positive = คอมเมนต์จริงถูกจับเป็น spam ถ้าลบทันทีจะสูญเสียไป |
| 7 | **B** | False Positive = คอมเมนต์จริงที่ Akismet จับว่าเป็น spam โดยผิดพลาด กด Not Spam เพื่อแก้ไข |
| 8 | **B** | สัปดาห์ละ 1 ครั้งเพียงพอ เพื่อตรวจ False Positives และล้าง spam ที่แน่ใจ |
| 9 | **Antispam Bee** | Antispam Bee ฟรี 100% ไม่ต้อง API Key เป็น privacy-friendly เหมาะกับ GDPR |
| 10 | **A** | Akismet กรอง spam ได้ทั้ง comment และ contact form (Contact Form 7, WPForms ฯลฯ) |
