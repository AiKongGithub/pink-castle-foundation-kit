| # | Insight (ข้อค้นพบ) | Detail (รายละเอียด) | Application (การนำไปใช้) | Impact Level (ระดับผลกระทบ) |
|---|---|---|---|---|
| 1 | Page Layout ต้องเลือกตามเป้าหมายของหน้า | Full-width เหมาะ Landing Page ที่ต้องการ impact สูง Boxed เหมาะเว็บข้อมูลทั่วไป Sidebar เหมาะบทความบล็อก | กำหนดเลย์เอาต์แต่ละหน้าตามวัตถุประสงค์ ไม่ใช่ใช้เลย์เอาต์เดียวกันทุกหน้า | สูง |
| 2 | Content Width 1200px เป็นค่ามาตรฐาน | กว้างเกินไปทำให้อ่านยากต้องกวาดตามาก แคบเกินไปดูอึดอัดเสียพื้นที่ 1200px สมดุลและรองรับหน้าจอส่วนใหญ่ | ตั้ง Content Width 1200px เป็นค่าเริ่มต้น ปรับเฉพาะหน้าที่ต้องการ Full-width เช่น Landing Page | กลาง |
| 3 | Line-height ส่งผลต่อความสามารถในการอ่าน | Line-height น้อยเกินทำให้ตัวอักษรเบียด มากเกินดูโปร่ง ค่าที่เหมาะสมทำให้อ่านสบายและดูเป็นมืออาชีพ | ตั้ง Body Line-height 1.5-1.7 และ Heading Line-height 1.2-1.3 ทดสอบกับเนื้อหาจริง | สูง |
| 4 | Heading H1-H6 ต้องลดหลั่นชัดเจน | ขนาดแต่ละระดับต้องแตกต่างกันอย่างเห็นได้ชัด สร้าง Visual Hierarchy ให้ผู้อ่านเข้าใจโครงสร้างเนื้อหา | กำหนดขนาดแต่ละ Heading ให้ลดลงอย่างสม่ำเสมอ เช่น H1: 36px, H2: 30px, H3: 24px ลดหลั่นลงไป | สูง |
| 5 | Color Palette ต้องมี 3 ระดับ | Primary (สีหลัก) Secondary (สีรอง) Accent (สีเน้น) ครบ 3 ระดับช่วยสร้างลำดับความสำคัญทางสายตาบนหน้าเว็บ | กำหนด Primary สำหรับส่วนหลัก Secondary สำหรับส่วนรอง Accent สำหรับ CTA buttons และจุดสำคัญ | สูง |
| 6 | Hover Effect ช่วยเพิ่ม Conversion | เมื่อเอาเมาส์ไปวางบนปุ่ม การเปลี่ยนสี/ขนาด/เงา บอกผู้ใช้ว่า "คลิกได้" ลดความสับสนและกระตุ้นการคลิก | กำหนด Hover Effect สำหรับทุกปุ่มและลิงก์ ให้มีการเปลี่ยนแปลงที่เห็นได้ชัดแต่ไม่รุนแรงเกินไป | สูง |
| 7 | Button Styles ต้องกำหนด 5 อย่าง | สีปกติ สี Hover สีขอบ ขนาด และมุมปุ่ม ทั้ง 5 อย่างต้องสม่ำเสมอทุกปุ่มทั้งเว็บ ปุ่มที่ไม่สม่ำเสมอทำให้ดูไม่เป็นระบบ | สร้างปุ่ม 2-3 แบบ (Primary Button, Secondary Button, Text Link) กำหนดสไตล์ชัดเจน ใช้ให้ตรงทั้งเว็บ | กลาง |
| 8 | Spacing มาตรฐานสร้างความเรียบร้อย | ระยะห่างที่ไม่สม่ำเสมอทำให้เว็บดูรกและไม่เป็นมืออาชีพ บางจุดห่างมากบางจุดชิดกันทำให้ดูผิดปกติ | กำหนดค่า Spacing มาตรฐาน เช่น Section spacing 60-80px, Element spacing 20-30px ใช้ให้สม่ำเสมอ | กลาง |
| 9 | Image Styles สร้างความเป็นหนึ่งเดียวให้ภาพ | รูปภาพที่มีสไตล์เดียวกัน (Border, Shadow, Aspect Ratio) ทำให้เว็บดูเป็นมืออาชีพ ต่างจากเว็บที่รูปคนละสไตล์ | เลือกสไตล์รูป 1 แบบ (เช่น มุมมน + เงาเบาๆ) และ Aspect Ratio เดียว ใช้ทั้งเว็บ | กลาง |
| 10 | Design Consistency คือสิ่งที่แยกมืออาชีพจากมือใหม่ | ทุกองค์ประกอบ (สี ฟอนต์ ปุ่ม ระยะห่าง รูปภาพ) ต้องสอดคล้องกันทุกหน้า สร้างความน่าเชื่อถือและความมั่นใจในแบรนด์ | สร้าง Style Guide บันทึกทุกค่าที่กำหนด ตรวจสอบทุกหน้าก่อนเปิดตัวว่าสอดคล้องกัน | สูงมาก |
