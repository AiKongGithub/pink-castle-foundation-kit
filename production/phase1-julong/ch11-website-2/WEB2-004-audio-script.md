## เปิดรายการ

**Host A:** สวัสดีครับทุกคน วันนี้เราจะมาปิดท้ายบทเรียนเรื่องการสร้างเว็บไซต์ Part 2 ด้วยหัวข้อที่ทำให้เว็บไซต์ของคุณดูเป็นมืออาชีพจริงๆ ครับ นั่นก็คือ Style Layout หรือการปรับแต่งสไตล์และเลย์เอาต์ครับ เป็นเรื่องของ Design System ที่ทำให้ทุกหน้าของเว็บมีความสอดคล้องกันครับ

**Host B:** ใช่ค่ะ ถ้าเปรียบเทียบ ตอนที่แล้วเราตั้งค่าสีและฟอนต์เหมือนเลือกสีทาบ้านกับแบบตัวอักษร วันนี้เราจะมาจัดห้อง วางเฟอร์นิเจอร์ ทำให้ทุกห้องในบ้านมีสไตล์เดียวกันค่ะ

## ช่วงที่ 1: Page Layout Options — รูปแบบเลย์เอาต์

**Host A:** มาเริ่มจากเรื่องพื้นฐานที่สุดก่อนครับ คือ Page Layout ว่าหน้าเว็บจะแสดงผลในรูปแบบไหน มีหลายตัวเลือกครับ ตัวแรกคือ Full-width เต็มหน้าจอ เนื้อหาจะกว้างเต็มที่ เหมาะกับ Landing Page ที่ต้องการ impact สูงครับ

**Host B:** ตัวที่สองคือ Boxed ค่ะ เนื้อหาจะอยู่ในกรอบมีขอบสองข้าง ดูเรียบร้อย เหมาะกับเว็บข้อมูลทั่วไป ตัวที่สามคือ Sidebar Left หรือ Sidebar Right ที่มีแถบด้านข้างซ้ายหรือขวาค่ะ

**Host A:** และยังมีเรื่อง Content Width ครับ ซึ่งกำหนดว่าเนื้อหาจะกว้างเท่าไร ส่วนใหญ่จะตั้งไว้ประมาณ 1200 พิกเซล ถ้ากว้างเกินไปคนอ่านจะต้องกวาดตามาก ถ้าแคบเกินไปก็ดูอึดอัด 1200 พิกเซลถือว่าเป็นค่ามาตรฐานที่ดีครับ

---

## ช่วงที่ 2: Typography System — ระบบฟอนต์

**Host B:** ต่อมาคือ Typography System ค่ะ ในตอนที่แล้วเราเลือกฟอนต์แล้ว แต่วันนี้จะเจาะลึกเรื่องขนาดและระยะห่างค่ะ Heading ตั้งแต่ H1 ถึง H6 แต่ละขนาดควรมีขนาดที่ชัดเจน H1 ใหญ่ที่สุดสำหรับหัวข้อหลัก H2 สำหรับหัวข้อรอง ลดหลั่นลงไปค่ะ

**Host A:** นอกจากขนาดแล้ว ยังมีเรื่อง Line-height ครับ คือระยะห่างระหว่างบรรทัด ถ้า Line-height น้อยเกินไป ตัวอักษรจะเบียดกัน อ่านยาก ถ้ามากเกินไปก็ดูโปร่งเกิน สำหรับเนื้อหา Body แนะนำ Line-height ประมาณ 1.5 ถึง 1.7 ครับ ส่วน Heading ประมาณ 1.2 ถึง 1.3 ครับ

**Host B:** ฟอนต์ Heading กับ Body ก็ควรเลือกให้เข้ากันค่ะ ไม่ใช่แค่สวยแยกกัน แต่ต้องดูดีด้วยกัน เคล็ดลับคือ เลือกฟอนต์ที่มีสไตล์ต่างกัน เช่น Heading เป็น Serif ส่วน Body เป็น Sans-serif จะสร้างความโดดเด่นแต่ก็กลมกลืนค่ะ

## ช่วงที่ 3: Color Palette & Button Styles — สีและปุ่ม

**Host A:** มาถึงเรื่อง Color Palette ครับ ในตอนที่แล้วเราเลือกสีหลักแล้ว วันนี้จะเป็นการจัดระบบสีให้ครบถ้วน ประกอบด้วย Primary Color สีหลักของแบรนด์ Secondary Color สีรอง และ Accent Color สีเน้นสำหรับจุดสำคัญ เช่น ปุ่ม CTA ครับ

**Host B:** นอกจากนั้นยังต้องกำหนดสีสำหรับ Button Styles ค่ะ ปุ่มเป็นองค์ประกอบที่สำคัญมากเพราะเป็นจุดที่เราอยากให้คนคลิก ต้องกำหนดสีปุ่มปกติ สีเมื่อ Hover คือเอาเมาส์ไปชี้ สีขอบ Border ขนาด และมุมของปุ่มค่ะ

**Host A:** เรื่อง Hover Effect ที่ Host B พูดถึงสำคัญมากครับ เมื่อเอาเมาส์ไปวางบนปุ่ม ปุ่มควรมีการเปลี่ยนแปลง เช่น เปลี่ยนสี เปลี่ยนขนาดเล็กน้อย หรือมีเงา สิ่งเล็กๆ แบบนี้บอกผู้ใช้ว่า "อันนี้คลิกได้" ช่วยเพิ่ม Conversion ได้เลยครับ

## ช่วงที่ 4: Spacing, Images & Design Consistency

**Host B:** สองเรื่องสุดท้ายค่ะ เรื่องแรกคือ Spacing and Padding มาตรฐาน ซึ่งเป็นระยะห่างระหว่างองค์ประกอบต่างๆ บนหน้าเว็บ ต้องกำหนดให้สม่ำเสมอค่ะ เช่น ระยะห่างระหว่างส่วนต่างๆ ของหน้า ระยะห่างระหว่างข้อความกับรูปภาพ ถ้าบางจุดห่างมาก บางจุดชิดกัน จะดูไม่เรียบร้อยค่ะ

**Host A:** เรื่องต่อมาคือ Image Styles ครับ กำหนดมาตรฐานสำหรับรูปภาพทั้งเว็บ เช่น ใส่ Border หรือไม่ มี Shadow เงาหรือเปล่า Aspect Ratio อัตราส่วนรูปภาพเป็นเท่าไร ถ้ารูปภาพบนเว็บมีสไตล์เดียวกันหมด จะดูเป็นมืออาชีพมากครับ

**Host B:** และเรื่องสุดท้ายที่เป็นหัวใจของทุกอย่างคือ Design Consistency หรือความสอดคล้องของการออกแบบทั้งเว็บไซต์ค่ะ ทุกหน้าต้องใช้สีเดียวกัน ฟอนต์เดียวกัน สไตล์ปุ่มเดียวกัน ระยะห่างเดียวกัน สไตล์รูปภาพเดียวกัน นี่คือสิ่งที่แยกเว็บมืออาชีพออกจากเว็บมือใหม่ค่ะ

## สรุปปิดรายการ

**Host B:** สรุปวันนี้นะคะ Style Layout ประกอบด้วย Page Layout เลือก Full-width หรือ Boxed, Typography System กำหนดขนาดและ Line-height, Color Palette กำหนดสีหลัก รอง และเน้น, Button Styles กำหนดสี Hover Effect ขนาด, Spacing มาตรฐาน และ Image Styles ทั้งหมดต้องสอดคล้องกันทุกหน้าค่ะ

**Host A:** ข้อคิดสำคัญคือ Design System ที่ดีทำให้ทุกหน้าของเว็บดูเป็นอันหนึ่งอันเดียวกัน ผู้เข้าชมรู้สึกมั่นใจและเชื่อถือแบรนด์มากขึ้น เป็นจบบทเรียนสร้างเว็บไซต์ Part 2 แล้วครับ หวังว่าทุกคนจะสร้างเว็บไซต์ที่สวยงามและขายได้จริงนะครับ

**Host B:** แล้วพบกันในบทเรียนถัดไปค่ะ สวัสดีค่ะ

**Host A:** สวัสดีครับ
