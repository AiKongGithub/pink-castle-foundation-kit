| # | Insight (ข้อค้นพบ) | Detail (รายละเอียด) | Application (การนำไปใช้) | Impact Level (ระดับผลกระทบ) |
|---|---|---|---|---|
| 1 | Permalink ต้องตั้งก่อนสิ่งอื่น | โครงสร้าง URL ส่งผลต่อ SEO โดยตรง การเปลี่ยนทีหลังทำให้ลิงก์เก่าพัง Google เสียอันดับ ต้องทำ Redirect ที่ซับซ้อน | เลือก Post name ทันทีหลังติดตั้ง WordPress ก่อนจะเริ่มเขียนบทความแม้แต่ชิ้นเดียว | สูงมาก |
| 2 | Site Title + Tagline แสดงใน Google | ชื่อเว็บและคำอธิบายปรากฏในผลการค้นหา เป็นสิ่งแรกที่ผู้คนเห็นก่อนคลิกเข้าเว็บ | ใส่ชื่อแบรนด์และคำอธิบายสั้นที่ชัดเจน สื่อสิ่งที่เว็บนำเสนอได้ภายใน 1-2 ประโยค | สูง |
| 3 | Timezone ผิดทำให้ดูไม่เป็นมืออาชีพ | เวลาที่แสดงในบทความจะเพี้ยนจากเวลาจริง เช่น โพสต์เช้าแต่แสดงเวลากลางคืน | ตั้ง Timezone เป็น Bangkok (UTC+7) ทันทีหลังติดตั้ง ตรวจสอบด้วยการดูเวลาในบทความทดสอบ | กลาง |
| 4 | หน้าแรกมี 2 แบบ ต้องเลือกให้เหมาะ | Latest posts เหมาะเว็บบล็อก Static page เหมาะเว็บธุรกิจ การเลือกผิดทำให้หน้าแรกไม่ตรงกับเป้าหมายเว็บ | เว็บธุรกิจ/Landing Page เลือก Static page ส่วนเว็บเนื้อหา/บล็อกเลือก Latest posts | สูง |
| 5 | Search Engine Visibility ลืมปิดเท่ากับเว็บหายจาก Google | ช่องติ๊ก Discourage search engines ทำให้ Google ไม่จัดอันดับ เว็บไม่ปรากฏในผลค้นหาเลย | ตอนพัฒนาเว็บอาจเปิดไว้ได้ แต่ต้องมี Checklist ก่อนเปิดตัวให้เอาออกทุกครั้ง | สูงมาก |
| 6 | Comment Moderation ป้องกันสแปม | เว็บ WordPress ที่ไม่กรองคอมเมนต์จะเต็มไปด้วยสแปม ลิงก์โฆษณา ทำให้เว็บดูไม่น่าเชื่อถือ | เปิด Comment Moderation ให้ตรวจสอบก่อนแสดง และพิจารณาติดตั้ง Akismet Anti-Spam เพิ่มเติม | กลาง |
| 7 | Media sizes สร้าง 3 สำเนาอัตโนมัติ | ทุกครั้งที่อัปโหลดรูป WordPress สร้าง Thumbnail (150px), Medium (300px), Large (1024px) กินพื้นที่โฮสติ้ง | ค่าเริ่มต้นใช้ได้ดี ไม่ต้องเปลี่ยน แต่ถ้าพื้นที่จำกัดให้ optimize รูปก่อนอัปโหลดด้วยเครื่องมือบีบอัด | ต่ำ |
| 8 | Default Post Category ป้องกันบทความไม่มีหมวด | หากลืมเลือกหมวดหมู่ขณะเขียนบทความ ระบบจัดเข้าหมวดเริ่มต้น (Uncategorized) ซึ่งดูไม่ดี | สร้างหมวดหมู่หลักก่อน แล้วตั้ง Default Post Category เป็นหมวดที่ใช้บ่อยที่สุด | ต่ำ |
| 9 | Post name URL ช่วย SEO 3 ทาง | (1) Google เข้าใจเนื้อหาจาก URL (2) ผู้ใช้เห็น URL แล้วรู้ว่าหน้าเกี่ยวกับอะไร (3) Click-through rate สูงขึ้นเพราะ URL น่าเชื่อถือ | ใช้ Post name permalink + ตั้งชื่อ slug เป็นภาษาอังกฤษที่สั้นและสื่อความหมาย | สูง |
| 10 | การตั้งค่าพื้นฐานต้องทำก่อนเนื้อหา | การตั้งค่าผิดตั้งแต่แรกสะสมเป็นปัญหาใหญ่ เช่น URL พัง เวลาผิด เว็บหายจาก Google การแก้ไขทีหลังเสียเวลาและทรัพยากรมาก | สร้าง Checklist การตั้งค่า 6 หมวด ทำให้ครบก่อนเริ่มเขียนบทความแม้แต่ชิ้นเดียว | สูง |
