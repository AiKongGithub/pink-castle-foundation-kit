## เปิดรายการ

**Host A:** สวัสดีครับทุกคน กลับมาพบกันอีกครั้งกับ PinkCastle Academy Podcast วันนี้เราจะมาพูดถึงปลั๊กอินตัวหนึ่งที่ถือว่าเป็นของจำเป็นสำหรับทุกเว็บ WordPress เลยครับ นั่นก็คือ Google Site Kit ปลั๊กอินทางการจาก Google ที่ช่วยเชื่อมบริการของ Google เข้ากับเว็บไซต์ของเราได้ในคลิกเดียว

**Host B:** ใช่ค่ะ ต้องบอกว่า Site Kit นี่เป็นปลั๊กอินที่ Google พัฒนาเองเลยนะคะ ไม่ใช่ของ third-party ทำให้มั่นใจได้ในเรื่องความปลอดภัยและความเข้ากันได้กับบริการต่างๆ ของ Google ไม่ว่าจะเป็น Analytics, Search Console, AdSense หรือ PageSpeed Insights เชื่อมได้หมดจากที่เดียวเลยค่ะ

---

## ช่วงที่ 1: Google Site Kit คืออะไร

**Host A:** มาเริ่มกันเลยครับ Google Site Kit คือปลั๊กอินอย่างเป็นทางการของ Google สำหรับ WordPress ซึ่งทำหน้าที่เป็นศูนย์กลางในการเชื่อมต่อบริการต่างๆ ของ Google เข้ากับเว็บไซต์ของเรา พูดง่ายๆ คือแทนที่คุณจะต้องไปตั้งค่าแต่ละบริการแยกกัน Site Kit รวมทุกอย่างมาไว้ในที่เดียวครับ

**Host B:** สิ่งที่สำคัญคือมันฟรีทั้งหมดค่ะ ไม่มีเวอร์ชัน Premium ให้ต้องจ่ายเพิ่ม และเพราะเป็นของ Google โดยตรง จึงมั่นใจได้ว่าข้อมูลที่แสดงนั้นถูกต้องแม่นยำ ไม่ต้องกังวลเรื่อง compatibility กับบริการ Google ค่ะ

**Host A:** ก่อนหน้าที่จะมี Site Kit ถ้าคุณอยากดูข้อมูล Analytics ก็ต้องเปิด Google Analytics ถ้าอยากดู Search Console ก็ต้องเปิดอีกแท็บ อยากดู PageSpeed ก็ต้องเข้าอีกเว็บ แต่ตอนนี้ Site Kit ดึงข้อมูลทุกอย่างมาแสดงบน Dashboard ของ WordPress เลยครับ สะดวกมาก

---

## ช่วงที่ 2: การติดตั้งและเชื่อมต่อ

**Host B:** มาดูขั้นตอนการติดตั้งกันค่ะ ง่ายมากเลย แค่ไปที่ Plugins > Add New แล้วค้นหา "Site Kit by Google" กดติดตั้งและ Activate ปลั๊กอินจะพาเราไปหน้า Setup Wizard ที่จะเชื่อมต่อกับ Google Account ของเราค่ะ

**Host A:** ขั้นตอน Setup เป็นแบบ One-click เลยครับ คุณแค่ Sign in ด้วย Google Account แล้ว Site Kit จะขออนุญาตเข้าถึงบริการต่างๆ ซึ่งในขั้นตอนนี้มันจะทำสิ่งสำคัญอย่างหนึ่งด้วยครับ คือการยืนยันความเป็นเจ้าของเว็บไซต์กับ Google หรือที่เรียกว่า Website Ownership Verification

**Host B:** ซึ่งปกติถ้าทำเอง การ Verify Ownership กับ Google ต้องไปใส่โค้ดพิเศษในเว็บไซต์ หรืออัปโหลดไฟล์ HTML ค่อนข้างยุ่งยากสำหรับมือใหม่ค่ะ แต่ Site Kit ทำให้อัตโนมัติ แค่คลิกอนุญาตก็เสร็จเลย ง่ายมากจริงๆ ค่ะ

**Host A:** หลังจาก Verify เสร็จ Site Kit จะเริ่มเชื่อมต่อบริการต่างๆ ให้ครับ เริ่มจาก Search Console ก่อนเป็นอันดับแรก จากนั้นก็ Analytics, AdSense และ PageSpeed Insights แต่ละตัวก็แค่กดเชื่อมต่อ ไม่ต้องไปตั้งค่าอะไรซับซ้อนเลยครับ

---

## ช่วงที่ 3: Dashboard และข้อมูลที่ได้

**Host B:** พอเชื่อมต่อเสร็จแล้ว มาดูกันค่ะว่า Dashboard ของ Site Kit แสดงข้อมูลอะไรบ้าง อย่างแรกเลยคือข้อมูลจาก Search Console ค่ะ ซึ่งจะแสดง 4 ค่าหลักๆ ได้แก่ Impressions คือจำนวนครั้งที่เว็บเราปรากฏในผลการค้นหา Clicks คือจำนวนคนที่คลิกเข้ามาจริงๆ CTR คืออัตราการคลิก และ Position คือตำแหน่งเฉลี่ยของเว็บเราในผลการค้นหาค่ะ

**Host A:** ข้อมูล 4 ตัวนี้สำคัญมากครับ ถ้า Impressions สูงแต่ Clicks ต่ำ แปลว่าคนเห็นเว็บเราแต่ไม่คลิก อาจต้องปรับ Title หรือ Meta Description ให้น่าสนใจขึ้น ถ้า Position สูงเกินไป เช่น อยู่หน้า 3 หน้า 4 ก็ต้องทำ SEO เพิ่มครับ Site Kit ช่วยให้เราเห็นภาพนี้ได้โดยไม่ต้องออกจาก WordPress เลย

**Host B:** นอกจากนี้ยังมีข้อมูล PageSpeed Insights ด้วยนะคะ ซึ่งจะวิเคราะห์ความเร็วของเว็บไซต์ทั้งบน Mobile และ Desktop บอกคะแนนให้เลย ถ้าต่ำกว่า 50 ก็ถือว่าต้องปรับปรุงด่วน 50-89 คือพอใช้ได้ และ 90-100 คือดีเยี่ยมค่ะ

## ช่วงที่ 4: Site Kit vs ติดตั้งแยก

**Host A:** หลายคนอาจสงสัยว่า แล้วทำไมไม่ติดตั้งปลั๊กอินแยกเป็นตัวๆ ไปเลย ต้องบอกว่าได้ครับ แต่ Site Kit มีข้อดีหลายอย่าง ข้อแรกคือเรื่อง Performance ครับ ปลั๊กอินตัวเดียวเบากว่าติดตั้งหลายตัว ข้อสองคือเรื่องความถูกต้องของข้อมูล เพราะเป็นของ Google โดยตรง จึงไม่มีปัญหาเรื่องข้อมูลคลาดเคลื่อน

**Host B:** ข้อที่สามคือเรื่องความสะดวกค่ะ Dashboard เดียวเห็นข้อมูลจากทุกบริการ ไม่ต้องสลับไปสลับมา และข้อที่สี่คือเรื่องการอัปเดต เพราะ Google ดูแลเอง จึงมั่นใจได้ว่าปลั๊กอินจะเข้ากับบริการ Google เวอร์ชันล่าสุดเสมอค่ะ

**Host A:** แต่ก็ต้องบอกตรงๆ ครับว่า Site Kit แสดงข้อมูลแบบสรุปภาพรวม ถ้าอยากดูข้อมูลลึกๆ ก็ต้องเข้าไปในหน้าเต็มของแต่ละบริการอยู่ดี แต่สำหรับการเช็คข้อมูลประจำวันแบบเร็วๆ Site Kit ทำได้ดีมากครับ

## สรุปปิดรายการ

**Host B:** สรุปวันนี้นะคะ Google Site Kit เป็นปลั๊กอินทางการจาก Google ที่รวมบริการ Analytics, Search Console, AdSense และ PageSpeed Insights ไว้ใน Dashboard เดียว ติดตั้งง่าย One-click Setup ช่วยยืนยันเจ้าของเว็บอัตโนมัติ และฟรีทั้งหมดค่ะ

**Host A:** ถ้าใครมีเว็บ WordPress ยังไม่ได้ติดตั้ง Site Kit แนะนำให้ทำเลยครับ เป็นปลั๊กอินตัวแรกๆ ที่ทุกเว็บควรมี ในตอนถัดไป เราจะมาเจาะลึกเรื่อง Google Analytics ว่าตั้งค่ายังไงให้ได้ข้อมูลที่เป็นประโยชน์สูงสุด อย่าพลาดนะครับ

**Host B:** แล้วพบกันตอนหน้าค่ะ สวัสดีค่ะ

**Host A:** สวัสดีครับ
