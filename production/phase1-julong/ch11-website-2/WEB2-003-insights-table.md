| 6 | Sidebar ต้องเลือกตามเป้าหมายของหน้า | บทความบล็อกมี Sidebar ช่วยแสดงเนื้อหาเกี่ยวข้อง แต่ Landing Page ไม่ควรมี Sidebar เพราะเสียสมาธิ | ตั้ง Sidebar สำหรับหน้าบล็อก (มี) และ Landing Page/Sales Page (ไม่มี) แยกกันชัดเจน | กลาง |
| 7 | Live Preview ประหยัดเวลาปรับแต่ง | WordPress Customizer แสดงผลทันทีขณะปรับ ไม่ต้องกดบันทึกแล้วเปิดหน้าเว็บดูซ้ำ ลดเวลาลอง-ผิดลอง-ถูก | ใช้ Live Preview ทุกครั้งที่ปรับแต่ง ทดสอบทั้ง Desktop และ Mobile View ก่อนกดบันทึก | กลาง |
| 8 | Menu Placement ส่งผลต่อ Navigation | ตำแหน่งเมนู (ข้างโลโก้ ใต้โลโก้ Hamburger) กำหนดว่าผู้ใช้จะนำทางได้ง่ายแค่ไหน | เว็บที่มีเมนูไม่เยอะใส่ข้างโลโก้ เว็บที่มีเมนูเยอะใช้ Dropdown หรือ Hamburger Menu | กลาง |
