## ตาราง 5: ฟีเจอร์เสริมอีเมลใน cPanel (Feature Table)

| ฟีเจอร์ | คำอธิบาย | ตัวอย่างการใช้งาน | ความสำคัญ |
|---------|---------|-----------------|----------|
| Email Forwarding | ส่งต่ออีเมลไปบัญชีอื่น | info@ ส่งต่อไป Gmail ส่วนตัว | สูง |
| Auto-Responders | ตอบกลับอัตโนมัติ | "ได้รับข้อความแล้ว จะตอบภายใน 24 ชม." | ปานกลาง |
| Email Filters | กรองอีเมลตามเงื่อนไข | อีเมลจาก VIP ไปโฟลเดอร์สำคัญ | ปานกลาง |
| Spam Filters (SpamAssassin) | กรองอีเมลขยะ | ตั้งค่าระดับความเข้มงวด | สูง |
| Mailing Lists | สร้างกลุ่มอีเมล | ส่งอีเมลหาทีมทั้งหมดพร้อมกัน | ต่ำ |
| Default Address | จัดการอีเมลที่ส่งมาไม่ตรง | อีเมลถึง typo@domain → ส่งต่อไป info@ | ต่ำ |

## ตาราง 8: ปัญหาที่พบบ่อยและวิธีแก้ไข (Troubleshooting Table)

| ปัญหา | สาเหตุ | วิธีแก้ไข |
|-------|--------|---------|
| เข้า cPanel ไม่ได้ | ลืมรหัสผ่าน / URL ผิด | ใช้ :2083 หรือติดต่อโฮสติ้ง |
| สร้างอีเมลไม่ได้ | เกินจำนวนบัญชีที่แพ็กเกจอนุญาต | อัปเกรดแพ็กเกจโฮสติ้ง |
| ส่งอีเมลไม่ได้ | รหัสผ่านผิด / เซิร์ฟเวอร์มีปัญหา | ตรวจสอบรหัสผ่าน ติดต่อโฮสติ้ง |
| อีเมลเข้า Spam | ไม่ได้ตั้ง SPF/DKIM | ตั้งค่า DNS records (เรียนในบทถัดไป) |
| พื้นที่เต็ม | Quota น้อยเกินไป | เพิ่ม Quota หรือลบอีเมลเก่า |
| Webmail ช้า | เซิร์ฟเวอร์โหลดสูง | เปลี่ยนเป็นใช้ IMAP ผ่านแอปแทน |
