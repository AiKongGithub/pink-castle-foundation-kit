## Executive Summary

Trip Wire เป็นกลยุทธ์การขายที่ใช้สินค้าราคาถูก ($7-47) เพื่อเปลี่ยน Free Subscriber ให้กลายเป็น Paying Customer ครั้งแรก โดยอาศัยหลักจิตวิทยา Commitment & Consistency เมื่อคนซื้อครั้งแรกแล้ว จะมีแนวโน้มซื้อสินค้าราคาสูงกว่าในอนาคต

**Key Insight:** คนที่ซื้อ Trip Wire แล้ว มีโอกาส 10-20% ที่จะซื้อ Core Offer เทียบกับ Cold Traffic ที่อาจแค่ 0.5-1%

---

## 1. Introduction

### 1.1 Trip Wire คืออะไร?

Trip Wire คือสินค้าหรือบริการราคาถูก (โดยทั่วไป $7-47 หรือ 200-1,500 บาท) ที่มีคุณค่าสูงมาก ออกแบบมาเพื่อ:

1. **เปลี่ยน Identity:** จาก "Free Subscriber" เป็น "Paying Customer"
2. **สร้าง Trust:** พิสูจน์ว่าสินค้าเราดี
3. **Generate Buyer's List:** รายชื่อคนที่พร้อมจ่ายเงิน

### 1.2 ตำแหน่งใน Sales Funnel

```
Lead Magnet (Free) → Trip Wire ($7-47) → Core Offer ($97-997) → High-Ticket ($1,000+)
```

Trip Wire อยู่ระหว่าง Lead Magnet และ Core Offer เป็น "สะพาน" เชื่อมระหว่างของฟรีกับของราคาสูง

### 1.3 ทำไมไม่ขาย Core Offer เลย?

| Approach | Conversion | Trust Level |
|----------|------------|-------------|
| ขาย Core เลย | 0.5-1% | Low |
| ผ่าน Trip Wire ก่อน | 10-20% | High |

เพราะ Core Offer ราคาสูง คนต้องไว้วางใจก่อนจึงจะซื้อ Trip Wire ช่วยสร้าง trust นั้น

## 2. Psychology Behind Trip Wire

### 2.1 Commitment & Consistency Principle

หลักการจากหนังสือ "Influence" ของ Robert Cialdini:
- เมื่อคนทำอะไรไปแล้ว (เช่น จ่ายเงิน) จะมีแนวโน้มทำต่อในทางเดียวกัน
- การซื้อ Trip Wire เป็น "small commitment" ที่นำไปสู่ "bigger commitment"

### 2.2 Identity Shift

| Before Trip Wire | After Trip Wire |
|------------------|-----------------|
| "ฉันแค่สมัครรับของฟรี" | "ฉันเป็นลูกค้าของเขา" |
| Passive Observer | Active Buyer |
| Low Engagement | High Engagement |

### 2.3 Loss Aversion

- คนกลัวเสียมากกว่าอยากได้
- "ถ้าไม่ซื้อตอนนี้ จะพลาดโอกาส" สร้าง urgency

## 3. Five Types of Trip Wire

### 3.1 Mini Course / Training

**รายละเอียด:**
- คอร์สสั้น 3-5 บทเรียน
- วิดีโอ หรือ text-based
- สอนทักษะเฉพาะเจาะจง

**Pricing:** $27-47 (700-1,500 บาท)

**ข้อดี:**
- Perceived value สูง
- แสดง expertise ได้ดี
- นำไปสู่ Core Course ได้ง่าย

**ตัวอย่าง:**
- "Mini Course: สร้าง Landing Page ใน 1 วัน"
- "Crash Course: Facebook Ads สำหรับมือใหม่"

---

### 3.2 Template Bundle

**รายละเอียด:**
- ชุด templates พร้อมใช้
- Excel, Canva, Notion, etc.
- Copy & paste ได้เลย

**Pricing:** $17-37 (490-990 บาท)

**ข้อดี:**
- Practical value ชัดเจน
- ประหยัดเวลาผู้ใช้มาก
- ง่ายต่อการสร้าง

**ตัวอย่าง:**
- "50 Email Templates พร้อมใช้"
- "Canva Template Pack: 100 Social Media Posts"

---

### 3.3 Toolkit / Swipe File

**รายละเอียด:**
- ชุดเครื่องมือ/ตัวอย่าง
- Headlines, Scripts, Formulas
- Reference material

**Pricing:** $7-27 (190-700 บาท)

**ข้อดี:**
- ราคาถูก impulse buy
- ใช้ได้ตลอด
- สร้างได้จาก expertise ที่มี

**ตัวอย่าง:**
- "100 Headlines ที่ขายดี"
- "Copywriting Swipe File"

---

### 3.4 eBook Premium

**รายละเอียด:**
- eBook ที่ลึกกว่า Lead Magnet
- 30-100 หน้า
- ความรู้เฉพาะทาง

**Pricing:** $7-17 (190-490 บาท)

**ข้อดี:**
- สร้างจาก content ที่มี
- ต้นทุนต่ำ
- Perceived value ดี

**ตัวอย่าง:**
- "คู่มือ A-Z สร้างธุรกิจ Dropshipping"
- "The Complete Guide to Instagram Marketing"

---

### 3.5 Workshop Recording

**รายละเอียด:**
- บันทึก workshop ที่เคยจัด
- 2-4 ชั่วโมง
- เนื้อหาคุณภาพ

**Pricing:** $17-47 (490-1,500 บาท)

**ข้อดี:**
- ใช้ content ที่มีอยู่แล้ว
- ดู professional
- High perceived value

**ตัวอย่าง:**
- "Recording: 3-Hour Marketing Workshop"
- "Full Day Masterclass: Product Launch"

## 4. One-Time Offer (OTO) Strategy

### 4.1 What is OTO?

One-Time Offer คือหน้าเว็บที่แสดงทันทีหลังจากคนสมัคร Lead Magnet มีลักษณะ:
- แสดงครั้งเดียว
- ราคาพิเศษเฉพาะหน้านี้
- สร้าง urgency

### 4.2 OTO Page Structure

```
┌─────────────────────────────────────────┐
│           WAIT! SPECIAL OFFER           │
│         "รอก่อน! ข้อเสนอพิเศษ"           │
├─────────────────────────────────────────┤
│                                         │
│  [Problem Reminder]                     │
│  คุณเพิ่งดาวน์โหลด X...                  │
│  แต่ถ้าอยากได้ผลลัพธ์เร็วขึ้น...          │
│                                         │
│  [Trip Wire Introduction]               │
│  แนะนำ [Product Name]                   │
│                                         │
│  [Value Stack]                          │
│  ✓ Item 1 - มูลค่า xxx บาท              │
│  ✓ Item 2 - มูลค่า xxx บาท              │
│  ✓ Item 3 - มูลค่า xxx บาท              │
│  รวมมูลค่า: x,xxx บาท                   │
│  วันนี้เท่านั้น: xxx บาท                 │
│                                         │
│  [Testimonials]                         │
│                                         │
│  [Urgency]                              │
│  ⏱️ ข้อเสนอนี้มีแค่หน้านี้เท่านั้น        │
│                                         │
│  [CTA Button] YES! ฉันต้องการ           │
│                                         │
│  No thanks, I'll pass                   │
│                                         │
└─────────────────────────────────────────┘
```

### 4.3 Value Stack Best Practices

**Do:**
- แสดงมูลค่าจริงของแต่ละ item
- รวมมูลค่าทั้งหมด
- เทียบกับราคาขาย
- แสดง % ที่ประหยัด

**Don't:**
- มูลค่าเกินจริง
- Item ที่ไม่มีคุณค่า
- ราคาขายสูงเกินไป

## 5. Pricing Strategy

### 5.1 Sweet Spot Pricing

| Price (THB) | Price (USD) | Psychology |
|-------------|-------------|------------|
| 190 | $7 | Impulse buy ไม่ต้องคิด |
| 490 | $17 | Value for money |
| 690 | $27 | Most popular |
| 990 | $37 | Premium feel |
| 1,490 | $47 | Max Trip Wire |

### 5.2 Pricing Tips

1. **ใช้เลข 7 หรือ 9:** $17, $27, $37 (ไม่ใช่ $20, $30)
2. **Anchor High:** "มูลค่า 6,000 บาท วันนี้แค่ 490"
3. **Compare to Daily:** "เท่ากับกาแฟ 2 แก้ว"
4. **Round Down:** 490 ดูถูกกว่า 500

## 6. Upsell & Downsell

### 6.1 Upsell (ถ้าซื้อ Trip Wire)

หลังจากซื้อ Trip Wire แล้ว เสนอ:
- **Bundle:** รวมหลาย products
- **Premium Version:** เวอร์ชั่นเต็ม
- **Done-For-You:** ทำให้แทน

**Pricing:** 2-3x Trip Wire

### 6.2 Downsell (ถ้าปฏิเสธ OTO)

ถ้าคนไม่ซื้อ OTO เสนอ:
- **Lite Version:** เวอร์ชั่นย่อ
- **Payment Plan:** แบ่งจ่าย
- **Smaller Offer:** ของที่ถูกกว่า

**Pricing:** 50-70% ของ Trip Wire

## 7. Conversion Benchmarks

### 7.1 Key Metrics

| Metric | Poor | Good | Great |
|--------|------|------|-------|
| OTO Conversion | < 5% | 8-12% | 15-20% |
| Upsell Take Rate | < 10% | 20-30% | 40%+ |
| Downsell Take Rate | < 5% | 10-15% | 20%+ |
| Trip Wire to Core | < 5% | 10-15% | 20%+ |
| Refund Rate | > 10% | 3-5% | < 3% |

### 7.2 Optimization Tips

1. **Test Headlines:** A/B test different approaches
2. **Adjust Price:** Try different price points
3. **Improve Value Stack:** Add/remove items
4. **Add Urgency:** Countdown timer
5. **Better Testimonials:** Specific results

Trip Wire เป็นเครื่องมือที่ทรงพลังในการ:
1. **เปลี่ยน Identity** จาก subscriber เป็น buyer
2. **สร้าง Trust** ก่อนขาย Core Offer
3. **Generate Revenue** เพื่อ cover ad costs
4. **Build Buyer's List** ที่มีค่ามาก
