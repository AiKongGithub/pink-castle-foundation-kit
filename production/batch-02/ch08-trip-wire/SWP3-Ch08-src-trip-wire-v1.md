## บทสรุปหลัก

### Trip Wire คืออะไร?

Trip Wire คือสินค้าราคาถูก ($7-47) ที่มีคุณค่าสูง ออกแบบมาเพื่อเปลี่ยน Free Subscriber ให้กลายเป็น Paying Customer ครั้งแรก เป็นขั้นตอนสำคัญใน Sales Funnel

**สูตร:** Lead Magnet (Free) → Trip Wire ($7-47) → Core Offer ($97-997)

---

## ทำไมต้องมี Trip Wire?

### 1. Psychology of First Purchase
- **เปลี่ยน Identity:** คนที่จ่ายเงินแล้วมีแนวโน้มจะซื้อซ้ำ
- **Commitment & Consistency:** เมื่อทำอะไรไปแล้วจะทำต่อ
- **Buyer's List:** รายชื่อคนที่จ่ายเงินแล้วมีค่ามากกว่า free list

### 2. ตัวเลขที่พูดแทน
- **Customer Acquisition Cost (CAC):** อาจ breakeven หรือกำไรเล็กน้อย
- **Lifetime Value (LTV):** คน 1 คนที่ซื้อ Trip Wire มีโอกาสซื้อ Core Offer 10-20%
- **ROI:** ได้ทุนคืนจาก ads ในขั้นตอนนี้

## 5 ประเภท Trip Wire

### 1. Mini Course / Training
- **รูปแบบ:** คอร์สสั้น 3-5 บทเรียน
- **ราคา:** $27-47
- **ตัวอย่าง:** "Mini Course: สร้าง Landing Page ใน 1 วัน"

### 2. Template Bundle
- **รูปแบบ:** ชุด templates พร้อมใช้
- **ราคา:** $17-37
- **ตัวอย่าง:** "50 Email Templates พร้อมใช้"

### 3. Toolkit / Swipe File
- **รูปแบบ:** ชุดเครื่องมือ/ตัวอย่าง
- **ราคา:** $7-27
- **ตัวอย่าง:** "Copywriting Swipe File 100 Headlines"

### 4. eBook Premium
- **รูปแบบ:** eBook เชิงลึกกว่า Lead Magnet
- **ราคา:** $7-17
- **ตัวอย่าง:** "คู่มือ A-Z สร้างธุรกิจออนไลน์"

### 5. Workshop Recording
- **รูปแบบ:** บันทึก workshop ที่เคยจัด
- **ราคา:** $17-47
- **ตัวอย่าง:** "Recording: 3-Hour Marketing Workshop"

## One-Time Offer (OTO) Page

### องค์ประกอบสำคัญ

1. **Headline:** "รอก่อน! มีข้อเสนอพิเศษสำหรับคุณ"
2. **Problem Reminder:** เตือนปัญหาที่เขามี
3. **Trip Wire Solution:** นำเสนอสิ่งที่ช่วยได้
4. **Value Stack:** แสดงมูลค่าจริง vs ราคาขาย
5. **Testimonials:** Social Proof
6. **Urgency:** "One-time only" / "หน้านี้เท่านั้น"
7. **CTA:** ปุ่มซื้อชัดเจน
8. **No Thanks:** ลิงก์ข้าม (ต้องมี)

### Value Stack Example

```
┌─────────────────────────────────────┐
│ มูลค่าที่คุณจะได้รับ:               │
├─────────────────────────────────────┤
│ ✓ Mini Course 5 บท      (2,970 บาท)│
│ ✓ Template 20 แบบ       (1,490 บาท)│
│ ✓ Swipe File 50 ตัวอย่าง  (990 บาท)│
│ ✓ Bonus: Private Group    (990 บาท)│
├─────────────────────────────────────┤
│ มูลค่ารวม:              6,440 บาท  │
│ วันนี้เท่านั้น:           490 บาท   │
│ ประหยัด:               5,950 บาท   │
└─────────────────────────────────────┘
```

## Pricing Psychology

### Sweet Spots

| Price | USD | Psychology |
|-------|-----|------------|
| 190 บาท | $7 | Impulse buy, ไม่ต้องคิด |
| 490 บาท | $17 | Value for money |
| 690 บาท | $27 | Most popular |
| 990 บาท | $37 | Premium feel |

### Pricing Tips
- ใช้เลข 7 หรือ 9 ลงท้าย ($17, $27, $37)
- แสดง Value Stack เทียบราคาจริง
- "เท่ากับกาแฟ 2-3 แก้ว"

## Upsell & Downsell

### Upsell (ถ้าซื้อ Trip Wire)
- เสนอ Bundle หรือ Premium Version
- ราคา: 2-3x Trip Wire
- ตัวอย่าง: "อัพเกรดเป็น Complete Package +690 บาท"

### Downsell (ถ้าไม่ซื้อ OTO)
- เสนอ Lite Version หรือ Payment Plan
- ราคา: ถูกกว่า OTO
- ตัวอย่าง: "ไม่พร้อมจ่ายเต็ม? รับ Lite Version 290 บาท"

## Common Mistakes

| ผิด | ถูก |
|-----|-----|
| ราคาแพงเกินไป (>$47) | Sweet spot $17-37 |
| คุณค่าไม่พอ | High perceived value |
| ไม่เกี่ยวกับ Core Offer | Related & leads to Core |
| ไม่มี Urgency | One-time offer |
| ไม่มี No Thanks link | ต้องมีทางออก |
| Over-promise | Deliver more than promise |

## Key Takeaways

1. **Trip Wire = First Purchase** - เปลี่ยน identity จาก subscriber เป็น buyer
2. **Price Sweet Spot:** $17-37 (490-990 บาท)
3. **Value > Price:** ต้องให้คุณค่ามากกว่าราคา
4. **One-Time Offer:** ใช้ urgency
5. **Lead to Core:** Trip Wire ต้องเตรียมทางไป Core Offer
