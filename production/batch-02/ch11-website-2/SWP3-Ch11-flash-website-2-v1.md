## Flashcard Set

### Card 1 (Definition)
**Q:** Visual Hierarchy คืออะไร?

**A:** Visual Hierarchy คือการจัดลำดับความสำคัญของ elements ด้วยสายตา สิ่งสำคัญที่สุดต้องเห็นก่อน ตัวใหญ่กว่า สีเด่นกว่า ทำให้ผู้ใช้รู้ว่าต้องดูอะไรก่อน

---

### Card 2 (Concept)
**Q:** ทำไม Mobile-First Design ถึงสำคัญ?

**A:**
- 60-70% ของ traffic มาจากมือถือ
- Google ใช้ Mobile-First Indexing
- ถ้าเว็บพังบนมือถือ = เสียลูกค้าไปมากกว่าครึ่ง

---

### Card 3 (Application)
**Q:** สีอะไรเหมาะกับปุ่ม CTA?

**A:**
- **แดง/ส้ม** - กระตุ้นการตัดสินใจ
- สิ่งสำคัญ: ต้อง**ตัดกับ background** ชัดเจน
- HubSpot ทดสอบ: ปุ่มแดงชนะเขียว 21%

---

### Card 4 (List)
**Q:** 7 ส่วนของ Landing Page มีอะไรบ้าง?

**A:**
1. **Hero Section** - Headline + CTA
2. **Problem** - ปัญหาที่ลูกค้าเจอ
3. **Solution** - วิธีแก้ปัญหา
4. **Benefits** - ประโยชน์ที่ได้
5. **Social Proof** - Testimonials
6. **FAQ** - คำถามที่พบบ่อย
7. **Final CTA** - ปิดการขาย

---

### Card 5 (Concept)
**Q:** AIDA Framework คืออะไร?

**A:**
- **A** = Attention (ดึงดูดความสนใจ)
- **I** = Interest (สร้างความสนใจ)
- **D** = Desire (กระตุ้นความต้องการ)
- **A** = Action (ให้ลงมือทำ)

ใช้เป็น flow สำหรับ Landing Page

---

### Card 6 (Comparison)
**Q:** Features vs Benefits ต่างกันอย่างไร?

**A:**
| Features | Benefits |
|----------|----------|
| คุณสมบัติของสินค้า | สิ่งที่ลูกค้าได้รับ |
| "มี 100 templates" | "ประหยัดเวลา 10 ชม./สัปดาห์" |
| What it IS | What it DOES for you |

**ในการขาย เน้น Benefits มากกว่า Features**

---

### Card 7 (List)
**Q:** Trust Elements ที่ต้องมีในเว็บมีอะไรบ้าง?

**A:**
1. **SSL Certificate** (https://)
2. **Testimonials** (รีวิวจากลูกค้าจริง)
3. **Case Studies** (ผลลัพธ์ที่พิสูจน์ได้)
4. **Contact Info** (ที่อยู่, เบอร์โทร)
5. **Guarantee** (รับประกันความพอใจ)
6. **Media Logos** (As seen on...)

---

### Card 8 (Application)
**Q:** ปุ่ม CTA ควรเขียนอย่างไรให้ดี?

**A:**
**ดี:**
- "Get Your Free Guide"
- "Start Saving Today"
- "Download Now"

**ไม่ดี:**
- "Submit" (เหมือนยอมจำนน)
- "Click Here" (ไม่บอก benefit)

**Formula:** Action Word + Benefit

---

### Card 9 (Definition)
**Q:** CRO คืออะไร?

**A:** **Conversion Rate Optimization** คือการปรับปรุงเว็บไซต์อย่างเป็นระบบเพื่อเพิ่ม conversion rate ผ่านการทดสอบ (A/B Test) และวิเคราะห์พฤติกรรมผู้ใช้

---

### Card 10 (Process)
**Q:** A/B Testing ทำอย่างไร?

**A:**
1. เลือก element ที่จะ test (เช่น headline)
2. สร้าง 2 versions (A และ B)
3. แบ่ง traffic 50/50
4. รันจนได้ผลที่มีนัยสำคัญ
5. ใช้ version ที่ชนะ
6. ทำซ้ำกับ element อื่น

---

### Card 11 (Statistics)
**Q:** Page Speed มีผลกับ Conversion อย่างไร?

**A:**
- ช้า 1 วินาที = **ลด conversion 7%**
- คนไม่รอเว็บช้าเกิน **3 วินาที**
- Target: โหลดเสร็จใน **< 3 วินาที**

---

### Card 12 (Application)
**Q:** Form Optimization ทำอย่างไร?

**A:**
- **ลด fields** ให้น้อยที่สุด (3-5 optimal)
- ลด 1 field = เพิ่ม conversion **10%+**
- ใช้ **multi-step** สำหรับ form ยาว
- แสดง **progress bar**
- เปิด **auto-fill**

---

### Card 13 (List)
**Q:** Google Analytics 4 Metrics สำคัญมีอะไรบ้าง?

**A:**
| Metric | Target |
|--------|--------|
| Sessions | - |
| Users | - |
| Bounce Rate | < 50% |
| Conversion Rate | 2-5% |
| Avg Session Duration | > 2 min |

---

### Card 14 (Definition)
**Q:** UTM Parameters คืออะไร?

**A:** UTM Parameters คือ tags ที่เพิ่มท้าย URL เพื่อ track แหล่งที่มาของ traffic

```
?utm_source=facebook
&utm_medium=paid
&utm_campaign=summer_sale
```

ช่วยบอกว่า traffic มาจากไหน campaign ไหน

---

### Card 15 (List)
**Q:** Website Maintenance Schedule ควรทำอะไรบ้าง?

**A:**
| เมื่อไหร่ | ทำอะไร |
|----------|--------|
| รายสัปดาห์ | Check analytics, backup |
| รายเดือน | Update plugins, security scan |
| รายไตรมาส | Content audit, speed test |
| รายปี | Renew domain/hosting |
