### SLIDE 4: Visual Hierarchy
```
👁️ VISUAL HIERARCHY

สิ่งสำคัญที่สุด = เห็นก่อน

ใช้:
• ขนาด - ใหญ่กว่า = สำคัญกว่า
• สี - เด่นกว่า = สำคัญกว่า
• ตำแหน่ง - บนซ้าย = เห็นก่อน

F-Pattern: คนอ่านเว็บตามรูปตัว F

Visual: F-pattern heatmap
```

### SLIDE 5: Mobile-First
```
📱 MOBILE-FIRST

60-70% ของ traffic มาจากมือถือ

Google ใช้ Mobile-First Indexing

ถ้าเว็บพังบนมือถือ =
เสียลูกค้าไปมากกว่าครึ่ง!

Checklist:
✅ ปุ่มใหญ่พอ (min 44x44px)
✅ Font ≥ 16px
✅ ไม่มี hover-only features

Visual: Desktop vs Mobile comparison
```

### SLIDE 7: Landing Page Anatomy
```
📄 LANDING PAGE = 7 ส่วน

1. 🎯 HERO - Headline + CTA
2. 😣 PROBLEM - ปัญหาลูกค้า
3. 💡 SOLUTION - วิธีแก้
4. ⭐ BENEFITS - ประโยชน์
5. 👥 SOCIAL PROOF - รีวิว
6. ❓ FAQ - คำถาม
7. 🔘 FINAL CTA - ปิดการขาย

Visual: Landing page wireframe
```

---

### SLIDE 8: AIDA Framework
```
📈 AIDA FRAMEWORK

A → ATTENTION
    Hero Section ดึงดูดสายตา

I → INTEREST
    Problem + Solution

D → DESIRE
    Benefits + Social Proof

A → ACTION
    CTA ชัดเจน

Visual: AIDA funnel diagram
```

### SLIDE 9: Features vs Benefits
```
⭐ FEATURES vs BENEFITS

| Features | Benefits |
|----------|----------|
| "100 templates" | "ประหยัด 10 ชม./สัปดาห์" |

💡 ในการขาย เน้น BENEFITS!
   บอกสิ่งที่ลูกค้าได้รับ
```

### SLIDE 10: Trust Elements
```
🔒 TRUST ELEMENTS

ทำไมคนไม่ซื้อ? → ไม่ไว้ใจ

Must-Have:
✅ SSL Certificate (https://)
✅ Testimonials + รูป + ชื่อ
✅ Case Studies (ผลลัพธ์จริง)
✅ Contact Info (ที่อยู่, เบอร์)
✅ Guarantee (คืนเงินได้)

Visual: Trust badges montage
```

### SLIDE 11: CTA Optimization
```
🔘 CTA BEST PRACTICES

Position:
• Above the Fold
• หลังทุก section สำคัญ

Design:
• สีตัดกับ background
• ใหญ่พอให้เห็นชัด

Copy:
✅ "Get Your Free Guide"
❌ "Submit" หรือ "Click Here"

FORMULA: Action Word + Benefit

Visual: CTA examples
```

### SLIDE 12: Key Statistics
```
📊 ตัวเลขที่ต้องรู้

⚡ ช้า 1 วินาที = -7% Conversion
📱 60-70% Traffic จาก Mobile
📝 ลด 1 Form Field = +10% Conversion
🎯 Target Conversion Rate: 2-5%
⏱️ Target Load Time: < 3 วินาที
📉 Target Bounce Rate: < 50%

Visual: Statistics infographic
```
