## Table 1: Design Principles

| Principle | Definition | Impact |
|-----------|------------|--------|
| Visual Hierarchy | จัดลำดับความสำคัญด้วยสายตา | ทำให้รู้ว่าดูอะไรก่อน |
| White Space | พื้นที่ว่างรอบ content | ดูโปร อ่านง่าย |
| Consistency | ใช้รูปแบบเดียวกันทั้งเว็บ | สร้าง Brand identity |
| Mobile-First | ออกแบบมือถือก่อน | รองรับ 60-70% traffic |

## Table 3: Landing Page 7 Sections

| Section | Purpose | Key Elements |
|---------|---------|--------------|
| Hero | First impression | Headline, Subheadline, CTA |
| Problem | แสดงความเข้าใจ | Pain points, Statistics |
| Solution | แนะนำสินค้า | Product overview, How it works |
| Benefits | ประโยชน์ที่ได้ | Benefits (not features), Icons |
| Social Proof | สร้างความเชื่อถือ | Testimonials, Case studies |
| FAQ | ลดความลังเล | Common questions, Objections |
| Final CTA | ปิดการขาย | CTA button, Urgency, Guarantee |

## Table 4: AIDA Framework

| Stage | Meaning | Landing Page Section |
|-------|---------|---------------------|
| A - Attention | ดึงดูดความสนใจ | Hero Section |
| I - Interest | สร้างความสนใจ | Problem + Solution |
| D - Desire | กระตุ้นความต้องการ | Benefits + Social Proof |
| A - Action | ให้ลงมือทำ | CTA |

## Table 5: Trust Elements

| Element | Importance | How to Implement |
|---------|------------|------------------|
| SSL Certificate | Critical | Let's Encrypt (free) |
| Testimonials | Very High | รูป + ชื่อ + ตำแหน่ง |
| Case Studies | Very High | ผลลัพธ์ + ตัวเลข |
| Contact Info | High | ที่อยู่ + เบอร์โทร + Email |
| Guarantee | High | Money-back policy |
| Media Logos | Medium | "As seen on..." |
| Certifications | Medium | ISO, รางวัล |

---

## Table 6: CTA Best Practices

| Element | Good | Bad |
|---------|------|-----|
| Position | Above the Fold | ซ่อนอยู่ล่างสุด |
| Color | ตัดกับ background | กลมกลืนกับ background |
| Size | ใหญ่พอให้เห็นชัด | เล็กจนมองข้าม |
| Copy | "Get Your Free Guide" | "Submit" |
| Copy | "Start Saving Today" | "Click Here" |

## Table 8: Conversion Statistics

| Metric | Impact | Target |
|--------|--------|--------|
| Page Speed | -7% conversion per 1 sec delay | < 3 seconds |
| Form Fields | +10% per field removed | 3-5 fields |
| Mobile Traffic | 60-70% of total | Optimized for mobile |
| Bounce Rate | Higher = worse | < 50% |
| Conversion Rate | Landing page average | 2-5% |
