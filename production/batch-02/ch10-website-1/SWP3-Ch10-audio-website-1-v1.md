## Audio Script

### OPENING (0:00 - 0:45)

**Host A:** สวัสดีครับ กลับมาพบกันอีกครั้ง วันนี้เราจะมาคุยกันเรื่อง Website Part 1 พื้นฐานการสร้างเว็บไซต์

**Host B:** เว็บไซต์เป็นสิ่งจำเป็นสำหรับธุรกิจออนไลน์เลยครับ มันคือ "บ้าน" ของธุรกิจเรา

**Host A:** มาเริ่มกันเลย

---

### SEGMENT 1: ทำไมต้องมีเว็บไซต์ (0:45 - 2:00)

**Host A:** ทำไมต้องมีเว็บไซต์?

**Host B:** เว็บไซต์เป็นทรัพย์สินที่คุณเป็นเจ้าของครับ ต่างจาก Social Media ที่อยู่บน platform คนอื่น

**Host A:** หมายความว่ายังไง?

**Host B:** ถ้า Facebook เปลี่ยน algorithm หรือ ban account ธุรกิจคุณก็พังได้ แต่เว็บไซต์คุณควบคุมได้ 100%

**Host A:** และมันช่วยเรื่อง credibility ด้วยใช่ไหม?

**Host B:** ใช่ครับ คนไว้วางใจธุรกิจที่มีเว็บไซต์มากกว่า

---

### SEGMENT 2: 3 องค์ประกอบหลัก (2:00 - 5:00)

**Host A:** เว็บไซต์มีกี่ส่วน?

**Host B:** มี 3 ส่วนหลักครับ

ส่วนแรก **Domain Name** คือที่อยู่เว็บไซต์ เช่น yourname.com เหมือนที่อยู่บ้าน

**Host A:** เลือกยังไงดี?

**Host B:** เลือกชื่อที่สั้น จำง่าย เกี่ยวข้องกับธุรกิจ และใช้ .com เป็นหลัก

ส่วนที่สอง **Web Hosting** คือพื้นที่เก็บไฟล์เว็บไซต์บน server เหมือนเช่าที่ดินปลูกบ้าน

**Host A:** ราคาเท่าไหร่?

**Host B:** เริ่มต้นประมาณ 100-300 บาทต่อเดือนสำหรับ Shared Hosting ครับ

ส่วนที่สาม **Platform** คือเครื่องมือสร้างเว็บ WordPress เป็นที่นิยมที่สุด ใช้ฟรี customize ได้เยอะ

---

### SEGMENT 3: WordPress Basics (5:00 - 8:00)

**Host A:** ทำไมถึงแนะนำ WordPress?

**Host B:** เพราะ 4 เหตุผลครับ

หนึ่ง **ฟรีและ Open Source** ไม่ต้องจ่ายค่า license

สอง **Plugins มากมาย** มีเครื่องมือเสริมเป็นหมื่นตัว

สาม **Customize ได้เต็มที่** ไม่มีข้อจำกัด

สี่ **Community ใหญ่** มีคนช่วยเหลือเยอะ

**Host A:** แล้ว WordPress.org กับ WordPress.com ต่างกันยังไง?

**Host B:** WordPress.org คือซอฟต์แวร์ที่ลงใน hosting เอง ควบคุมได้ 100% ส่วน WordPress.com คือ hosted version มีข้อจำกัด แนะนำ WordPress.org ครับ

**Host A:** Plugins ที่ต้องมีมีอะไรบ้าง?

**Host B:** 10 ตัวหลักครับ Elementor สำหรับ page builder, Yoast SEO สำหรับ SEO, WPForms สำหรับ contact form, UpdraftPlus สำหรับ backup, Wordfence สำหรับ security และอื่นๆ

---

### SEGMENT 4: Setup & หน้าที่ต้องมี (8:00 - 10:30)

**Host A:** ขั้นตอนการ setup มีอะไรบ้าง?

**Host B:** 8 ขั้นตอนครับ

1. ซื้อ Domain
2. ซื้อ Hosting
3. ติดตั้ง WordPress ด้วย 1-Click Install
4. เลือก Theme
5. ติดตั้ง Essential Plugins
6. ตั้งค่า Settings พื้นฐาน
7. สร้างหน้าสำคัญ
8. ตั้งค่า Security และ Backup

**Host A:** หน้าสำคัญมีอะไรบ้าง?

**Host B:** 5 หน้าครับ Home คือหน้าแรก, About แนะนำตัว, Services หรือ Products สินค้าบริการ, Contact ติดต่อ, และ Blog สำหรับ content

---

### SEGMENT 5: Speed & Security (10:30 - 11:30)

**Host A:** Speed สำคัญแค่ไหน?

**Host B:** สำคัญมากครับ เว็บช้า 1 วินาที conversion ลด 7% และ Google ชอบเว็บเร็ว

**Host A:** ทำยังไงให้เร็ว?

**Host B:** ใช้ Caching Plugin, optimize รูป, ใช้ CDN, และเลือก hosting ที่ดี

**Host A:** แล้ว Security?

**Host B:** ต้องใช้ password แข็งแรง, ติดตั้ง security plugin, อัพเดทสม่ำเสมอ, backup เป็นประจำ, และใช้ SSL Certificate
