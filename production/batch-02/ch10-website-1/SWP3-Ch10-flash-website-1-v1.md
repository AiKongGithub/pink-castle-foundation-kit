## Flashcard Set

### Card 1 (List)
**Q:** 3 องค์ประกอบหลักของเว็บไซต์มีอะไรบ้าง?

**A:**
1. **Domain Name** - ที่อยู่เว็บ (เช่น yourname.com)
2. **Web Hosting** - พื้นที่เก็บไฟล์เว็บบน server
3. **Platform** - เครื่องมือสร้างเว็บ (เช่น WordPress)

---

### Card 2 (Definition)
**Q:** Domain Name คืออะไร?

**A:** Domain Name คือที่อยู่ของเว็บไซต์บน Internet เปรียบเหมือนที่อยู่บ้าน เช่น google.com หรือ facebook.com ราคาประมาณ $9-15 ต่อปี

---

### Card 3 (Definition)
**Q:** Web Hosting คืออะไร?

**A:** Web Hosting คือพื้นที่เก็บไฟล์เว็บไซต์บน Server เปรียบเหมือนเช่าที่ดินปลูกบ้าน มีหลายประเภท: Shared ($3-10/mo), VPS ($20-80/mo), Dedicated ($100+/mo)

---

### Card 4 (Comparison)
**Q:** WordPress.org vs WordPress.com ต่างกันอย่างไร?

**A:**
| WordPress.org | WordPress.com |
|--------------|---------------|
| ต้องหา hosting เอง | hosting รวมมาแล้ว |
| ควบคุมได้ 100% | มีข้อจำกัด |
| Plugins ไม่จำกัด | Plugins จำกัด |
| Monetize ได้เลย | ต้องจ่าย Premium |

**แนะนำ:** WordPress.org

---

### Card 5 (List)
**Q:** 5 Essential Plugins ที่ต้องติดตั้งมีอะไรบ้าง?

**A:**
1. **Elementor** - Page Builder
2. **Yoast SEO** - SEO Optimization
3. **WPForms** - Contact Forms
4. **UpdraftPlus** - Backup
5. **Wordfence** - Security

---

### Card 6 (List)
**Q:** 5 หน้าที่เว็บไซต์ต้องมีคืออะไร?

**A:**
1. **Home** - หน้าแรก, First impression
2. **About** - แนะนำตัว, สร้าง trust
3. **Services/Products** - สินค้า/บริการ
4. **Contact** - ติดต่อ
5. **Blog** - Content, SEO

---

### Card 7 (Concept)
**Q:** ทำไม Speed ของเว็บไซต์ถึงสำคัญ?

**A:**
- **Google Ranking:** เว็บเร็ว rank สูงกว่า
- **Conversion:** ช้า 1 วินาที = ลด conversion 7%
- **Bounce Rate:** คนไม่รอเว็บช้าเกิน 3 วินาที

---

### Card 8 (List)
**Q:** วิธีทำให้เว็บเร็วมีอะไรบ้าง?

**A:**
1. ใช้ Caching Plugin (WP Super Cache)
2. Optimize รูปภาพ (Smush)
3. ใช้ CDN
4. เลือก Hosting ที่เร็ว
5. Minify CSS/JS
6. ลดจำนวน Plugins

---

### Card 9 (Concept)
**Q:** ทำไมต้องมีเว็บไซต์ ไม่ใช้แค่ Social Media?

**A:** เว็บไซต์เป็นทรัพย์สินที่คุณเป็นเจ้าของ ควบคุมได้ 100% ไม่ขึ้นกับ algorithm ต่างจาก Social Media ที่อยู่บน platform คนอื่น ถูก ban หรือ algorithm เปลี่ยนได้

---

### Card 10 (Application)
**Q:** Permalinks ควรตั้งค่าเป็นแบบไหน?

**A:** ควรตั้งเป็น "Post name" (Settings > Permalinks)

ตัวอย่าง: yoursite.com/my-blog-post/
ไม่ใช่: yoursite.com/?p=123

เพราะอ่านง่าย และดีต่อ SEO

---

### Card 11 (List)
**Q:** Security Basics ที่ต้องทำมีอะไรบ้าง?

**A:**
1. **Strong Password** - ใช้ Password Manager
2. **Limit Login Attempts** - Wordfence
3. **Keep Updated** - Auto-updates
4. **Regular Backups** - UpdraftPlus
5. **SSL Certificate** - Let's Encrypt (ฟรี)
6. **Hide Login URL** - WPS Hide Login

---

### Card 12 (Application)
**Q:** Free WordPress Theme ที่แนะนำมีอะไรบ้าง?

**A:**
1. **Astra** - เร็ว, ยืดหยุ่น, All-purpose
2. **GeneratePress** - เบามาก, Performance
3. **Kadence** - Modern, Feature-rich
4. **OceanWP** - Feature เยอะ, E-commerce

---

### Card 13 (Concept)
**Q:** SSL Certificate คืออะไร และทำไมต้องมี?

**A:** SSL ทำให้เว็บแสดง https:// และรูปกุญแจ หมายความว่าข้อมูลถูกเข้ารหัส

ต้องมีเพราะ:
- Google rank เว็บ HTTPS สูงกว่า
- ลูกค้าไว้วางใจมากกว่า
- จำเป็นสำหรับ payment

---

### Card 14 (Application)
