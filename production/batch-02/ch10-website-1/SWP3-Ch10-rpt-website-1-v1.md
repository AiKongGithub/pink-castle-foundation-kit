## Executive Summary

เว็บไซต์เป็นทรัพย์สินที่สำคัญที่สุดของธุรกิจออนไลน์ เป็น "บ้าน" ที่คุณเป็นเจ้าของ ต่างจาก Social Media ที่อยู่บน platform คนอื่น บทนี้ครอบคลุมพื้นฐานการสร้างเว็บไซต์ตั้งแต่ Domain, Hosting, Platform ไปจนถึงการตั้งค่าพื้นฐาน

**Key Insight:** เว็บไซต์ที่ดี = Domain ดี + Hosting เร็ว + WordPress + 5 หน้าสำคัญ + Speed & Security

---

## 1. Why You Need a Website

### 1.1 เว็บไซต์ vs Social Media

| Aspect | Website | Social Media |
|--------|---------|--------------|
| Ownership | คุณเป็นเจ้าของ | Platform เป็นเจ้าของ |
| Control | 100% | จำกัดตาม rules |
| Algorithm | ไม่มี | เปลี่ยนบ่อย |
| Risk | ต่ำ | Account ถูก ban ได้ |
| Credibility | สูง | ปานกลาง |
| Monetization | ยืดหยุ่น | มีข้อจำกัด |

### 1.2 Benefits of Having a Website

1. **Ownership:** ทรัพย์สินที่ขายต่อได้
2. **Credibility:** ดูเป็นมืออาชีพ
3. **Control:** ไม่ขึ้นกับ algorithm
4. **SEO:** Rank บน Google ได้
5. **Flexibility:** Customize ได้ตามต้องการ

## 2. Three Components of a Website

### 2.1 Domain Name

**Definition:** ที่อยู่เว็บไซต์บน Internet (เช่น yourname.com)

**วิธีเลือก Domain:**
- ✅ สั้น จำง่าย
- ✅ เกี่ยวข้องกับธุรกิจ
- ✅ ใช้ .com เป็นหลัก
- ❌ หลีกเลี่ยงตัวเลขและขีด
- ❌ หลีกเลี่ยงชื่อที่สะกดยาก

**Domain Providers:**

| Provider | Price/Year | Pros |
|----------|------------|------|
| Namecheap | $9-15 | ถูก, Free WhoisGuard |
| Cloudflare | $8-9 | ถูกที่สุด, Fast DNS |
| Google Domains | $12 | ง่าย, เชื่อมต่อ Google |
| GoDaddy | $12-20 | Brand ใหญ่ |

**แนะนำ:** Namecheap หรือ Cloudflare

---

| Dedicated | $100+ | Traffic สูงมาก |

**Hosting Providers:**

| Provider | Price | Pros | Best For |

| Kinsta | $35/mo | Premium | High-traffic |

**แนะนำสำหรับมือใหม่:** Hostinger หรือ SiteGround

### 2.3 Platform

**Definition:** ซอฟต์แวร์/เครื่องมือที่ใช้สร้างและจัดการเว็บไซต์

**Platform Comparison:**

| Platform | Cost | Difficulty | Best For |
|----------|------|------------|----------|
| WordPress.org | Free* | Medium | ทุกอย่าง |
| Wix | $16-45/mo | Easy | มือใหม่สุด |
| Squarespace | $16-49/mo | Easy | Portfolio |
| Shopify | $29-299/mo | Easy | E-commerce |
| Webflow | $12-36/mo | Hard | Design-focused |

*Free software, but need hosting cost

**แนะนำ:** WordPress.org เพราะ:
- Free และ Open Source
- 60,000+ Plugins
- Unlimited Customization
- Huge Community
- Sell-able Asset

## 3. WordPress Deep Dive

### 3.1 WordPress.org vs WordPress.com

| Aspect | WordPress.org | WordPress.com |
|--------|--------------|---------------|
| Hosting | ต้องหาเอง | รวมมาแล้ว |
| Cost | Hosting cost only | Free-$45/mo |
| Freedom | 100% | จำกัด |
| Plugins | Unlimited | จำกัด (Free plan) |
| Themes | Unlimited | จำกัด |
| Monetization | ได้เลย | ต้องจ่าย Premium |
| Ads | ไม่มี | มี (Free plan) |

**สรุป:** ใช้ WordPress.org + Self-hosting เสมอ

## 4. Website Setup Guide

### 4.1 Step-by-Step Process

```
Step 1: ซื้อ Domain ($10-15/year)
   ↓
Step 2: ซื้อ Hosting ($3-15/month)
   ↓
Step 3: ติดตั้ง WordPress (1-Click)
   ↓
Step 4: ตั้งค่า Settings พื้นฐาน
   ↓
Step 5: เลือกและติดตั้ง Theme
   ↓
Step 6: ติดตั้ง Essential Plugins
   ↓
Step 7: สร้าง 5 หน้าสำคัญ
   ↓
Step 8: ตั้งค่า Security & Backup
```

### 4.2 Essential WordPress Settings

| Setting | Location | Action |
|---------|----------|--------|
| Site Title | Settings > General | ใส่ชื่อเว็บ |
| Tagline | Settings > General | คำอธิบายสั้น |
| Timezone | Settings > General | Asia/Bangkok |
| Permalinks | Settings > Permalinks | Post name |
| Comments | Settings > Discussion | ปิดถ้าไม่ต้องการ |
| Homepage | Settings > Reading | Static page |

## 5. Five Essential Pages

### 5.1 Page Priority

| Page | Purpose | Priority | Must-Have Elements |
|------|---------|----------|-------------------|
| Home | First impression | Critical | Value prop, CTA |
| About | Build trust | High | Story, Team |

### 5.2 Home Page Best Practices

1. **Above the Fold:**
   - Clear headline
   - Value proposition
   - Primary CTA button

2. **Below the Fold:**
   - Benefits/Features
   - Social proof
   - Secondary CTA

## 6. Speed Optimization

### 6.1 Why Speed Matters

- Google Ranking: เว็บเร็ว rank สูงกว่า
- Conversion: 1 วินาทีช้า = -7% conversion
- Bounce Rate: 3 วินาที+ = 50% bounce

### 6.2 Speed Optimization Checklist

| Action | Impact | Difficulty |
|--------|--------|------------|
| Use Caching Plugin | High | Easy |
| Optimize Images | High | Easy |
| Use CDN | High | Medium |
| Minify CSS/JS | Medium | Easy |
| Choose Fast Hosting | High | Medium |
| Limit Plugins | Medium | Easy |
