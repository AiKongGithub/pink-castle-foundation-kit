## Key Concepts

### 1. Why Facebook Ads?
- **Reach**: 3 พันล้านผู้ใช้ต่อเดือน
- **Targeting**: ละเอียดที่สุดในโลก
- **Formats**: หลากหลายรูปแบบ
- **Cost**: เริ่มต้นได้ทุก Budget

### 2. Campaign Structure
```
Campaign (วัตถุประสงค์)
    └── Ad Set (กลุ่มเป้าหมาย + งบ)
            └── Ad (ชิ้นงานโฆษณา)
```

### 3. Campaign Objectives
**Awareness:**
- Brand Awareness
- Reach

**Consideration:**
- Traffic
- Engagement
- App Installs
- Video Views
- Lead Generation
- Messages

**Conversion:**
- Conversions
- Catalog Sales
- Store Traffic

### 4. Targeting Options

**Demographics:**
- Age
- Gender
- Location
- Language
- Education
- Job Title

**Interests:**
- Pages liked
- Activities
- Hobbies
- Shopping behavior

**Behaviors:**
- Purchase behavior
- Device usage
- Travel patterns

**Custom Audiences:**
- Website visitors (Pixel)
- Customer list (Email/Phone)
- App users
- Engagement (Video, Page)

**Lookalike Audiences:**
- คนที่คล้ายกับลูกค้าปัจจุบัน
- 1-10% similarity
- แนะนำเริ่มที่ 1%

### 5. Ad Formats

| Format | Best For | Specs |
|--------|----------|-------|
| Image | Simple message | 1080x1080 หรือ 1200x628 |
| Video | Engagement | < 15 sec, 4:5 หรือ 1:1 |
| Carousel | Multiple products | 2-10 cards |
| Collection | E-commerce | Catalog + Video |
| Stories | Mobile | 1080x1920 |
| Reels | Young audience | 9:16 |

### 6. Facebook Pixel

**คืออะไร:** Code ติดเว็บไซต์เพื่อ track พฤติกรรม

**Standard Events:**
- PageView
- ViewContent
- AddToCart
- InitiateCheckout
- Purchase
- Lead
- CompleteRegistration

**Benefits:**
- Track Conversions
- Build Custom Audiences
- Optimize for Conversions
- Measure ROI

### 7. Ad Copy Formula

**PAS Formula:**
```
Problem: พูดถึงปัญหา
Agitate: ขยายความเจ็บปวด
Solution: เสนอวิธีแก้
```

**AIDA Formula:**
```
Attention: ดึงความสนใจ
Interest: สร้างความสนใจ
Desire: กระตุ้นความต้องการ
Action: บอกให้ทำอะไร
```

### 8. Budgeting Strategies

**Daily Budget:**
- ใช้ทุกวัน
- เหมาะกับ testing
- แนะนำเริ่ม 200-500 บาท/วัน

**Lifetime Budget:**
- ใช้ตลอด campaign
- Facebook จัดการให้
- เหมาะกับ limited time offers

### 9. Bidding Strategies

| Strategy | Best For |
|----------|----------|
| Lowest Cost | เริ่มต้น, Testing |
| Cost Cap | ควบคุม CPA |
| Bid Cap | ควบคุม bid สูงสุด |
| ROAS | E-commerce |

### 10. Testing Framework

**What to Test:**
- Audiences (ทดสอบก่อน)
- Creatives (Images, Videos)
- Copy (Headlines, Body)
- Placements
- Bid strategies

**How to Test:**
1. ทดสอบ 1 variable ต่อครั้ง
2. Budget เท่าๆ กัน
3. รันอย่างน้อย 3-7 วัน
4. ดูผลแบบ statistical significance

### 11. Key Metrics

| Metric | Definition | Good |
|--------|------------|------|
| CPM | Cost per 1000 impressions | $5-15 |
| CPC | Cost per click | $0.5-2 |
| CTR | Click-through rate | > 1% |
| CPA | Cost per acquisition | ต้นทุนต่ำกว่ากำไร |
| ROAS | Return on ad spend | > 3x |
| Frequency | ครั้งที่เห็นโฆษณา | < 3 |

### 12. Common Mistakes

1. **ไม่ติด Pixel** ก่อนยิงโฆษณา
2. **Audience กว้างเกินไป** หรือแคบเกินไป
3. **ไม่ทดสอบ** Creative หลายแบบ
4. **ปิดเร็วเกินไป** ไม่รอ learning phase
5. **ไม่ดู Frequency** ยิงซ้ำคนเดิมเยอะ
6. **Landing Page ไม่ดี** โฆษณาดีแต่เว็บพัง

## Funnel Strategy

### Cold Audience (ไม่รู้จัก)
- **Objective:** Traffic, Video Views
- **Content:** ให้คุณค่า, แนะนำตัว
- **Budget:** 60-70%

### Warm Audience (สนใจ)
- **Objective:** Engagement, Lead
- **Content:** Case studies, Testimonials
- **Budget:** 20-30%

### Hot Audience (พร้อมซื้อ)
- **Objective:** Conversions
- **Content:** Offer, Urgency
- **Budget:** 10-20%
