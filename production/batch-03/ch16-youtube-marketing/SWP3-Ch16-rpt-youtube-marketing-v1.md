## 2. YouTube Algorithm 2026

### 2.1 Three Main Factors

```
1. Watch Time (เวลาที่คนดู)
2. CTR (Click-Through Rate)
3. Engagement (Likes, Comments, Shares)
```

### 2.2 Watch Time

**Definition:** เวลาทั้งหมดที่คนดูวิดีโอ

**Why Important:**
- YouTube ต้องการให้คนอยู่บน Platform นานๆ
- ยิ่งวิดีโอถูกดูนาน ยิ่งได้ Promote

**How to Increase:**
- เนื้อหาน่าสนใจตลอด
- ใช้ Pattern Interrupt
- เพิ่ม Retention Hooks

### 2.3 CTR (Click-Through Rate)

**Definition:** % ที่คนเห็น Thumbnail แล้วคลิก

**Formula:** (Clicks / Impressions) x 100

**Benchmarks:**
| Level | CTR |
|-------|-----|
| Poor | < 2% |
| Average | 2-4% |
| Good | 4-7% |
| Excellent | > 7% |

**How to Improve:**
- Thumbnail ที่ดึงดูด
- Title ที่น่าสนใจ
- A/B Testing

### 2.4 Engagement

**Types:**
- **Likes** - Signal คุณภาพ
- **Comments** - Active community
- **Shares** - Viral potential
- **Subscribe** - Channel growth

**How to Increase:**
- Ask for engagement (CTA)
- ตอบ Comments 1 ชั่วโมงแรก
- สร้าง Community

### 2.5 Audience Retention

**Definition:** % ที่คนดูวิดีโอจบ

**Benchmarks:**
| Level | Retention |
|-------|-----------|
| Poor | < 30% |
| Average | 30-40% |
| Good | 40-50% |
| Excellent | > 50% |

**How to Improve:**
- Hook ใน 30 วินาทีแรก
- Pattern Interrupts
- เนื้อหาตรงประเด็น

## 4. Video SEO

### 4.1 Title Optimization

**Formula:**
```
Target Keyword + Modifier + Hook
```

**Examples:**
- "YouTube SEO 2026: วิธีติดหน้าแรก [ทำตามได้เลย]"
- "5 วิธีทำเงินจาก YouTube (ได้จริง!)"

**Best Practices:**
- Keyword อยู่ด้านหน้า
- 60-70 characters
- ใส่ปี (2026)
- ใช้ตัวเลข

### 4.2 Description Optimization

**Structure:**
```
บรรทัด 1-2: Summary + Keyword (แสดงก่อน See More)
บรรทัด 3-5: รายละเอียดวิดีโอ

TIMESTAMPS:
0:00 Intro
1:30 Topic 1
5:00 Topic 2
...

LINKS:
Website: [URL]
Instagram: [URL]
Facebook: [URL]

#Hashtag1 #Hashtag2 #Hashtag3
```

### 4.3 Tags

| Type | Example | Amount |
|------|---------|--------|
| Primary | youtube marketing | 1-2 |
| Secondary | youtube seo, youtube tips | 3-5 |
| Long-tail | วิธีทำเงินจาก youtube | 3-5 |
| Brand | ชื่อช่อง | 1 |

**Total:** 5-15 tags

### 4.4 Thumbnail Best Practices

**Specs:**
- Size: 1280 x 720 px
- Ratio: 16:9
- File: < 2MB

**Elements:**
| Element | Why |
|---------|-----|
| Face with emotion | ดึงดูดสายตา |
| Big text (3-5 words) | อ่านง่ายใน feed |
| High contrast | โดดเด่น |
| Bright colors | ดึงความสนใจ |

### 4.5 Thumbnail A/B Testing

YouTube มี feature "Test & Compare"

**How to Test:**
1. สร้าง Thumbnail 2-3 แบบ
2. ใช้ Test & Compare
3. รอ 7-14 วัน
4. ดูผล CTR

**Requirements:**
- 1,000 Subscribers
- 4,000 Watch Hours (12 เดือน)
- OR 10 Million Shorts Views (90 วัน)

| Membership | Channel Members | 5-20% |
| Super Chat | Live donations | 2-10% |
| Merchandise | ขายของ | 5-15% |

## 10. Growth Strategies

### 10.1 Organic Growth

| Strategy | Description | Priority |
|----------|-------------|----------|
| SEO | Optimize ทุกวิดีโอ | High |
| Consistency | อัพโหลดสม่ำเสมอ | High |
| Thumbnails | A/B Test CTR | High |
| Engagement | ตอบ Comments | Medium |
| Collaboration | ทำกับ YouTubers | Medium |
| Shorts | ดึงคนใหม่ | High |
| Playlists | เพิ่ม Session Time | Medium |
| Community | Community Tab | Low |

### 10.2 Content Strategy (Hero-Hub-Help)

| Type | Frequency | Purpose |
|------|-----------|---------|
| Hero | 1/เดือน | เนื้อหาใหญ่ ดึงคนใหม่ |
| Hub | 1/สัปดาห์ | เนื้อหาประจำ สร้าง Community |
| Help | 2-3/สัปดาห์ | How-to ให้คนค้นหาเจอ |

### 10.3 Common Mistakes

| Mistake | Impact | Solution |
|---------|--------|----------|
| ไม่ Optimize SEO | คนหาไม่เจอ | Research keywords |
| Thumbnail ไม่ดึงดูด | CTR ต่ำ | A/B Test |
| ไม่สม่ำเสมอ | Algorithm ไม่ผลัก | ตารางที่ทำได้จริง |
| ไม่ตอบ Comments | Engagement ต่ำ | ตอบภายใน 1 ชม. |
| ไม่ทำ Shorts | พลาดคนใหม่ | Repurpose content |
| Quality ต่ำ | คนไม่ดูจบ | ลงทุนอุปกรณ์ |
