## Key Concepts

### 1. Why YouTube Marketing?

- **Search Engine**: Search Engine อันดับ 2 ของโลก (รองจาก Google)
- **Users**: 2.5 พันล้านผู้ใช้ต่อเดือน
- **Watch Time**: 1 พันล้านชั่วโมงต่อวัน
- **Longevity**: วิดีโอมีอายุยืนยาว ทำเงินได้หลายปี
- **Trust**: วิดีโอสร้างความไว้วางใจได้ดีกว่าข้อความ

### 2. YouTube Algorithm 2026

**3 ปัจจัยหลัก:**
```
1. Watch Time (เวลาที่คนดู)
2. CTR (Click-Through Rate)
3. Engagement (Likes, Comments, Shares)
```

**Ranking Factors:**
- **Watch Time** - ยิ่งคนดูนาน ยิ่งดี
- **Session Time** - คนดูวิดีโออื่นต่อไหม
- **CTR** - คลิกจาก Thumbnail + Title
- **Engagement Rate** - Like, Comment, Share, Subscribe
- **Upload Frequency** - ความสม่ำเสมอ
- **Audience Retention** - คนดูจบหรือไม่

### 3. Video Types

| Type | Duration | Best For |
|------|----------|----------|
| Tutorial | 8-15 min | แนะนำวิธีทำ How-to |
| Vlog | 10-20 min | สร้าง Connection |
| Review | 5-15 min | สินค้า บริการ |
| Listicle | 8-12 min | "5 วิธี...", "Top 10..." |
| Interview | 15-30 min | Authority Building |
| Short | 15-60 sec | Viral, Discovery |
| Live | 30-120 min | Q&A, Launch |
| Documentary | 15-45 min | Deep Content |

### 4. Channel Setup & Optimization

**Channel Art:**
- Banner: 2560 x 1440 px
- Profile: 800 x 800 px
- Watermark: 150 x 150 px

**Channel Description:**
```
บรรทัดที่ 1-2: บอกว่าช่องเกี่ยวกับอะไร
บรรทัดที่ 3-4: อัพโหลดวันไหน บ่อยแค่ไหน
บรรทัดที่ 5+: Links, Contact
```

**Channel Keywords:**
- ใส่ 5-10 keywords หลัก
- ใช้ทั้งภาษาไทยและอังกฤษ

**Channel Sections:**
- Uploads (Latest)
- Popular Videos
- Playlists
- Featured Channels

### 5. Video SEO

**Title Optimization:**
```
Target Keyword + Modifier + Hook
ตัวอย่าง: "YouTube SEO 2026: วิธีติดหน้าแรก [ทำตามได้เลย]"
```

**Description:**
```
บรรทัด 1-2: Summary + Keyword (แสดงก่อนคลิก See More)
บรรทัด 3-5: รายละเอียดวิดีโอ
Timestamps: 0:00 Intro, 1:30 Topic 1...
Links: Social media, Website
Hashtags: #Keyword1 #Keyword2 #Keyword3
```

**Tags:**
- Primary keyword (exact match)
- Secondary keywords (variations)
- Long-tail keywords
- Brand name
- Related topics

**Thumbnails:**
```
Size: 1280 x 720 px (16:9)
File: < 2MB (PNG/JPG)
Elements:
  - ใบหน้าคน (Emotion)
  - Text ใหญ่ อ่านง่าย (3-5 คำ)
  - Contrast สูง
  - สีสดใส
```

### 6. YouTube Shorts Strategy

**Specs:**
- Duration: 15-60 วินาที
- Ratio: 9:16 (Vertical)
- Hook: 1-3 วินาทีแรกสำคัญมาก

**Best Practices:**
1. เริ่มเร็ว ไม่มี Intro
2. เนื้อหาเข้าใจง่าย
3. Loop ได้ (จบแล้วดูซ้ำ)
4. Text บนจอ (ดูไม่มีเสียง)
5. CTA สั้นๆ

**Content Ideas:**
- Behind the scenes
- Quick tips
- Teasers ของ Long-form
- Repurpose content
- Trending topics

### 7. Monetization

**YouTube Partner Program (YPP):**
- 1,000 Subscribers
- 4,000 Watch Hours (12 เดือน) หรือ
- 10 Million Shorts Views (90 วัน)

**Revenue Streams:**
| Source | Description | Typical % |
|--------|-------------|-----------|
| AdSense | โฆษณาจาก Google | 50-80% |
| Sponsorship | Brand ว่าจ้าง | 10-30% |
| Affiliate | ลิงก์ขายของ | 5-15% |
| Membership | Channel Members | 5-20% |
| Super Chat | Live donations | 2-10% |
| Merchandise | ขายของ | 5-15% |

**CPM (Cost per 1000 views):**
- Thailand: $0.30-1.50
- USA: $3-8
- Finance/Business: สูงสุด
- Entertainment: ต่ำสุด

### 10. YouTube Ads

**Ad Types:**
| Type | Description | Best For |
|------|-------------|----------|
| Skippable In-stream | ข้ามได้หลัง 5 วิ | Awareness, Traffic |
| Non-skippable | 15-20 วิ ข้ามไม่ได้ | Brand message |
| Bumper | 6 วินาที | Brand recall |
| Discovery | แสดงในผลค้นหา | Targeted reach |
| Masthead | Banner หน้าแรก | Mass awareness |

**Targeting:**
- Demographics (อายุ, เพศ)
- Interests (หมวดหมู่ที่สนใจ)
- Keywords (คำค้นหา)
- Placements (ช่อง/วิดีโอเฉพาะ)
- Custom Audiences

**Campaign Objectives:**
1. Awareness (Reach, Brand awareness)
2. Consideration (Views, Website traffic)
3. Conversion (Sales, Leads)
