## 2. Instagram Algorithm 2026

### 2.1 Ranking Factors

| Factor | Description | Weight |
|--------|-------------|--------|
| Interest | ความสนใจของผู้ใช้ | High |
| Relationship | ความสัมพันธ์กับ Account | High |
| Timeliness | ความใหม่ของ Content | Medium |
| Frequency | ความถี่ในการใช้งาน | Medium |
| Following | จำนวนคนที่ติดตาม | Low |
| Session Time | ระยะเวลาใช้งาน | Low |

### 2.2 Algorithm by Format

| Format | Primary Factors | Reach Potential |
|--------|-----------------|-----------------|
| Feed | Relationship + Interest | Medium (Followers) |
| Stories | Recency + Engagement | Low (Followers only) |
| Reels | Entertainment Value | High (Non-followers) |
| Explore | New Discovery + Interest | High (Non-followers) |

### 2.3 Key Insight

**Reels** ได้รับ Algorithm Push มากที่สุด เพราะ Instagram แข่งขันกับ TikTok ทำให้เป็น Format ที่ดีที่สุดในการเข้าถึงคนใหม่

## 3. Content Types

### 3.1 Feed Posts

| Type | Dimensions | Best For |
|------|------------|----------|
| Single Image (Square) | 1080x1080 | Simple message |
| Single Image (Portrait) | 1080x1350 | Mobile-optimized |
| Video | 1080x1350, < 60 sec | Engagement |
| Carousel | 1080x1080, 2-10 slides | Education, Storytelling |

### 3.2 Stories

**Specifications:**
- Size: 1080x1920 (9:16)
- Duration: 15 seconds per story
- Lifespan: 24 hours

**Interactive Elements:**
| Element | Use Case |
|---------|----------|
| Poll | This or That |
| Quiz | Test knowledge |
| Questions | Q&A |
| Slider | Rating |
| Countdown | Launch events |

### 3.3 Reels

**Specifications:**
- Size: 1080x1920 (9:16)
- Duration: 15-90 seconds (7-15 optimal)
- Lifespan: Permanent

**Best Practices:**
- Hook ใน 1 วินาทีแรก
- ใช้ Trending audio
- Add text overlay (85% ดูไม่มีเสียง)
- Strong CTA ท้ายคลิป

### 3.4 Lives

**Specifications:**
- Max Duration: 4 hours
- Can save to IGTV
- Best For: Q&A, Product launches, Behind-the-scenes

### 3.5 Posting Frequency Recommendations

| Format | Frequency | Best Times (Thailand) |
|--------|-----------|----------------------|
| Feed | 3-5 ครั้ง/สัปดาห์ | 7-9 AM, 12-1 PM, 7-9 PM |
| Stories | 5-10 ครั้ง/วัน | Throughout the day |
| Reels | 4-7 ครั้ง/สัปดาห์ | 7-9 PM |
| Lives | 1-2 ครั้ง/สัปดาห์ | 7-9 PM |

## 4. Profile Optimization

### 4.1 Profile Components

| Component | Specification | Best Practice |
|-----------|---------------|---------------|
| Username | 30 characters max | จำง่าย, searchable |
| Name | 30 characters max | ใส่ keywords |
| Profile Picture | 320x320 | ใบหน้าชัด หรือ Logo |
| Bio | 150 characters | Value proposition + CTA |
| Link | 1 link | Linktree or Landing page |
| Highlights | Unlimited | จัดหมวดหมู่ Stories |

### 4.2 Bio Formula

```
Line 1: Who you are (ชื่อ/บทบาท)
Line 2: What you do (ช่วยอะไร)
Line 3: Why follow (ประโยชน์ที่จะได้)
Line 4: CTA (ทำอะไรต่อ)
```

**Example:**
```
Kong | Marketing Coach
ช่วย SME เพิ่มยอดขาย 3x
Tips การตลาดทุกวัน
คลิก Link เรียนฟรี
```

### 4.3 Highlight Categories

| Category | Purpose |
|----------|---------|
| About/Intro | แนะนำตัว, Brand story |
| Products/Services | แสดงสินค้า/บริการ |
| Testimonials | รีวิวจากลูกค้า |
| FAQ | คำถามที่พบบ่อย |
| Behind-the-scenes | เบื้องหลัง, ความเป็นมนุษย์ |

## 5. Hashtag Strategy

### 5.1 The 30 Hashtags Formula

| Category | Posts Range | Purpose | Example |
|----------|-------------|---------|---------|
| Broad | 500K-5M | Wide reach | #marketing #business |
| Niche | 50K-500K | Targeted audience | #thaimarketing #contentcreator |

| Mix ตาม Formula | Balance reach และ relevance |

## 6. Engagement Strategies

### 6.1 Increase Engagement

| Strategy | Why It Works |
|----------|--------------|
| ตอบ Comment ใน 60 นาทีแรก | Algorithm ชอบ early engagement |
| ใช้ CTA ใน Caption | บอกให้คน interact |
| Post เวลาที่ Audience online | ได้ engagement ทันที |
| ใช้ Interactive Stories | เพิ่ม interaction |
| Engage กับ Account อื่น | Reciprocity |

### 6.2 Caption Formula

```
HOOK (ดึงความสนใจ)
    ↓
VALUE (ให้คุณค่า)
    ↓
CTA (บอกให้ทำอะไร)
```

**Example:**
```
HOOK: รู้ไหมว่า 90% ของคนไม่รู้เรื่องนี้?

VALUE: [Content ที่ให้ความรู้/คุณค่า]

CTA: กด Save ไว้อ่านทีหลัง และ Comment บอกว่าคิดยังไง
```

### 6.3 Engagement Rate Benchmarks

| Followers | Good | Excellent |
|-----------|------|-----------|
| < 10K | > 5% | > 8% |
| 10K-100K | > 3% | > 5% |
| 100K-1M | > 1.5% | > 3% |
| 1M+ | > 1% | > 2% |

## 11. Common Mistakes

| Mistake | Impact | Solution |
|---------|--------|----------|
| ไม่สม่ำเสมอ | Algorithm ลดการแสดง | สร้าง Content Calendar |
| Hashtag เดิมซ้ำๆ | Shadowban risk | เปลี่ยนทุกโพสต์ |
| ไม่ตอบ Comment | Engagement ลด | ตอบใน 60 นาที |
| Focus แค่ Followers | ไม่มี engagement | Focus คุณภาพ |
| ไม่ใช้ Reels | พลาด reach | Post Reels 4-7/สัปดาห์ |
| Bio ไม่ชัด | คนไม่ Follow | Optimize Bio |

## Key Takeaways

1. **Reels = ทางลัดสู่ Growth** - Algorithm push มากที่สุด
2. **30 Hashtags Strategy** - Mix Broad + Niche + Branded
3. **Profile = First Impression** - Bio, Link, Highlights ต้องครบ
4. **Engagement ภายใน 60 นาที** - Algorithm ชอบ early engagement
5. **Consistency > Perfection** - โพสต์สม่ำเสมอสำคัญกว่าสมบูรณ์แบบ
6. **Analyze Weekly** - ดู Insights ปรับปรุงตลอด
7. **Community First** - Engage กับคนอื่นก่อน
