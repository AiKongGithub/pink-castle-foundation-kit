## Key Points Covered

### 1. ทำไมต้อง Follow-up?
- 80% ของยอดขายเกิดจาก Follow-up ครั้งที่ 5-12
- คนส่วนใหญ่เลิก Follow-up หลังครั้งที่ 2
- Follow-up = โอกาสทอง
- สร้าง relationship ระยะยาว

### 2. Follow-up Timing ที่เหมาะสม
- **Day 1:** Thank you + recap
- **Day 2-3:** Additional value
- **Day 4-5:** Case study / testimonial
- **Day 7:** Reminder + urgency
- **Day 14:** Last chance / new offer

### 3. Follow-up Automation
- ใช้ Email Autoresponder (ConvertKit, ActiveCampaign)
- Set trigger-based sequences
- Personalization tokens
- A/B testing subject lines

---

## Key Takeaways

1. ความพยายามใน Follow-up คือกุญแจสู่ยอดขาย
2. Timing สำคัญมาก — ไม่เร็วเกินไป ไม่ช้าเกินไป
3. Automation ช่วยให้ consistent และ scalable
4. ทุก Follow-up ต้องให้ value ไม่ใช่แค่ขาย
