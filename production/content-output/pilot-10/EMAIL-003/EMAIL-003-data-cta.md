## Table 1: 4C Framework Reference

| C | Principle | Description | Example |
|---|-----------|-------------|---------|
| Clear | ชัดเจน | บอกให้ชัดว่าต้องทำอะไร | "ดาวน์โหลด" ไม่ใช่ "คลิกที่นี่" |
| Compelling | น่าสนใจ | มีเหตุผลให้คลิก | "รับฟรี มูลค่า 997 บาท" |
| Visible | เห็นชัด | สีเด่น ขนาดใหญ่ หาเจอง่าย | ปุ่มสี contrast |
| Single | เดียว | มี CTA หลักแค่ 1 อัน | ไม่ใส่หลาย CTA |

## Table 2: Bad vs Good CTA Examples

| Bad CTA ❌ | Good CTA ✅ | Why Better? |
|-----------|------------|-------------|
| คลิกที่นี่ | ดาวน์โหลด Ebook ฟรี | บอก action + benefit |
| ดูเพิ่มเติม | ดู Case Study ยอด 3 ล้าน | specific + curiosity |
| ติดต่อเรา | นัดคุยฟรี 30 นาที | บอก value ที่ได้รับ |
| Submit | ส่งใบสมัครรับของแถม | บอก incentive |
| Learn More | เรียนรู้วิธีเพิ่มยอด 200% | result-focused |
| Buy Now | ซื้อเลย ลด 50% วันนี้เท่านั้น | urgency + discount |

## Table 3: CTA Types Comparison

| Type | Format | Purpose | Quantity | Example |
|------|--------|---------|----------|---------|
| Primary | Button (large) | Main action | 1 per email | "สมัครเลย" |
| Secondary | Text link | Alternative | 1-2 per email | "อ่านเพิ่มเติมก่อน" |
| P.S. CTA | Text in P.S. | Reminder | 1 per email | "P.S. หมดเวลาพรุ่งนี้!" |

## Table 4: CTA by Email Type

## Table 5: Urgency Techniques

## Table 6: CTA Button Design Specs

| Element | Recommendation | Avoid |
|---------|----------------|-------|
| Color | Contrast กับพื้นหลัง | กลมกลืนกับพื้น |
| Size | Min 44x44px (mobile) | เล็กจนกดยาก |
| Shape | Rounded corners 4-8px | เหลี่ยมเกินไป |
| Padding | 12-16px vertical | แน่นเกินไป |
| Font | Bold, 14-16px | เล็กจนอ่านยาก |
| Whitespace | มีพื้นที่ว่างรอบ | ติดกับ element อื่น |

## Table 8: CTA Formula Components

| Component | Options | Examples |
|-----------|---------|----------|
| Action Verb | ดาวน์โหลด, สมัคร, ซื้อ, ดู, เข้าร่วม, รับ | "ดาวน์โหลด" |
| Benefit | ฟรี, ส่วนลด, เพิ่มยอด, ประหยัด | "Ebook ฟรี" |
| Urgency | วันนี้เท่านั้น, เหลือ X ที่, ก่อนหมดเวลา | "วันนี้เท่านั้น" |

### Formula:
```
[Action Verb] + [Benefit] + [Urgency (optional)]
```
