## 1. CTA คืออะไร?

### 1.1 นิยาม
Call to Action (CTA) คือประโยคหรือองค์ประกอบในอีเมลที่บอกให้ผู้อ่านทำสิ่งที่เราต้องการ เช่น คลิก, ซื้อ, สมัคร, ดาวน์โหลด

### 1.2 ความสำคัญ
| Without CTA | With CTA |
|-------------|----------|
| ผู้อ่านไม่รู้ว่าต้องทำอะไร | มีทิศทางชัดเจน |
| Conversion = 0% | Conversion วัดผลได้ |
| เสียเวลาอ่าน | นำไปสู่ action |

## 2. 4C Framework

### 2.1 Clear (ชัดเจน)

**หลักการ:** บอกให้ชัดว่าต้องทำอะไร

**ตัวอย่าง:**
- ❌ "ข้อมูลเพิ่มเติม"
- ✅ "ดาวน์โหลด Price List"

**Tips:**
- ใช้ verb เปิด (Action verb)
- บอกสิ่งที่จะได้รับ
- หลีกเลี่ยงคำกำกวม

### 2.2 Compelling (น่าสนใจ)

**หลักการ:** ให้เหตุผลว่าทำไมต้องคลิก

**เทคนิค:**
1. **Benefit-driven:** "ดาวน์โหลด Ebook ฟรี มูลค่า 997 บาท"
2. **Curiosity:** "ดูความลับที่ไม่เคยเปิดเผย"
3. **Urgency:** "รับส่วนลดก่อนหมดเวลา"
4. **Exclusivity:** "เฉพาะสมาชิก VIP เท่านั้น"

### 2.3 Visible (เห็นชัด)

**หลักการ:** CTA ต้องโดดเด่น หาง่าย

**Design Guidelines:**
| Element | Recommendation |
|---------|----------------|
| Color | Contrast กับพื้นหลัง |
| Size | ใหญ่พอให้เห็นชัด |
| Whitespace | มีพื้นที่ว่างรอบปุ่ม |
| Position | Above the fold หรือหลังเนื้อหาหลัก |

### 2.4 Single (เดียว)

**หลักการ:** มี CTA หลักแค่ 1 อย่าง

**Research:**
- อีเมลที่มี 1 CTA มี Click Rate สูงกว่า 371% (Campaign Monitor)
- ผู้อ่านมี Decision Fatigue เมื่อมีตัวเลือกเยอะ

## 3. ประเภทของ CTA

### 3.1 Primary CTA (หลัก)
- **รูปแบบ:** ปุ่มขนาดใหญ่
- **วัตถุประสงค์:** การกระทำหลักที่ต้องการ
- **จำนวน:** 1 ต่ออีเมล

### 3.2 Secondary CTA (รอง)
- **รูปแบบ:** Text link
- **วัตถุประสงค์:** สำหรับคนที่ยังไม่พร้อม
- **ตัวอย่าง:** "อ่านเพิ่มเติมก่อน", "ดู FAQ"

---

## 4. CTA Formula

```
[Action Verb] + [Benefit] + [Urgency/Exclusivity]
```

### ตัวอย่าง:

| Formula | Result |
|---------|--------|
| ดาวน์โหลด + Checklist ฟรี | ดาวน์โหลด Checklist ฟรี |
| สมัคร + รับส่วนลด 50% + วันนี้เท่านั้น | สมัครรับส่วนลด 50% วันนี้เท่านั้น |
| ซื้อ + ประหยัด 3,000 บาท | ซื้อเลย ประหยัด 3,000 บาท |

---

## 5. Common Mistakes

| Mistake | Problem | Solution |
|---------|---------|----------|
| Too many CTAs | สับสน | เลือก 1 หลัก |
| Vague wording | ไม่มีแรงจูงใจ | ใช้ benefit |
| Hidden button | หาไม่เจอ | สีเด่น, ตำแหน่งดี |
| No urgency | ไม่รีบคลิก | เพิ่ม deadline |

## 6. CTA by Email Type

| Email Type | Primary CTA | Example |
|------------|-------------|---------|
| Newsletter | Read more | อ่านบทความเต็ม |
| Promotional | Buy/Shop | ซื้อเลย รับส่วนลด |
| Educational | Download | ดาวน์โหลด Ebook |
| Webinar | Register | ลงทะเบียน ฟรี |
| Survey | Participate | ร่วมตอบแบบสอบถาม |
