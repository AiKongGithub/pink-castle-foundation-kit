## Branch Details

### 1. 4C Rules (กฎ 4C)
- **Clear (ชัดเจน)**
  - บอกให้ชัดว่าต้องทำอะไร
  - ใช้ verb เปิด: ดาวน์โหลด, สมัคร, ซื้อ

- **Compelling (น่าสนใจ)**
  - มีเหตุผลให้คลิก
  - บอกประโยชน์ที่ได้รับ
  - สร้าง urgency

- **Visible (เห็นชัด)**
  - ปุ่มสีเด่น contrast กับพื้นหลัง
  - ขนาดใหญ่พอให้เห็น
  - ตำแหน่งที่เหมาะสม

- **Single (เดียว)**
  - มี CTA หลักแค่ 1 ต่ออีเมล
  - ไม่สร้างความสับสน

### 2. Types of CTA (ประเภท)
- **Primary CTA (หลัก)**
  - ปุ่มใหญ่ สีเด่น
  - วัตถุประสงค์หลักของอีเมล
  - มีได้ 1 อัน

- **Secondary CTA (รอง)**
  - Text link
  - สำหรับคนที่ยังไม่พร้อม
  - เช่น "อ่านเพิ่มเติม"

### 3. Common Mistakes (ข้อผิดพลาด)
- **Too Many CTAs**
  - มีหลาย CTA ทำให้สับสน
  - ผู้อ่านไม่รู้ว่าต้องทำอะไรก่อน

- **Vague CTA**
  - "คลิกที่นี่" ไม่บอกประโยชน์
  - "ดูเพิ่มเติม" ไม่มีแรงจูงใจ

- **No Benefit**
  - ไม่บอกว่าจะได้อะไร
  - ขาดเหตุผลให้คลิก

- **Hidden CTA**
  - สีกลมกลืนกับพื้นหลัง
  - ตำแหน่งที่หาไม่เจอ

## CTA Formula

```
┌─────────────────────────────────────────────────┐
│  ACTION VERB + BENEFIT + URGENCY (optional)     │
├─────────────────────────────────────────────────┤
│  ดาวน์โหลด + Ebook ฟรี + วันนี้เท่านั้น          │
│  สมัคร + รับส่วนลด 50% + เหลือ 10 ที่             │
│  ซื้อ + ประหยัด 3,000 บาท + ก่อนราคาขึ้น         │
└─────────────────────────────────────────────────┘
```
