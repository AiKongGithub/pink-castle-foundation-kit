## Table 3: On-page SEO Checklist

| Element | Best Practice | Character Limit |
|---------|---------------|-----------------|
| Title Tag | Keyword at start | 60 |
| Meta Description | Summary + CTA | 155 |
| URL Slug | Short, keyword | 50-60 |
| H1 | One per page | No limit |
| H2/H3 | Structure content | No limit |
| Image Alt | Descriptive | 125 |
| Internal Links | Related content | 3-5 per post |
| External Links | Authoritative | 2-3 per post |
