## Table 2: Storytelling Framework

| Step | Thai | Purpose | Example |
|------|------|---------|---------|
| Problem | ปัญหา | สร้าง empathy | "ผมเคยล้มเหลวกับ video..." |

## Table 3: Why Stories Work

| Factor | Impact | Data |
|--------|--------|------|
| Memory Retention | จำได้นานกว่า | 22x vs facts alone |
| Emotional Connection | สร้าง trust เร็ว | 65% prefer story content |
| Engagement | Watch time สูงขึ้น | +30% avg watch time |
| Relatability | เห็นตัวเองใน story | Higher conversion |

## Table 4: Hook Types & Examples

| Type | Description | Example | Best For |
|------|-------------|---------|----------|
| Question | ถามคำถามที่อยากรู้คำตอบ | "เคยสงสัยไหมว่าทำไม...?" | Curiosity-driven |
| Statistic | ใช้ตัวเลขน่าตกใจ | "82% ของ..." | Data-driven |
| Story | เริ่มด้วยเรื่องเล่า | "เมื่อ 2 ปีก่อน..." | Emotional |
| Curiosity | สร้างความอยากรู้ | "สิ่งที่ไม่ค่อยบอก..." | Mystery |

---

## Table 5: Hook Formula Components

| Component | Description | Example |
|-----------|-------------|---------|
| Attention Grabber | สิ่งที่หยุดการ scroll | "82% ของ internet traffic คือ video" |
| Promise | สิ่งที่จะได้เรียนรู้ | "3 วิธีเพิ่ม view 10x" |
| Credibility | ทำไมควรฟังเรา | "ใช้สร้าง channel 100K" |

## Table 6: CTA Types by Goal

| CTA Type | Goal | When to Use |
|----------|------|-------------|
| Subscribe | Build audience | End of every video |
| Like | Boost engagement | Mid + End |
| Comment | Engagement | After question |
| Link in bio | Lead generation | Offer videos |
| Watch next | Retention | Series content |
| Share | Viral | Emotional content |

## Table 7: Video Drop-off Points

| Timestamp | % Leaving | Reason |
|-----------|-----------|--------|
| 0-10 sec | 20% | Bad hook |
| 10-30 sec | 15% | No clear value |
| Mid-video | 10-20% | Boring content |
| Before CTA | 5% | Too long |
