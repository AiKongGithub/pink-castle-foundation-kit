## Branch Details

### 1. Script Structure (Hook-Content-CTA)

- **Hook (15 วินาทีแรก)**
  - จับความสนใจทันที
  - สร้าง curiosity
  - บอก benefit

- **Content (เนื้อหาหลัก)**
  - Point 1, 2, 3
  - Examples / Stories
  - Data / Proof
  - Visual support

- **CTA (30 วินาทีสุดท้าย)**
  - Subscribe / Like
  - Link in description
  - Next video
  - Offer

- **Transitions**
  - บอกล่วงหน้าว่าจะพูดอะไร
  - สรุปก่อนไปหัวข้อใหม่
  - ใช้ visual cues

### 2. Storytelling Framework

- **Problem**
  - ปัญหาที่ผู้ชมเผชิญ
  - สร้าง empathy
  - Relatable situation

- **Struggle**
  - ความพยายามแก้ปัญหา
  - ความล้มเหลว
  - Frustration

- **Discovery**
  - ค้นพบวิธีใหม่
  - Turning point
  - Aha moment

- **Transformation**
  - ผลลัพธ์ที่ได้
  - Before/After
  - Proof

- **Lesson**
  - บทเรียนที่ได้
  - Key takeaway
  - Actionable advice

### 3. Hook Techniques

- **Question Hook**
  - "เคยสงสัยไหมว่า...?"
  - "รู้ไหมว่าทำไม...?"
  - สร้าง curiosity

- **Statistic Hook**
  - "82% ของ..."
  - ตัวเลขน่าตกใจ
  - Data-driven

- **Story Hook**
  - "เมื่อ 2 ปีก่อน..."
  - Personal experience
  - Emotional connection

- **Curiosity Hook**
  - "สิ่งที่คนส่วนใหญ่ไม่รู้..."
  - "ความลับของ..."
  - Open loop
