## 1. Script Structure: Hook-Content-CTA

### 1.1 Overview

| Component | Duration | Purpose |
|-----------|----------|---------|
| Hook | 15 sec | จับความสนใจ |
| Content | Main body | ให้ value |
| CTA | 30 sec | บอกขั้นตอนถัดไป |

### 1.2 Hook (15 วินาทีแรก)

**Critical Importance:**
- 20% ของ viewers ออกใน 10 วินาทีแรก
- Hook ต้องจับความสนใจทันที
- บอกว่าจะได้อะไรจาก video นี้

**Hook Elements:**
- Bold statement / Claim
- Question
- Teaser of value
- Visual attention grabber

### 1.3 Content Structure

```
Intro (What you'll learn)
  ↓
Point 1 (+ Example/Story)
  ↓
Transition
  ↓
Point 2 (+ Example/Story)
  ↓
Transition
  ↓
Point 3 (+ Example/Story)
  ↓
Summary
```

### 1.4 CTA (Call to Action)

| CTA Type | When to Use |
|----------|-------------|
| Subscribe | Building audience |
| Like/Comment | Engagement boost |
| Link in bio | Lead generation |
| Watch next | Retention |

## 2. Storytelling Framework

### 2.1 5-Step Story Arc

```
Problem → Struggle → Discovery → Transformation → Lesson
```

### 2.2 Detailed Breakdown

| Step | Purpose | Example |
|------|---------|---------|
| Problem | สร้าง empathy | "ผมเคยล้มเหลวกับ video..." |
| Struggle | แสดง journey | "ลองหลายวิธี ไม่ได้ผล..." |
| Discovery | จุดเปลี่ยน | "จนกระทั่งเจอเทคนิคนี้..." |
| Transformation | ผลลัพธ์ | "ตอนนี้ได้ view เพิ่ม 10x..." |
| Lesson | Key takeaway | "สิ่งที่เรียนรู้คือ..." |

### 2.3 Why Stories Work

| Factor | Impact |
|--------|--------|
| Emotional connection | Trust เร็วขึ้น |
| Memory retention | จำได้นาน 22x |
| Engagement | Watch time สูงขึ้น |
| Relatability | ผู้ชมเห็นตัวเองใน story |

## 3. Hook Techniques

### 3.1 Four Main Types

| Type | Description | Example |
|------|-------------|---------|
| Question | ถามคำถามที่อยากรู้คำตอบ | "เคยสงสัยไหมว่าทำไม video บางอันดัง?" |
| Statistic | ใช้ตัวเลขน่าตกใจ | "82% ของ internet traffic คือ video" |
| Story | เริ่มด้วยเรื่องเล่า | "เมื่อ 2 ปีก่อน ผมเกือบเลิกทำ video..." |
| Curiosity | สร้างความอยากรู้ | "สิ่งที่ YouTuber ไม่ค่อยบอกคือ..." |

### 3.2 Hook Formula

```
[Attention Grabber] + [Promise/Benefit] + [Credibility]
```

**Example:**
"82% ของ internet traffic คือ video (Attention) แต่ส่วนใหญ่ทำผิดวิธี วันนี้ผมจะแชร์ 3 วิธีที่ช่วยให้ view เพิ่ม 10x (Promise) ซึ่งเป็นสิ่งที่ผมใช้สร้าง channel 100K (Credibility)"

## 4. Script Templates

### 4.1 How-to Video

```
Hook: "ปัญหาที่คุณเจอ + สิ่งที่จะได้เรียนรู้"
Step 1: [Action] + [Why] + [Example]
Step 2: [Action] + [Why] + [Example]
Step 3: [Action] + [Why] + [Example]
Summary: "สรุป 3 ขั้นตอน"
CTA: "Subscribe สำหรับ tips เพิ่มเติม"
```

### 4.2 Story Video

```
Hook: "เริ่มจากจุดน่าสนใจของเรื่อง"
Problem: "ปัญหาที่เผชิญ"
Struggle: "ความพยายามและความล้มเหลว"
Discovery: "จุดเปลี่ยน"
Transformation: "ผลลัพธ์"
Lesson: "บทเรียนสำหรับผู้ชม"
CTA: "Share if this resonates"
```
