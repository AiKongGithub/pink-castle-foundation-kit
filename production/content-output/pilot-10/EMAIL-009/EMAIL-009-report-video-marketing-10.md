## 2. Video Types by Funnel

### 2.1 TOFU (Awareness)

| Type | Length | Goal |
|------|--------|------|
| Shorts/Reels | 15-60 sec | Viral, discovery |
| How-to | 5-10 min | Answer questions |

### 2.2 MOFU (Consideration)

|------|--------|------|

| Type | Length | Goal |
|------|--------|------|

| Sales video | 10-30 min | Close sale |

## 3. Production Essentials

### 3.1 Priority Order

```
1. Audio (Most Important)
2. Lighting
3. Camera
4. Editing
```

### 3.2 Audio Setup

| Level | Equipment | Budget |
|-------|-----------|--------|
| Basic | Phone mic | Free |
| Good | Lavalier | 500-1,500 THB |
| Great | USB condenser | 2,000-5,000 THB |
| Pro | XLR + Interface | 10,000+ THB |

### 3.3 Lighting Setup

| Setup | Description | Best For |
|-------|-------------|----------|
| Natural | หน้าหน้าต่าง | Budget-friendly |
| Ring light | วงกลม LED | Face close-ups |
| Softbox | แสงกระจาย | Pro look |
| 3-point | Key, Fill, Back | Studio quality |

### 3.4 Camera Options

| Option | Quality | Notes |
|--------|---------|-------|
| Smartphone | 1080p-4K | เพียงพอสำหรับเริ่มต้น |
| Webcam | 720p-1080p | สะดวกสำหรับ live |
| DSLR | 1080p-4K | Professional look |

## 4. Platform Strategy

### 4.1 Platform Comparison

| Platform | Strength | Best Content | Audience |

| YouTube | Search, Long-form | Tutorials, Vlogs | All ages |
| TikTok | Discovery, Viral | Shorts, Trends | Gen Z |

```
Create for TikTok → Repurpose to IG Reels → Long version for YouTube
```

## 5. Video Script Framework

### 5.1 Hook-Content-CTA

```
HOOK (0:00-0:15): จับความสนใจ
CONTENT (0:15-X:XX): ให้คุณค่า
CTA (End): บอกให้ทำอะไร
```

### 5.2 Example Structure

| Section | Time | Content |
|---------|------|---------|
| Hook | 15 sec | Bold statement / Question |
| Intro | 30 sec | What they'll learn |
| Point 1 | 2 min | First key concept |
| Point 2 | 2 min | Second key concept |
| Point 3 | 2 min | Third key concept |
| Summary | 1 min | Recap |
| CTA | 30 sec | Subscribe / Link |
