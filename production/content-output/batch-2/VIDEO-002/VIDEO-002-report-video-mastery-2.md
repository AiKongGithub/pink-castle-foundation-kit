4. อธิบายวิธีการวิเคราะห์กลุ่มเป้าหมาย

### 1.3 Methodology

การวิจัยนี้ใช้การรวบรวมข้อมูลจาก:

## 4. Topic Research Methods

### 4.1 Five-Method Framework

#### Method 1: Audience-First Approach

| Source | What to Look For | Action |
|--------|-----------------|--------|
| Comments | คำถามซ้ำๆ | รวบรวมเป็น FAQ video |
| DMs | ปัญหาที่แชร์ | สร้าง Solution content |
| Polls/Surveys | ความต้องการ | ทำตามที่ Vote สูง |
| Live Q&A | Real-time questions | บันทึกและทำ video ตอบ |
| Community posts | Engagement patterns | ดูว่าหัวข้อไหน Popular |

#### Method 2: Search-Based Approach

| Tool | How to Use | Output |
|------|------------|--------|
| YouTube Autocomplete | พิมพ์ keyword ดู suggestions | Long-tail keywords |
| Google Trends | เปรียบเทียบ trends | Seasonal topics |
| Answer The Public | ดูคำถามที่คนถาม | Question-based content |
| TubeBuddy | Keyword research | SEO-optimized titles |
| VidIQ | Competitor analysis | Trending topics |
| Ahrefs/SEMrush | Deep keyword research | Content gaps |

#### Method 3: Competitor Analysis

| What to Analyze | Why | How |
|-----------------|-----|-----|
| Top performing videos | ดูว่าอะไร Works | Sort by views |
| Comment themes | ดูว่าคนต้องการอะไร | Read top comments |
| Content gaps | หา opportunity | Note what's missing |
| Publishing schedule | เข้าใจ rhythm | Track for 1 month |
| Engagement patterns | Learn best practices | Analyze metrics |

#### Method 4: Content Pillars

| Pillar Type | Description | Example (Cooking Channel) |
|-------------|-------------|---------------------------|
| Core | Main expertise | สูตรอาหาร |
| Educational | Teaching value | เทคนิคการทำ |
| Entertainment | Fun content | Challenge, BTS |
| Promotional | Product-related | รีวิวอุปกรณ์ |
| Community | Audience connection | Q&A, Collab |

#### Method 5: Content Repurposing

| Original Format | Repurposed To | Effort Level |
|-----------------|---------------|--------------|

| Webinar | Multiple tutorials | Medium |
