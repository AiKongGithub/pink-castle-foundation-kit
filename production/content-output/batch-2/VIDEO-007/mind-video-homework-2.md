## Branch 1: SCRIPT REVIEW FRAMEWORK

```
SCRIPT REVIEW FRAMEWORK
├── 1. Hook Analysis
│   ├── มาใน 3-5 วินาทีไหม?

│

│
├── 4. CTA Evaluation
│   ├── ชัดเจนไหม?
```

## Branch 3: COMMON MISTAKES

```
COMMON MISTAKES
├── 1. Weak Hook
│   ├── Problem: "สวัสดีครับ..."
│   ├── Impact: คน Skip ทันที
│   └── Fix: เริ่มด้วย Question/Promise
│
├── 2. Long Introduction
│   ├── Problem: 1-2 นาที Intro
│   ├── Impact: 40% Drop-off
│   └── Fix: Max 15 วินาที
│
├── 3. No Clear Structure
│   ├── Problem: พูดวนไปวนมา
│   ├── Impact: คนสับสน
│   └── Fix: ใช้ AIDA/BAB
│
├── 4. Poor Audio
│   ├── Problem: เบา/Noise/Echo
│   ├── Impact: 60% ปิดวิดีโอ
│   └── Fix: External Mic + Noise Removal
│
└── 5. Too Many CTAs
    ├── Problem: 5+ CTAs
    ├── Impact: Conversion ต่ำ
    └── Fix: เลือก 1 Primary CTA
```

## Review Scoring Matrix

```
┌─────────────────────────────────────────┐
│         REVIEW SCORING MATRIX           │
├──────────────┬──────┬───────────────────┤
│ Category     │Points│ Criteria          │
├──────────────┼──────┼───────────────────┤
│ Hook         │ /20  │ Strong, 3-5 sec   │
│ Structure    │ /20  │ Clear AIDA/BAB    │
│ Storytelling │ /15  │ Personal + BAB    │
│ Audio        │ /15  │ Clear, No Noise   │
│ Visual       │ /10  │ Good Lighting     │
│ Editing      │ /10  │ Clean, Paced      │
│ CTA          │ /10  │ Clear, Single     │
├──────────────┼──────┼───────────────────┤
│ TOTAL        │ /100 │                   │
└──────────────┴──────┴───────────────────┘

Grades:
90-100 = Excellent (พร้อม Publish)
80-89  = Good (Minor fixes)
70-79  = Fair (Some work needed)
60-69  = Needs Improvement
<60    = Major Revision
```
