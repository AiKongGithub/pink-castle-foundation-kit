**[SCREEN RECORDING - YouTube Settings]**

**VISUAL**: Split screen แสดง Personal vs Brand Channel

**NARRATION:**
"YouTube มี 2 ประเภท Channel"

**[GRAPHIC - Personal Channel]**
- ใช้ชื่อ Google Account
- เหมาะกับ Creator รายบุคคล
- จัดการได้คนเดียว

**[GRAPHIC - Brand Channel]**
- แยกเป็น Brand ต่างหาก
- เพิ่ม Manager ได้หลายคน

### Scene 2: Essential Channel Settings (2:30-4:30)

**[SCREEN RECORDING - YouTube Studio]**

**VISUAL**: Navigate ผ่าน Settings

**NARRATION:**
"Settings ที่ต้องตั้งค่าทันที"

**[HIGHLIGHT - Basic Info]**
- Channel Name: ชัดเจน จดจำง่าย
- Handle: @yourname สำหรับ mention
- Description: อธิบาย Channel ใน 3-4 ย่อหน้า

**[HIGHLIGHT - Upload Defaults]**
- Title: Template สำหรับชื่อวิดีโอ
- Description: Default description
- Tags: Tags ที่ใช้ทุกวิดีโอ
- Visibility: Public/Private/Unlisted

**[HIGHLIGHT - Permissions]**
- เพิ่ม Manager และ Editor
- กำหนดสิทธิ์แต่ละคน

### Scene 3: Visual Branding (4:30-7:30)

**[DESIGN SOFTWARE DEMO]**

**VISUAL**: Canva/Photoshop creating assets

**NARRATION:**
"Visual Branding มี 3 องค์ประกอบหลัก"

**[DEMO - Creating Banner]**
- ใช้ Template ที่มี Safe Area

- Preview ใน TV, Desktop, Mobile

**[GRAPHIC 3 - Video Watermark]**
- ขนาด: 150x150 pixels
- ช่วย Subscribe จากในวิดีโอ
- แสดงมุมล่างขวา

### Scene 5: Channel Optimization (9:30-11:30)

**[SCREEN RECORDING - YouTube Studio]**

**VISUAL**: Settings > Channel section

**NARRATION:**
"Optimization ช่วยให้ Channel ถูกค้นพบ"

**[HIGHLIGHT - Channel Keywords]**
- ไป Settings > Channel > Basic Info
- ใส่ Keywords หลัก 5-10 คำ
- เกี่ยวข้องกับเนื้อหา

**[HIGHLIGHT - About Section]**
- ย่อหน้าแรก: บอกว่า Channel เกี่ยวกับอะไร
- ย่อหน้าสอง: ผู้ชมจะได้อะไร
- ย่อหน้าสาม: ตารางอัปโหลด
- ย่อหน้าสี่: Contact และ Links

**[HIGHLIGHT - Channel Sections]**
- Customize channel > Add section
- Featured Video สำหรับ New Visitors
- Popular Uploads
- Playlists ตามหมวดหมู่

### Scene 6: Analytics Overview (11:30-13:30)

**[SCREEN RECORDING - YouTube Analytics]**

**VISUAL**: Dashboard และ Metrics

**NARRATION:**
"Analytics บอกทุกอย่างที่ต้องรู้"

**[HIGHLIGHT - Key Metrics]**

**Watch Time:**
- สำคัญที่สุดสำหรับ Algorithm
- YouTube ให้ความสำคัญกับเวลารวม

**Average View Duration:**
- ผู้ชมดูนานแค่ไหน
- เป้าหมาย: 50%+ ของความยาววิดีโอ

**CTR (Click-Through Rate):**
- % ที่คลิกจาก Impressions
- เป้าหมาย: 4-10%

**[HIGHLIGHT - Traffic Sources]**

"รู้ว่าผู้ชมมาจากไหน"
- YouTube Search
- Suggested Videos
- Browse Features
- External Sources
