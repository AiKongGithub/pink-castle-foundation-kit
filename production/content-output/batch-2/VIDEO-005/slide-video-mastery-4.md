## Slide 3: AIDA Structure

### Content
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│              📐 AIDA STRUCTURE                      │
│                                                     │
│   A │ ATTENTION │ 0-5 วินาที    │ ดึงความสนใจ       │
│   ──┼───────────┼───────────────┼─────────────────  │
│   I │ INTEREST  │ 5-35 วินาที   │ สร้างความสนใจ     │
│   ──┼───────────┼───────────────┼─────────────────  │
│   D │ DESIRE    │ เนื้อหาหลัก    │ สร้างความต้องการ   │
│   ──┼───────────┼───────────────┼─────────────────  │
│   A │ ACTION    │ ตอนท้าย       │ CTA              │
│                                                     │
│   ┌─────────────────────────────────────────────┐   │
│   │         ทุกวิดีโอต้องมี 4 ส่วนนี้              │   │
│   └─────────────────────────────────────────────┘   │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Speaker Notes
- Attention: ช่วง Make or Break
- Interest: ทำให้คนรู้สึก "นี่คือปัญหาของเรา"
- Desire: แสดง Solution และ Value
- Action: บอกให้ทำอะไร ต้องชัดเจน

## Slide 4: Hook - The Critical First Seconds

### Content
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│           🎣 HOOK: 3-5 วินาทีแรก                    │
│                                                     │
│   ┌─────────────────────────────────────────────┐   │
│   │                                             │   │
│   │        20% ของผู้ชม                         │   │
│   │        ตัดสินใจ Skip/Watch                  │   │
│   │        ใน 3 วินาทีแรก                        │   │
│   │                                             │   │
│   └─────────────────────────────────────────────┘   │
│                                                     │
│   ❌ Bad Hook:                                      │
│   "สวัสดีครับ วันนี้ผมจะมาพูดเรื่อง..."              │
│                                                     │
│   ✅ Good Hook:                                     │
│   "80% ของคน Skip วิดีโอภายใน 3 วินาที              │
│    คุณกำลังจะได้เรียนรู้วิธีไม่ให้โดน Skip"          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Speaker Notes
- เปรียบเทียบ Bad vs Good Hook
- เน้นว่าห้ามเริ่มด้วย "สวัสดีครับ" เฉยๆ
- ต้องมีสิ่งที่ดึงดูดทันที

### Content
```
┌─────────────────────────────────────────────────────┐

│          📖 BAB STORYTELLING FRAMEWORK              │
│                                                     │
│                                                     │
│      ╔═══════════════════════════════════╗          │
│      ║  B E F O R E                      ║          │
│      ║  😰 "คุณใช้เวลา 3 ชม.ตัดต่อวิดีโอ   ║          │
│      ║     5 นาที รู้สึกเหนื่อยและท้อ"     ║          │
│      ╚═══════════════════════════════════╝          │
│                      ↓                              │
│      ╔═══════════════════════════════════╗          │

│      ║  😊 "ลองนึกว่าตัดได้ใน 30 นาที      ║          │
│      ║     และมีเวลาทำเรื่องอื่น"          ║          │
│      ╚═══════════════════════════════════╝          │
│                      ↓                              │
│      ╔═══════════════════════════════════╗          │
│      ║  B R I D G E                      ║          │
│      ║  🌉 "วันนี้ผมจะสอน Workflow        ║          │
│      ║     ที่ช่วยประหยัดเวลา 80%"        ║          │
│      ╚═══════════════════════════════════╝          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Speaker Notes
- BAB เป็น Framework ที่ง่ายและได้ผลที่สุด
- Before: ต้อง Relatable กับผู้ชม
- After: สร้างภาพผลลัพธ์ที่น่าดึงดูด
- Bridge: คุณคือทางเชื่อม
