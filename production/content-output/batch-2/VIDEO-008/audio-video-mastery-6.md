## Episode: Thumbnails, Titles & Video SEO - ทำให้คนคลิกวิดีโอคุณ

### Opening (1 นาที)

**Host**: สวัสดีครับทุกคน ยินดีต้อนรับสู่ Pink Castle Podcast วันนี้เราจะมาพูดถึงสิ่งที่ทำให้คนตัดสินใจคลิกวิดีโอของคุณ นั่นคือ **Thumbnails, Titles และ SEO** คุณอาจมีเนื้อหาดีที่สุดในโลก แต่ถ้าไม่มีใครคลิกดู ก็ไม่มีความหมาย มาเรียนรู้วิธีทำให้วิดีโอคุณถูกค้นพบและถูกคลิกกันครับ

### Segment 1: ทำไม Thumbnail และ Title สำคัญ (2 นาที)

**Host**: มาดูว่าทำไมสองสิ่งนี้ถึงสำคัญมาก...

**Key Points**:
- **CTR (Click-Through Rate)** กำหนดว่า Algorithm จะ Push วิดีโอหรือไม่
- YouTube แนะนำวิดีโอที่ CTR สูงให้คนดูมากขึ้น
- Thumbnail + Title = First Impression = ตัดสินภายใน 1-2 วินาที
- Thumbnail ดี + Title ดี = CTR สูง = Views สูง

**Statistics**:
- 90% ของวิดีโอที่ Performance ดี มี Custom Thumbnail
- Thumbnail ที่ดีเพิ่ม CTR ได้ถึง 150%
- Title ที่ Optimized เพิ่ม Search visibility 40%

### Segment 2: Thumbnail Design Principles (4 นาที)

**Host**: มาดูหลักการออกแบบ Thumbnail ที่ได้ผลกัน

**Principle 1: High Contrast & Bold Colors**
- ใช้สีที่ตัดกับ Background ของ YouTube (สีขาว)
- สีเหลือง, แดง, เขียว เห็นชัด
- หลีกเลี่ยงสีเทา, สีเข้มเกินไป

**Principle 2: Face with Emotion**
- ใบหน้าดึงดูดสายตา (Human instinct)
- Emotion ที่ชัดเจน: ประหลาดใจ, ตื่นเต้น, Curious
- Eye contact กับ Viewer
- ใบหน้าใหญ่ 40%+ ของ Thumbnail

**Principle 3: Minimal Text**
- ข้อความไม่เกิน 3-5 คำ
- Font ใหญ่ อ่านง่ายในขนาดเล็ก
- Text เสริม Title ไม่ซ้ำ Title

**Principle 4: Visual Curiosity**
- สร้างคำถามในหัว "นี่คืออะไร?"
- Before/After
- Unexpected Elements
- Arrow หรือ Circle ชี้สิ่งสำคัญ

**Pro Tip**: "ย่อ Thumbnail ลงเป็นขนาด Mobile ถ้ายังเห็นชัด = ดี"

### Segment 3: Title Writing Formula (4 นาที)

**Host**: มาดูสูตรเขียน Title ที่ทำให้คนคลิกกัน

**Formula 1: Number + Benefit + Timeframe**
- "5 วิธีเพิ่ม Views ใน 7 วัน"
- "10 เทคนิค Thumbnail ที่ได้ผลทันที"

**Formula 2: How To + Specific Result**
- "วิธีเขียน Script วิดีโอให้คนดูจบ"
- "How to ทำ Thumbnail ให้คลิกเยอะ"

**Formula 3: Question + Promise**
- "ทำไมวิดีโอไม่มีคนดู? (และวิธีแก้)"
- "YouTube Algorithm ทำงานอย่างไร?"

**Formula 4: Listicle + Power Word**
- "7 Secrets ที่ YouTuber ระดับโลกไม่บอก"
- "Ultimate Guide to Video SEO"

**Title Best Practices**:
- ความยาว: 50-60 ตัวอักษร (ไม่ถูกตัด)
- Keyword อยู่ข้างหน้า
- ไม่ Clickbait (ต้อง Deliver สิ่งที่สัญญา)
- ใช้ Brackets [สิ่งที่น่าสนใจ]

### Segment 4: Video Description & Tags (3 นาที)

**Host**: มาดู Description และ Tags กัน

**Description Best Practices**:
- **First 150 Characters สำคัญที่สุด** (เห็นใน Search)
- Keyword หลักอยู่ 2-3 บรรทัดแรก
- Include timestamps (Chapters)
- Links ที่เกี่ยวข้อง
- CTA: Subscribe, Playlist, Website

**Description Template**:
```
[Keyword-rich summary ใน 1-2 ประโยค]

ในวิดีโอนี้คุณจะได้เรียนรู้:
- Point 1
- Point 2
- Point 3

TIMESTAMPS:
00:00 - Intro
01:30 - Topic 1
...

LINKS:
[Related videos, resources]

#Hashtag1 #Hashtag2 #Hashtag3
```

**Tags Tips**:
- ใช้ Primary Keyword เป็น Tag แรก
- Mix: Broad + Specific tags
- 8-15 tags ก็พอ (ไม่ต้องเยอะเกิน)
- ดู Tags ของวิดีโอ Competitor

### Segment 5: YouTube SEO Fundamentals (3 นาที)

**Host**: มาเข้าใจ SEO สำหรับ YouTube กัน

**Ranking Factors**:
1. **Watch Time**: ยิ่งคนดูนาน ยิ่งดี
2. **CTR**: Click-Through Rate
3. **Engagement**: Likes, Comments, Shares
4. **Keywords**: ใน Title, Description, Tags
5. **Relevance**: เนื้อหาตรงกับ Search Intent

**Keyword Research วิธีง่าย**:
1. **YouTube Search Suggest**: พิมพ์ Keyword แล้วดู Suggestions
2. **Competitor Analysis**: ดูว่า Top Videos ใช้ Keyword อะไร
3. **TubeBuddy/VidIQ**: Tools ช่วยวิเคราะห์

**SEO Checklist**:
- [ ] Keyword ใน Title (ข้างหน้า)
- [ ] Keyword ใน Description (บรรทัดแรก)
- [ ] Keyword ใน Tags
- [ ] Keyword พูดในวิดีโอ (Transcript)
- [ ] Custom Thumbnail
- [ ] Chapters/Timestamps

### Segment 6: Thumbnail vs Title Synergy (2 นาที)

**Host**: Thumbnail และ Title ต้องทำงานด้วยกัน

**The Partnership Rule**:
- Thumbnail บอก "What" (อะไร)
- Title บอก "Why" (ทำไมต้องดู)
- สองอย่างเสริมกัน ไม่ซ้ำกัน

**Bad Example**:
- Thumbnail: "5 Tips" (text)
- Title: "5 Tips สำหรับ..."
- **ปัญหา**: ซ้ำกัน ไม่เพิ่ม Information

**Good Example**:
- Thumbnail: หน้าตกใจ + ลูกศรชี้ Graph ขึ้น
- Title: "วิธีที่ทำให้ Channel ผมโต 500% ใน 30 วัน"
- **ดี**: Thumbnail สร้าง Curiosity, Title บอก Benefit

### Closing (1 นาที)

**Host**: สรุปวันนี้ เราได้เรียนรู้ว่า Thumbnail และ Title คือ First Impression หลักการ Thumbnail: High Contrast, Face, Minimal Text, Curiosity หลักการ Title: Formula ที่ได้ผล, Keyword ข้างหน้า, ไม่ Clickbait และ SEO Basics ลองนำไปปรับใช้กับวิดีโอของคุณดู แล้วเจอกันตอนหน้าครับ
