**[OBS DEMO]**

"มาตั้งค่า OBS กัน"

1. **Sources**
   - Video Capture Device (Camera)
   - Audio Input Capture (Mic)
   - Display Capture (Screen Share)

2. **Scenes**
   - Full Camera
   - Screen + Camera Corner
   - Starting Soon Screen
   - Ending Screen

3. **Settings**
   - Video: 1080p 30fps
   - Audio: 44.1kHz
   - Streaming: YouTube RTMP

**[YOUTUBE STUDIO DEMO]**

"ตั้งค่าใน YouTube Studio"
- Go Live > Stream
- Copy Stream Key to OBS
- Set Title, Description, Thumbnail
- Schedule if needed

### Scene 3: Live Streaming Best Practices (5:30-8:00)

**[VISUAL]: Host หน้ากล้อง demonstrating

**HOST:**
"เคล็ดลับก่อน ระหว่าง และหลัง Live"

**[THREE COLUMNS GRAPHIC]**

**Before Live:**
- ทดสอบ 30 นาทีก่อน
- Backup Internet (Hotspot)
- แจ้ง Audience 24h ก่อน
- เตรียม Outline ไม่ใช่ Script
- ตั้ง Starting Soon Screen

**During Live:**
- ทักทายคนดู (เรียกชื่อ)
- อ่านและตอบ Comments
- ขอ Like/Subscribe
- ขอบคุณ Super Chat
- ประกาศ Timestamps

**After Live:**
- Thank You Post
- ตอบ Comments ที่พลาด
- ดู Analytics
- Plan Next Live
- Edit for Highlights (Optional)

### Scene 4: Webinar Integration (8:00-11:00)

**[VISUAL]: Webinar platform demo

**HOST:**
"Webinar ต่างจาก Live Stream ตรงที่เน้น Education และ Sales"

**[COMPARISON TABLE]**

| Feature | Live Stream | Webinar |
|---------|-------------|---------|
| Purpose | Community | Sales/Education |
| Platform | YouTube/FB | Zoom/WebinarJam |
| Audience | Public | Registered |
| Follow-up | Optional | Email Sequence |

**[WEBINAR STRUCTURE GRAPHIC]**

**5-Part Framework:**

```
1. Opening (5 min)
   ├── ต้อนรับ
   ├── แนะนำตัว
   └── Preview

2. Content (30-40 min)
   ├── Section 1
   ├── Section 2
   └── Section 3

3. Q&A (10-15 min)
   └── ตอบคำถาม

4. Offer (5-10 min)
   ├── Product/Service
   ├── Limited Offer
   └── Bonuses

5. Closing (5 min)
   ├── สรุป
   └── Next Steps
```

**[REGISTRATION FLOW]**

"ขั้นตอนสำคัญ:"
1. Landing Page (เก็บ Email)
2. Thank You Page (Calendar Link)
3. Reminder Emails (24h, 1h, 15min)
4. Live Webinar
5. Replay (ถ้ามี)
6. Follow-up Sequence

### Scene 5: Real-time Engagement (11:00-14:00)

**[VISUAL]: Host demonstrating techniques

**HOST:**
"4 Techniques สำหรับ Engage ผู้ชม"

**[TECHNIQUE 1 - Call and Response]**

"ถามคำถามให้พิมพ์ตอบ"
- "พิมพ์ 1 ถ้าเคยลอง"
- "Where are you watching from?"
- "ใครอยากให้พูดเรื่อง X?"

**[TECHNIQUE 2 - Shoutouts]**

"เรียกชื่อคนดู"
- "ขอบคุณ [ชื่อ] สำหรับ Super Chat!"
- "เห็น [ชื่อ] เป็น Regular เลยนะ"
- "Welcome [ชื่อ] to the stream!"

**[TECHNIQUE 3 - Interactive Content]**

"ทำให้มีส่วนร่วม"
- Polls
- Q&A Sessions
- Live Demo/Drawing
- Games/Challenges

**[TECHNIQUE 4 - Timestamps]**

"บอกว่าจะเกิดอะไร"
- "5 นาทีจะเริ่มพูดเรื่อง X"
- "ตอนท้ายมี Surprise"
- สร้างความคาดหวัง

**[HANDLING TROLLS]**

- มี Moderators
- Keyword Blocking
- ไม่ Engage
- Focus Positive
