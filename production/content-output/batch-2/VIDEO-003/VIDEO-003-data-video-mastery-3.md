## Table 7: Common Lighting Mistakes

| Mistake | Problem | Solution | Prevention |
|---------|---------|----------|------------|
| Backlit subject | Silhouette effect | Face the light source | Check before recording |
| Overhead light only | Under-eye shadows | Add front light | Test with phone camera |
| Mixed color temps | Unnatural skin tones | Use one color temp | Match all light sources |
| Too much contrast | Harsh shadows | Add fill light | Use reflector |
| Too flat | No dimension | Add back light | Create depth |

## Table 13: Composition Guidelines

| Rule | Description | Application | Common Mistake |
|------|-------------|-------------|----------------|
| Rule of Thirds | Grid of 9 sections | Subject on intersections | Always centering |
| Headroom | Space above head | 10-20% of frame | Too much/too little |
| Lead room | Space in look direction | More space in front | Cramped framing |
| Eye line | Camera at eye level | Direct connection | Too high/low angle |
| Background | Clean, uncluttered | Remove distractions | Busy backgrounds |
| Depth | Layers in frame | Subject separated from BG | Flat against wall |
