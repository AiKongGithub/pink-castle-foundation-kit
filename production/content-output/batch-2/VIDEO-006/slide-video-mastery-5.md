## Slide 5: Cut, Trim & Delete

### Content
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│            ✂️ CUT, TRIM & DELETE                    │
│                                                     │
│   ┌─────────────────────────────────────────────┐   │
│   │ 🔪 SPLIT (Cut)                              │   │
│   │    • กด S ณ ตำแหน่ง Playhead                │   │
│   │    • แบ่งคลิปเป็น 2 ส่วน                    │   │
│   │    • ใช้เมื่อต้องการลบตรงกลาง                │   │
│   └─────────────────────────────────────────────┘   │
│                                                     │
│   ┌─────────────────────────────────────────────┐   │
│   │ 📏 TRIM                                     │   │
│   │    • ลากขอบคลิปเพื่อย่อ                     │   │
│   │    • Non-destructive (ดึงกลับได้)           │   │
│   │    • ใช้สำหรับหัว/ท้าย                      │   │
│   └─────────────────────────────────────────────┘   │
│                                                     │
│   ┌─────────────────────────────────────────────┐   │
│   │ 🗑️ RIPPLE DELETE                            │   │
│   │    • ลบแล้วชิดกันอัตโนมัติ                   │   │
│   │    • ไม่เหลือช่องว่าง                        │   │
│   │    • คลิกขวา > Ripple Delete               │   │
│   └─────────────────────────────────────────────┘   │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Speaker Notes
- สาธิตการตัด "อืม" "เอ่อ" ออก
- แสดงความแตกต่าง Delete vs Ripple Delete
- Trim vs Cut ใช้ต่างกันอย่างไร

## Slide 7: Transitions Do's and Don'ts

### Content
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│            🎭 TRANSITIONS GUIDE                     │
│                                                     │
│   ✅ DO (ใช้)                                       │
│   ┌─────────────────────────────────────────────┐   │
│   │ • Cut (No Transition) - 80% ของเวลา        │   │
│   │ • Fade - สำหรับเปลี่ยน Scene               │   │
│   │ • Dissolve - สำหรับ Time passing           │   │
│   │ • Duration: 0.3-0.5 วินาที                 │   │
│   └─────────────────────────────────────────────┘   │
│                                                     │
│   ❌ DON'T (หลีกเลี่ยง)                             │
│   ┌─────────────────────────────────────────────┐   │
│   │ • Star Wipe - ดูเก่าโบราณ                   │   │
│   │ • Spin/Rotate - ทำให้เวียนหัว              │   │
│   │ • 3D Cube - ไม่ Professional              │   │
│   │ • ใช้หลายแบบเกินไป - ดู Amateur            │   │
│   └─────────────────────────────────────────────┘   │
│                                                     │
│   🎯 "ถ้าสงสัยว่าควรใส่ Transition ไหม = ไม่ใส่"    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Speaker Notes
- แสดงตัวอย่าง Good vs Bad Transitions
- เน้นว่า Less is More
- วิดีโอโปรใช้ Cut เป็นหลัก
