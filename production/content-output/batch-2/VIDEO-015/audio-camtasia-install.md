## Segment 1: Camtasia คืออะไร (2 นาที)

**Host:** ก่อนที่เราจะติดตั้ง มาทำความรู้จัก Camtasia กันก่อนครับ

**Camtasia คืออะไร?**

Camtasia เป็น Software จาก TechSmith ที่รวม 2 สิ่งเข้าด้วยกัน:
1. **Screen Recorder** - บันทึกหน้าจอคอมพิวเตอร์
2. **Video Editor** - ตัดต่อวิดีโอได้ในตัว

**ทำไมต้อง Camtasia?**

- **All-in-one** - ไม่ต้องใช้หลาย software
- **ง่ายต่อการใช้งาน** - Interface ไม่ซับซ้อน
- **Professional Results** - ผลลัพธ์คุณภาพดี
- **Built-in Assets** - มี Template, Music, Effects ให้ใช้ฟรี

**ใครควรใช้ Camtasia?**
- ครู อาจารย์ ที่ทำ Online Course
- YouTuber ที่ทำ Tutorial
- นักการตลาดที่ทำ Demo Video
- คนทำงานที่ต้อง Present บ่อยๆ

## Segment 2: System Requirements (2 นาที)

**Host:** ก่อนติดตั้ง เราต้องเช็คก่อนว่าคอมพิวเตอร์เราพร้อมหรือยัง

**Windows Requirements:**

- **OS:** Windows 7, 8, 8.1, 10 (64-bit only)
- **Processor:** Intel 6th Gen หรือใหม่กว่า / AMD Ryzen 1000 series หรือใหม่กว่า
- **RAM:** ขั้นต่ำ 8 GB (แนะนำ 16 GB)
- **Storage:** 4 GB สำหรับติดตั้ง + พื้นที่สำหรับ Project
- **Display:** 1366 x 768 ขั้นต่ำ (แนะนำ 1920 x 1080)

**Mac Requirements:**

- **OS:** macOS 10.13 (High Sierra) หรือใหม่กว่า
- **Processor:** Intel processor with 64-bit support
- **RAM:** ขั้นต่ำ 8 GB (แนะนำ 16 GB)
- **Storage:** 4 GB สำหรับติดตั้ง

**Tips สำคัญ:**

**Host:** มีเรื่องสำคัญที่ต้องบอกครับ ถ้าคุณจะทำ 4K video หรือ Long-form content RAM 16 GB ขึ้นไปจะทำให้การ render เร็วขึ้นมาก

และที่สำคัญ ควรมี SSD เป็น Storage เพราะจะช่วยให้ Preview และ Export ไวขึ้นหลายเท่า

## Segment 3: ขั้นตอนการติดตั้ง (4 นาที)

**Host:** มาถึงส่วนสำคัญแล้วครับ การติดตั้ง Camtasia 2018

**Step 1: Download**

ไปที่ techsmith.com/camtasia แล้วกด Download Free Trial หรือถ้ามี License แล้วก็ Login เข้าไป Download ได้เลย

ไฟล์จะมีขนาดประมาณ 500 MB สำหรับ Windows และ 300 MB สำหรับ Mac

**Step 2: Run Installer**

Windows: Double-click ไฟล์ .exe
Mac: Double-click ไฟล์ .dmg แล้วลาก Camtasia ไปใส่ folder Applications

**Step 3: Accept License Agreement**

อ่าน License Agreement แล้วกด I Accept ครับ ตรงนี้สำคัญ ถ้าไม่กดก็ติดตั้งต่อไม่ได้

**Step 4: Choose Installation Options**

- **Typical:** ติดตั้งทุกอย่าง (แนะนำสำหรับคนส่วนใหญ่)
- **Custom:** เลือกว่าจะติดตั้งอะไรบ้าง

แนะนำให้เลือก Typical ครับ เพราะจะได้ครบทุก feature

**Step 5: Installation Location**

ปกติจะติดตั้งที่ C:\Program Files\TechSmith\Camtasia 2018 ซึ่งไม่ต้องเปลี่ยนอะไร

**Step 6: Wait for Installation**

รอประมาณ 5-10 นาที ขึ้นอยู่กับความเร็วของเครื่อง

**Step 7: Finish & Launch**

กด Finish แล้วเปิด Camtasia ได้เลย!

## Segment 4: การ Activate License (2 นาที)

**Host:** หลังติดตั้งเสร็จ ถ้าคุณมี License Key ก็ต้อง Activate ครับ

**วิธี Activate:**

1. เปิด Camtasia
2. ไปที่ Help > Sign In / Enter Software Key
3. ใส่ Email ที่ซื้อ License
4. ใส่ Software Key (25 ตัวอักษร)
5. กด Activate

**ถ้าใช้ Trial:**

ถ้ายังไม่มี License สามารถใช้ Free Trial ได้ 30 วัน
- ได้ใช้ทุก feature เต็มที่
- แต่ video ที่ export จะมี Watermark

**Tips:** ถ้าจะซื้อ License แนะนำให้รอช่วง Promotion เช่น Black Friday หรือ Back to School จะได้ส่วนลดถึง 25-30%

## Segment 5: แนะนำ Interface เบื้องต้น (3 นาที)

**Host:** ติดตั้งเสร็จแล้ว มาดู Interface กันครับ

**หน้าจอหลักของ Camtasia มี 4 ส่วนหลัก:**

**1. Media Bin (ซ้ายบน)**
- ที่เก็บไฟล์ที่ Import เข้ามา
- Video, Audio, Images ทั้งหมดอยู่ตรงนี้

**2. Canvas (ตรงกลาง)**
- หน้าจอ Preview video
- สามารถ Drag วัตถุมาวางได้

**3. Timeline (ล่าง)**
- แถบเวลาสำหรับตัดต่อ
- มีหลาย Track สำหรับวาง Media

**4. Properties Panel (ขวา)**
- ปรับค่าต่างๆ ของ Element ที่เลือก
- Visual Properties, Audio, Behaviors

**Tools สำคัญ:**

- **Record** - ปุ่มสีแดงสำหรับบันทึกหน้าจอ
- **Import** - นำไฟล์เข้ามา
- **Export** - บันทึกเป็น Video file
- **Library** - Asset ต่างๆ ที่ Camtasia มีให้

**Host:** ผมแนะนำให้ลองคลิกไปเรื่อยๆ ก่อนครับ จะได้รู้ว่าแต่ละปุ่มทำอะไร ไม่ต้องกลัวพัง เพราะมีปุ่ม Undo ช่วยเสมอ!
