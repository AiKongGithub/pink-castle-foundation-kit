## Flashcard 1

**Front:**
Camtasia คืออะไร?

**Back:**

## Flashcard 6

**Front:**
มีกี่ขั้นตอนในการติดตั้ง Camtasia 2018?

**Back:**
**7 ขั้นตอนหลัก:**

1. Double-click installer
2. Accept UAC prompt (Windows)
3. Read License Agreement
4. Accept License Agreement

6. Select Install Location
7. Wait & Finish

**ระยะเวลา:** ประมาณ 5-10 นาที

## Flashcard 7

**Front:**
Typical vs Custom Installation ต่างกันอย่างไร?

**Back:**
**Typical Installation:**
- ติดตั้งทุกอย่าง
- แนะนำสำหรับคนส่วนใหญ่
- ใช้พื้นที่ ~2 GB

**Custom Installation:**
- เลือกติดตั้งเฉพาะที่ต้องการ
- สำหรับ Advanced users
- ใช้พื้นที่ตามที่เลือก

**แนะนำ:** เลือก Typical

## Flashcard 9

**Front:**
วิธี Activate License ของ Camtasia ทำอย่างไร?

**Back:**
**ขั้นตอน Activation:**

1. เปิด Camtasia
2. ไปที่ **Help > Enter Software Key**
3. ใส่ **Email** ที่ใช้ซื้อ
4. ใส่ **Software Key** (25 ตัวอักษร)
5. กด **Activate**

**ต้องการ:** Internet connection

## Flashcard 10

**Front:**
Interface หลักของ Camtasia มีกี่ส่วน อะไรบ้าง?

**Back:**
**4 ส่วนหลัก:**

1. **Media Bin** (ซ้ายบน) - เก็บไฟล์ที่ Import
2. **Canvas** (ตรงกลาง) - Preview video
3. **Timeline** (ล่าง) - แถบเวลาสำหรับตัดต่อ
4. **Properties** (ขวา) - ปรับค่าต่างๆ

## Flashcard 11

**Front:**
ปุ่ม Record ใน Camtasia ทำหน้าที่อะไร และอยู่ตรงไหน?

**Back:**
**ปุ่ม Record:**

- **หน้าที่:** เริ่มบันทึกหน้าจอคอมพิวเตอร์
- **ลักษณะ:** ปุ่มสีแดง
- **ตำแหน่ง:** มุมซ้ายบนของ Interface
- **Shortcut:** Ctrl+R

สามารถบันทึก: หน้าจอทั้งหมด, บางส่วน, Webcam, เสียง

## Flashcard 12

**Front:**
Timeline ใน Camtasia ใช้ทำอะไร?

**Back:**
**Timeline Functions:**

- **จัดเรียง Clips** ตามลำดับเวลา

- **Cut/Split** ตัดแบ่ง clips
- **Add Transitions** ระหว่าง clips
- **Markers** ทำเครื่องหมายจุดสำคัญ
- **Zoom** ปรับ view ละเอียด/กว้าง

## Flashcard 14

**Front:**
ถ้า Installation ล้มเหลว ควรทำอย่างไร?

**Back:**
**วิธีแก้ไข:**

1. **Run as Administrator** - คลิกขวา > Run as Admin
2. **ปิด Antivirus** ชั่วคราว
3. **เช็ค Disk Space** - ต้องมีพื้นที่เพียงพอ
4. **Download ใหม่** - ไฟล์อาจ corrupt
5. **Restart Computer** แล้วลองใหม่

## Flashcard 15

**Front:**
หลังติดตั้ง Camtasia เสร็จ ควรทำอะไรก่อน?

**Back:**
**Post-Installation Checklist:**

1. **Check for Updates** - อัพเดทให้ล่าสุด
2. **Set Default Project Settings**
   - Resolution: 1920 x 1080
   - Frame Rate: 30 fps
3. **Configure Recording Settings**
4. **Explore Library** - ดู Built-in Assets
5. **ทำ Test Recording** - ทดสอบว่าทำงานปกติ

1. **ลองติดตั้งจริง** - Practice makes perfect

4. **อย่ากลัวพัง** - มี Undo เสมอ (Ctrl+Z)
