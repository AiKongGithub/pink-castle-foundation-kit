## Flashcard Set

### CARD 1
```
[FRONT]
4 Phases ของ 0-100K Journey คืออะไร?

[BACK]
1. Foundation (0-10K) - Validate idea
2. Traction (10-30K) - Repeatable system
3. Growth (30-70K) - Optimize & Scale
4. Scale (70-100K+) - Automate

ต้องโฟกัสที่ Phase ปัจจุบัน อย่าข้ามขั้น
```

---

### CARD 2
```
[FRONT]
Phase 1 (0-10K) เป้าหมายหลักคืออะไร?

[BACK]
Validation = พิสูจน์ว่า idea ขายได้

สิ่งที่ต้องทำ:
• เลือก Niche
• สร้าง Lead Magnet
• Landing Page พื้นฐาน
• First Sale

ยังไม่ต้องสมบูรณ์แบบ เน้น speed
```

### CARD 3

• Customer Rating: 4.5+ stars

Focus: Repeatable process + Social Proof

### CARD 5
```

[BACK]

สิ่งที่ต้องทำ:

Work ON business, not IN it
```

[BACK]

### CARD 7
```
[FRONT]
Membership Model รายได้คำนวณอย่างไร?

[BACK]
รายได้ = จำนวนสมาชิก × ราคา/เดือน

ตัวอย่าง:
• 100 × $47 = $4,700/เดือน
• 300 × $47 = $14,100/เดือน

ข้อดี: Recurring, คาดการณ์ได้
ข้อเสีย: ต้องสร้าง content ต่อเนื่อง
```

### CARD 8

ข้อดี: High-ticket, ลูกค้าน้อยก็ OK
ข้อเสีย: Trade time for money

### CARD 15
```
[FRONT]
"First 100K is the hardest" หมายความว่าอย่างไร?

[BACK]
100K แรกยากที่สุดเพราะ:
• ต้องเรียนรู้ทุกอย่าง
• ต้อง build systems จาก 0
• ยังไม่มี social proof

แต่พอผ่าน 100K แรกได้:
• มี systems แล้ว
• มี proof แล้ว
• มี audience แล้ว
→ Next 100K ง่ายกว่ามาก
```
