# Flashcards: 8 ขั้นตอนสร้างความมั่งคั่ง

> **Format:** Flashcard Set
> **Source:** SWP3 Chapter 2
> **Total Cards:** 16
> **Production ID:** SWP3-Ch02-001-FLASH

---

## Flashcard Set

### CARD 1
```
[FRONT]
Niche คืออะไร?

[BACK]
ตลาดเฉพาะที่เราเลือกจะเจาะ
เป็น intersection ของ:
• Passion (สิ่งที่ชอบ)
• Expertise (สิ่งที่เก่ง)
• Market Demand (สิ่งที่คนต้องการ)
```

---

### CARD 2
```
[FRONT]
Customer Avatar คืออะไร?

[BACK]
ลูกค้าในอุดมคติของเรา ประกอบด้วย:
• Demographics (ข้อมูลพื้นฐาน)
• Psychographics (ความคิด ความกลัว ความฝัน)
• Behavior (พฤติกรรม)
```

---

### CARD 3
```
[FRONT]
Lead Magnet คืออะไร?

[BACK]
ของฟรีที่มีคุณค่า ใช้แลกกับ email
ตัวอย่าง: eBook, Checklist, Video training
หลักการ: แก้ปัญหาเล็กๆ ให้ได้ผลลัพธ์จริง
```

---

### CARD 4
```
[FRONT]
Trip Wire คืออะไร? ราคาเท่าไร?

[BACK]
สินค้าราคาถูก ($1-47)
เป้าหมาย: เปลี่ยน subscriber เป็น buyer
ตัวอย่าง: Mini-course, Template pack
สร้าง Buyer's List สำหรับขาย Core Offer
```

---

### CARD 5
```
[FRONT]
Core Offer คืออะไร? ราคาเท่าไร?

[BACK]
สินค้าหลักที่สร้างรายได้หลัก ($197-997)
ประเภท: Online course, Membership, Coaching
หลักการ: แก้ปัญหาใหญ่ให้ครบวงจร
```

---

### CARD 6
```
[FRONT]
Traffic System มีกี่ประเภทหลัก?

[BACK]
2 ประเภท:
• Organic: YouTube, Content Marketing, SEO
  (ฟรี แต่ใช้เวลา)
• Paid: FB Ads, Google Ads, YouTube Ads
  (เร็ว แต่ต้องจ่ายเงิน)
```

---

### CARD 7
```
[FRONT]
Sales Funnel คืออะไร?

[BACK]
ระบบอัตโนมัติที่พาลูกค้าจากรู้จัก→ซื้อ
Flow: Traffic → Landing Page → Lead Magnet
     → Trip Wire → Core Offer → Upsell
ทำงาน 24/7 โดยไม่ต้องมีเรา
```

---

### CARD 8
```
[FRONT]
Scale & Automate หมายถึงอะไร?

[BACK]
Scale = ขยายธุรกิจ
• เพิ่ม Traffic
• เพิ่ม Conversion Rate
• เพิ่ม Average Order Value

Automate = ทำให้อัตโนมัติ
Email sequences, Chatbots, Webinar funnels
```

---

### CARD 9
```
[FRONT]
Wealth Formula คืออะไร?

[BACK]
Wealth = Traffic × Conversion × Price × Frequency

• Traffic = จำนวนคนเข้ามา
• Conversion = % ที่ซื้อ
• Price = ราคาสินค้า
• Frequency = ซื้อซ้ำบ่อยแค่ไหน
```

---

### CARD 10
```
[FRONT]
80/20 Rule ในธุรกิจคืออะไร?

[BACK]
• 80% ของรายได้ มาจาก 20% ของลูกค้า
• 80% ของผลลัพธ์ มาจาก 20% ของกิจกรรม

สิ่งที่ต้องทำ: หา 20% ที่สำคัญ แล้วทุ่มเทกับมัน
```

---

### CARD 11
```
[FRONT]
"Done is better than perfect" หมายความว่าอย่างไร?

[BACK]
ทำเลยดีกว่ารอให้สมบูรณ์แบบ
• Imperfect action > Perfect planning
• เริ่มก่อน ปรับทีหลัง
• Speed to market สำคัญมาก
```

---

### CARD 12
```
[FRONT]
ทำไมต้องทำ 8 ขั้นตอนตามลำดับ?

[BACK]
เพราะทุกขั้นตอนสร้างต่อกัน:
• ไม่มี Niche → ไม่รู้จะขายใคร
• ไม่มี Avatar → พูดไม่ตรงกลุ่ม
• ไม่มี Lead Magnet → ไม่มี email list
• ไม่มี Funnel → ต้องขายเองตลอด
```

---

### CARD 13
```
[FRONT]
Demographics vs Psychographics ต่างกันอย่างไร?

[BACK]
Demographics = ข้อมูลภายนอก
(อายุ, เพศ, อาชีพ, รายได้, ที่อยู่)

Psychographics = ข้อมูลภายใน
(ความกลัว, ความฝัน, ปัญหา, ค่านิยม)
```

---

### CARD 14
```
[FRONT]
ทำไม Trip Wire สำคัญ?

[BACK]
3 เหตุผล:
1. พิสูจน์ว่าคนยินดีจ่ายเงิน
2. สร้าง Buyer's List (คนที่เคยซื้อ)
3. ลด Customer Acquisition Cost
   (ได้เงินกลับมาบางส่วน)
```

---

### CARD 15
```
[FRONT]
ทำไมควรเลือก Traffic Channel แค่ 1 ช่องก่อน?

[BACK]
เพราะ:
• แต่ละช่องมีความซับซ้อน
• ถ้าทำหลายช่องพร้อมกัน จะไม่เชี่ยวชาญสักช่อง
• เชี่ยวชาญ 1 ช่องก่อน → สร้างระบบ → ค่อยขยาย
```

---

### CARD 16
```
[FRONT]
วิธี Scale ธุรกิจมี 3 ทางหลักคืออะไร?

[BACK]
1. เพิ่ม Traffic
   (ใส่เงิน ads มากขึ้น, เปิดช่องใหม่)

2. เพิ่ม Conversion Rate
   (optimize funnel, improve copy)

3. เพิ่ม Average Order Value
   (upsell, cross-sell, bundles)
```
