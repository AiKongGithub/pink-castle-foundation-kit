## Flashcard Set

### CARD 1
```
[FRONT]
Marketing ต่างจาก Selling อย่างไร?

[BACK]
Marketing = ทำให้คนรู้จัก สนใจ เชื่อใจ
Selling = ปิดการขาย

Marketing ที่ดี ทำให้ Selling ง่ายขึ้น
ลูกค้าจะมาหาเราเอง
```

---

### CARD 2
```
[FRONT]
3M ของ Marketing คืออะไร?

[BACK]
• Message = สิ่งที่จะสื่อสาร
• Market = คนที่จะรับสาร
• Media = ช่องทางในการสื่อสาร

สูตร: Right Message + Right Market + Right Media = Results
```

---

### CARD 3
```
[FRONT]
AIDA Model คืออะไร?

[BACK]
A = Attention (ดึงดูดความสนใจ)
I = Interest (สร้างความสนใจ)
D = Desire (สร้างความต้องการ)
A = Action (กระตุ้นการกระทำ)

เป็นกระบวนการพาลูกค้าจากรู้จัก→ซื้อ
```

---

### CARD 4
```
[FRONT]
วิธีดึงดูด Attention มีอะไรบ้าง?

[BACK]
• Headline ที่ดึงดูด
• Visual ที่สะดุดตา
• Hook ที่ทำให้อยากรู้ต่อ

ตัวอย่าง Hook:
- คำถามกระตุ้น
- ตัวเลขน่าตกใจ
- ความขัดแย้ง
```

---

### CARD 5
```
[FRONT]
วิธีสร้าง Interest มีอะไรบ้าง?

[BACK]
• เล่าเรื่องที่เกี่ยวกับปัญหาของเขา
• ให้ข้อมูลที่มีคุณค่า
• แสดงว่าเราเข้าใจเขา

หลักการ: พูดถึง "เขา" ไม่ใช่ "เรา"
```

---

### CARD 6
```
[FRONT]
Benefits vs Features ต่างกันอย่างไร?

[BACK]
Features = คุณสมบัติของสินค้า
(เช่น "คอร์สมี 50 บท")

Benefits = ผลลัพธ์ที่ลูกค้าจะได้
(เช่น "สร้างรายได้ 6 หลักใน 90 วัน")

ลูกค้าซื้อ Benefits ไม่ใช่ Features
```

---

### CARD 7
```
[FRONT]
Content Pillars 4 ประเภทคืออะไร?

[BACK]
1. Educational = ให้ความรู้ (How-to)
2. Inspirational = สร้างแรงบันดาลใจ
3. Entertaining = ความบันเทิง
4. Promotional = ส่งเสริมการขาย

สัดส่วน: 80% Value, 20% Promo
```

---

### CARD 8
```
[FRONT]
Content Repurposing คืออะไร?

[BACK]
การนำ content 1 ชิ้นไปใช้หลายที่

ตัวอย่าง YouTube 30 นาที:
• ตัดเป็น Shorts 3 ชิ้น
• ถอดเสียงเป็น Blog
• สรุปเป็น Carousel
• Audio เป็น Podcast
```

---

### CARD 9
```
[FRONT]
Organic vs Paid Traffic ต่างกันอย่างไร?

[BACK]
Organic:
• ฟรี แต่ใช้เวลา 3-6 เดือน
• ยั่งยืน ทำครั้งเดียวได้ผลยาว
• เช่น YouTube, Blog, SEO

Paid:
• จ่ายเงิน แต่ได้ผลทันที
• หยุดจ่าย = หยุดได้
• เช่น FB Ads, Google Ads
```

---

### CARD 10
```
[FRONT]
Landing Page ที่ดีต้องมีอะไรบ้าง?

[BACK]
1. Headline ดึงดูด + บอก benefit
2. Subheadline ขยายความ
3. Hero Image/Video
4. Benefits 3-5 ข้อ
5. Social Proof (Testimonials)
6. CTA Button ชัดเจน
7. FAQ ตอบข้อสงสัย
```

---

### CARD 11
```
[FRONT]
CTA (Call-to-Action) คืออะไร?

[BACK]
คำสั่งบอกให้ลูกค้าทำอะไร

ตัวอย่าง:
• "คลิกเลย"
• "สมัครฟรี"
• "ดาวน์โหลดตอนนี้"
• "เริ่มต้นใช้งาน"

ต้องชัดเจน เห็นได้ชัด
```

---

### CARD 12
```
[FRONT]
A/B Testing คืออะไร?

[BACK]
การทดสอบเปรียบเทียบ 2 version
เพื่อดูว่าอันไหนดีกว่า

ทดสอบทีละอย่าง:
• Headline
• CTA (สี, ข้อความ)
• Images
• Price
```

---

### CARD 13
```
[FRONT]
Opt-in Rate ที่ดีควรเป็นเท่าไร?

[BACK]
20-40% ถือว่าดี

Opt-in Rate = จำนวน Leads / จำนวน Visitors × 100

ถ้าต่ำกว่า 20% = ต้องปรับปรุง
• Headline ไม่ดึงดูดพอ
• Offer ไม่น่าสนใจ
• Form ซับซ้อนเกินไป
```

---

### CARD 14
```
[FRONT]
Return on Ad Spend (ROAS) ที่ดีคือเท่าไร?

[BACK]
2-4x ถือว่าดี

ROAS = รายได้ / ค่าโฆษณา

ตัวอย่าง: จ่าย ads 10,000 ได้รายได้ 30,000
ROAS = 3x (ดี)

ต่ำกว่า 2x = ขาดทุน
```

---

### CARD 15
```
[FRONT]
"Content is king, but distribution is queen"
หมายความว่าอย่างไร?

[BACK]
Content ดีอย่างเดียวไม่พอ
ต้องกระจาย (distribute) ให้คนเห็นด้วย

Content ที่ดี + Distribution ที่ดี = ผลลัพธ์

ถ้า content ดีแต่ไม่มีคนเห็น = เสียเปล่า
```
