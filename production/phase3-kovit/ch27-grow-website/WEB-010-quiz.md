### Q2: CMS หลักที่ใช้ในบทเรียนคืออะไร?

A) Wix
B) Squarespace
C) WordPress
D) Shopify

✅ **คำตอบ: C) WordPress**

**อธิบาย:** WordPress เป็น CMS หลักที่ใช้ตลอดทั้ง Series เพราะยืดหยุ่นและมี Plugin มาก

### Q3: ก่อนสร้างเว็บต้องทำอะไรก่อน?

A) ซื้อ Theme ราคาแพง
B) วางแผนโครงสร้างและ Sitemap
C) เริ่มเขียนโค้ดทันที
D) ลงโฆษณา

✅ **คำตอบ: B) วางแผนโครงสร้างและ Sitemap**

**อธิบาย:** ต้องรู้ว่าเว็บจะมีกี่หน้า แต่ละหน้าทำอะไร ก่อนลงมือสร้าง

---

### Q4: SSL Certificate สำคัญอย่างไร?

A) ทำให้เว็บสวยขึ้น
B) เข้ารหัสข้อมูลและเพิ่มความน่าเชื่อถือ
C) เพิ่มความเร็วเว็บ
D) ไม่จำเป็น

✅ **คำตอบ: B) เข้ารหัสข้อมูลและเพิ่มความน่าเชื่อถือ**

**อธิบาย:** SSL ทำให้เว็บเป็น HTTPS ปกป้องข้อมูล และ Google ให้คะแนน SEO เพิ่ม

---

### Q5: ทำไม Responsive Design สำคัญ?

A) เพราะดูเท่
B) เพราะ 70%+ ของผู้ใช้เข้าเว็บผ่าน Mobile
C) ไม่สำคัญ
D) เพราะ Google บังคับ

✅ **คำตอบ: B) เพราะ 70%+ ของผู้ใช้เข้าเว็บผ่าน Mobile**

**อธิบาย:** ถ้าเว็บไม่ Responsive จะเสียผู้ใช้มากกว่าครึ่ง

---

### Q6: Plugin กลุ่มไหนต้องติดตั้งก่อน?

A) Social Media Plugins
B) Animation Plugins
C) Security + SSL + Backup
D) Font Plugins

✅ **คำตอบ: C) Security + SSL + Backup**

**อธิบาย:** ความปลอดภัยและ Backup เป็นพื้นฐานที่ต้องมีก่อนทำอย่างอื่น

---

### Q7: Branding บนเว็บต้องสม่ำเสมอหมายถึงอะไร?

A) ใช้สีเดียวตลอด
B) สี ฟอนต์ โลโก้ เหมือนกันทุกหน้า
C) ใส่โลโก้แค่หน้าแรก
D) ไม่ต้องมี Branding

✅ **คำตอบ: B) สี ฟอนต์ โลโก้ เหมือนกันทุกหน้า**

**อธิบาย:** ความสม่ำเสมอสร้างความเป็นมืออาชีพและความน่าเชื่อถือ

---

### Q8: WooCommerce ใช้ทำอะไร?

A) สร้าง Blog
B) ระบบ E-commerce สำหรับขายสินค้าบน WordPress
C) สร้าง Social Media
D) ทำ SEO

✅ **คำตอบ: B) ระบบ E-commerce สำหรับขายสินค้าบน WordPress**

**อธิบาย:** WooCommerce เป็น Plugin ฟรีที่เปลี่ยน WordPress ให้เป็นร้านค้าออนไลน์

---

### Q9: ก่อนเปิดขายจริงต้องทำอะไร?

A) ลงโฆษณาทันที
B) ทดลองสั่งซื้อสินค้าเอง
C) ไม่ต้องทำอะไร
D) เปลี่ยน Theme

✅ **คำตอบ: B) ทดลองสั่งซื้อสินค้าเอง**

**อธิบาย:** ต้อง Test ระบบสั่งซื้อ-ชำระเงินจนมั่นใจก่อนเปิดขายจริง

---

### Q10: ทำไมต้อง Backup ทุกวัน?

A) เพราะ Google บังคับ
B) เพราะข้อมูลหาย = เริ่มใหม่ทั้งหมด
C) เพราะเว็บจะเร็วขึ้น
D) ไม่จำเป็น

✅ **คำตอบ: B) เพราะข้อมูลหาย = เริ่มใหม่ทั้งหมด**

**อธิบาย:** Backup คือประกัน ถ้าเว็บโดน Hack หรือข้อมูลเสียหาย Restore ได้ทันที

---

### Q11: SEO พื้นฐาน 4 อย่างที่ต้องมีคืออะไร?

A) Facebook, Instagram, TikTok, YouTube
B) Title Tag, Meta Description, H-tags, Alt Text
C) Logo, Color, Font, Layout
D) Speed, Security, SSL, CDN

✅ **คำตอบ: B) Title Tag, Meta Description, H-tags, Alt Text**

**อธิบาย:** 4 องค์ประกอบ SEO พื้นฐานที่ทุกหน้าต้องมี

---

### Q12: สิ่งที่ต้องทำหลังสร้างเว็บเสร็จคืออะไร?

A) ลืมมันไป
B) ทดสอบทุกหน้า ทุกลิงก์ ทุกฟอร์ม ทั้ง Desktop + Mobile
C) เริ่มสร้างเว็บใหม่
D) ลบ Plugin ทั้งหมด

✅ **คำตอบ: B) ทดสอบทุกหน้า ทุกลิงก์ ทุกฟอร์ม ทั้ง Desktop + Mobile**

**อธิบาย:** Testing End-to-end สำคัญมาก ต้องเช็คทุกจุดก่อนเปิดใช้จริง
