## SLIDE 9: SEO พื้นฐาน
```
╔══════════════════════════════════════════╗
║         SEO พื้นฐาน                      ║
╚══════════════════════════════════════════╝
```

- Title Tag — ชื่อหน้า (60 ตัวอักษร)
- Meta Description — คำอธิบาย (160 ตัวอักษร)
- H1-H3 Tags — โครงสร้างหัวข้อ
- Alt Text — คำอธิบายรูปภาพ
